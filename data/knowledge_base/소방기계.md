# 소방기계 지식 베이스

## 초기 기준
스프링클러 헤드 살수반경, 배관 레이어, 소방법 민감 구간, 알람밸브/펌프 연계 조건을 확인한다.


## 소방기계 계통 기본 분류 — 역할·유체 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: fire-mechanical,fluid,classification,sprinkler,suppression

소방기계는 화재 시 불을 끄거나 연기를 제어하는 설비다. 유체 종류에 따라 계통이 구분된다.

스프링클러 계통 (물 소화):
- 습식(Wet Pipe): 배관 내 항상 물 충만 → 헤드 개방 즉시 방수. 동결 우려 없는 일반 구역.
- 건식(Dry Pipe): 배관 내 가압 공기 → 화재 시 공기 배출 후 물 충전 → 방수. 동결 우려 구역(주차장·옥외).
- 준비작동식(Pre-action): 감지기 신호 + 헤드 개방 2중 조건 충족 시 방수. 오작동 방지 중요 (전산실·전기실).
- 일제살수식(Deluge): 개방형 헤드 + 감지기 신호로 전 헤드 동시 방수. 위험물 창고·항공기 격납고.

연결송수관 (Standpipe):
- 유체: 소방차에서 가압 공급하는 물
- 역할: 소방대원이 건물 내부에서 직접 호스 연결·방수 (화재 진압용)
- 설치 대상: 11층 이상 또는 연면적 5,000㎡ 이상

옥내소화전:
- 유체: 물
- 역할: 초기 화재 진압용. 재실자가 직접 사용 가능.
- 배관: 스프링클러 배관과 독립 계통 (일부 겸용 가능)

가스계 소화 계통 (비수계 소화):
- CO₂(이산화탄소): 전기실·발전기실. 산소 농도 희석으로 소화. 사람 대피 후 방사.
- HFC-125/HFC-227ea(청정소화약제): 전산실·통신실. 정밀 기기에 무해. 잔존물 없음.
- IG-541(INERGEN, 질소·아르곤·CO₂ 혼합): 미술관·문서보관실. 비도전성·비부식성.
- 주의: 가스계 소화 구역은 방호구역 기밀 필수, 환기 댐퍼 자동 폐쇄 연동.

소방 배관 일반 기준:
- 재료: 배관용 탄소강관(KS D 3507), 동관, 스테인리스
- 색상: 빨간색 (Red) 전체 도장
- 위치: 다른 설비 배관 하부 배치 원칙 (화재 시 충돌 최소화)
- 방화구획 관통: 방화 슬리브 + 내화채움재 필수

## 스프링클러 헤드 살수반경 및 배치 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: fire-mechanical,sprinkler,coverage

표준형 헤드 살수 커버: 최대 9㎡(3×3m 격자), 위험도 높은 구역 6.25㎡(2.5×2.5m).
헤드 간 최대 거리: 표준 위험 3.6m, 고위험 2.5m.
헤드와 벽 이격: 헤드 살수 반경의 1/2 이상(최소 0.45m).
Add-in은 헤드 살수 커버리지를 평면으로 계산하고 미커버 구역을 자동 감지해 보고서에 표기한다.


## 소방 배관 압력 및 관경 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: fire-mechanical,pipe,pressure

습식 스프링클러 배관 내 압력: 최소 0.1MPa, 최대 1.2MPa.
주 배관(급수 헤더): DN80~DN150, 교차 배관: DN25~DN50, 가지 배관: DN25.
말단 헤드 방수압: 0.1MPa 이상, 방수량: 80L/min 이상.
Add-in은 배관 계통 끝단 압력 손실 경로를 표시하고 최소 방수압 미달 위험 구간을 경고한다.


## 소방법 민감 구간 및 협의 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19

소방 전용 공간(소화펌프실·수조실): 다른 설비 배관 원칙적 불가.

## 소방배관·소방기계 적용 근거문서 및 법규 기준 (2026-05-22)

### 국내 법령
- 소방시설 설치 및 관리에 관한 법률 (법률 제18522호, 2022.12.1 전부 개정 시행): 소방시설 설치 의무, 성능 기준 위임
- 소방시설 설치 및 관리에 관한 법률 시행령: 소방시설 설치 대상 및 면제 기준
- 소방기본법: 소방시설의 유지·관리 의무
- 건축법 제49조·제64조: 방화구획, 스프링클러 설치 의무 건축물 규정

- NFTC 102 (구 NFSC 102) 옥내소화전설비의 화재안전기술기준
- NFTC 103 (구 NFSC 103) 스프링클러설비의 화재안전기술기준: 습식·건식·준비작동식·일제살수식 설치 기준 전부
- NFTC 104 (구 NFSC 104) 물분무소화설비의 화재안전기술기준
- NFTC 105 (구 NFSC 105) 포소화설비의 화재안전기술기준
- NFTC 106 (구 NFSC 106) 이산화탄소소화설비의 화재안전기술기준
- NFTC 107A (구 NFSC 107A) 할로겐화합물 및 불활성기체소화설비의 화재안전기술기준
- NFTC 109 (구 NFSC 109) 옥외소화전설비의 화재안전기술기준
- NFTC 402 (구 NFSC 402) 소화수조 및 저수조의 화재안전기술기준
- NFTC 502 (구 NFSC 502) 연결송수관설비의 화재안전기술기준
- NFTC 503 (구 NFSC 503) 연결살수설비의 화재안전기술기준

### KS 규격
- KS D 3507 배관용 탄소강 강관: 소방 배관 주 재질
- KS D 3562 압력배관용 탄소강 강관: 고압 소방 계통
- KS B 6536 소방용 펌프: 소방펌프 성능 기준

### 국제 기준 (설계 참고)
- NFPA 13 Standard for the Installation of Sprinkler Systems: 스프링클러 헤드 간격·압력·관경 상세 기준
- NFPA 14 Standard for the Installation of Standpipe and Hose Systems
- NFPA 20 Standard for the Installation of Stationary Pumps for Fire Protection
- NFPA 2001 Standard on Clean Agent Fire Extinguishing Systems: 청정소화약제

### BIM/설계 실무 적용 시 우선 확인 법규
1. 스프링클러 헤드 살수 커버리지·간격 → NFTC 103 + NFPA 13 (설계 기준이 다를 수 있음 — 인허가 담당자 확인)
2. 가스계 소화 방호구역 기밀 요구 → NFTC 106/107A (구획 면적·개구부 폐쇄 연동)
3. 소방 배관 방화구획 관통 처리 → 건축법 + NFTC 각 기준 관통부 기준 조항
4. 소방전용 수조·펌프실 → NFTC 402 + 소방시설법 시행령 [별표] 소방 전용 구역 기준
5. 소방법 판정은 반드시 해당 프로젝트 인허가 소방서 기준 확인 필수 (기준 해석이 지역/시기별 상이)


## 스프링클러 헤드 주변 BIM 확인 체크리스트 (2026-05-23)
- Source: LUA BIM LABS internal fire-mechanical BIM QA baseline
- Tags: fire-mechanical,sprinkler,head,bim,clash,coverage
- Links: [[소방전기]], [[공조덕트]], [[설비시공조율]]

헤드 주변 BIM에서 확인해야 할 항목 — 물리 충돌 외에 살수 장애와 법규 관점까지 포함한다.

살수 장애 검토:
- 헤드 하부 360° 살수 반경 내 덕트, 대구경 배관, 빔, 방음재, 장비가 있는지 확인한다.
- 일반 헤드(수하형)는 헤드 디플렉터 기준 하부 450mm 이내에 장애물이 있으면 살수 방해 이슈다.
- 덕트나 장비가 헤드 옆으로 지나갈 때는 살수 반경을 반으로 줄이는 차단 면적(Shadow Effect)을 확인한다.
- NFPA 13 기준: 헤드 디플렉터에서 장애물 하단까지 수직 거리, 장애물 폭에 따라 추가 헤드 설치 여부 결정.

헤드 위치 기준:
- 천장 마감면 아래 헤드 디플렉터 위치: 수하형 25~305mm (NFTC 103 기준).
- 헤드와 벽 이격: 헤드 살수 반경의 1/2 이상(최소 450mm, 국내 기준 일부 상이 — 확인 필요).
- 헤드 간격: 표준위험 3.6m 이하, 고위험 2.5m 이하.
- 헤드와 빔 간격: 보 하단이 천장보다 낮으면 보 한쪽에만 헤드 배치 시 살수 미커버 확인 필수.

다른 설비와의 조율:
- 덕트: 헤드 위에 덕트가 지나가면 살수 각도를 막는다. 덕트 측면 통과 시 이격 필요.
- 배관: 소방 배관 가지관이 헤드 바로 위에서 분기되므로 가지관 레벨이 헤드 디플렉터 위치에 영향.
- 천장 마감: 천장 마감 시공 후 헤드 돌출 높이가 기준에 맞는지 확인. 내화천장은 헤드 종류 변경 필요 가능.
- 감지기: 준비작동식 또는 건식은 구역 내 감지기와 세트로 동작 → 감지기 설치 위치와 헤드 구역 매칭.

Revit BIM 확인 항목:
- 헤드의 Offset(레벨 기준 높이)이 천장 마감고와 일치하는지 확인한다.
- 헤드 패밀리가 수하형/측벽형/상향형 구분에 맞게 설치되어 있는지 확인한다.
- 가지 배관(DN25)이 헤드까지 정확히 연결되어 있고, 계통(Sprinkler System)이 올바르게 지정되어 있는지 확인한다.
- 헤드 주변 Navisworks 간섭 검토 시 단순 물리 충돌뿐 아니라 "살수 장애 예비 공간"을 별도 체크 항목으로 분류한다.

자주 발생하는 이슈:
- 조명기구가 헤드 바로 옆에 붙는 경우: 조명 갓이 살수 장애가 되지 않는지 확인.
- CCTV 돔 카메라가 헤드 인근 천장에 부착: 살수 방향 방해 여부 확인.
- 스프링클러 헤드 직상부에 공조 덕트 수평 통과: 덕트 폭과 이격 거리로 추가 헤드 필요 여부 판단.
- 방화셔터 하강 구역과 헤드 위치 충돌: 셔터 하강 후 헤드가 차단되지 않는 위치 확인.


## 소방기계 계통별 BIM 모델링 기준 (2026-05-23)
- Source: LUA BIM LABS internal fire-mechanical BIM modeling baseline
- Tags: fire-mechanical,bim,revit,modeling,sprinkler,standpipe
- Links: [[설비시공조율]], [[소방전기]], [[설비도면해석]]

Revit에서 소방 계통 모델링 시 자주 틀리는 항목:

스프링클러:
- 계통 지정: Revit Fire Protection Wet / Dry / Preaction / Deluge로 구분. 혼용 시 수량 집계 오류.
- 헤드 패밀리: 수하형(Pendant), 상향형(Upright), 측벽형(Sidewall), 은폐형(Concealed) 구분.
- 가지관 연결: 헤드가 독립된 패밀리로 놓이고 배관이 연결되지 않은 경우 → 간섭 검토에서 누락 위험.
- 경보밸브(알람밸브): 계통별로 1대 이상, 습식 전용 구역에 설치 위치와 접근성 확인.

옥내소화전:
- Revit 카테고리: Fire Protection 또는 Specialty Equipment.
- 설치 높이: 호스 접속구 바닥에서 1.5m 이하(NFTC 102).
- 보행 거리: 각 소화전에서 보행 거리 25m(바닥면적 1,000㎡ 초과 시 소방청 기준 확인).

소화펌프:
- 소화펌프실은 별도 방화구획 구역에 배치.
- 주펌프 + 예비펌프 + 충압펌프 3대 구성이 일반적.
- 수조와 펌프 사이 흡수관 연결: 편심 리듀서로 공기 고임 방지.
- 자동제어: 압력 스위치, 유량계, 방재 연동 신호를 소방전기와 연계.

가스계 소화:
- 방호구역 기밀: 바닥, 천장, 벽체 개구부 폐쇄 댐퍼 자동 연동.
- Revit에서 소화약제 용기, 기동장치, 방출헤드, 선택밸브 패밀리 배치와 배관 연결 확인.
- 소화약제 방출 전 경보 30초 지연 → 소방전기 연동 시나리오 확인 필수.

### 소화펌프 설치 기준
- 기초: 콘크리트 기초 위에 설치. 기초 높이 바닥면에서 100mm 이상.

- 소화펌프실: 별도 방화구획. 전용 수전 회로 또는 비상전원 연결.
- 압력 스위치 설정: 충압펌프 기동 압력 > 주펌프 기동 압력. 시스템 설계 압력의 1.1배 이하로 정지 압력 설정.
