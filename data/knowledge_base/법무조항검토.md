## EULA 핵심 조항 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: legal,eula,terms

EULA 필수 조항:
1. 라이선스 범위: 구독 기간 내 지정 사용자 1인 사용 허용(비독점, 비양도)
2. 금지 행위: 역공학, 소스코드 추출, 재배포, 서브라이선스 금지
3. 책임 제한: 소프트웨어 결함으로 인한 간접 손해 배상 제외, 최대 배상액 = 구독료 3개월치
4. 지식재산권: 소프트웨어 및 문서의 모든 IP는 판매자 소유
5. 업데이트/지원: 구독 기간 내 업데이트 제공, 지원 응답 SLA 명시
6. 계약 해지: 약관 위반 시 즉시 해지 가능


## 제3자 라이브러리 라이선스 검토 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: legal,third-party,license

라이선스 유형별 상용 판매 가능 여부:
- MIT/BSD/Apache 2.0: 상용 사용 허용, 저작권 고지 필요
- LGPL: DLL 링크 방식으로 사용 시 허용, 수정 배포 시 소스 공개 필요
- GPL: 소스 공개 의무 발생 → 상용 Add-in에 사용 금지
- 독점 라이선스: 계약서 확인 후 사용 범위 준수
Revit/Navisworks API 자체는 Autodesk 이용약관 준수 필수(상용 재배포 허용, 비공개 API 사용 금지).

### 제1조 라이선스 부여
공급자는 구독 기간 동안 사용자에게 본 소프트웨어의 비독점적·비양도적·단일 사용자 라이선스를 부여합니다.

### 제2조 금지 행위
사용자는 다음 행위를 할 수 없습니다.
- 소프트웨어의 역공학, 디컴파일, 소스코드 추출
- 제3자에게 재배포, 대여, 서브라이선스 부여
- Autodesk 비공개 API를 통한 무단 접근

### 제3조 지식재산권
본 소프트웨어 및 관련 문서의 모든 지식재산권은 공급자에게 귀속됩니다.

### 제4조 책임 제한
소프트웨어 사용으로 발생한 간접·특별·파생 손해에 대해 공급자는 책임을 지지 않습니다. 최대 배상액은 사용자가 지급한 최근 3개월 구독료로 제한됩니다.

구독 기간 중 업데이트를 제공합니다. 기술 지원 응답 SLA: 영업일 기준 ___ 일 이내.

약관 위반 시 공급자는 라이선스를 즉시 해지할 수 있습니다.

### 제7조 준거법
본 계약은 대한민국 법률에 따릅니다.

## 법무조항검토 페르소나 업무 기준 (2026-05-26)
- Source: LUA BIM LABS 조직 역할 정의 2026-05-26
- Tags: persona,role,workflow,legal

**BIM 용역 계약서 검토 핵심 항목:**
- 지식재산권(IP): 납품 BIM 모델, 패밀리, 스크립트의 소유권이 LUA BIM LABS에 귀속되는지 또는 발주처에 이전되는지 명확히 확인. 이전 조건(범위/시점/조건) 구체화 필요.
- 납품물 정의: 납품 파일 형식(RVT/IFC/NWD), LOD 기준, 파일 버전, 납품 일정 구체적으로 기재 여부 확인. 모호한 "BIM 모델 납품" 표현은 구체화 협상 요청.
- 하자 기간: 준공 BIM 납품 후 하자 책임 기간 명시 여부 (권장: 준공 후 1년, 최대 2년 이내). 하자의 정의와 보정 범위 구체화 필요.
- 변경 관리: 발주처 요청으로 범위 변경 시 추가 비용 청구 근거 조항 포함 여부 확인.

**소프트웨어 EULA 검토 기준 (Autodesk 정책 준수):**
- Autodesk API 이용약관 준수: 공개 API만 사용, 비공개 API 사용 조항 없을 것
- 재판매 금지 조항: LUA BIM LABS Add-in이 Autodesk 제품 번들로 재판매되지 않도록 명시
- Autodesk 상표 사용: "Works with Autodesk Revit®" 문구만 허용, 제품명에 Revit/Autodesk 포함 금지
- EULA 검토 체크리스트: 라이선스 범위, 금지 행위, 책임 제한, IP 귀속, 해지 조건, 준거법 6개 항목 전체 확인

**개인정보 처리 동의 및 보안 조항 확인 사항:**
- 수집 데이터 목록: Machine ID, 오류 로그 외 추가 수집 항목 없을 것. 수집 항목 변경 시 법무 재검토.
- 정보 주체 동의: 라이선스 활성화 시 개인정보 수집 동의 UI 제공 여부 확인
- 데이터 보관 및 파기: 구독 만료 후 보관 기간(최대 1년) 및 파기 절차 명시
- 국외 이전: 라이선스 서버가 해외 소재인 경우 국외 이전 동의 조항 추가 필요
- 보안 사고 대응: 데이터 침해 발생 시 72시간 이내 고객 통지 조항 포함 여부

**분쟁 발생 시 법무 대응 프로세스:**
- 1단계 (접수): 분쟁 내용 서면 접수 → 법무조항검토 담당 및 CEO에게 즉시 공유
- 2단계 (사실 확인): 계약서, 납품물, 커뮤니케이션 이력 수집. 72시간 이내 완료.
- 3단계 (대응 방향 결정): CEO/COO와 협의하여 협상/조정/법적 대응 방향 결정
- 4단계 (외부 법률 자문): 계약 금액 USD 10,000 이상 또는 형사적 이슈 포함 시 외부 변호사 선임 검토
- 5단계 (기록 보존): 분쟁 관련 모든 문서와 커뮤니케이션 이력 5년 이상 보관
