```
건축전기설비설계기준 기준:
- 강전(전력) 케이블 트레이 ↔ 약전(통신·제어) 케이블 트레이: 최소 300mm 이격
- 이격 불가 시: 격벽(금속제 차폐판) 설치로 대체 가능
- 강전 트레이와 배관(소화·위생)과의 이격: 최소 150mm
- 분전반 전면 점검 공간: 최소 600mm 확보
- 수배전반(큐비클) 전면 조작 공간: 최소 1,000mm
```

|---------|---------|-----------|-------------|---------|
| 200 | 큐비클 개략 위치 | 층별 분전반 위치 | 메인 경로 | 조명 배치 구역 |
| 300 | 형식·용량·치수 | 회로 구성·용량 | 단면·폭·경로 | 기구 타입·위치 |

| 400 | 전장품·배선 상세 | 내부 결선 | 볼트·행거 | 전구 소켓 상세 |

**전기설비 BIM 간섭검토 우선순위:**
1. **강전/약전 케이블 트레이 이격**: 300mm 이격 또는 격벽 필수
2. **분전반 전면 점검 공간 확보**: 600mm 미만 → 유지보수 불가
3. **수배전반(큐비클) 위치**: 물 배관과 최소 1m 이격 (누수 위험)
4. **케이블 트레이 곡률반경**: 케이블 허용 굽힘 반경 이상 (케이블 손상 방지)
5. **조명기구 하부 덕트 간섭**: 천장 마감 높이 결정 후 BIM 확인

**국내 스마트빌딩 전기 자동화 현황 (2026):**
- EMS(Energy Management System): 분전반별 실시간 전력 모니터링
- 조명 DALI 제어: 존별 조도·색온도 자동 조절, BIM 조명 구역과 연동
- ESS(에너지저장시스템): 태양광 발전 + ESS + BMS 통합 제어
- EV 충전 인프라: 건물 분전반 → EV 충전기 전력 부하 계산 필요

**건축전기설비설계기준 핵심 수치:**
- 조명 조도: 사무실 500 lux, 복도 100 lux, 주차장 30 lux (KS A 3011)
- 비상조명: 피난경로 10 lux 이상 (예비전원 20분 이상)
- 수변전: 한전 22.9kV 수전 → 변압기(Delta/Y 결선) → 3.3kV 또는 380/220V
- 접지저항: A종(100Ω), B종(150/Ig Ω), C종(10Ω), D종(100Ω)

## 전기 설비 계통 기본 분류 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: electrical,classification,strong-current,weak-current,fire-electrical

전기 설비는 전압 크기에 따라 강전(Strong Current)과 약전(Weak Current/Low Voltage)으로 구분한다.

강전 (Strong Current, 전력 계통):
- 수변전: 한전 수전(특고압 22.9kV) → 변압기 → 배전반 → 각 층 분전반
- 배전: 고압(1kV 초과), 저압(1kV 이하 — 일반 220V/380V)
- 동력: 모터 구동 (펌프, 팬, 엘리베이터, 에스컬레이터)
- 조명: 일반 조명, 비상 조명
- 비상전원: 발전기(Generator), UPS(무정전전원장치)
- 케이블 트레이: 강전 전용 (약전과 최소 300mm 이격 또는 격벽)

약전 (Weak Current, 신호·통신 계통):
- 통신/LAN: Cat6, 광케이블 (MDF-IDF 체계)
- 전화: 일반 전화, 인터폰
- 방송: 공공 방송(PA), 비상방송 (내화 케이블 FR 필수)
- 영상: CCTV, 동축케이블 RG-59 (최대 300m), HD-SDI (100m)
- 보안: 출입통제, 카드리더
- 주의: 약전 케이블과 전력선 최소 150mm 이격 (EMI 방지)

소방전기 (Fire Electrical):
- 화재감지기: 차동식·연기식·열감지식 스포트형
- 수신기/발신기: 화재 신호 수집·발신
- 방재연동: 방화셔터·제연팬·소방펌프 자동 기동
- 배선: 내화 케이블(FR-8) 필수 — 화재 시 30분 이상 기능 유지
- 비상조명·유도등: 예비전원 30분 지속

전기 계통 Revit 카테고리:
- 강전: Conduit(전선관), Cable Tray(케이블 트레이), Electrical Equipment(분전반·배전반)
- 약전: 별도 Conduit 계통 또는 통신 전용 트레이

## 케이블 트레이 규격 및 이격 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: electrical,cable-tray,clearance

케이블 트레이 폭: 200~1200mm, 깊이 50~150mm. 충전율 50% 이하 유지.
강전(고압/저압) 트레이와 통신(약전) 트레이: 최소 300mm 이격, 또는 격벽 설치.
전력 케이블 트레이와 접지 트레이: 동일 계통 가능, 타 계통과 분리.
Add-in은 강전/약전 트레이 이격 부족 구간을 EMI 위험 항목으로 분류하고, 이격 거리를 자동 측정해 보고서에 표기한다.


## 분전반 접근 공간 및 설치 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: electrical,panel,access

분전반 전면 작업 공간: 최소 1,000mm(KEC 기준 고압 1,200mm).
분전반 상단 케이블 인입 여유: 최소 300mm.
분전반은 습기·열원·진동 발생원에서 이격, 소방설비 전용 분전반은 별도 구획.
Add-in은 분전반 전면 공간 침해, 인입 케이블 경로 간섭, 소방/일반 혼용 분전반 설치를 경고 항목으로 표시한다.


## 누수·침수 위험 구역 이격 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: electrical,water-protection,zone

전기 기기·분전반은 배수 배관·급수 배관 직하 설치 금지.
누수 위험 구역(위생기구 상부, 지하 집수정 주변) 반경 1,000mm 이내 전기 기기 설치 시 방수 등급 IP44 이상.

- KEC 400 분산형 전원 설비 (태양광·ESS 연계)

### 국내 고시 및 기준
- 건축전기설비 설계기준 (국토교통부 고시): 조명·동력·접지·피뢰 설계 기준
- 지능형 건축물 인증기준 (국토교통부 고시): 건물 자동화 전기 배선 기준
- 소방시설 설치 및 관리에 관한 법률: 비상전원·발전기 용량 기준

### KS 규격
- KS C 8305 금속제 전선관 (두께·허용 전류)
- KS C 8459 금속제 케이블 트레이 및 래더 시스템
- KS C IEC 60364 시리즈 저압전기설비 (KEC 기반 설계 참고)
- KS C IEC 60529 외함 보호 등급 (IP 코드): 방수·방진 등급 기준

### 국제 기준 (설계 참고)
- IEC 60364 Low-voltage electrical installations (KEC의 기반 국제 표준)
- IEEE 1100 Powering and Grounding Electronic Equipment: 데이터센터·IT 장비 접지
- NFPA 70 National Electrical Code (NEC): 미국 기준 (해외 프로젝트 참고)

### BIM/설계 실무 적용 시 우선 확인 법규
1. 케이블 트레이 재질·치수 → KEC 220 + KS C 8459
2. 강전/약전 이격 → KEC 220.10 (전자기 간섭 방지 이격 기준)
3. 분전반 전면 공간 → KEC 210 (저압 전기설비 점검 공간 규정)

5. 비상전원·발전기 → 소방시설법 시행령 [별표 6] 비상전원 용량·종류 기준


## 2026-06-04 KEC 공식 출처 기반 전기 BIM 기준 게이트
- Source: `docs/knowledge_updates/daily/2026-06-04_LUA_BIM_LABS_KEC_ELECTRICAL_STANDARD_UPDATE.md`
- Tags: electrical,KEC,KDS32,KCS32,official-source,review-gate

전기 BIM 검토는 KEC 단독으로 판단하지 않고 전기설비기술기준, KDS/KCS 32, KS/IEC, NFTC/NFPC, 발주처 특기시방서를 함께 확인한다. KEC는 전기사업법 제67조 및 시행령 제43조 기반의 전기설비 안전 기준 축이며, 최신 공고 목록과 시행일을 확인한 뒤 프로젝트 기준으로 반영한다.

운영 기준:
- KEC 최신 공고 번호와 시행일 확인
- 실제 KEC 전문의 장/절/조항번호 확인
- KDS/KCS 32 또는 발주처 특기시방서와의 우선순위 확인
- 소방전기, 통신, 방재, EV 충전, ESS, UPS 관련 기준 충돌 여부 확인
- 수치 기준을 Revit 파라미터나 검토표에 넣기 전 근거 URL 또는 문서번호 기록

주의:
- 이 파일의 자동 수집 KEC 조항번호와 수치 문장은 공식 전문 확인 전 확정 기준으로 사용하지 않는다.
- 분전반 접근 공간, 강전·약전 이격, 트레이 충전율, 접지저항, 전선 색상, 곡률 반경은 프로젝트 특기시방서와 전기 설계자 확인을 거쳐 적용한다.

## 전기 케이블 트레이 BIM 확인 기준 상세 (2026-05-23)
- Source: LUA BIM LABS internal electrical BIM QA baseline
- Tags: electrical,cable-tray,bim,clearance,emi,bend-radius
- Links: [[통신]], [[설비시공조율]]

케이블 트레이에서 확인해야 하는 기본 기준:

트레이 규격:
- 폭: 200mm / 300mm / 450mm / 600mm / 750mm / 900mm / 1,200mm (강전 주 트레이 600mm 이상 일반적).
- 깊이: 50mm / 75mm / 100mm / 150mm.
- 충전율: 50% 이하 유지. 케이블 단면적 합계 ÷ 트레이 내부 단면적 × 100%.

강전/약전 이격:
- 강전 트레이(220V~380V 전력)와 통신 트레이(LAN, CCTV, 약전): 최소 300mm 이격 또는 금속 격벽.
- 강전 트레이 상층부 → 통신 트레이 하층부 원칙 (열, EMI 분리).
- 트레이가 교차할 때: 강전이 통신보다 위에 있으면 교차 지점에서 최소 300mm 수직 이격.

곡률 반경(Bend Radius):
- 단심 전력 케이블: 외경 × 15배 이상.
- 다심 전력 케이블: 외경 × 12배 이상.
- LAN Cat6 케이블: 외경 × 4배 이상.
- 광케이블: 외경 × 20~30배 이상 (케이블 사양 확인).
- 엘보·분기 피팅 위치에서 곡률 반경이 확보되는지 BIM에서 확인.

배선 방식:
- 고압 케이블: 트레이 최상층 또는 별도 독립 트레이.
- 저압 전력 케이블: 중간층.
- 약전 케이블: 최하층 또는 별도 약전 트레이 (강전과 물리적 분리).
- 같은 트레이에 강전·약전 혼재 금지.

물 배관 이격:

## 전기 배선 설계 기준 추가 (2026-05-23)

전선관(Conduit) 배선:
- 배선 방식: 전선관(금속·합성수지) 안에 전선을 넣어 보호.
- 용도: 소구경 전로, 분기 배선, 벽 매립 배선.
- 충전율: 전선 단면적 합계가 전선관 내부 단면적의 32% 이하.
- 금속 전선관: 접지 연속성 확보 용이. 기계실, 산업 환경에 적합.
- 합성수지 전선관(CD관, HI-C관): 가벼워 시공성 양호. 매립 배선에 많이 사용.

버스덕트(Bus Duct):
- 대용량 전력 수송 (500A 이상). 배전반 ↔ MCC 간 주 배선에 사용.
- 장점: 단락 용량 크고 열방산 유리. 케이블 대비 접속이 용이.
- BIM에서 버스덕트 경로, 지지 간격, 분기 박스 위치를 확인한다.
- 방화구획 관통 시 내화 차단 처리 필요 (버스덕트 전용 방화 피팅 사용).

발전기·UPS:
- 발전기: 비상전원. 정전 후 10~30초 내 기동. 연료(경유) 저장탱크, 냉각, 배기, 소음 검토.
- UPS(무정전전원장치): 순간 정전 보호. 서버실, 통신실, 의료 장비. 정전 시 수 분에서 수 시간 공급.
- 비상전원 대상: 비상조명, 소방설비, 피난설비, 통신설비, 의료기기(법정 비상전원 의무 대상 확인).

  - 일반 분기 회로: 2.5mm² 이상 (KEC 212.4).
  - 콘센트 회로: 2.5mm² 이상.
  - 조명 회로: 1.5mm² 이상 (220V 단상).
  - 동력 회로: 부하 전류의 1.25배 이상으로 선정 (KEC 212.5).

## 전기 마스터급 경험 지식 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: electrical,transformer,grounding,cable-tray,power-factor,field-experience

변압기 과부하 발생 시 응급 조치는 부하 재분배가 핵심이다. 단상 부하가 특정 상(R/S/T)에 집중되어 불평형이 발생하는 경우 중성선 과부하까지 동반되므로, 상별 전류를 클램프 미터로 측정해 편차가 20% 이상이면 분기 차단기 배치를 재조정한다. 응급 단계에서는 비핵심 부하(광고 조명, 예비 콘센트 회로)의 차단기를 내려 부하를 줄이고, 변압기 온도 감지기(Winding Temp) 알람을 모니터링하면서 교체 일정을 계획한다.

접지 저항 측정값이 10Ω 초과 시 추가 접지 방법: 기존 접지극의 저항이 높은 원인은 주로 토양 건조, 암반층 존재, 접지극 부식이다. 수평 보조 접지극(메시 접지)을 추가로 매설하거나 심타 공법(Deep Ground Rod, 6m 이상)을 적용한다. 도심 밀집 지역에서 공간이 부족할 때는 저저항 접지재(벤토나이트 혼합물)를 접지극 주변에 충전하면 저항을 50% 이상 낮출 수 있다. 접지 보조선을 설비 금속 구조물(케이블 트레이, 배관)과 등전위 본딩하는 것도 효과적이다.

케이블 트레이 허용 전류 계산 시 충전율(Fill Factor) 50% 초과 오류는 설계 단계에서 케이블 추가 여유분을 반영하지 않았을 때 발생한다. 트레이 충전율이 50%를 넘으면 열 축적으로 허용 전류가 감소하고 절연 열화가 빨라진다. 기존 트레이에 케이블을 추가할 때는 단면적 합계를 재계산하고, 충전율이 80%를 넘으면 트레이 폭을 상위 규격으로 교체하거나 병렬 트레이를 추가한다.

역률이 0.85 미만으로 측정되어 한전 요금 가산이 발생하는 경우 역률 개선 콘덴서 용량을 계산하여 설치한다. 콘덴서 용량(kVAR) = 유효전력(kW) × (tan θ1 - tan θ2) 공식으로 목표 역률(0.95 이상)까지 개선할 용량을 산출한다. 고조파 발생 부하(인버터, UPS, 서버 등)가 많은 건물에서는 일반 콘덴서가 공진 위험이 있으므로 직렬 리액터 부착형 콘덴서(Detuned Filter) 설치가 표준이다.

- EV 충전 의무: 신축 건물 주차면의 5%(2026.01.27 시행)
- KEC에서 EV 충전 회로는 전용 분기회로 필수 (기존 일반 콘센트 회로와 분리)
- BIM 파라미터:
  ```
  Pset_EVCharging_KR:
    - EV_Charger_Level: Level1(완속7kW)/Level2(급속50kW)/Level3(초급속350kW)
    - Dedicated_Circuit: Yes (전용 분기회로 여부)
    - Smart_Charging: Yes/No (스마트충전 기능)
    - Charging_Protocol: CCS/CHAdeMO/AC3Phase
    - Circuit_Breaker_Rating: A (차단기 용량)
  ```

**케이블트레이 BIM LOD 파라미터 체계 (KEC + KS C IEC 60364-5-52):**
```
Pset_CableTray_KR:
  - Material: GalvanizedSteel / AluminumAlloy / FRP / HDPE (KEC 2024 비금속 허용)
  - Width: mm (케이블트레이 폭 — 100/150/200/300/450/600mm)
  - Depth: mm (깊이 — 50/75/100mm)
  - Fill_Ratio: % (케이블 점유율 — KEC 기준 40% 이하 권장)
  - Fire_Rating: 내화(IEC 60332-3)/내열(IEC 60332-1)/일반
  - Grounding: Yes/No (트레이 접지 여부)
  - Cover: Yes/No (커버 설치 여부 — 습기·부식 환경)
```
