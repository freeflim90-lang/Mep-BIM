## 외주관리 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28

- **계약 필수 항목**: 작업 범위(SOW)·납품물 목록·일정·단가 및 지급 조건·IP 소유권(work-for-hire)·NDA·하자보수(3개월 이상)를 계약서에 반드시 포함한다. 착수금 30~50%·중간 40%·납품 완료 10~20% 분할 지급을 기본 구조로 적용하며, 납기 지연 시 지연 일수당 계약금 0.1% 페널티 조항을 삽입한다.
- **납품 검수 기준**: 소스코드 저장소 접근 권한 이전, README 기준 빌드 재현 가능, 단위 테스트 커버리지 60% 이상, 지원 Revit 버전 전체 smoke test 통과, 설치 가이드·API 명세 문서 완비를 검수 통과 조건으로 설정한다. 하자 발생 시 원 개발사에 10영업일 이내 수정 의무를 부여하며, 미이행 시 잔금을 보류한다.
- **보안 및 IP 관리**: 고객 프로젝트 모델은 외주사에 제공하지 않으며, 테스트 전용 더미 모델만 제공한다. 하드코딩 비밀값(API 키·패스워드) 없음, 개인정보 무단 수집 없음, Autodesk 공식 API만 사용 여부를 검수 체크리스트에 포함한다.
- **협력사 품질 등급 관리**: 납기 준수율·검수 통과율·하자 발생 건수를 기준으로 A/B/C 등급을 반기 1회 평가한다. C등급 연속 2회 시 신규 과업 발주를 중단하고 협력사안부와 협의하여 관계 재검토를 진행한다.
- **진행 중 모니터링**: 주간 진행 보고를 의무화하고 Milestone 기준으로 중간 지급을 처리한다. 일정 지연 징후 발생 시 경영지원에 즉시 보고하고 리스크 대응 계획을 수립한다.

## 외주관리 마스터급 경험 지식 (2026-05-29)
- Source: claude-code-enhanced 2026-05-29
- Tags: 외주관리, 비정형건물외주, 협력사선정, 실패패턴, 품질관리

### 비정형 건물 BIM 외주 특수 관리 기준

비정형 건물은 일반 BIM 외주사에게 의뢰 불가 → 전문 협력사 별도 관리 필수:

**비정형 외주사 선정 기준 (5점 척도):**

| 평가 항목 | 배점 | 평가 방법 |
|---------|-----|---------|
| 비정형 BIM 포트폴리오 | 30% | 실제 사례 3건 이상 + 납품 결과물 확인 |
| Dynamo/Grasshopper 역량 | 25% | 실기 테스트 (2시간) |
| IFC 납품 경험 | 20% | 비정형 부재 IFC Export 성공 사례 |
| 납기 준수율 | 15% | 이전 협력사 평가서 요청 |
| 보안/기밀 관리 | 10% | NDA 서명 이력 확인 |

### 외주관리 실패 패턴 4가지

**1. 비정형 경험 없는 일반 BIM 외주사 발주 → 납기 2배 초과**
- 해결: 비정형 프로젝트는 반드시 비정형 경험 보유 외주사만 발주

**2. 중간 산출물 검수 없이 최종 납품만 검수 → 대량 재작업**
- 해결: 주간 WIP(Work-In-Progress) 파일 공유 의무화, 클래시 이슈 조기 발견

**3. 저가 외주사 선택 → 모델 품질 불량 → 재작업 비용 원가 초과**
- 해결: 견적의 70% 이하 외주사 제외, 품질 보증 항목 계약 필수

**4. 외주 완료 후 파일 소유권 불명확 → 추가 요청 시 재비용**
- 해결: 계약서에 "납품 완료 시 모든 BIM 데이터 저작권 발주처(LUA BIM LABS) 귀속" 명기
