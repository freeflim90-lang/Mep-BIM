## 외주 개발 계약 및 검수 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: outsourcing,contract,acceptance

계약서 필수 항목: 작업 범위(SOW), 납품물 목록, 일정, 단가/지급 조건, IP 소유권(work-for-hire), NDA, 하자 보수(3개월 이상).
납품물 검수 기준:
□ 소스코드 저장소 접근 권한 이전
□ 빌드 재현 가능 (README 기준 빌드 성공)
□ 단위 테스트 커버리지 60%↑
□ 지원 Revit 버전 smoke test 통과
□ 문서(설치 가이드, API 명세) 완비
하자 발생 시 원 개발사 우선 수정 의무, 10영업일 내 미이행 시 대금 보류 조항 삽입.
