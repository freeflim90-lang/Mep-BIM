## 2026-06-04 BF/편의시설 공식 기준 기반 건축 BIM 게이트
- Source: `docs/knowledge_updates/daily/2026-06-04_LUA_BIM_LABS_BF_ACCESSIBILITY_STANDARD_UPDATE.md`
- Tags: architecture,BF,accessibility,barrier-free,official-source

건축 BIM에서 BF와 편의시설 검토는 주출입구 접근로, 장애인전용주차구역, 주출입구 높이차이 제거, 출입구, 문, 복도, 계단/승강기, 장애인등 이용 가능 화장실, 점자블록, 안내설비, 경보·피난설비, 객실·관람석·접수대와 연결된다.

운영 기준:
- 장애인등편의법, 시행령 별표, 시행규칙 별표, BF 인증 규칙은 `KST01 공식확인` 출처로 관리한다.
- 대상시설 여부, 시설 용도, 규모, 공공/민간 여부, 예비인증/본인증 단계, 지자체 사전검사 조건 확인 전에는 프로젝트 적용을 `KST03 적용주의`로 둔다.
- 자동 수집된 문폭, 경사, 단차, 회전반경, 화장실 세부 치수 문장은 `KST04 자동수집`으로 보고 공식 전문과 프로젝트 기준 대조 전 납품 판정 기준으로 쓰지 않는다.

BIM 모델 추가 확인 필드:
- BF 대상시설 여부
- 의무/권장 편의시설 구분
- 접근로, 주차, 출입구, 복도, 승강기, 화장실, 안내설비, 피난설비 모델링 여부
- 자동 검토 가능 항목과 수동 검토 필요 항목
- BF 예비인증/본인증 문서 또는 RFI 연결

## 2026-06-04 BF QA 체크리스트 모델러 샘플 반영
- Source: `docs/training_curriculum/team_distribution/samples/2026-06-04_BIM_MODELER_BF_QA_CHECKLIST_SAMPLE.md`
- Tags: architecture,BF,QA,checklist,training-sample

BIM 모델러는 BF/편의시설 검토를 모델 누락 찾기와 적합 판정으로 혼동하지 않는다. 문, 복도, 화장실, 주차, 접근로, 안내설비는 자동 검토 후보가 될 수 있지만, 대상시설 여부, 의무/권장, 예비인증/본인증, 지자체 조건, 특기시방서 해석은 수동 검토로 분리한다.

모델러 산출물에는 다음을 포함한다.
- 자동 검토 후보 항목
- 수동 검토 필요 항목
- KST 상태 코드
- 도면/모델 뷰/BCF/RFI 증빙
- PM 또는 전문가 확인 필요 여부

## 2026-06-04 BF 기준 미확정 리스크의 건축 견적 연결
- Source: `docs/training_curriculum/team_distribution/samples/2026-06-04_ESTIMATOR_BF_SCOPE_RISK_MEMO_SAMPLE.md`
- Tags: architecture,BF,estimate,risk,KST

건축 BIM에서 BF/편의시설 항목은 모델 누락 검토와 견적 책임 범위가 함께 움직인다. 대상시설, 의무/권장, 예비인증/본인증, 지자체 조건, 특기시방서가 확정되지 않으면 건축 모델링 범위에 BF 인증 대응을 포함했다고 해석하지 않는다.

건축 파트는 견적심사원에게 다음 정보를 넘긴다.
- BF 대상 가능성이 있는 시설 용도와 규모
- 모델에서 확인 가능한 편의시설 항목
- 자동검수 후보와 수동 검토 필요 항목
- 설계자 또는 인증기관 확인이 필요한 항목

## 방화구획 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: architecture,fire-compartment,code

국내 건축법 기준 방화구획은 바닥면적 1,000㎡(스프링클러 설치 시 3,000㎡)마다 구획한다.
관통부(배관·덕트·케이블)는 내화채움재·방화댐퍼로 막아야 하며, Add-in 보고서에는 구획 경계 교차 여부를 별도 분류해야 한다.
피난 계단·복도는 다른 용도 공간과 방화구획으로 분리하고, MEP 관통은 해당 구획 등급(1·2시간) 유지가 필수다.


## 천장고 및 마감 두께 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: architecture,clearance,ceiling

법정 최소 천장고: 거실 2.1m, 피난계단 2.1m, 주차장 2.3m(기계식 2.0m), 의료시설 2.4m.
마감 두께: 경량칸막이 75~150mm, 석고보드 양면 12.5mm×2, 바닥마감 40~100mm.
Add-in은 천장고 계산 시 구조체 하단에서 마감+단열+천장틀 전체 두께를 뺀 순 유효 천장고를 보고서에 표기해야 한다.
MEP 설비 돌출이 천장고 기준을 침해할 경우 건축 승인 필요 이슈로 분류한다.


## 실면적 및 피난 동선 검토 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: architecture,space,evacuation

피난 동선은 막힌 복도(dead-end corridor) 길이 제한(일반 6m, 스프링클러 설치 12m)이 있다.
비상구까지 보행거리: 일반 30m 이하, 스프링클러 50m 이하.
Add-in은 MEP 간섭이 피난 동선 폭(최소 1.2m) 또는 비상구 전실을 침범하면 최우선 경고로 표시한다.
실 분할·확장 시 방화구획 재검토 필요 여부도 보고서 항목으로 포함한다.


## 건축 모델 명명 규칙 및 BIM 데이터 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: architecture,naming,bim-data

레벨명: FL_B2, FL_1F 형식 권장. 실(Room) 명칭은 용도+번호(예: 회의실_201) 표준화.
공유 파라미터: 실용도, 방화구획ID, 피난경로여부, 마감두께, 천장고(설계/실측) 필수.
Add-in이 읽는 패밀리 타입 파라미터는 Revit 기본값 아닌 프로젝트 공유 파라미터 기준으로 한다.
뷰 템플릿은 구조체/마감/가구를 별도 레이어로 관리해 MEP 협의도 추출이 가능하게 한다.


## 건축 공종과 MEP 조율 기준 (2026-05-23)
- Source: LUA BIM LABS internal architectural-MEP coordination baseline
- Tags: architecture,mep,coordination,bim,ceiling,shaft
- Links: [[설비시공조율]], [[설비도면해석]]

건축과 MEP 조율에서 자주 충돌하는 항목:

천장 관련:
- 법정 최소 천장고: 거실 2.1m, 의료시설 2.4m, 주차장 2.3m. MEP 설비가 천장을 침범하면 법적 기준 확인 필요.
- 천장 마감 방식에 따라 MEP 점검 가능 여부가 달라진다. 시스템 천장(T-Bar)은 점검구 없이 패널 탈거 가능.
- 석고보드 직부 천장: 설비 점검구 위치를 확정해야 하며 시공 후 변경이 어렵다.
- 설비 돌출로 천장고가 낮아지면 인허가 재검토 또는 구조 레벨 조정 협의가 필요하다.

방화구획 관통 처리:
- MEP 배관·덕트가 방화구획을 관통하면 내화채움재(Firestop) 또는 방화댐퍼가 필수.
- 방화구획 면적 기준: 일반 건물 1,000㎡, 스프링클러 설치 시 3,000㎡ (건축법 시행령 제46조).
- BIM에서 방화구획 경계를 교차하는 모든 MEP 요소를 자동 감지하고 처리 방법을 확인한다.
- 내화채움재는 관 외경 기준으로 슬리브 크기와 내화 성능(1시간, 2시간 등)이 결정된다.

피난 경로 침해:
- 피난 통로 유효 폭: 직통계단은 1.2m 이상, 피난계단 전실은 1.5m 이상.
- 비상구까지 보행 거리: 일반 30m, 스프링클러 50m.
- MEP 배관, 트레이, 덕트가 피난 경로 위로 지나갈 때 하단고가 2.1m 이상인지 확인.

마감 두께와 MEP 이격:
- 벽체 마감 두께(석고보드 12.5mm×2 + 경량 칸막이 50~100mm): 배관 접속구 위치에 영향.
- 바닥 마감 두께(40~100mm): 바닥 배관 슬리브 위치와 돌출 높이에 영향.
- 천장 마감(천장틀 + 석고보드 = 약 100~150mm): 설비 레벨 계산 시 포함해야 함.

창호와 설비:
- 커튼월 또는 창호 주변 공조 배관(FCU), 라디에이터, 덕트 배치 시 창호 프레임 간섭 확인.
- 환기구(루버)와 외기 덕트 접속 위치: 건축 파사드 디자인과 MEP 기능 요건이 충돌하는 구간.

Revit 연동 확인:
- Room/Space 경계: MEP Space는 건축 Room과 경계를 공유해야 정확한 부하 계산이 된다.
- 레벨 공유: 건축 레벨과 MEP 레벨이 불일치하면 조율 혼선이 생긴다.
- 링크 모델 방식: 건축 모델을 MEP 모델에 링크로 참조. 링크 업데이트 주기와 담당자를 확인한다.
