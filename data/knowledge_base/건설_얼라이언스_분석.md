### 비정형 건물 특화 얼라이언스 전략

비정형 건물 BIM 사업은 단독 수행 역량이 부족한 영역 → 전략적 파트너십 필수:

| 파트너 유형 | 협력 내용 | LUA BIM LABS 제공 | 파트너 제공 |
|-----------|---------|----------------|----------|
| 구조 설계사 | 비정형 구조 BIM 공동 수행 | MEP BIM + Add-in | 구조 전문성 + 발주처 네트워크 |
| Rhino/GH 전문 디자인 스튜디오 | 파라메트릭 설계 → BIM 변환 | Revit 변환 도구 | 디자인 모델 |
| BIM 소프트웨어 리셀러 | 기술 지원 + 교육 | 전문 교육 커리큘럼 | 유통 채널 |
| 공공 발주처 BIM 컨설팅 | BIM 수행 계획 작성 | EIR 템플릿 + 도구 | 공공 레퍼런스 |

### 얼라이언스 실패 패턴 4가지

**1. 역할 분담 불명확 → 납품 분쟁**
- 원인: "함께 하자" 합의만 있고 계약서 없음
- 해결: MOU 체결 시 산출물 목록, 수익 배분율, 하자 책임 범위 명기

**2. 발주처 직접 접촉 방지 조항 누락**
- 원인: 파트너가 소개한 발주처에 직접 접근 → 관계 파탄
- 해결: 3년 비경쟁 조항(Non-Compete) + 소개 수수료 체계 명기

**3. 기술 이전 없이 파트너 의존**
- 원인: 파트너사 고유 기술에 종속 → 파트너 이탈 시 사업 불가
- 해결: 기술 문서화 + 내부 역량 병행 개발; 파트너 의존도 50% 이하 유지

**4. 비정형 건물 얼라이언스 파트너 역량 미검증**
- 원인: "Rhino 가능" → 실제 Grasshopper→Revit 변환 경험 없음
- 해결: PoC(개념 검증) 프로젝트로 역량 검증 후 정식 파트너 계약
