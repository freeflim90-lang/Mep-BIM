## Revit Add-in 설치 경로 및 매니페스트 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: deployment,manifest,install-path

설치 경로(시스템 전체): C:\ProgramData\Autodesk\Revit\Addins\[버전]\
설치 경로(사용자): %AppData%\Autodesk\Revit\Addins\[버전]\
.addin 파일과 DLL은 같은 폴더 또는 .addin의 Assembly 태그에 전체 경로 명시.
버전별 폴더 분리 필수(2023, 2024, 2025, 2026). MSI 설치 시 각 버전 폴더에 자동 배포.

# [제품명] v_._._ 설치 가이드

## 사전 요구사항

| 항목 | 버전 |
|---|---|
| Autodesk Revit / Navisworks | ___ 이상 |
| Windows | 10 / 11 64-bit |
| .NET Framework / Runtime | ___ |
| 관리자 권한 | 필요 / 불필요 |

## 설치 절차

### MSI 설치 (권장)
1. [다운로드 링크]에서 `[제품명]_v_._._Setup.msi` 다운로드
2. 관리자 권한으로 실행
3. 설치 경로 확인 후 [설치] 클릭
4. Revit / Navisworks 재시작
5. 리본 탭 [탭명] 표시 확인

### 수동 설치 (MSI 미사용 환경)
1. 압축 파일 해제 후 DLL을 `%APPDATA%\Autodesk\Revit\Addins\[버전]\` 에 복사
2. `.addin` 파일을 동일 폴더에 복사
3. Revit 재시작

## 설치 확인

| 확인 항목 | 예상 결과 |
|---|---|
| Revit 리본에 탭 표시 | [탭명] 확인 |
| 기본 명령 실행 | 정상 동작 |
| 라이선스 상태 | 활성화 / 체험판 |

## 제거 방법
Windows 설정 → 앱 → [제품명] → 제거

## 문제 해결

| 증상 | 해결 방법 |
|---|---|
| 리본 미표시 | `.addin` 파일 경로 확인, Revit 재시작 |
| 설치 중단 | 관리자 권한으로 재실행 |
| 라이선스 오류 | 인터넷 연결 확인 후 재시도 |

- Docs-as-Code 파이프라인: Markdown 원본을 Git에서 관리하고, GitHub Actions로 MkDocs 또는 Docusaurus v3를 빌드하여 GitHub Pages에 자동 배포한다. 버전 태그(`git tag v2.1.0`) 생성 시 릴리스 노트가 자동으로 Autodesk App Store 업데이트 초안으로 변환되는 스크립트를 Python 3.12로 구현한다.
- 설치 가이드 표준화: Windows 10/11 기준, .NET 8 Runtime 설치 여부 사전 확인 → MSI 실행 → Revit Add-in Manager에서 로드 확인의 3단계로 구성한다. Revit 2025의 경우 `%APPDATA%\Autodesk\Revit\Addins\2025\` 경로에 `.addin` 파일이 배치되어야 한다.

## 배포문서 마스터급 경험 지식 (2026-05-29)
- Source: claude-code-enhanced 2026-05-29
- Tags: 배포문서, 비정형기능릴리스, 릴리스노트, 배포실패패턴

### 비정형 건물 BIM 기능 릴리스 시 배포 문서 특이사항

**비정형 기능 릴리스 노트 필수 포함 항목:**
1. 지원 비정형 수준 명시 (경사/복합경사/자유곡면/전면비정형)
2. 지원 Revit 버전 (비정형 API는 버전별 차이 큼)
3. Dynamo/Grasshopper 연동 버전 요구사항
4. 알려진 제한사항 (특정 곡면 유형에서 오류 발생 등)
5. 마이그레이션 노트 (이전 버전 비정형 프로젝트 호환성)

**릴리스 노트 템플릿:**
```markdown
## v{버전} 릴리스 노트 ({날짜})

### 신규 기능
- [비정형] 자유 곡면 면적 산출 (오차 ±2% 이내)
- [비정형] Grasshopper → Revit 패널 분할 자동화

### 지원 환경
- Revit 2022/2023/2024/2025
- Dynamo 2.17+, Rhino 8+
- 비정형 수준: 경사/복합경사/자유곡면

### 알려진 제한사항
- 완전 폐합곡면(예: 구형 지붕) IFC 수출 시 면 분할 필요

### 마이그레이션
- v{이전버전}에서 업그레이드 시 비정형 패밀리 재로드 필요
```

### 배포 실패 패턴 3가지

**1. 릴리스 노트 없는 배포 → CS 문의 폭발**
- 해결: 릴리스 노트 없으면 배포 금지 원칙

**2. Revit 버전 미기재 → 호환성 오류 CS**
- 해결: 배포 전 체크리스트: Revit 2022/2023/2024/2025 각각 스모크 테스트

**3. 비정형 제한사항 미공개 → 고객 신뢰 손상**
- 해결: 알려진 제한사항은 솔직하게 공개, 다음 버전 로드맵 함께 제시
