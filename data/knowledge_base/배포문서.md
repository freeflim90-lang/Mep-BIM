## Revit Add-in 설치 경로 및 매니페스트 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: deployment,manifest,install-path

설치 경로(시스템 전체): C:\ProgramData\Autodesk\Revit\Addins\[버전]\
설치 경로(사용자): %AppData%\Autodesk\Revit\Addins\[버전]\
.addin 파일과 DLL은 같은 폴더 또는 .addin의 Assembly 태그에 전체 경로 명시.
버전별 폴더 분리 필수(2023, 2024, 2025, 2026). MSI 설치 시 각 버전 폴더에 자동 배포.

# [제품명] v_._._ 설치 가이드

## 사전 요구사항

| 항목 | 버전 |
|---|---|
| Autodesk Revit / Navisworks | ___ 이상 |
| Windows | 10 / 11 64-bit |
| .NET Framework / Runtime | ___ |
| 관리자 권한 | 필요 / 불필요 |

## 설치 절차

### MSI 설치 (권장)
1. [다운로드 링크]에서 `[제품명]_v_._._Setup.msi` 다운로드
2. 관리자 권한으로 실행
3. 설치 경로 확인 후 [설치] 클릭
4. Revit / Navisworks 재시작
5. 리본 탭 [탭명] 표시 확인

### 수동 설치 (MSI 미사용 환경)
1. 압축 파일 해제 후 DLL을 `%APPDATA%\Autodesk\Revit\Addins\[버전]\` 에 복사
2. `.addin` 파일을 동일 폴더에 복사
3. Revit 재시작

## 설치 확인

| 확인 항목 | 예상 결과 |
|---|---|
| Revit 리본에 탭 표시 | [탭명] 확인 |
| 기본 명령 실행 | 정상 동작 |
| 라이선스 상태 | 활성화 / 체험판 |

## 제거 방법
Windows 설정 → 앱 → [제품명] → 제거

## 문제 해결

| 증상 | 해결 방법 |
|---|---|
| 리본 미표시 | `.addin` 파일 경로 확인, Revit 재시작 |
| 설치 중단 | 관리자 권한으로 재실행 |
| 라이선스 오류 | 인터넷 연결 확인 후 재시도 |
