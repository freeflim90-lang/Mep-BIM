## 모니터링 방법 — 공식 소스별 확인 경로

### 소방청 (NFTC 계통 — 최우선 확인)
- 소방청 법령정보: https://www.law.go.kr 에서 "화재안전기술기준" 검색
- 소방청 고시 목록 확인 경로: 국가법령정보센터 → 행정규칙 → 소방청
- 개정 현황 포인트: NFSC → NFTC 전환 이후 계통별 고시번호 변경됨 → 구 NFSC번호로 검색 시 누락 위험

### 국토교통부 (기계설비, 건축에너지, 건축법)
- 국가법령정보센터: https://www.law.go.kr → 부령·고시 → 국토교통부
- 기계설비법 관련 하위법령: 시행령·시행규칙·기술기준 고시 모두 별도 확인

### 산업통상자원부 (KEC, 에너지이용합리화)
- 한국전기기술기준위원회: www.kec.go.kr (KEC 최신판 다운로드 가능)
- e-나라표준인증 (eStandard): https://www.standard.go.kr → KS 규격 검색

### 환경부 (수도법, 실내공기질)
- 국가법령정보센터 → 환경부 고시

### 과학기술정보통신부 (통신, 방송설비)
- 국가법령정보센터 → 과기정통부 고시
- 방통위 법령정보: https://www.kcc.go.kr

### 한국산업표준 KS 규격
- e-나라표준인증: https://www.standard.go.kr → 규격 검색 → 개정이력 확인

### 법규 변경 감지 자동화 (RSS/스크래핑)

```python
import httpx, feedparser, hashlib, json
from pathlib import Path

# 국토부 공지사항 RSS 모니터링
FEEDS = {
    "국토부_법령정보": "https://www.molit.go.kr/rss/rss_law.xml",
    "조달청_공고": "https://www.pps.go.kr/rss/notice.xml",
    "소방청_법령": "https://www.nfa.go.kr/rss/law.xml",
}

def check_new_items(feed_url: str, cache_file: Path) -> list[str]:
    feed = feedparser.parse(feed_url)
    seen = json.loads(cache_file.read_text()) if cache_file.exists() else []
    new_items = []
    for entry in feed.entries:
        item_id = hashlib.md5(entry.link.encode()).hexdigest()
        if item_id not in seen:
            new_items.append(entry.title)
            seen.append(item_id)
    cache_file.write_text(json.dumps(seen[-1000:]))  # 최근 1000개 유지
    return new_items

# 매일 실행 → 새 항목 발견 시 텔레그램 알림
```
