# 법규변경모니터링 지식 베이스

## 개요
- Source: LUA BIM LABS regulatory watch baseline
- Tags: regulation,monitoring,update,legal-change
- Last verified: 2026-05-22

MEP·소방·전기·통신 분야 법규는 수시로 개정된다. 이 파일은 변경 빈도가 높은 법규를 추적하고, 지식베이스 업데이트 시 어떤 문서를 먼저 확인해야 하는지 안내한다.


## 변경 빈도별 법규 분류

### 🔴 변경 빈도 높음 — 분기~연 1회 이상 개정
| 법규/고시 | 주관 기관 | 비고 |
|-----------|-----------|------|
| NFTC (화재안전기술기준) 전 계통 | 소방청 | 수시 고시 개정, 기술기준 세분화 계속 진행 중 |
| 화재안전성능기준 (NFPC) | 소방청 | 2023년 이후 NFTC와 병행 운용 중 — 혼용 주의 |
| 건축물 에너지절약 설계기준 | 국토교통부 | 에너지 정책에 따라 단열·효율 기준 수시 갱신 |
| 실내공기질 관리법 시행규칙 | 환경부 | 오염물질 기준 항목·수치 지속 강화 |
| 다중이용시설 환기 기준 | 환경부/국토부 | 코로나 이후 환기량 기준 강화 추세 |

### 🟡 변경 빈도 중간 — 1~3년 주기
| 법규/고시 | 주관 기관 | 비고 |
|-----------|-----------|------|
| 한국전기설비규정 (KEC) | 산업통상자원부 | 2021 전면 시행 후 개정판 출간 중 |
| 건축전기설비 설계기준 | 국토교통부 | 에너지·친환경 설비 기준 반영 |
| 기계설비법 시행규칙 | 국토교통부 | 2021년 신법, 시행규칙 보완 계속 |
| 초고속정보통신건물 인증기준 | 과기정통부 | 등급 기준 재편 가능성 있음 |
| 방송통신설비 기술기준 규정 | 과기정통부 | 5G·광통신 반영 개정 진행 중 |
| 수도법 시행규칙 | 환경부 | 급수관 재질·수질 기준 갱신 |

## 모니터링 방법 — 공식 소스별 확인 경로

### 소방청 (NFTC 계통 — 최우선 확인)
- 소방청 법령정보: https://www.law.go.kr 에서 "화재안전기술기준" 검색
- 소방청 고시 목록 확인 경로: 국가법령정보센터 → 행정규칙 → 소방청
- 개정 현황 포인트: NFSC → NFTC 전환 이후 계통별 고시번호 변경됨 → 구 NFSC번호로 검색 시 누락 위험

### 국토교통부 (기계설비, 건축에너지, 건축법)
- 국가법령정보센터: https://www.law.go.kr → 부령·고시 → 국토교통부
- 기계설비법 관련 하위법령: 시행령·시행규칙·기술기준 고시 모두 별도 확인

### 산업통상자원부 (KEC, 에너지이용합리화)
- 한국전기기술기준위원회: www.kec.go.kr (KEC 최신판 다운로드 가능)
- e-나라표준인증 (eStandard): https://www.standard.go.kr → KS 규격 검색

### 환경부 (수도법, 실내공기질)
- 국가법령정보센터 → 환경부 고시

### 과학기술정보통신부 (통신, 방송설비)
- 국가법령정보센터 → 과기정통부 고시
- 방통위 법령정보: https://www.kcc.go.kr

### 한국산업표준 KS 규격
- e-나라표준인증: https://www.standard.go.kr → 규격 검색 → 개정이력 확인


## 지식베이스 법규 업데이트 원칙

### 업데이트 트리거 (즉시 업데이트)
1. 팀원이 현장에서 "기준이 바뀐 것 같다"고 보고할 때
2. 관할 소방서·인허가 기관에서 새 기준 적용을 요구할 때
3. NFTC/법령 개정 공포 소식을 확인했을 때
4. 프로젝트 인허가 과정에서 기존 지식과 다른 기준이 제시될 때

### 정기 검토 주기
- NFTC 계통: **분기 1회** 소방청 고시 신규·개정 여부 확인
- KEC·국토부 고시: **반기 1회** 개정 여부 확인
- KS 규격·국제기준: **연 1회** 개정이력 확인

### 업데이트 방법
1. 해당 계통 지식베이스 파일에 새 섹션 추가 (`## [업데이트 내용] (날짜)`)
2. 변경된 근거문서 섹션에서 구버전 고시번호·조항에 "⚠️ [날짜] 개정됨" 표시
3. 이 파일(`법규변경모니터링.md`)의 "최근 업데이트 이력"에 기록


## 최근 주요 변경 이력 (2026-05-22 기준 파악분)

| 날짜 | 계통 | 변경 내용 |
|------|------|-----------|
| 2022.12 | 소방 전 계통 | NFSC → NFTC/NFPC 체계로 전면 개편 (법률 제18522호 시행) |
| 2021.01 | 전기 | KEC(한국전기설비규정) 전면 시행 — 구 전기설비기술기준·판단기준 통합 폐지 |
| 2021.04 | 기계설비 전 계통 | 기계설비법 시행 (신설) — 공조·위생·소방기계 포함 |
| 2023 | 공조·에너지 | 건축물에너지절약설계기준 고시 제2023-88호 개정 (단열·효율 기준 강화) |

> ⚠️ 위 이력은 파악된 범위 내 정리이며, 개정 후 지식베이스에 반영 시 이 표를 갱신한다.
