| LOD | MDF/IDF | 케이블 트레이 | 약전 기기 |
|-----|--------|------------|---------|
| 200 | 층별 IDF 위치 | 주 경로 구분 | CCTV·방송 구역 |
| 300 | 함체 치수·위치 | 단면·폭·경로 | 기기 위치·타입 |
| 350 | 배선 경로·단자 | 피팅·지지물 | 케이블 트레이 연결 |
| 400 | 내부 패치패널 | 볼트·행거 | 전체 배선 상세 |

**통신설비 BIM 간섭검토 우선순위:**
1. **MDF/IDF 접근 공간**: 전면 1,000mm — 패치 작업 필수 공간
2. **강전/약전 트레이 이격**: 300mm 또는 격벽 — 데이터 신호 노이즈 방지
3. **CCTV 카메라 시야각**: 죽은 각도(Dead Zone) 없이 커버리지 확인
4. **방송 스피커 위치**: 비상방송 가청 범위 60dB 이상 확인
5. **UTP 케이블 최대 길이**: Cat.6 100m 이내 (MDF→단말 거리)

| 홈네트워크 | 세대별 통합제어 | 세대 단자함·허브 위치 |

**스마트빌딩 통신 인프라 트렌드 (2026):**
- Wi-Fi 6E/7: 6GHz 대역 추가 → 전관 무선 커버리지 BIM 시뮬레이션
- PoE(Power over Ethernet): UTP로 전력·데이터 동시 전송 → 별도 전원배선 감소
- 건설기준 디지털화(국토부 2022~2026): BIM 환경에서 통신 기준 즉시 검색 가능

## 약전 트레이 이격 및 EMI 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: telecom,emi,cable-tray

통신 케이블(Cat6, 광케이블)과 전력선(AC 220V~380V): 최소 150mm 이격(TIA-569 기준).
변압기·UPS·형광등 안정기 근처 300mm 이내는 통신 배선 금지.
약전 트레이 내부: 전화·LAN·방송·CCTV 계통을 격벽 또는 별도 트레이로 분리.
Add-in은 약전 트레이와 전력 배관 간격을 자동 측정하고 EMI 위험 구간을 통신 전용 경고로 분류한다.


## 2026-06-04 KEC 연계 강전·약전 이격 확인 게이트
- Source: `docs/knowledge_updates/daily/2026-06-04_LUA_BIM_LABS_KEC_ELECTRICAL_STANDARD_UPDATE.md`
- Tags: telecom,KEC,EMI,electrical-clearance,official-source,review-gate

통신 BIM에서 강전·약전 이격은 통신 기준만으로 판단하지 않는다. KEC, KDS/KCS 32, 방송통신설비 기술기준, TIA/ISO 기준, 발주처 특기시방서를 함께 확인한다.

운영 기준:
- 강전 트레이·전선관과 통신 트레이·통신 배관의 이격 기준 출처를 명시한다.
- 격벽, 금속관, 교차부, 병행부처럼 대체 조건이 있는 경우 전기·통신 설계자 확인을 받는다.
- 자동 수집된 이격 수치는 공식 전문 확인 전 확정 기준으로 사용하지 않는다.

## MDF·IDF 설치 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: telecom,mdf,idf

MDF(주배선반): 건물 당 1개소, 통신실 전용 공간, 바닥하중 500kg/㎡ 이상.
IDF(중간배선반): 층당 1개소, 서비스 구역 반경 90m 이내(UTP 케이블 100m 제한).
통신실 냉방: 상시 18~27℃, 상대습도 30~60%.
Add-in은 IDF에서 최원단 통신 단말까지의 배선 길이를 계산하고 100m 초과 구간을 보고한다.


## CCTV·방송·비상방송 배선 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: telecom,cctv,broadcast

CCTV 동축 케이블(RG-59): 최대 300m(HD-SDI 100m), 광케이블로 연장 가능.
비상방송 케이블: 내화 케이블(FR) 사용 필수, 소방 계통 배선과 동일 내화 등급.
CCTV 카메라 전원: PoE 최대 100m, 별도 전원 공급 시 트레이 분리.
Add-in은 CCTV 케이블 길이 초과 구간과 비상방송 비내화 배선 구간을 보안·소방 협의 항목으로 분류한다.


## 통신 적용 근거문서 및 법규 기준 (2026-05-22)
- Source: LUA BIM LABS regulatory reference baseline
- Tags: telecom,regulation,standard,legal-basis

### 국내 법령
- 정보통신공사업법 (법률 제18428호): 통신공사 시공·감리 자격, 설계·시공 의무 등록
- 방송통신발전 기본법: 방송통신 설비 설치 기본 원칙
- 방송통신설비의 기술기준에 관한 규정 (과학기술정보통신부령): 통신 배선·단자·설비 기술 기준
- 전기통신사업법: 통신 사업자 설비 연결 기준 (건물 인입점, MDF 연결)
- 정보통신망 이용촉진 및 정보보호 등에 관한 법률: 보안 카메라·CCTV 관련 개인정보 처리 기준

### 국내 고시 및 인증
- 초고속정보통신건물 인증제도 기준 (과학기술정보통신부 고시): 홈네트워크·특등급·1등급·2등급 인증 배선 기준
- 지능형 홈네트워크 설비 설치 및 기술기준 (과학기술정보통신부 고시): 세대 단자함, 홈네트워크 배선 기준
- 공동주택 통신설비 설치기준: 세대 내 정보통신 배선, TV·인터넷·인터폰 배관 기준
- 방재실 및 종합방재실 설치 기준: 방재·감시 통합 통신선로 기준

### KS 규격
- KS X 4603 정보기술 — 건물 내 일반 배선 (ISO/IEC 11801 국내 채택)
- KS C 4715 광케이블 배선 시스템: 광케이블 사양 및 배선 기준
- KS C 2504 비차폐 쌍선 케이블(UTP) 기준

### 국제 기준 (설계 참고)
- TIA-568.2-D Balanced Twisted-Pair Cabling Components: UTP Cat6/Cat6A 성능 기준
- TIA-569-D Commercial Building Standard for Telecommunications Pathways and Spaces: 트레이 경로·통신실(MDF/IDF) 치수 기준
- TIA-942-B Telecommunications Infrastructure Standard for Data Centers: 데이터센터 통신 배선 기준
- ISO/IEC 11801 Generic Cabling for Customer Premises: 건물 내 일반 배선 국제 표준
- ANSI/BICSI 002 Data Center Design and Implementation Best Practices

### BIM/설계 실무 적용 시 우선 확인 법규
1. MDF/IDF 통신실 치수·환경 → TIA-569-D (면적·냉방·바닥하중 기준)
2. UTP 케이블 최대 길이 → TIA-568 (수평배선 90m + 패치코드 10m = 100m)
3. 초고속 인증 등급별 배선 기준 → 초고속정보통신건물 인증기준 고시
4. 비상방송 내화배선 → NFTC 202 (소방청 고시) — 통신이 아닌 소방 기준 우선 적용
5. CCTV 개인정보 → 개인정보보호법 + 정보통신망법 (설치 위치·저장·열람 기준)


## 통신 배선 체계 상세 (2026-05-23)
- Source: LUA BIM LABS internal telecom BIM baseline
- Tags: telecom,lan,cabling,mdf,idf,horizontal,vertical
- Links: [[전기]], [[설비시공조율]]

건물 내 통신 배선 구조 — 수직 간선과 수평 배선으로 나눈다.

수직 간선(Backbone Cabling):
- MDF(Main Distribution Frame, 주배선반) → IDF(Intermediate Distribution Frame, 층별 배선반).
- 케이블 종류: 광케이블(Single Mode 또는 Multi Mode). 거리 제한 없음(수백 m~km 가능).
- 설치: 수직 샤프트(EPS/TPS) 통해 층간 배선. 광케이블 최소 곡률 반경 준수 (일반 광 30mm).
- 용량: 처음부터 여유 코어를 확보. 장래 증설을 위한 여분 코어(Spare Fibers) 포함.

수평 배선(Horizontal Cabling):
- IDF → 각 층 말단 단자(정보통신 콘센트, 액세스포인트, 카메라 등).
- 케이블 종류: UTP Cat6 (최대 수평 배선 90m + 패치코드 10m = 총 100m 이내).
- 고속 환경: Cat6A (10Gbps 지원, 굵고 무거우므로 트레이 폭/중량 확인).
- 배관: CD관(16~28mm), 금속 전선관, 약전 전용 트레이.

IDF 위치 선정 기준:
- 서비스 반경: 수평 배선 90m 이내 도달 가능 위치.
- 면적: 랙 1대 이상 + 작업 공간 최소 600mm.
- 냉방: 발열 있는 Active 장비 설치 시 상시 냉방 필요.
- 전원: 전용 회로 + 소용량 UPS (네트워크 장비 정전 보호).


## CCTV 시스템 구성 상세 (2026-05-23)
- Source: LUA BIM LABS internal telecom baseline
- Tags: telecom,cctv,camera,nvr,poe,bim
- Links: [[전기]], [[소방전기]]

CCTV 시스템 구성 요소:
- IP 카메라: 현재 주류. PoE(Power over Ethernet)로 전원과 데이터 동시 전송. Cat6 1가닥으로 해결.
- NVR (Network Video Recorder): 디지털 영상 저장 장치. 용량은 카메라 수, 해상도, 녹화 일수로 결정.
- PoE 스위치: 카메라에 전원 공급 + 데이터 수집. PoE+ (30W/포트)이면 PTZ 카메라도 지원.
- VMS (Video Management System): 영상 관제 소프트웨어. 다수 카메라 통합 모니터링.

케이블 길이 기준:
- Cat6 PoE 방식: 최대 100m (수평 배선 90m + 패치코드 10m).
- 100m 초과 시: 광케이블 + 미디어 컨버터, 또는 PoE 익스텐더.
- 동축 케이블(구형): RG-59 최대 300m, HD-SDI 100m. 신규 설치는 IP 방식 권장.

카메라 배치 기준:
- 감시 목적: 출입구, 주차장, 복도, 엘리베이터 내부, 계단.
- 개인정보보호: 화장실, 탈의실, 의무실 내부 설치 금지 (개인정보보호법).
- 천장 높이: 2.5~4m 권장. 너무 높으면 해상도 저하, 너무 낮으면 사각지대 증가.
- PTZ(Pan/Tilt/Zoom) 카메라: 넓은 공간 1대로 커버 가능. 주차장 상부, 로비에 사용.

BIM 확인 항목:
- 카메라 위치와 시야각을 BIM 모델에서 확인 (시야 원추형 표현 가능).
- PoE 스위치에서 카메라까지 Cat6 케이블 배선 경로와 길이 확인.
- NVR 설치 공간: 전용 랙, 냉방, 전원, 배선 접근성.


## 홈네트워크 및 세대 배선 (2026-05-23)
- Source: LUA BIM LABS internal telecom baseline
- Tags: telecom,home-network,apt,세대배선
- Links: [[전기]]

공동주택 세대 내 통신 배선 기준:
- 세대 단자함: 세대별 통신 배선의 허브. 인터넷·TV·전화·인터폰 배선 집결.
- 배관: 세대 단자함 → 각 방 콘센트 배관은 CD관 16mm 이상 독립 배관.
- 홈네트워크: 세대 내 월패드(Wall Pad) 연결. 인터폰, 도어락, 일부 가전 제어 통합.

초고속정보통신건물 인증:
- 특등급: 기가비트 이상 속도 보장. 광케이블 세대 직접 인입(FTTH).
- 1등급: 100Mbps 이상. Cat5e 이상 배선.
- 2등급: 50Mbps 이상.
- 인증 등급에 따라 설계 단계에서 배관, 케이블 사양, 단자함 사양이 결정된다.

- 약전 트레이와 강전 트레이 이격: 동일 경로 병행 시 최소 150mm(TIA-569 기준), 변압기·형광등 안정기 등 고발열 전기기기에서 300mm 이상.

- 굴곡(Bend): 최소 굴곡 반경은 트레이 폭의 3배 이상. 광케이블 단독 포설 트레이는 광 굴곡 반경 기준 우선.

- MDF(주배선반): 건물 당 1개소, 전용 통신실. 바닥 내하중 500kg/㎡ 이상. 외부 통신사 인입 케이블 집결점.
- IDF(중간배선반): 서비스 반경 수평 배선 90m 이내 위치. 최소 전용 랙 1대 + 전면 600mm 작업 공간 확보.
- 통신실 환경 기준: 온도 18~27℃(상시 냉방), 상대습도 30~60%, 정전기 방지 바닥재.
