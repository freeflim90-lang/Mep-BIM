## 경영지원 마스터급 경험 지식 (2026-05-29)

### BIM 프로젝트 계약 경영지원 특이사항

**비정형 건물 BIM 계약 시 추가 검토 항목:**
1. **공수 가산 근거 명기**: 비정형 건물 가산율(10~70%)을 계약서 특기사항에 명시, 구두 합의만으로 계약 금지
2. **LOD 검수 기준 정량화**: "LOD 350 이상" 문구 대신 납품물 목록 + Solibri 검증 기준 첨부
3. **IFC 납품 면책 조항**: 비정형 부재 IFC 변환 시 발주처 뷰어 미지원으로 인한 납품 오류는 재작업 별도 발주 조항 삽입
4. **하자보수 범위 제한**: 비정형 모델 수정은 '원설계 범위 내'로 한정, 설계 변경에 따른 모델 수정 제외

### 경영지원 운영 실패 패턴 3가지

**1. 계약서 검토 없이 착수 → 분쟁**
- 원인: "일단 시작하고 계약서는 나중에" 관행
- 해결: 착수 전 계약서 서명 완료 철칙, LOI(착수의향서)라도 받기

**2. 세금계산서 발행 타이밍 오류**
- 원인: 중간 납품 시점 아닌 프로젝트 완료 시 일괄 발행 → 현금흐름 악화
- 해결: 마일스톤 기준 세금계산서 발행 (설계 30%, 시공 50%, 준공 20%)

**3. 외주 협력사 4대보험 처리 누락**
- 비정형 전문 인력 외부 채용 시 도급 vs 고용 구분 오류
- 해결: 법무·회계 자문 후 계약 형태 결정

- 계약 관리는 명확한 계약 조건과 서면화가 필요합니다. 특히 인력 관련 계약에서는 근로기준법을 준수해야 하며, 최소 임금 기준은 2025년 현재 서울 기준으로 월 1,640,000원입니다.
- 인사 행정 체계는 직원의 효율적인 관리를 위해 중요합니다. 2025년 1월 기준, 팁스R&D 일반트랙 창업기업 선정 결과에 따르면, 인력 규모가 30명 미만인 회사는 근로시간을 40시간으로 제한해야 합니다.
