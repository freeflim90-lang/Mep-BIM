# 설계 지침서 지식 베이스

## 초기 기준
설계 단계별 BIM 수행 기준, 도면 표준, 설계 문서 체계, 설계-시공 인수인계, 품질 검토 체크리스트, KDS/KCS 연계 기준을 통합 관리한다.


## 설계 BIM 수행 일반 기준 — 단계별 설계 (2026-05-26)
- Source: LUA BIM LABS curated baseline, 국토교통부 건축BIM 활성화 로드맵, KDS/KCS 기준 2026-05-26
- Tags: design,bim,phase,planning,schematic,dd,cd

설계 단계는 기획설계 → 계획설계 → 기본설계 → 실시설계 4단계로 구분하며, 각 단계별 BIM 수행 범위와 납품 LOD가 다르다.

기획설계 (Conceptual Design):
- 목적: 사업 타당성 검토, 규모 산정, 사업비 개산 견적.
- BIM 활용: 매스 모델(LOD 100) 작성, 용적률·건폐율·층수 시뮬레이션.
- 납품물: 매스 BIM 모델, 기획 설계도서(배치도, 층별 면적표).
- 검토 항목: 법정 용적률 초과 여부, 일조권 사선 제한, 도로 사선 제한, 용도지역 허용 범위.

계획설계 (Schematic Design):
- 목적: 기본 공간 배치 확정, 외관 디자인 결정, 인허가 사전 검토.
- BIM 활용: 건축 매스 + 주요 구조 그리드 모델(LOD 200), 공간 프로그램 BIM 분석.
- 납품물: 계획설계도서(배치도, 평면도, 입면도, 단면도), BIM 매스 모델.
- 검토 항목: 실 구성 및 면적 배분, 코어 위치, 주차 대수, 피난 동선 초기 검토.

기본설계 (Design Development, DD):
- 목적: 설계 내용 구체화, 공종별 기본 시스템 결정, 인허가 도서 완성.
- BIM 활용: 공종별 BIM 모델(LOD 300) 작성 — 건축, 구조, MEP 각각.
- 납품물: 기본설계도서(건축/구조/기계/전기/소방 도면), BIM 협업 모델, 개산 공사비.
- 검토 항목: 공종 간 주요 간섭 검토(Coordination), 구조 그리드 확정, 주요 MEP 경로 확정.

실시설계 (Construction Documents, CD):
- 목적: 시공을 위한 완성된 도면 및 시방서 작성.
- BIM 활용: LOD 350~400 수준의 상세 모델, 도면 직접 추출(Sheet 뷰), 물량 산출.
- 납품물: 실시설계도서(전 공종 상세 도면, 시방서, 물량내역서), BIM 모델(LOD 350 이상).
- 검토 항목: 전 공종 BIM 간섭 검토 완료, 시공 상세(Connection Detail) 반영, 사양 파라미터 입력 완료.

단계별 모델 인계 규칙:
- 상위 단계 BIM 모델을 하위 단계에서 반드시 계승하여 사용한다(재작성 금지).
- 단계 전환 시 BIM 모델 상태(LOD, 완성도, 간섭 해소 현황)를 인수인계 체크리스트로 기록한다.


## 도면 표준 — CAD/BIM 레이어 네이밍, 스케일, 시트 번호 (2026-05-26)
- Source: LUA BIM LABS internal drawing standard baseline, KS F 1602, 국토교통부 건설CALS 도면 작성 기준 2026-05-26
- Tags: design,drawing,layer,scale,sheet,naming

레이어 네이밍 원칙 (ISO 13567 / KS F 1602 기반):
- 형식: [공종코드]-[요소코드]-[상태코드]
  - 공종코드: A(건축), S(구조), M(기계), P(배관), E(전기), F(소방), C(토목)
  - 요소코드: WALL(벽체), BEAM(보), PIPE(배관), DUCT(덕트), TRAY(트레이), ANNO(주석), DIM(치수), GRID(그리드)
  - 상태코드: E(기존), N(신설), D(철거), P(예정)
- 예시: A-WALL-N (건축 신설 벽체), M-DUCT-E (기계 기존 덕트), S-BEAM-N (구조 신설 보)
- 공통 레이어: 0(기준선), DEFPOINTS(치수 보조선), VIEWPORT(뷰포트)는 수정하지 않는다.

BIM 뷰/시트 네이밍:
- 평면도: [공종]-[레벨]-PLAN (예: A-1F-PLAN, M-B1-PLAN)
- 단면도: [공종]-SECTION-[번호] (예: A-SECTION-01)
- 상세도: [공종]-DETAIL-[번호] (예: S-DETAIL-05)
- 협의도: COORD-[공종조합]-[구역]-[레벨] (예: COORD-MS-ZONE-A-3F)

도면 스케일 기준:
- 배치도: 1/300, 1/500, 1/600 (대지 규모에 따라 선택)
- 평면도·입면도·단면도: 1/100, 1/200
- 상세도·접합부 도면: 1/20, 1/30, 1/50
- 시공 상세도(Connection Detail): 1/5, 1/10, 1/20
- 매설 배관 종·횡단면도: 1/100(수평), 1/200(수직) — 축척 불일치 시 "NS(Non-Scale)" 표기

시트 번호 체계:
- [공종코드]-[도면종류코드]-[일련번호] (3자리)
  - 예: A-DR-001(건축 도면), S-DR-001(구조 도면), M-DR-001(기계 도면)
- 공종코드: A(건축), S(구조), M(기계), P(배관/위생), E(전기), FA(소방), C(토목), L(조경)
- 도면종류코드: DR(도면), DT(상세도), SC(시방서), CA(계산서)
- 표지/목차: 000, 001 고정 번호 사용


## 설계 문서 체계 — 도면목록, 설계설명서, 계산서 구성 (2026-05-26)
- Source: LUA BIM LABS internal design document standard, 건축법 시행규칙 별표 2 설계도서 기준 2026-05-26
- Tags: design,document,drawing-list,specification,calculation

설계도서 구성 체계 (건축법 시행규칙 제6조 기준):
1. 건축계획서: 대지위치, 지역·지구, 건물용도, 면적·높이 개요
2. 배치도: 대지 경계, 건물 배치, 외부 인입 계통
3. 평면도: 각 층 공간 배치, 면적 표기
4. 입면도: 4면 이상 (주 입면 + 측·후 입면)
5. 단면도: 주요 단면 2개소 이상
6. 구조도: 기초, 골조, 단면 상세
7. 설비도: 기계, 전기, 소방, 통신, 위생 각 계통도 및 평면도
8. 시방서: 각 공종별 재료 기준, 시공 기준
9. 계산서: 구조계산서, 열관류율 계산, 설비 용량 계산

도면목록(Drawing List) 관리 기준:
- 도면목록은 프로젝트 착수 시 작성하고, 설계 진행에 따라 버전 관리.
- 필수 항목: 도면번호, 도면명, 축척, 작성일, 개정이력(Rev.0 ~ Rev.N), 담당자.
- BIM 기반 프로젝트는 BIM 모델 파일명과 도면번호를 1:1 매핑하여 추적성 확보.

설계설명서 필수 항목:
- 프로젝트 개요 (대지면적, 연면적, 층수, 구조종별, 용도)
- 적용 법규 목록 (건축법, 소방법, 에너지절약설계기준 등)
- 재료 사양 (콘크리트 강도 fck, 철근 강도 fy, 외장재 사양)
- 설계 하중 기준 (고정하중, 활하중, 적설하중, 풍하중, 지진하중)
- BIM 수행 기준 (사용 소프트웨어, LOD, 파일 관리 방법)

계산서 구성 기준:
- 구조계산서: KDS 14 20 기준, 하중 조합(1.2D+1.6L, 1.0D+1.0E 등) 명시
- 기계 용량 계산서: 냉난방 부하 계산(ASHRAE/KS 기준), 환기량(0.5 AC/h 이상), 급수 압력
- 전기 용량 계산서: 수전 용량(VA/㎡ 기준), 수변전 용량, 비상발전 용량
- 소방 계산서: 스프링클러 방수량(80L/min × 10개 = 800L/min), 방수압력(0.1MPa 이상)
- 에너지 성능 계산: 에너지절약계획서(연면적 500㎡ 이상 건물 의무)


## 설계-시공 간 정보 인수인계 기준 (2026-05-26)
- Source: LUA BIM LABS internal design-to-construction handover baseline, 국토교통부 BIM 시행지침 2026-05-26
- Tags: design,construction,handover,bim,information-management

인수인계 대상 정보 유형:
1. BIM 모델 파일: Revit 원본(.rvt), 링크 모델 전체, 패밀리 파일(.rfa) 포함
2. 도면 파일: PDF, DWG(CAD 내보내기), 도면목록
3. 시방서: 공종별 시방서 PDF 및 편집 가능 원본
4. 계산서: 서명·날인된 PDF
5. BIM 실행계획서(BEP): 최종 승인본
6. 파라미터 정의서: 공유 파라미터 파일(.txt), 사용 목적 설명
7. 모델 좌표계 정보: 프로젝트 원점, 측량 좌표 기준점 데이터

인수인계 절차:
1단계 — 설계사 BIM 모델 완성도 자체 점검 (LOD 검증, 파라미터 입력 완성도)
2단계 — 발주처/CM 검토 (BIM 납품 기준 적합성 확인)
3단계 — 시공사 인수 검토 (모델 열람 확인, 좌표계 일치 확인, 패밀리 로드 여부 확인)
4단계 — 인수인계 확인서 서명 및 보관
5단계 — 시공 BIM 팀 모델 인수 후 이상 보고 (10영업일 이내)

인수인계 품질 기준:
- 링크 모델 경로: 상대 경로(Relative Path) 설정 필수 — 절대 경로 사용 금지.
- 누락 패밀리: 0건 (모델 열람 시 경고 없는 상태).
- 미완성 요소: LOD 기준 미달 요소는 목록화하여 별도 첨부.
- 좌표계: 프로젝트 기준점과 측량 기준점 좌표 일치 확인서 첨부.
- 파일 크기: 단일 Revit 파일 500MB 초과 시 모델 분리 여부 협의.


## 설계 품질 검토 체크리스트 — BIM 기반 (2026-05-26)
- Source: LUA BIM LABS internal BIM quality review checklist, Autodesk BIM 360 QA 기준 2026-05-26
- Tags: design,quality,checklist,bim,coordination,clash

BIM 설계 품질 검토는 모델 자체 검토(Model Check)와 공종 간 협의 검토(Coordination Check)로 구분한다.

모델 자체 검토 항목:
- [ ] LOD 기준 적합성: 단계별 LOD 기준표 대비 완성도 확인
- [ ] 파라미터 입력 완성도: 필수 공유 파라미터 값 미입력 요소 0건 목표
- [ ] 패밀리 사용 적합성: 표준 패밀리 라이브러리 외 개인 패밀리 사용 시 사전 승인
- [ ] 레벨/그리드 기준 일치: 전 공종 레벨 및 그리드 기준 동일 여부
- [ ] 좌표계 일치: 공유 좌표(Shared Coordinates) 설정 일치 여부
- [ ] 경고 메시지(Revit Warnings): 중복 요소, 룸 미연결 등 치명 경고 0건
- [ ] 링크 모델 상태: 모든 링크 모델 정상 로드 여부
- [ ] 파일명 준수: 표준 파일 명명규칙 준수 여부

공종 간 협의 검토(BIM 간섭 검토) 항목:
- [ ] 구조체 관통 검토: MEP vs 구조(보, 기둥, 슬래브) 간섭 — 허용/불허 판정
- [ ] 천장고 침해 검토: MEP 하단고 vs 법정 최소 천장고
- [ ] 피난 동선 침해: MEP 하단고 vs 피난 통로 유효 높이 2.1m
- [ ] 방화구획 관통 처리: 방화구획 경계 교차 MEP 요소 방화 처리 확인
- [ ] MEP 공종 간 충돌: 기계 vs 전기 vs 소방 vs 위생 상호 간섭
- [ ] 유지관리 공간 확보: 주요 설비 점검구, 분전반 전면 800mm 이상
- [ ] 배관/덕트 구배: 배수관 1/100 이상, 증기관 1/200 이상 구배 준수

검토 결과 분류 기준:
- Critical(치명): 법령 위반, 구조 안전 관련 → 즉시 수정, 다음 단계 진행 불가
- Major(중요): 시공 불가, 설비 성능 미달 → 해당 단계 완료 전 해결
- Minor(경미): 시공 조정 가능, 현장 협의 처리 가능 → 목록화 후 시공 단계 인계


## KDS/KCS 연계 기준 (2026-05-26)
- Source: LUA BIM LABS curated baseline, 국가건설기준센터(KCSC) KDS/KCS 최신본 기준 2026-05-26
- Tags: design,kds,kcs,code,standard,regulation

KDS(한국설계기준) 주요 적용 기준:
- KDS 14 20 (콘크리트 구조 설계): fck 24MPa 이상(일반 RC), 피복두께(보 40mm, 슬래브 20mm, 기둥 40mm)
- KDS 14 30 (강구조 설계): 사용 강재 SS275~SS400, 내화 피복 두께 기준
- KDS 41 10 (건축 구조 하중): 기본 풍속, 지진구역계수(S), 지반분류(SA~SF)
  - 서울 기준 지진구역 1, 지역계수 S=0.11g
  - 풍하중 기준풍속: 지역별 30~45 m/s
- KDS 31 00 (기계설비 설계): 냉난방 설계 조건, 환기량 기준
- KDS 55 10 (전기설비 설계): 수전 전압, 간선 설계 기준

KCS(한국건설기준) 주요 적용 기준:
- KCS 14 20 (콘크리트 공사): 배합 강도, 양생 기간(표준 28일), 슬럼프 기준(120±25mm)
- KCS 14 31 (철골 공사): 용접 규격, 고력볼트 조임 기준(F10T, F13T)
- KCS 31 20 (공조설비 공사): 덕트 기밀 시험, 풍량 조정 기준
- KCS 55 10 (전기설비 공사): 절연저항 측정 기준(1MΩ 이상), 접지 저항

BIM 모델과 KDS/KCS 연계 방법:
- 구조 부재 파라미터에 KDS 기준 설계 강도(fck, fy) 입력 필수.
- 방화 관련 요소(방화문, 방화댐퍼)에 내화 성능 등급(1시간, 2시간) 파라미터 입력.

## 설계 단계 LOD 전환 마스터급 경험 지식 (2026-05-28)

- Tags: LOD전환, 설계충돌, 발주처변경요청, 납기압박, LOD재협상

기본설계(LOD 200~300)에서 실시설계(LOD 300~350)로 LOD를 상향할 때 실무에서 가장 자주 충돌하는 시나리오와 대응법을 정리한다.

**상황 1 — 건축 도면 수정 지시가 BIM 모델보다 선행될 때**: 건축사가 CAD 도면에서 벽체를 이동하거나 창호 크기를 변경한 지시가 구두 또는 이메일로 먼저 내려오고, BIM 모델 업데이트는 며칠 뒤에 이루어지는 사이 MEP 팀이 구버전 BIM 기준으로 간섭 검토를 완료해버리는 상황이 발생한다. 대응법: BIM 모델을 단일 진실 출처(Single Source of Truth)로 설정하고, 도면 수정은 반드시 Revit Sheet 뷰에서 직접 수행하도록 BEP에 명시. 건축 변경 발생 시 BIM 코디네이터가 변경 범위를 ACC Issue로 등록하고 영향받는 공종 담당자에게 알림. 구두 지시는 효력 없음 원칙을 계약에 포함.

**상황 2 — 발주처 중도 요구사항 변경 시 LOD 재협상 전략**: 실시설계 70% 진행 시점에 발주처가 "기계실 장비를 실제 제조사 모델로 바꿔달라(LOD 350→LOD 400)"고 요구하는 케이스가 잦다. LOD 상향은 모델링 공수 증가뿐 아니라 파라미터 재입력, 패밀리 교체, IFC 재익스포트, 간섭 검토 재수행이 수반된다. 협상 전략: ① 추가 요구 LOD 범위를 명확히 문서화한다. ② 공수 산정 후 변경 계약 또는 계약 외 작업 확인서를 발행한다. ③ 납기 연장이 불가피할 경우 단계적 납품(기존 LOD 300 납품 후 LOD 400 보완 납품)을 제안한다. ④ LOD 400 이상 작업은 시공 단계로 구분하여 시공사 BIM 팀 범위로 전가 협의.

**상황 3 — 납기 압박 시 LOD 타협 한계선**: 납기 직전 LOD를 낮추는 타협은 FM 연동, 발주처 검수, 후속 시공 BIM 인수인계 품질을 떨어뜨린다. LUA BIM LABS 기준 LOD 타협 한계선: 파라미터 입력 완결성(LOI)은 납기 조건에 무관하게 95% 이상 유지. 기하학적 정밀도는 주요 구조 및 MEP 주 계통 기준 LOD 300, 소부재·디테일은 LOD 200 허용 가능. 타협 범위는 BIM 코디네이터와 발주처가 서면으로 합의하고 BEP 부록에 기록. "LOD 낮춰도 괜찮다"는 구두 승인은 추후 납품 거절 사유가 될 수 있으므로 반드시 이메일 또는 ACC Issue로 기록 남길 것.
