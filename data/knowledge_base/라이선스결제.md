## BIM Command Center 구독 가격 정책 (2026-05-19 09:37:53)
- Source: `docs/autodesk_store/SUBSCRIPTION_PRICING.md`, Autodesk App Store submission docs checked 2026-05-19
- Tags: subscription,pricing,entitlement

첫 Autodesk Store 판매 가격은 `BIM Command Center for Revit` 기준 USD 19/month 또는 USD 190/year를 권장한다. 상품은 Individual, Team 5-Pack, Enterprise 세 가지로 구성한다. 5인 팀팩은 USD 79/month 또는 USD 790/year를 내부 기준으로 둔다. 30일 무료 체험을 제공하려면 Autodesk App Store Entitlement API 연동이 필요하며, 구독 상태 확인 전에는 유료 명령을 잠금 처리한다. 첫 상용 패키지에서 `RevitAddin.MepBim` 관련 명령, 설치 파일, 구독 권한은 제외한다.

유료/구독 제품은 가격, 결제 제공자, 라이선스 상태별 UX, 오프라인/네트워크 실패 시 동작을 명확히 해야 한다. Autodesk App Store 자료는 유료 앱 결제에 PayPal 및 BlueSnap 사용을 언급한다. Entitlement/API 연계가 필요한 경우 별도 검증한다.


## Autodesk Entitlement API 연동 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: license,entitlement,api

Autodesk App Store 유료 앱은 Entitlement API로 구독 상태 검증.
엔드포인트: GET https://apps.autodesk.com/webservices/checkentitlement
파라미터: userid(Autodesk ID), appid(Store App ID)
응답: isValid(true/false), message
Add-in 시작 시 검증 → 실패 시 유료 기능 잠금 + 구매 페이지 링크 표시.
오프라인 시 마지막 검증 결과를 암호화해 로컬 캐시(최대 7일)로 허용.


## 라이선스 상태별 UX 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: license,ux,freemium

상태별 동작:
- 유효 구독: 전체 기능 활성화
- 체험판(Trial): 기능 제한 또는 실행 횟수 제한 + 업그레이드 안내
- 만료: 유료 기능 비활성화 + 갱신 링크
- 오프라인 캐시 유효: 전체 기능 허용 (캐시 기간 내)
- 오프라인 캐시 만료: 읽기 전용 모드 + 온라인 연결 안내
구독 갱신 버튼은 AppStore 제품 페이지 URL로 직접 연결한다.
