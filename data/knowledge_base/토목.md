**AI 즉시 답변 패턴 — "토목 분야 BIM(CIM)과 건축 BIM 차이가 뭔가요?"**
```
CIM(Civil Information Modeling) vs 건축 BIM 차이:
- 공간 규모: CIM은 선형(도로·철도), 광역(유역·지구) 단위
- 소프트웨어: Civil 3D(도로), OpenRoads(교량), 건축은 Revit 중심
- IFC 버전: 토목은 IFC 4.3(IfcAlignment·IfcBridge) 필요
- 납품 기준: 건축 BIM → 건물 3D, 토목 BIM → 종·횡단면 + 3D 통합
- 한국 의무화: 1,000억↑ SOC 공사 2024년부터 BIM 의무 → 2026년 500억↑
```

**국내 토목 BIM 지침 체계 (2025~2026):**
| 발주처 | 지침명 | 주요 내용 |
|--------|--------|---------|

| 한국도로공사 | 고속도로 BIM 적용지침(2023) | WBS·CBS·OBS 객체분류, 납품관리시스템 |
| 국가철도공단 | 국가철도 BIM 적용지침(2023) | 교량·터널·역사 LOD 300~350 |
| LH공사 | 공공주택 BIM 설계지침(2024) | 단지 토목·조경 BIM 납품 |

**토목 BIM 핵심 공종별 요건:**
| 공종 | 소프트웨어 | IFC 버전 | LOD |
|------|---------|---------|-----|
| 도로 | Civil 3D / OpenRoads | IFC 4.3 (IfcAlignment) | LOD 300 |
| 교량 | Revit / Tekla / Midas | IFC 4.3 (IfcBridge) | LOD 350 |
| 터널 | Civil 3D / Revit | IFC 4 또는 4.3 | LOD 300 |
| 토공 | Civil 3D | LandXML/IFC | LOD 200~300 |
| 배수구조물 | Revit | IFC 4 | LOD 300 |

**건설기준 디지털화 사업 (국토부 2022~2026):**
- 교량·건축·도로·철도·터널 기준을 BIM 환경에서 즉시 활용 가능한 데이터로 변환
- AI가 건설기준을 읽고 해석하는 '디지털 건설기준' 시스템 구축 (2025년 12월 성과발표)
- BIM 납품 성과품과 설계기준이 연동되어 자동 적합성 검토 가능

**토목-건축 인터페이스 BIM 관리 포인트:**
- GL(Ground Level)↔FL(Floor Level): 토목 GL 확정 전 건축 기초 레벨 미결정
- 외부 배관 인입구: 오수·우수·급수 관로 위치를 토목 BIM과 건축 MEP BIM 연동
- 대지 경계선: 건물 외곽선과 도로 경계 이격 토목도면 기준 확인
- 부지 정지: 절토·성토 범위 확정 후 건축 터파기 레벨 결정

## 외부 인입 및 대지 경계 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: civil,site,invert-level

외부 인입 하수관 최소 구배: 오수 1/100, 우수 1/200 (관경 300mm 기준).
인입 인버트(invert) 레벨은 건물 집수정 바닥보다 최소 150mm 낮아야 역류 방지.
대지 경계에서 건물 외벽까지 이격(건폐율 제외 여유): 인입관 굴착 폭 600mm + 작업 여유 300mm 확보.
Add-in은 건물 내부 배관 레벨 변경 시 외부 인입 레벨과의 고도차를 자동 재계산하여 경고한다.


## 우수·오수·상수 관로 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: civil,drainage,utility

분류식 하수도: 오수관과 우수관 분리 필수. 합류식은 지자체 승인 필요.
상수 인입: 계량기 기준 아파트 DN50~100, 업무시설 DN100~150.
매설 깊이: 동결선 이하(중부 1.0m, 남부 0.6m), 차량 하중 구간 최소 1.2m.
Add-in은 건물 내부 위생 배관 GL 이하 구간을 자동 감지하고 외부 인입 레벨 충돌 가능성을 보고한다.


## GL/FL 관계 및 레벨 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: civil,level,gl-fl

GL(Ground Level)과 1F FL 차이: 일반 건물 300~600mm, 침수 위험 지역 1,000mm 이상.
지하층 바닥: 지하수위 + 방수층 두께 + 마감을 고려한 레벨 설정.
Add-in 레벨 검토 시 Revit 프로젝트 기준점(Project Base Point)과 측량 기준점(Survey Point) 일치 여부 먼저 확인.
외부 인입 관로 구배 계산은 GL 기준, 내부 설비는 FL 기준으로 레벨을 이중 표기한다.


## 외부 MEP 인입 계통별 기준 (2026-05-23)
- Source: LUA BIM LABS internal civil-MEP interface baseline
- Tags: civil,utility,invert-level,drainage,water,gas,power
- Links: [[위생]], [[설비시공조율]], [[건축]]

건물로 들어오는 외부 인입 계통별 핵심 확인 항목:

상수도(급수) 인입:
- 계량기(수도계량기) 위치: 대지 경계 인근 맨홀 또는 계량기함.
- 배관 재질: 외부 매설은 PE관(폴리에틸렌관). 동결선 이하 매설(중부 1.0m, 남부 0.6m).
- 건물 진입 후 저수조 또는 직결 가압.
- 급수관 인입 크기: 일반 업무시설 DN100~150, 주거 DN50~100.

하수(오수) 인입:
- 건물 집수정 오수 → 외부 오수관 → 공공하수관 접속.
- 외부 오수관 구배: 1/100 이상 (관경 DN300 기준).
- 인버트(관 내저면) 레벨: 외부 공공하수관 인버트보다 건물 집수정 바닥이 높아야 자연유하 가능.
- 높이 확보 불가 시: 오수 펌프로 강제 압송 후 접속.

우수 인입:
- 옥상·주차장 우수 → 우수받이 → 우수관 → 집수정 → 외부 우수관.
- 오수와 법적 분리 의무 (분류식 하수도 기준).
- 외부 우수관 구배: 1/200 이상.
- 집중 호우 대비: 100년 빈도 강우량 기준 설계.

전력 인입:
- 한전 변압기 → 지중 케이블 → 건물 수변전실 또는 수전함.
- 지중 케이블 매설 깊이: 차량 통행 구간 1.2m 이상.
- 케이블 트렌치 또는 PVC 전선관(28mm 이상)으로 보호.

가스 인입:
- 도시가스 배관은 지역 도시가스사 기준 적용.
- 매설 깊이: 0.6m 이상 (일반 구간).
- 건물 진입 후 가스계량기 → 압력 조정기 → 각 층 배관.
- LNG(도시가스)는 공기보다 가벼워 위로 확산. LPG는 아래로 가라앉음 → 용도와 환기 계획 다름.

통신 인입:
- 통신사 관로 → 맨홀 → 건물 통신실 MDF.
- 건물 외벽 관통 후 관로 방수처리 필수.
- 여유 관로 확보: 장래 통신사 추가, 광케이블 증설 대비.

## 토목 시방서 핵심 기준 (2026-05-26)
- Source: KCS 한국건설기준, 국내 표준 시방서 2026-05-26
- Tags: civil,specification,construction-spec

### 토공사 기준
- 다짐도: 수정 프록터 최대 건조밀도(KS F 2312 D법) 대비 95% 이상. 구조물 뒤채움은 90% 이상.
- 함수비 관리: 최적 함수비(OMC) ±2% 범위 내 다짐. 과다 함수비 시 지반 교란 방지를 위해 건조 후 시공.
- 다짐 층 두께: 불도저·컴팩터 사용 시 1층 300mm 이하(완성 후 200mm). 소형 다짐기 150mm 이하.
- 다짐 횟수: 장비·토질에 따라 시험 시공으로 결정(일반 점성토 6~8회, 사질토 4~6회).
- 절토 비탈면: 토질 기준 경사 — 경암 1:0.3~0.5, 연암 1:0.5~0.8, 토사 1:1.0~1.5(높이에 따라 단계식 설계).
- 성토 비탈면: 사질토 1:1.5 이상, 점성토 1:1.8 이상 기준.

### 콘크리트 구조물 기준 (PC BOX, 암거, 옹벽)
- PC BOX 암거: 설계 하중 — 도로설계기준 DB-24(24톤 표준트럭) 기준. 제작 공장 KS 인증 제품 사용.
- 암거 기초 처리: 지지력 확보를 위해 치환 또는 개량. 말뚝 기초 시 지지력 확인 시험 실시.
- 옹벽 콘크리트: fck 24MPa 이상(중력식·반중력식), 배근 옹벽 21MPa 이상. 전면 마감 줄눈 10mm 이하.
- 신축줄눈: RC 구조물 10~15m 간격, PC BOX 6m 간격. 지수판(waterstop) 설치.
- 뒷채움: 옹벽 뒤채움 투수성 양질 재료(투수계수 1×10⁻³cm/s 이상). 구조물에서 300mm 이상은 소형 다짐기 사용.

### 도로 포장 기준
- 아스콘 혼합물 온도: 공장 출하 시 150~175℃, 현장 도착 시 140℃ 이상, 초기 전압 시 110℃ 이상.
- 다짐 횟수: 1차 다짐(Breakdown) — 머캐덤 롤러 2~3회 왕복. 2차(중간) — 타이어 롤러 4~6회. 마무리 — 탠덤 롤러 2~3회.
- 평탄성: 3m 프로파일미터 기준 표준편차 σ ≤ 2.4mm(일반도로), σ ≤ 1.8mm(고속도로).
- 다짐도: 코어 채취 밀도가 이론 최대밀도(기준밀도)의 96% 이상.
- 재료 분리 방지: 아스콘 포설 시 피니셔 스크리드 진동 조정, 이음부(Joint) 수직 절단 후 도포.
- 콘크리트 포장(CRCP): 두께 250~300mm, fck 27MPa 이상, 철근 배근 간격 150~200mm.

### 상하수도 배관 기준
- 토피 깊이: 일반 구간 — 오수관·우수관 최소 0.6m(남부), 1.0m(중부/북부). 차량 통행 구간 1.2m 이상.
- 기초재 다짐: 배관 하부 기초재(모래 또는 쇄석) 두께 100mm 이상 포설 후 다짐. 배관 측면도 기초재 충전(Haunching).
- 관 이음부: 고무링 이음부 각도 이탈 허용 범위 — 관경 400mm 이하 3°, 500mm 이상 2° 이내.
- 배관 시험: 수압 시험(Hydrostatic) — 오수관 0.03MPa 30분 유지 누수 없음. 공기압 시험 대체 가능.
- 관 재질: 상수도 — 주철관(KS D 4308), 강관(KS D 3565), 덕타일주철관. 하수도 — 흄관(KS F 4401), 파형강관, PVC관.
- 접속 맨홀 간격: 직선 구간 최대 150m(관경 DN500 이하). 방향·경사 변경점, 지류 합류점에 맨홀 설치.

### 지반 개량 기준
- 그라우팅: 주입 압력 — 얕은 지반(5m 이내) 0.2~0.5MPa, 깊은 지반 최대 1.0MPa 이하. 시멘트 주입비(I.G.R) 기준 충진율 80% 이상.
- SCP(Sand Compaction Pile): 개량 모래 말뚝 직경 700mm 표준. 타설 간격은 지반 조건별 계산. 개량 후 CPT/SPT로 효과 확인.
- DCM(Deep Cement Mixing): 시멘트 혼합비 80~200kg/m³. 개량체 압축강도 — 연약 점토 0.3~1.0MPa 이상. 품질관리 코어 채취 1/100 개소 이상.
- 지반 개량 품질 확인: 재하 시험(PBT), 표준관입시험(SPT), 현장 전단 시험. 개량 후 7~28일 강도 확인.
- 개량 범위: 구조물 외측 1.0~1.5m 이상 확대 개량. 액상화 방지 목적은 구조물 폭 × 0.5 이상.

### 토목 BIM 모델 요구사항 (지형, 지하매설물, 토공 LOD)
- 지형 LOD: LOD 200 — 일반적 지형 윤곽(TIN 메쉬). LOD 300 — 절토·성토 영역 구분, 측점 데이터 반영. LOD 350 — 옹벽·암거·포장 구조물 상세 위치.
- 지하매설물: 상수도·하수도·가스·전력·통신 매설물을 각 계통별 레이어 분리. 심도(invert 레벨), 관경, 재질 파라미터 기록.

**토목 BIM 조율 실패 패턴 5가지:**
1. **건축 BIM과 좌표계 불일치**: 건축팀의 Revit 공유 좌표(Survey Point)와 토목팀 Civil 3D 좌표가 다를 때 외부 인입 배관이 건물 외벽을 관통하지 못하는 위치로 오배치. 프로젝트 착수 전 측량 성과물(SHP 또는 DWG) 기반 공통 기준점(BCP) 합의 및 문서화 필수
2. **오수 인입 레벨 오산**: 대지 경계 맨홀 인버트 레벨(IL)을 확인하지 않고 오수관을 설계하면 역구배 발생. 관할 지자체 하수도 대장에서 공공 하수관거 IL 확인 후 BIM 모델에 반영
3. **굴착 시 지하매설물 충돌**: 기존 지하 통신·전기·가스 매설물 위치가 토목 BIM에 미반영된 상태에서 굴착 → 파손 사고. 착공 전 지하안전관리에관한특별법에 따른 지하안전조사 및 탐사 결과를 BIM에 레이어로 등록
4. **흙막이 버팀대 MEP 간섭**: H-Pile + 띠장 + 버팀대 구조가 지하 레벨 MEP 인입 배관과 충돌. 흙막이 설계 초기에 MEP 인입 경로를 BIM에서 확인하고 버팀대 설치 위치 협의
5. **성토 지반 침하로 옥외 배관 꺾임**: 성토 구간 위에 설치된 옥외 배수·위생 배관이 부등침하로 이음부 파손. 성토 구간 배관은 가요성 이음(플렉시블 커플링) 또는 강성 보강 설계 필요

**Civil 3D → Revit IFC 연계 워크플로:**
- Civil 3D Corridor → IFC 4.3 Export: IfcAlignment 기반 도로 선형 → Revit로 링크 후 건물 BIM과 병합
- 지형 데이터: Civil 3D Surface → `Toposurface`(Revit) 또는 IFC IfcGeographicElement로 변환
- 공통 BIM 환경: ACC(Autodesk Construction Cloud) 또는 BIM 360에서 토목+건축+MEP 모델 통합 관리, 클래시 검토
