## 보·기둥 관통 및 개구부 보강 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,penetration,opening

일반 RC보: 높이의 1/3 이하, 보 단부에서 보 높이 2배 이상 이격한 위치에만 관통 허용.
관통 직경은 보 높이의 1/4 이하, 연속 관통 간격은 관통경의 3배 이상.
슬래브 관통: 단변 길이의 1/6, 장변의 1/10 이하면 보강 없이 가능(설계 검토 필요).
Add-in은 관통 요청을 위치·크기·이격 기준으로 자동 분류하고 구조 검토 필요/불필요를 초기 판단값으로 표시한다.


## 전이보·전이 슬래브 처리 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,transfer-beam,slab

전이보는 상부 하중을 받는 핵심 부재로 절대 관통 금지. Add-in 태그로 TRANSFER_BEAM 분류 필수.
전이 슬래브 두께는 일반 슬래브 대비 1.5~2배 이상이므로 레벨 계산 시 실제 두께 데이터 활용.
Add-in 보고서에는 '전이 부재 여부' 필드를 별도 컬럼으로 표기하고 담당 구조기술사 확인 요청 플래그를 붙인다.


## 구조 모델 파라미터 및 데이터 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,parameter,bim-data

공유 파라미터 필수: 부재유형(보/기둥/슬래브/벽), 설계강도(fck), 관통허용여부, 보강필요여부.
Revit 구조 카테고리: Structural Framing(보), Structural Columns(기둥), Floors(슬래브).
Add-in이 충돌 감지 시 BuiltInCategory로 구조 부재를 필터링하고 관통 위치의 BoundingBox 좌표를 기록한다.
개구부 보강 정보(보강근 규격, 길이)는 모델 파라미터로 직접 저장하거나 연결 문서로 링크한다.


## 구조 관통 요청 처리 절차 (2026-05-23)
- Source: LUA BIM LABS internal structural coordination baseline
- Tags: structure,penetration,opening,bim,rfi,coordination
- Links: [[설비시공조율]], [[건축]], [[설비도면해석]]

MEP에서 구조체 관통이 필요할 때 처리 순서:

1단계 — 필요성 확인:
- 관통이 꼭 필요한지 우회 경로 먼저 검토한다. 보를 관통하는 것보다 보 아래로 우회하는 것이 구조적으로 안전하다.
- 관통 크기, 위치(보에서의 거리), 이격 거리, 관통 개수를 먼저 결정한다.

2단계 — 관통 가능 여부 초기 판단 (BIM에서 자동 확인 가능):
- RC보 관통 허용 범위: 관통 위치가 보 단부에서 보 높이 2배 이상 이격, 관통 직경이 보 높이 1/3 이하, 1/4 이하가 안전.
- 슬래브 관통: 단변 1/6, 장변 1/10 이하면 보강 없이 가능 (설계자 확인 필수).
- 전이보·내력벽: 원칙적으로 관통 금지. 무조건 구조기술사 확인.

3단계 — 구조 설계자 협의 (설계 질의, RFI):
- 관통 위치 좌표, 크기, 목적 공종, 관통 방향을 RFI로 제출한다.
- 허용 가능하면 보강 방법(보강근 규격, 배근 방법)을 받는다.
- 불가 판정이면 대체 경로를 협의한다.

4단계 — BIM 모델 반영:
- 승인된 관통 위치에 Opening 요소 또는 Sleeve를 삽입한다.
- 보강 정보를 구조 모델 파라미터에 기록하거나 연결 문서로 남긴다.
- Navisworks 간섭 검토에서 구조 관통 승인 여부를 이슈 상태(Resolved/Pending)로 관리한다.

자주 발생하는 구조 관통 이슈:
- 보 단부(기둥 접합부 근처) 관통: 구조적으로 가장 위험한 위치 → 기본적으로 불가.
- 대구경 배관이 보를 직각이 아닌 사선으로 관통: 개구부가 타원형이 되어 보강이 복잡해짐.
- 여러 공종 배관이 같은 위치 요청: 관통 간격 기준(관통경의 3배 이상)을 만족하도록 조정.
- 이미 시공된 구조체에 관통 추가: 코어 드릴링 전 구조기술사 현장 확인 필수.


## 구조 FAQ (2026-05-23)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: structure,faq,bim,penetration,beam

보를 관통할 수 있는지 물을 때:
- 관통 위치가 보 중앙부이고 크기가 보 높이 1/4 이하면 일반적으로 가능성이 있지만 반드시 구조 설계자 확인이 필요하다.
- 보 단부(기둥 근처)는 원칙적으로 불가하고, 전이보는 무조건 금지다.
- BIM에서는 먼저 관통 가능 범위를 초기 판단하고 경계 케이스는 구조 질의(RFI)를 낸다.

슬래브를 관통해야 할 때 어떻게 하는지 물을 때:
- 슬래브 관통은 보 관통보다 상대적으로 자유도가 높지만 슬리브 크기와 위치에 따라 보강이 필요하다.
- 기둥·보와 가까운 슬래브 구간은 슬래브 응력이 집중하므로 주의.
- 슬리브는 시공 전 구조도면의 배근 상태와 비교해 주근을 끊지 않도록 위치를 정한다.

구조 BIM 모델에서 보 크기가 안 맞을 때 어떻게 하는지 물을 때:
- 구조 링크 모델의 최신 버전 여부를 확인한다.
- 구조 모델 개정 이력과 구조 도면의 단면 크기를 비교한다.
- 차이가 있으면 구조팀에 확인하고 MEP 관통 계획을 재검토한다.


## 구조 협의 보고서 필수 항목 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,report,coordination

보고서 필수 컬럼: 부재ID, 부재유형, 레벨, 관통위치(X/Y/Z), 관통직경/크기, 요청 공종, 이격거리, 판정(허용/검토필요/불가).
판정 기준은 구조 기준값 자동 적용 후 경계 케이스(허용값 ±10%)는 '구조 검토 필요'로 분류.
Add-in 출력 형식: Revit 일람표 + Excel 보고서 + 모델 상 마커(색상 코드: 녹=허용, 황=검토, 적=불가).
