## 2. 기준 공수 산출 (MM 기반)

> MM(Man-Month) = 1인이 1개월(22일, 176시간) 투입하는 공수

### 2.1 단계별 MM 계수 (연면적 1,000㎡ · 공종 1개 기준)

| BIM 단계 | LOD | MM 계수 | 비고 |
|---|---|---|---|
| 기획 BIM | 100~200 | 0.05~0.10 | 공간 구성·볼륨 위주 |
| 기본설계 BIM | 200~300 | 0.10~0.20 | 공종별 기본 모델 |
| 실시설계 BIM | 300~350 | 0.25~0.45 | 상세 모델 + 파라미터 |
| 시공 BIM | 350 | 0.15~0.30 | 시공 모델 갱신 |
| 준공 BIM | 400 | 0.08~0.15 | 준공 반영 + FM 속성 |

**계산식:**
```
기준 공수(MM) = 연면적(㎡) ÷ 1,000 × 공종 수 × 단계별 계수
```

**예시 — 연면적 20,000㎡, 건축+구조+MEP(3공종), 실시설계:**
```
20,000 ÷ 1,000 × 3공종 × 0.35(중간값) = 21 MM
```

### 견적 자동화 로직 (Python 기반)

```python
# BIM 프로젝트 견적 자동 산정
UNIT_RATE = {  # 일/100㎡ by (공종, 등급)
    ("건축", 1): 0.8, ("건축", 2): 1.2, ("건축", 3): 2.0,
    ("기계", 1): 1.0, ("기계", 2): 1.8, ("기계", 3): 3.0,
    ("전기", 1): 0.6, ("전기", 2): 1.2, ("전기", 3): 2.0,
}
DAILY_RATE = 500_000  # 원/일 (모델러 기준)
COMPLEXITY_FACTOR = {"일반": 1.0, "복잡": 1.3, "초복잡": 1.6}

def estimate(area_m2: float, disciplines: list[tuple], complexity: str) -> dict:
    factor = COMPLEXITY_FACTOR[complexity]
    total_days = sum(
        area_m2 / 100 * UNIT_RATE[(disc, grade)] * factor
        for disc, grade in disciplines
        if (disc, grade) in UNIT_RATE
    )
    buffer_days = total_days * 0.18  # 변경 대응 버퍼
    cost = (total_days + buffer_days) * DAILY_RATE
    return {"공수(일)": round(total_days, 1), "버퍼(일)": round(buffer_days, 1), "견적(원)": round(cost, -4)}
```
