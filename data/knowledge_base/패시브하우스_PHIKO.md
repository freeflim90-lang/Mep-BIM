## 패시브하우스 정의 및 핵심 성능 기준

### 국제 패시브하우스 인증 기준 (Passivhaus Institut, Darmstadt)

| 항목 | 기준값 | 비고 |
|------|--------|------|
| 난방에너지요구량 | ≤ 15 kWh/㎡·a | 연간 단위면적당 |
| 냉방에너지요구량 | ≤ 15 kWh/㎡·a | 또는 피크냉방부하 ≤ 10 W/㎡ |
| 1차에너지소요량 | ≤ 120 kWh/㎡·a | 전체 에너지 합산 |
| 최대 난방부하 | ≤ 10 W/㎡ | 피크 부하 |
| 기밀성능 (n50) | ≤ 0.6 /h | 50Pa 차압 조건 블로어도어 테스트 |

### 패시브플러스(Passive House Plus) 기준
- 1차에너지 생산량 ≥ 60 kWh/㎡·a (재생에너지 포함 시 패시브 기준 달성 가능)

### 패시브 프리미엄(Passive House Premium) 기준
- 1차에너지 생산량 ≥ 120 kWh/㎡·a

## 외피 열성능 상세 기준

### 부위별 열관류율 (U-value) 목표값

| 부위 | 목표 U-value | 한국 실정 적용값 |
|------|-------------|-----------------|
| 외벽 | ≤ 0.15 W/㎡K | 0.10~0.13 W/㎡K 권장 |
| 지붕/최상층 바닥 | ≤ 0.15 W/㎡K | 0.10~0.12 W/㎡K |
| 바닥/최하층 | ≤ 0.15 W/㎡K | 0.10~0.15 W/㎡K |
| 창호 (Uw) | ≤ 0.80 W/㎡K | 0.7~0.8 W/㎡K |
| 현관문 | ≤ 0.80 W/㎡K | - |

### 창호 태양열 취득률 (g-value)
- 남향: g ≥ 0.5 (태양열 취득 최대화)
- 동/서향: g ≥ 0.4
- 북향: 개구부 최소화, g값 무관하게 U-value 우선

### 열교 (Thermal Bridge) 기준
- **임계 표면온도**: ≥ 12.6℃ (결로·곰팡이 방지 기준; fRsi ≥ 0.70)
- 열교 부위 선형열관류율 (Ψ): ≤ 0.01 W/mK 권장
- 주요 열교 부위: 발코니 연결부, 창호 프레임 주변, 기초-벽체 접합부

## 기밀성능 (Air Tightness)

| 항목 | 기준 | 비고 |
|------|------|------|
| n50 (기밀도) | ≤ 0.6 /h | 패시브하우스 기준 |
| n50 (일반 건물) | ≤ 3.0 /h | 독일 에너지절약법 기준 |
| 한국 제로에너지 | ≤ 5.0 /h | 현행 한국 인증 기준 |

**블로어도어 테스트(Blower Door Test) 절차:**
1. 건물 50Pa 가압/감압 조건 조성
2. 누기량(ACH) 측정
3. 열화상 카메라 병행 → 누기 부위 시각화

## 환기시스템 (HRV/ERV) 성능 기준

| 항목 | 기준값 |
|------|--------|
| 열회수 효율 | ≥ 75% (현열 기준) |
| 전기 효율 | ≤ 0.45 Wh/㎥ |
| 소비전력 | 50~100 W (주택 기준) |
| 필터 등급 | F7 이상 권장 (ISO 16890 기준) |
| 환기횟수 | 0.3~0.5 ACH (필요 외기량 기준) |

**HRV vs ERV 선택 기준:**
- 한국 기후: ERV(전열교환) 권장 — 여름 제습, 겨울 수분 보존 효과
- 한냉지(중부1 지역): HRV로도 충분, ERV는 결로 위험 존재

|---------|---------|------|
| 전기 | 2.75 | 발전 손실 포함 |


## 한국 법규 단열 기준 (건축물 에너지절약설계기준)

### 기후 구분 (3개 지역)
- **중부1**: 강원 산간, 경기북부 일부 (가장 엄격)
- **중부2**: 서울, 경기 대부분, 강원 해안, 충청 (중간)  
- **남부**: 대구, 부산, 광주 등 남부 (완화)
- **제주**: 제주도 (가장 완화)

### 지역별 부위별 열관류율 기준 [W/㎡K] (공동주택 기준)

| 부위 | 중부1 | 중부2 | 남부 | 제주 |
|------|-------|-------|------|------|
| 거실 외벽 (외기 직접) | 0.210 | 0.260 | 0.320 | 0.430 |
| 거실 외벽 (외기 간접) | 0.300 | 0.370 | 0.450 | 0.600 |
| 지붕 (외기 직접) | 0.150 | 0.180 | 0.220 | 0.290 |
| 최하층 바닥 (외기 직접) | 0.170 | 0.200 | 0.250 | 0.330 |
| 창호 | 1.200 | 1.400 | 1.600 | 2.000 |
| 현관문 | 1.400 | 1.600 | 1.800 | 2.200 |

> 패시브하우스 달성 시 위 법규 기준 대비 2~5배 엄격한 성능 요구됨

## 표면열전달저항 (Surface Thermal Resistance)

| 부위 | 방향 | Rsi 실내 [m²K/W] | Rse 실외 [m²K/W] |
|------|------|------------------|------------------|
| 거실 외벽 | 수평 | 0.11 | 0.043 |
| 최상층 지붕 | 상향 | 0.10 | 0.043 |
| 최하층 바닥 | 하향 | 0.15 | 0.043 |
| 내벽/내부바닥 | 수평 | 0.11 | 0.11 |

**열관류율 계산:**
```
U = 1 / (Rsi + R1 + R2 + ... + Rn + Rse)
R_layer = 두께(m) / 열전도율(W/mK)
```

## 주요 단열재 열전도율 (λ값)

| 단열재 종류 | 열전도율 [W/mK] | 특징 |
|------------|----------------|------|
| 비드법 단열재 1호 | 0.036 | EPS, 가격 저렴 |
| 비드법 단열재 2호 | 0.032 | 고성능 EPS |
| 압출법 단열재 (XPS) | 0.028~0.034 | 수분 저항 우수 |
| 경질우레탄폼 | 0.023~0.024 | 최고 성능, 가격 높음 |
| 미네랄울 (암면) | 0.036~0.040 | 불연, 흡음 |
| 유리면 | 0.036~0.044 | 불연, 저렴 |
| 셀룰로오스 | 0.040 | 친환경, 흡습성 주의 |
| 에어로젤 | 0.015~0.018 | 초고성능, 매우 고가 |
| 진공단열패널 (VIP) | 0.007~0.010 | 최고 단열, 내구성 주의 |

**동일 열저항 달성 시 두께 비교 (기준: EPS 1호 100mm):**
- EPS 1호 (λ=0.036): 100mm
- 경질우레탄 (λ=0.024): 67mm (33% 절약)
- VIP (λ=0.008): 22mm (78% 절약)

## 패시브하우스 설계 5대 원칙 (PHIKO 기준)

1. **고단열 (Super-insulation)**: 외피 전체 U ≤ 0.15 W/㎡K
2. **고성능 창호**: 3중 유리 + 슈퍼스페이서, Uf ≤ 0.80 W/㎡K
3. **열교 없는 시공**: 단열 연속성 확보, Ψ → 0
4. **고기밀 시공**: n50 ≤ 0.6/h, 기밀층 연속성
5. **기계환기 (열회수)**: HRV/ERV 효율 ≥ 75%

## Revit과의 연계 활용 포인트

### 에너지 해석 (Revit Energy Analysis)
- 패시브하우스 설계 시 Revit 에너지 모델 → **OpenStudio** 또는 **PHPP** 연계
- Revit 에너지 분석용 `Space` 설정 필수: OccupancyType, HVAC System, 재실자수
- Room/Space의 `Energy Analysis` 탭에서 U-value 설정 확인

```csharp
// Revit API: 공간 에너지 설정 확인
var spaces = new FilteredElementCollector(doc)
    .OfClass(typeof(SpatialElement))
    .OfType<Space>();

foreach (var space in spaces)
{
    var heatGain = space.get_Parameter(
        BuiltInParameter.ROOM_CALCULATED_COOLING_LOAD_PER_AREA);
    // kW/㎡ 단위 반환
}
```

### 열관류율 계산 자동화 포인트
- Revit `CompoundStructure` 레이어 두께 + 열전도율 → 자동 U-value 계산 가능
- Material의 `ThermalProperties` → `Conductivity` 파라미터 활용

```csharp
// 재료 열전도율 추출
var material = doc.GetElement(materialId) as Material;
var thermalAsset = doc.GetElement(
    material.ThermalAssetId) as PropertySetElement;
var conductivity = thermalAsset?.get_Parameter(
    BuiltInParameter.PHY_MATERIAL_PARAM_THERMAL_CONDUCTIVITY);
// W/mK 단위
```

## 패시브하우스 마스터급 경험 지식 (2026-05-29)

### 현장에서 자주 발생하는 패시브하우스 시공 실패 패턴 5가지

**1. 기밀층 연속성 단절**
- 원인: 전기 콘센트 박스, 설비 관통부, 지붕-벽 접합부 처리 누락
- 증상: 블로어도어 n50 > 2.0/h (목표 0.6 초과)
- 해결: 시공 전 기밀층 도면 별도 작성 (레드라인), 콘센트 기밀박스 사용, 모든 관통부 EPDM 슬리브+테이프 처리

**2. 열교 설계는 했으나 시공 미반영**
- 원인: 설계도서의 단열 연속성 상세가 현장 반영 안 됨 (발코니 단열 브라켓, 창호 설치 위치 오류)
- 증상: 열화상 촬영 시 접합부 표면온도 < 12.6℃ → 결로/곰팡이
- 해결: 시공 전 열화상 점검 체크리스트 + 단계별 검측 (기초완료/외벽완료/창호설치후)

**3. HRV 덕트 설계 오류**
- 원인: 급기/배기 불균형 (±10% 초과), 덕트 누기율 과다, 필터 미교체
- 증상: CO2 농도 상승, 실내 과습/과건조
- 해결: 세대별 풍량 밸런싱 필수 (commissioning), 덕트 기밀등급 Class B 이상, 6개월 필터 교체 주기

**4. 창호 설치 위치 오류 (단열층 외부 설치)**
- 원인: 기존 시공 관행대로 벽체 중간에 창호 설치 → 프레임 주변 열교 발생
- 해결: 창호는 단열층과 동일 평면 또는 단열층 내부에 설치, 프레임 Uf ≤ 1.4 W/㎡K 확인

**5. 냉방 미고려 설계**
- 원인: 패시브하우스 = 난방만 고려한다는 오해 → 남향 창 면적 과다, 차양 미설치
- 증상: 여름 실내온도 28℃ 초과, 에어컨 과부하
- 해결: 여름 태양고도각 기준 수평 차양 깊이 계산 (서울 기준 여름 고도 72°), 남향 창은 SHGC 조절

### 한국 패시브하우스 시장 특수성

- **인증 이중 부담**: PHIKO 인증 + 녹색건축 ZEB 인증 병행 요구 증가 (공공사업 조건)
- **법규 vs 패시브 갭**: 법규 기준(중부1 외벽 0.21)과 패시브 기준(0.15) 사이 설계자 혼선
- **시공 단가**: 패시브하우스 추가 공사비 ㎡당 15~30만원 수준 (일반 건물 대비 10~20% 추가)
- **PHPP 사용**: 한국 기후데이터(기상청 TMY) 적용 시 서울 기준 난방도일 HDD≈2,500 (독일 프랑크푸르트 HDD≈3,000보다 낮음)
