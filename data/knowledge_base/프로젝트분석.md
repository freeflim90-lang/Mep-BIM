## 프로젝트분석 페르소나 업무 기준 (2026-05-26)

**BIM 프로젝트 사전 분석 기준:**
- 규모: 연면적, 공종 수, 모델 파일 수, 참여 팀 수로 대/중/소 분류 (대: 연면적 30,000m² 이상 또는 3개 이상 공종)
- 공종 범위: 건축/구조/MEP(기계/전기/소방/통신) 포함 여부 및 공종별 BIM 요구 수준(LOD) 확인
- 발주처 요구사항: BEP 제출 여부, IFC 납품 여부, Autodesk BIM 360/ACC CDE 사용 여부, 준공 BIM 요구 여부
- 사전 분석 결과는 견적심사원 및 조율차장에게 공유하여 공수 산정 및 리소스 배분에 활용

**견적 근거 분석 프레임워크 (공종별 BIM 투입 공수 산정):**
- 건축: 기본 모델링 공수 + 패밀리 제작 공수 + 도서 작성 공수 합산
- 구조: 구조 모델링 공수 + 간섭 검토 참여 공수
- MEP(기계/전기/소방): 공종별 모델링 + 간섭 검토 + 물량 산출 + CDE 업로드 공수
- 공수 산정 기준: [[BIM_등급별_투입일_기준표]] 참조, 발주처 요구 LOD에 따라 보정 계수 적용
- 견적 산출 후 ±15% 범위 초과 불확실성 존재 시 견적심사원에게 재검토 요청

**프로젝트 리스크 분류 기준:**
- 기술 리스크: 요구 LOD 대비 팀 역량 부족, 미사용 소프트웨어 플랫폼, 기존 모델 품질 불량
- 일정 리스크: 발주처 설계 변경 빈도, 중간 납품 기한 촉박, 다른 프로젝트와 인력 충돌
- 예산 리스크: 견적 산정 오차, 범위 변경(Scope Creep) 가능성, 외주 비용 증가
- 발주처 리스크: BIM 경험 부족한 발주처(요구사항 불명확), 계약서 납품물 정의 모호
- 각 리스크는 HIGH/MEDIUM/LOW 분류 + 대응 방안 1줄 포함하여 분석 보고서에 기재

**분석 보고서 표준 형식:**
- 섹션 1: 프로젝트 개요 (발주처, 용도, 규모, 공종, 납품 기한)
- 섹션 2: BIM 요구사항 정리 (LOD, CDE, 납품물 목록, 발주처 BIM 기준문서)
- 섹션 3: 공종별 투입 공수 산정 (표 형식: 공종 / 업무 항목 / 산정 공수 / 비고)
- 섹션 4: 리스크 분류 (HIGH/MEDIUM/LOW + 대응 방안)
- 섹션 5: 권고 의견 (수주 가능성 평가, 주의사항, 조율차장 전달 사항)

## BIM 프로젝트 사전분석 및 공수산정 전략 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: 프로젝트분석,공수산정,리스크분류,BIM,사전분석

- BIM 프로젝트 사전 분석 5대 항목: ①발주처 BIM 요구수준(LOD/LOI 기준) ②납품 포맷(IFC/RVT/NWD) ③참여 공종 수 ④기존 2D 도면 완성도 ⑤공정 일정 여유율. 이 5개 항목 점수화(각 1~3점, 총 15점 만점)로 프로젝트 복잡도 등급을 산정한다.
- 공수 산정 기준표(MEP BIM 기준): 연면적 1,000m² 당 MEP 모델링 공수는 기계 45시간, 전기 35시간, 배관 40시간이 업계 평균이다. 간섭 검토 및 조율은 전체 모델링 공수의 20~30%를 추가로 산정해야 하며, 발주처 검토 대응 공수는 별도로 10~15% 가산한다.
- 리스크 분류 체계: High(착공 지연, 설계 변경 빈발, BIM 미경험 발주처) / Medium(납품 기준 불명확, 공종 간 소통 체계 미흡) / Low(일정 여유 충분, 유사 프로젝트 경험 보유). High 리스크 항목이 2개 이상이면 계약금 비율을 40% 이상으로 협상한다.
- 프로젝트 수익성 분석: 목표 영업이익률 25% 이상 확보를 위해 인건비(총 공수 × 시간당 단가)가 계약금액의 55% 이하여야 한다. 시간당 단가는 BIM 모델러 45,000원, BIM 코디네이터 65,000원, BIM 관리자 90,000원 기준을 적용한다.
- Revit Add-in 도구 활용으로 공수 절감이 가능한 작업: 요소 자동 배치(30% 절감), 간섭 리스트 자동 추출(50% 절감), 수량 산출(40% 절감). 이 절감분을 견적서에 경쟁력 있는 단가로 반영하면 수주 경쟁에서 우위를 점할 수 있다.

## 프로젝트분석 마스터급 경험 지식 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: 프로젝트분석,수주판단,손익역산,중도포기,Post-mortem

수주 가능성 80% 이상 프로젝트 선별 기준: ①발주처와 1회 이상 사전 미팅 경험 보유 ②LUA BIM LABS 레퍼런스가 동일 건물 유형(상업/주거/의료)과 일치 ③경쟁사 대비 가격 차이 15% 이내 ④발주처 BIM 담당자가 의사결정권자(과장급 이상)와 직접 소통 가능. 이 4개 기준 중 3개 이상 충족 시 수주 가능성 80% 이상으로 평가하고 제안 인력 투입을 승인한다. 미충족 항목이 2개 이상이면 제안 투자(제안서 작성 공수 40시간)를 CEO 승인 후에만 진행한다.

손익분기 공수 역산법: 목표 영업이익률 25% 기준 계산식 = 계약금액 × 0.75 ÷ 평균 시간당 단가. 예시: 계약금 3,000만 원, 평균 단가 60,000원/시간 → 최대 투입 공수 375시간. 실제 견적 공수가 375시간을 10% 이상 초과하면 계약 단가 협상 또는 수주 포기를 검토한다. 추가 발주 가능성(후속 프로젝트)이 있는 경우에는 최초 계약 손익분기를 15%까지 낮춰도 전략적 수주가 정당화된다.

프로젝트 중도 포기 판단 기준(매몰비용 오류 방지): 이미 투입된 공수가 아니라 "남은 기간 동안 추가로 투입될 공수 × 기회비용"을 기준으로 판단한다. 잔여 계약금보다 남은 투입 공수 비용이 20% 이상 초과하고, 발주처와 범위 협상이 실패했을 때 중도 계약 해지(계약서 해지 조항 근거)를 선택하는 것이 합리적이다. 중도 포기 선언 전에 항상 법무조항검토 담당과 계약 해지 위약금 조항을 사전 확인한다.

사후 회고(Post-mortem) 구조화 방법: 프로젝트 종료 후 7일 이내 60분 회고 세션을 진행하며, 구성은 ①예측 대비 실제 공수 차이(원인 분석) ②발주처 만족도 체크(5점 척도) ③재발 방지 액션 아이템 3건 이내(담당자·기한 명시) ④다음 유사 프로젝트 견적 보정 계수 도출 순서로 진행한다. 회고 결과는 프로젝트분석 KB에 누적하여 견적 정확도를 연 2회 이상 개선한다.
