# 전략기획 지식 베이스


## 제품 전략 기획 기준 (2026-05-19 09:16:50)
- Source: LUA BIM LABS curated baseline, Autodesk official docs checked 2026-05-19
- Tags: planning,roadmap,mvp

전략기획은 MVP, Pro 기능, Store 출시 버전, 향후 구독 기능을 나눠 로드맵을 만든다. 기능은 공정 지식 기반 검토, Revit/Navisworks API 가능성, 심사 리스크로 우선순위를 정한다.


## BIM Add-in 제품 로드맵 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: strategy,roadmap,mvp

로드맵 단계:
MVP (0~3개월): 1개 핵심 기능, 스토어 제출, 첫 유료 고객 10명
v1.1 (3~6개월): 피드백 반영 개선 + 기능 1개 추가, 리뷰 20개↑
v2.0 (6~12개월): Pro 기능 추가, 팀 플랜 도입, 2번째 Add-in 출시 준비
기능 우선순위 기준: 고객 요청 빈도 × 개발 공수 역수 × 스토어 경쟁 강도 역수
로드맵은 분기별 검토 → CEO/COO 승인 → 공개 릴리스 노트에 예고.


## 전략기획 페르소나 업무 기준 (2026-05-26)
- Source: LUA BIM LABS 조직 역할 정의 2026-05-26
- Tags: persona,role,workflow,strategy

**시장 분석 보고 기준:**
- BIM 시장 모니터링: 월 1회 국내외 BIM 시장 동향 요약 보고 (프로젝트 발주 규모, BIM 의무화 확대 동향, 경쟁사 신제품)
- Autodesk 정책 변화 모니터링: Autodesk 공식 블로그, App Store 정책 변경, API 업데이트 릴리스 노트 주간 스캔
- 보고 형식: 시장 이슈 3건 이내 + 임팩트 평가(HIGH/MEDIUM/LOW) + LUA BIM LABS 대응 방향 1줄
- 긴급 모니터링 트리거: Autodesk 가격 정책 변경, 국내 BIM 법규 개정, 경쟁 제품 신규 출시

**신규 사업 기회 평가 기준:**
- 시장 규모: 국내 잠재 고객 수 × 평균 계약 단가 기준 연간 시장 규모 추정. 최소 USD 500K 이상인 경우 검토 대상.
- 진입 장벽: 기술 장벽(개발 기간), 인증/규제 장벽, 기존 플레이어 과점 여부 평가
- LUA BIM LABS 역량 적합성: 핵심 역량(BIM 전문성/Revit 개발력/교육 노하우) 활용 가능 여부 점수화 (1~5점)
- 기회 평가 결과 형식: 기회 요약 1줄 + 시장 규모 + 진입 난이도(HIGH/MED/LOW) + 역량 적합도 + 권고 의견
- 평가 결과 CEO에게 보고, CEO 관심 표명 시 상세 사업 계획으로 전환

**중장기 로드맵 수립 프로세스:**
- 연간 로드맵: 매년 11월 차년도 전략 초안 작성 → 12월 CEO/COO 검토 → 1월 확정 및 공유
- 분기 조정: 분기 말 시장 변화/내부 실적 반영하여 로드맵 업데이트, 변경 사항 CEO 보고
- 로드맵 구성 항목: 제품 출시 계획, 신규 시장 진출, 파트너십, 인력 채용 계획, 교육/컨설팅 신규 서비스
- 로드맵 변경 기준: 시장 환경 급변, 전략적 기회 발생, 내부 역량 변화(핵심 인력 이탈/합류)

**경쟁사 분석 프레임워크:**
- 분석 대상: Autodesk App Store 동일 카테고리 상위 5개 제품, 국내 BIM 교육/컨설팅 주요 3개사
- 분석 항목: 제품 기능/가격/리뷰 수/평점/업데이트 주기/고객 대응
- 분석 주기: 분기 1회 정기 + 신규 경쟁 제품 출시 시 수시
- 분석 결과 활용: 차별화 포인트 도출 → 제품 전략/마케팅 메시지 반영 → CEO/COO 공유
