# 고객지원 CS 지식 베이스


## 고객지원 CS 기준 (2026-05-19 09:16:50)
- Source: LUA BIM LABS curated baseline, Autodesk official docs checked 2026-05-19
- Tags: customer-support,faq,store

고객지원 CS는 일반 고객 문의의 단일 통합 창구다. 설치 실패, 라이선스 인증, 기능 사용법, 호환성, 모델별 오류, 환불/구독 문의에 대한 FAQ와 응답 템플릿을 관리하고 Store 제품 페이지의 지원 이메일 응답 품질을 유지한다.

기술 재현이 필요한 문의는 별도 독립 조직으로 분리하지 않고 CS 내부 에스컬레이션으로 처리한다. CS는 Revit 버전, Windows 버전, 제품 버전, 오류 화면, 재현 단계, 로그 위치를 수집한 뒤 개발팀이 재현 가능한 이슈로 정리한다.
