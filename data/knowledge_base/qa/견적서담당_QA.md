## 견적서담당 FAQ (2026-05-24)
- Source: LUA BIM LABS internal knowledge baseline
- Tags: quote,faq,bim,pricing

BIM 용역 견적서 작성 시 공수 산정을 물을 때:
- 공종별 모델링 공수 = (BIM 모델 요소 수 × 공종 난이도 계수) / 1인 일 생산성
- LOD 300 기준 공조 배관 단일 계통: 숙련자 기준 1인·일 40~60개 요소
- 간섭 검토 1회: 1~2인·일 (프로젝트 규모 따라 조정)
- 물량 산출 추가 시 전체 모델링 공수의 20~30% 추가

견적서에 반드시 포함해야 할 항목을 물을 때:
- 기본 개발/모델링, 지원 버전/공종, UI/UX, QA, 패키징, 스토어 지원, 라이선스 연동, 유지보수
- 유효 기간 (발행 후 30일 명시 필수)
- 지급 조건 (착수 40%, 중도 40%, 잔금 20% 권장)
- VAT 별도 표기 필수

범위 변경 발생 시 견적 처리를 물을 때:
- 서면 합의 후 변경 견적서를 재발행한다.
- 변경 전 착수된 업무는 기존 단가로 정산, 추가 범위만 신규 견적 적용.
- 미합의 추가 업무는 수행 의무 없음을 계약서에 명시.


## Q: BIM 모델링 견적 단가 기준이 있어? (2026-05-24)
- Source: LUA BIM LABS internal knowledge baseline
- Tags: quote,qa,pricing,unit-cost

LUA BIM LABS 내부 단가 참고 기준 (공개 기준, 프로젝트별 조정):

| 업무 항목 | 단위 | 내부 기준가 | 비고 |
|---|---|---|---|
| MEP BIM 모델링 (LOD 300) | 연면적 m²당 | [내부 단가] | 공종·난이도 따라 ±30% |
| 간섭 검토 보고서 | 회차 | [내부 단가] | 공종 수 따라 조정 |
| BIM 수행계획서 작성 | 식 | [내부 단가] | |
| Revit Add-in 기능 개발 | 기능 1개 | [내부 단가] | 복잡도 따라 ±50% |
| MSI 패키징 | 식 | [내부 단가] | |
| 유지보수 (연간) | 년 | 개발비의 15~20% | |

주의: 위 기준가는 내부 참고용이며, 실제 견적은 범위·기간·위험도를 반영하여 담당자가 확정한다.


## Q: 소프트웨어 개발 견적과 BIM 용역 견적의 차이는? (2026-05-24)
- Source: LUA BIM LABS internal knowledge baseline
- Tags: quote,qa,software,bim-service

소프트웨어 개발 견적 특징:
- 기능 단위로 분리 가능 (기능 1개당 단가 × 수량)
- 지원 Revit 버전 수에 따라 추가 단가 발생
- QA·테스트·패키징·Store 제출이 별도 항목
- 유지보수는 연간 계약으로 별도 체결

BIM 용역 견적 특징:
- 공종·구역·LOD 기준으로 모델링 범위 분리
- 간섭 검토 회차·물량 산출 여부 별도 항목
- 인력·기간 기반 공수 산정이 주 방식
- 설계 변경 대응을 별도 조항으로 명시 필요
