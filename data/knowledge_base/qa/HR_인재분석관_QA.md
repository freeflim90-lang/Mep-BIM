# HR_인재분석관 Q&A 지식

텔레그램 질문·자동 수집 응답 저장소 — 기준 가이드라인(HR_인재분석관.md)과 분리 운영.

## 2026-06-09 HR 인재분석 실전 Q&A
- Source: LUA BIM LABS HR 운영 기준, 채용 실무 경험
- Tags: hr,recruitment,resume,talent,qa,2026

**Q: 이력서 분석 시 어떤 기준으로 평가하나요?**
A: LUA BIM LABS 채용 평가 기준 (BIM 직군): ① 필수 역량: Revit 실무 경험 연수, IFC/LOD 이해도, 공종별 BIM 모델링 경험 ② 우대: Dynamo 자동화, Add-in 개발 경험, 간섭검토 참여 이력 ③ 인성/협업: 프로젝트 규모, 팀 규모, 역할 변화. AI 등급은 최종 채용 판단으로 확정하지 않는다. 인간 면접관의 최종 판단이 필요하다.

**Q: 이력서 PDF에서 개인정보를 어떻게 처리하나요?**
A: 개인정보 처리 원칙: ① 후보자 이름·연락처·주민번호는 외부 AI API로 전송하지 않는다 ② 로컬 PDF 파싱만 허용 ③ 분석 결과에서 개인 식별 정보를 제거한 뒤 채용 대시보드에 입력 ④ 분석 완료 후 임시 파일 삭제 ⑤ 이전 회사명·고객사명은 외부 공개 자료에 포함하지 않는다. HR_인재분석관 에이전트는 개인정보 비식별화를 먼저 수행한다.

**Q: 채용 대시보드에는 어떤 항목을 포함하나요?**
A: 채용 대시보드 항목: ① 후보자 코드 (실명 대신) ② 지원 직군 ③ 경력 연수 ④ 필수 역량 충족 여부 (체크리스트) ⑤ 우대 역량 수 ⑥ 서류 통과 여부 (Yes/No/Pending) ⑦ 면접 일정 ⑧ 면접관 의견 (면접 후 입력). 대시보드는 채용 과정 전반을 추적하는 도구이며 최종 합격 결정은 기록하지 않는다 (CEO/COO 별도 결정).

**Q: 역량 매트릭스는 어떻게 활용하나요?**
A: 역량 매트릭스는 직군별 필요 역량(기술/도메인/협업)을 등급(초급/중급/고급/전문가)으로 정의한 문서다. HR_인재분석관이 이력서를 매트릭스와 대조해 부족한 역량을 식별한다. 부족 역량은 교육컨설팅 에이전트에 전달해 온보딩 커리큘럼을 보강한다.

**Q: 채용 과정에서 AI 판단의 한계는?**
A: AI 분석은 이력서 텍스트 기반 패턴 매칭이다. 한계: ① 동기, 태도, 팀워크 평가 불가 ② 이력서에 없는 역량 파악 불가 ③ 거짓 기재 감지 불가 ④ 조직문화 적합성 판단 불가. HR_인재분석관은 "후보자 이력서 기준 추정치"를 제공하는 것이며, 최종 판단은 면접과 인간 판단이 필수다.
