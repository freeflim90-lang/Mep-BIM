# 러닝콘텐츠디자이너 Q&A 지식

## 2026-06-09 러닝콘텐츠디자이너 실전 Q&A
- Source: LUA BIM LABS 콘텐츠 제작 기준, L&D 실무
- Tags: learning-design,content,curriculum,instructional-design,qa,2026

**Q: BIM 교육 콘텐츠의 기본 구성 단위는?**
A: 콘텐츠 단위(Module): ① 학습 목표 (1~3개 동사형 문장) ② 개념 설명 (텍스트 + 시각자료) ③ 예시/시나리오 ④ 퀴즈 (2~5문항) ⑤ 실습 가이드 또는 체크리스트. 1모듈 권장 학습 시간: 15~20분. 텔레그램 AI 튜터용 콘텐츠는 메시지 분량 기준으로 3~5개 메시지로 압축한다.

**Q: 텔레그램 봇용 학습 메시지는 어떻게 작성하나요?**
A: 텔레그램 학습 메시지 원칙: ① 첫 문장에 핵심 개념 ② 예시는 실무 사례로 (이론 위주 금지) ③ 1메시지 = 1개념 (2개 이상 섞지 않음) ④ 마지막 줄에 확인 질문 또는 다음 내용 예고 ⑤ 메시지 길이: 300자 이내 권장. 교육컨설팅이 커리큘럼 개요를 제공하면 러닝콘텐츠디자이너가 실제 메시지를 작성한다.

**Q: 퀴즈 문항은 어떻게 설계하나요?**
A: 퀴즈 설계 기준: ① 암기 테스트(단순 정의) 비율 30% 이하 ② 적용 문제(실무 상황 적용) 50% 이상 ③ 오답 선지는 흔한 오개념 기반으로 작성 ④ 오답 시 해설 필수 ⑤ 4지선다 또는 OX. 정답 공개 형식: 즉시 피드백(텔레그램 봇은 정답 직후 해설 메시지 전송).

**Q: 기존 콘텐츠를 업데이트해야 할 때는?**
A: 업데이트 트리거: ① BIM 법규/기준 변경 (법규변경모니터링 에이전트 알림) ② Revit 신버전 출시 ③ 학습자 오답률 > 40%인 문항 ④ 교육컨설팅이 커리큘럼 개편 요청. 업데이트 시 기존 버전을 보관하고 변경 내역을 기록한다. 대규모 개편은 교육컨설팅과 협의 후 진행한다.

**Q: 영어/한국어 이중 언어 콘텐츠는 어떻게 관리하나요?**
A: 기본 콘텐츠는 한국어로 먼저 작성 → 영어 번역은 테크니컬_라이터가 검토 → 글로벌_유통기획관이 현지화 검토. 번역 품질 기준: BIM 전문 용어는 Autodesk 공식 용어집을 따른다. 번역과 원본의 의미 차이가 발생하면 한국어 원본을 기준으로 수정한다.
