- **협력사 정기 안부 및 관계 관리**: 월 1회 협력사 담당자에게 정기 안부 연락을 실시하며, 분기 1회 현황 공유 미팅(대면 또는 화상)을 진행한다. 안부 연락 내용은 협력사 DB에 기록하여 관계 이력을 누적 관리한다.
- **협력사 평가 기준(100점 만점)**: 품질(납품물 검수 통과율·하자 발생 건수) 40점, 납기 준수율 30점, 커뮤니케이션(응답 속도·보고 충실도) 15점, 보안 준수(NDA·접근 권한 관리) 15점으로 반기 1회 평가한다. 70점 미만 시 개선 계획 요구, 60점 미만 2회 연속 시 협력 관계 종료를 외주관리와 협의한다.
- **신규 파트너 발굴 기준**: BIM 전문성(Revit/Navisworks/Dynamo 실무 경험), AEC 업계 레퍼런스(납품 이력), 재무 안정성(사업자등록·보험 가입 여부), 보안 준수 가능 여부를 신규 파트너 자격 기준으로 설정한다. 후보 협력사는 소규모 파일럿 과업으로 기술력과 협업 방식을 검증한 후 정식 등록을 진행한다.
- **보안 관리 기준**: 고객 프로젝트 모델 공유 금지, 테스트 전용 더미 모델만 제공한다. 협력사 접근 권한은 최소 권한 원칙으로 부여하며, 작업 종료 즉시 접근 권한을 해제한다. 보안 서약서 미서명 협력사에는 민감 정보를 제공하지 않는다.
- **글로벌 파트너 관리**: 해외 파트너사는 글로벌_유통기획관과 협력하여 계약 조건·판매 지역·지원 의무를 관리한다. 현지 규정(개인정보 처리·계약법)을 사전 검토하여 리스크를 최소화한다.

## 협력사 관계 관리 심화 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: partner,vendor,evaluation,contract,relationship-management

**협력사 평가 100점 루브릭 상세:**

| 평가 항목 | 배점 | 세부 기준 |
|---|---|---|
| 품질 | 40점 | 결함률 2% 이하(40점) / 5% 이하(30점) / 10% 이하(20점) |
| 납기 | 30점 | 납기 준수율 98%+(30점) / 95%+(20점) / 90%+(10점) |
| 커뮤니케이션 | 15점 | 이슈 24시간 내 보고(15점) / 48시간(10점) / 72시간+(5점) |
| 보안 준수 | 15점 | NDA 이행·코드 보안 심사 합격(15점) / 미흡사항 1건(10점) |

**협력사 등급별 관리 정책:**
- **A등급 (85점+)**: 우선 발주, 단가 인상 협의 가능, 연간 파트너 인증서 발급
- **B등급 (70~84점)**: 일반 발주, 분기 1회 역량 점검 미팅
- **C등급 (55~69점)**: 조건부 발주, 월 1회 모니터링, 6개월 내 B등급 달성 요구
- **D등급 (54점 이하)**: 신규 발주 중단, 기존 계약 완료 후 관계 재검토

**계약서 필수 조항:**
- 지식재산권: 납품 결과물 LUA BIM LABS 귀속 명시
- 보안: NDA + 소스코드 외부 공유 금지 + 개인정보 처리 동의
- 하자보수: 납품 후 3개월 무상 수정 의무 (중요 버그 30일 이내)
- 페널티: 납기 1일 지연 시 계약금액의 0.1%, 최대 10%
- 분쟁: 서울중앙지방법원 전속 관할
