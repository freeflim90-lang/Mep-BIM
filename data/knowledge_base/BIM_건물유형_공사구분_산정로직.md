## 전체 산정 공식 (통합)

```
실 투입 공수(MM)
  = 기준 공수(MM)
    × 건물유형계수(T)
    × 공사구분계수(C)
    + 현황파악 추가공수(S)   ← 개보수·증축 시만 적용
    × EIR수준계수(β)
    × 납기계수(γ)
    × (1 − AI절감율)

기준 공수(MM) = 연면적(㎡) ÷ 1,000 × 공종별 가중 합산 × 단계별 MM계수
```

> 각 계수의 범위에서 **값 선택 기준**: 하한 = 도면·정보 완비, 협조적 발주처, AI 도구 최대 활용 / 상한 = 정보 불완전, EIR 엄격, 협의 빈번

## 1. 건물 유형별 계수 (T)

### 1.1 유형 분류 및 계수

| 건물 유형 | T 계수 | MEP 복잡도 | 주요 특이사항 |
|---|---|---|---|
| **물류창고 / 공장 (단순)** | 0.5~0.7 | 낮음 | MEP 최소, 구조 위주, 층고 높음 |
| **주차장 (독립)** | 0.4~0.6 | 매우 낮음 | 배기팬·소방 위주 |
| **공동주택 (아파트)** | 0.7~0.9 | 낮음~보통 | 세대 반복 효과, 단위 세대 단순 |
| **교육시설 (초중고)** | 0.8~1.0 | 보통 | 반복 평면, 표준 MEP |
| **업무시설 (일반 오피스)** | **1.0** | **기준** | 표준 복잡도 |
| **공공청사 / 행정시설** | 1.0~1.2 | 보통 | EIR 엄격, 보안 요건 |
| **상업시설 (판매·백화점)** | 1.2~1.6 | 보통~높음 | 대공간, 임대 구획 가변, 천장 복잡 |
| **숙박시설 (호텔)** | 1.1~1.4 | 보통~높음 | 객실 반복 + 공용부(연회장·수영장) 복잡 |
| **복합시설 (주상복합 등)** | 1.2~1.5 | 높음 | 용도 혼재, 수직 조닝 복잡 |
| **문화시설 (공연장·박물관)** | 1.4~1.8 | 높음 | 대공간 구조, 음향·특수 조명 |
| **연구시설 (연구소·실험실)** | 1.4~2.0 | 높음 | 흄후드, 특수 배기·가스 배관 |
| **의료시설 (종합병원)** | 1.8~2.5 | 매우 높음 | 의료가스, 음압실, 방사선 차폐, 이중화 |
| **데이터센터** | 2.0~3.0 | 최고 | 이중화 전원·냉각, UPS, 정밀 공조, EMC |
| **플랜트 / 산업시설** | 별도 협의 | — | P&ID 기반, 전용 도구(PDMS·E3D) 필요 |

### 1.2 건물 유형 내 추가 보정

#### 공동주택 — 반복 세대 할인

| 세대 수 | 반복 할인율 | 실 계수 |
|---|---|---|
| 50세대 미만 | 0% | T 그대로 |
| 50~200세대 | −10% | T × 0.90 |
| 200~500세대 | −15% | T × 0.85 |
| 500세대 초과 | −20% | T × 0.80 |

> 단, 주동 종류(타입)가 3가지 이상이면 할인 미적용.

#### 의료시설 — 병상 수 추가 공수

| 병상 수 | 추가 MM |
|---|---|
| 100병상 미만 | 기준 T 적용 |
| 100~300병상 | +2~4 MM (의료가스·간호 시스템) |
| 300~700병상 | +5~10 MM |
| 700병상 초과 | +10 MM 이상, 별도 협의 |

#### 데이터센터 — 티어(Tier) 수준

| Tier | 계수 조정 |
|---|---|
| Tier I (단일 계통) | T = 2.0 |
| Tier II | T = 2.3 |
| Tier III (이중화) | T = 2.6 |
| Tier IV (완전 이중화) | T = 3.0+ |

## 2. 공사 구분별 계수 (C) 및 현황파악 추가공수 (S)

### 2.1 공사 구분 정의

| 구분 | 정의 |
|---|---|
| **신축** | 기존 건물 없음, 새로 지음 |
| **증축** | 기존 건물에 연접·연결하여 면적 추가 |
| **전면 리모델링** | 기존 외피(골조) 유지, 내부 전체 교체 |
| **부분 리모델링** | 특정 층·공간만 개보수, 나머지 존치 |
| **용도 변경** | 기존 건물 용도를 법적으로 바꾸는 개보수 |
| **내진 보강** | 구조 보강 공사 (단독 또는 리모델링 병행) |

### 2.2 공사 구분별 계수

| 공사 구분 | C 계수 | 비고 |
|---|---|---|
| **신축** | **1.0** | 기준 |
| **증축** (기존 BIM 있음) | 1.1~1.2 | 인터페이스 조율 공수만 추가 |
| **증축** (기존 BIM 없음) | 1.3~1.5 | 기존 부분 현황 BIM 작성 필요 |
| **전면 리모델링** (도면 양호) | 1.3~1.5 | 현황 확인 후 전체 재모델링 |
| **전면 리모델링** (도면 불량·부재) | 1.6~2.0 | 현장 실측 + 전면 재모델링 |
| **부분 리모델링** (범위 명확) | 1.2~1.4 | 해당 구간만 교체 |
| **부분 리모델링** (범위 불명확) | 1.5~1.8 | 공사 진행 중 범위 변경 리스크 |
| **용도 변경** | 1.4~1.8 | 전면 리모델링 준용 + 법규 재검토 |
| **내진 보강 (단독)** | 구조 공종만 ×1.5~2.5 | 타 공종 영향 최소 |
| **내진 보강 + 리모델링** | 전체 공수 ×1.4~1.8 | 구조 변경이 MEP 경로에 영향 |

### 2.3 현황파악 추가공수 (S) — 개보수·증축 전용

> 개보수·증축 시 기존 건물의 현황을 파악하는 방법에 따라 추가 공수가 발생한다.
> S는 기준 공수(MM)의 비율로 산정한다.

| 현황파악 방법 | 추가 공수(S) | 조건 |
|---|---|---|
| 기존 BIM 모델 제공 (정확도 높음) | +0% | 발주처가 검증된 BIM 제공 시 |
| CAD 도면 제공 (최신·정확) | +10~20% | 도면과 실제 일치 확인만 필요 |
| CAD 도면 있으나 오래됨·부정확 | +30~50% | 주요 구간 현장 대조 필수 |
| 도면 부재, 현장 실측 전담 | +50~80% | 전 층 직접 측정 |
| 3D 레이저 스캔 활용 | 스캔 비용 별도 + 처리 공수 +20~30% | 포인트 클라우드 → BIM 변환 공수 |

#### 3D 레이저 스캔 비용 참고

| 범위 | 스캔 비용 (외부 발주 시) | 처리 기간 |
|---|---|---|
| 단일 층 (1,000㎡ 기준) | 150~300만원 | 1~2주 |
| 전체 건물 (10,000㎡) | 1,000~2,500만원 | 3~6주 |
| 전체 건물 (30,000㎡↑) | 별도 협의 | 별도 협의 |

### 2.4 사용 중 공사 (Occupied Renovation) 추가 계수

> 입주자·사용자가 있는 상태에서 공사가 진행되면 접근 제한, 분산 공정, 소음 민원으로 BIM 조율 부담이 커진다.

| 상황 | 추가 계수 |
|---|---|
| 비어 있는 공간 (공실) | +0% |
| 일부 사용 중 (층별 순차 공사) | +15~25% |
| 전체 사용 중 (야간·주말 한정 작업) | +30~50% |

## 3. 층수 및 지하층 보정

| 조건 | 보정 | 사유 |
|---|---|---|
| 지상 10층 이하 | ×1.0 | 기준 |
| 지상 11~20층 | ×1.1 | 코어·수직 계통 복잡도 증가 |
| 지상 21~40층 (고층) | ×1.2~1.3 | 피난 계획, 구조 전이, 수직 간섭 증가 |
| 지상 41층 이상 (초고층) | ×1.4~1.6 | 아웃리거, 피난 안전구역, 이중 외피 |
| 지하 2층 이하 | +0% | 기준 |
| 지하 3~4층 | +10~15% | 흙막이·지하 MEP 복잡도 |
| 지하 5층 이상 | +20~30% | 구조·토목·MEP 심층 조율 |

## 4. 통합 산정 공식 정리

```
① 기준 공수(MM)
   = (연면적 ÷ 1,000) × Σ(공종별 가중치) × 단계별 MM계수

② 건물유형·규모 보정
   = 기준 공수 × T(건물유형) × 층수보정

③ 공사구분 보정
   = ② × C(공사구분)

④ 현황파악 추가공수 (개보수·증축만)
   = ② × S(현황파악 방법)

⑤ 사용 중 공사 추가 (해당 시)
   = (③ + ④) × 사용 중 공사 계수

⑥ 발주처 요건·납기 보정
   = ⑤ × β(EIR수준) × γ(납기)

⑦ AI 절감 차감
   = ⑥ × (1 − AI절감율 20~35%)

⑧ 실 투입 공수(MM) = ⑦
```

## 5. 케이스별 계산 예시

### Case A — 신축 업무시설, 중형

- 연면적: 15,000㎡ / 지상 12층 / 지하 2층
- 공종: 건축·구조·공조·위생·소방·전기·통신 (7공종, 가중치 합 4.6)
- BIM 단계: 실시설계 (MM계수 0.35)
- 건물유형: 업무시설 (T = 1.0)
- 공사구분: 신축 (C = 1.0)
- EIR 수준: 표준 (β = 1.1)
- 납기: 표준 (γ = 1.0)
- AI 절감: 25%

```
기준 공수    = 15 × 4.6 × 0.35 = 24.2 MM
유형·층수    = 24.2 × 1.0 × 1.1(12층) = 26.6 MM
공사구분    = 26.6 × 1.0 = 26.6 MM
EIR·납기    = 26.6 × 1.1 × 1.0 = 29.3 MM
AI 차감     = 29.3 × 0.75 = 21.9 MM ≈ 22 MM
→ 투입 인력 예: 시니어 1(40%) + 중급 2(80%×2) × 5개월 ≈ 22 MM
```

**참고 제안가 (시니어 750만 + 중급 480만×2):**
```
인건비 = (750×0.4 + 480×0.8×2) × 5개월 = (300 + 768) × 5 = 5,340만원
간접·리스크·마진(×1.5) → 약 8,000만원
```

---

### Case B — 전면 리모델링, 의료시설 (도면 부재)

- 연면적: 8,000㎡ / 지상 6층 / 병원 (250병상)
- 공종: 건축·구조·공조·위생·소방·전기·통신 + 의료가스 (8공종, 가중치 합 5.2)
- BIM 단계: 실시설계 (MM계수 0.38)
- 건물유형: 의료시설 (T = 2.1) + 250병상 추가 +5 MM
- 공사구분: 전면 리모델링, 도면 부재 (C = 1.8)
- 현황파악: 현장 실측 (S = +70%)
- 사용 중 공사: 일부 (×1.20)
- EIR: 엄격 (β = 1.3)
- 납기: 다소 촉박 (γ = 1.2)
- AI 절감: 15% (개보수·의료 특수 조건으로 절감 낮음)

```
기준 공수    = 8 × 5.2 × 0.38 = 15.8 MM
유형·층수    = 15.8 × 2.1 × 1.0 = 33.2 MM
공사구분    = 33.2 × 1.8 = 59.8 MM
현황파악 추가 = 33.2 × 0.70 = +23.2 MM → 소계 83.0 MM
의료가스 추가 = +5 MM → 88.0 MM
사용 중 공사  = 88.0 × 1.20 = 105.6 MM
EIR·납기    = 105.6 × 1.3 × 1.2 = 164.7 MM
AI 차감     = 164.7 × 0.85 = 140 MM
→ 대형 프로젝트 — 수석 1 + 시니어 2 + 중급 3 × 12개월 이상
```

---

### Case C — 증축, 공동주택 (기존 BIM 있음)

- 연면적(증축부): 5,000㎡ / 300세대 추가
- 공종: 건축·구조·MEP 4공종 (가중치 합 3.2)
- BIM 단계: 기본+실시 통합 (MM계수 0.28)
- 건물유형: 공동주택 (T = 0.85) × 300세대 반복 할인(×0.85) = T = 0.72
- 공사구분: 증축, 기존 BIM 있음 (C = 1.15)
- 현황파악: 기존 BIM 제공 (S = +0%)
- EIR: 자체 기준 (β = 1.0) / 납기: 표준 (γ = 1.0)
- AI 절감: 35%

```
기준 공수    = 5 × 3.2 × 0.28 = 4.5 MM
유형·층수    = 4.5 × 0.72 = 3.2 MM
공사구분    = 3.2 × 1.15 = 3.7 MM
EIR·납기    = 3.7 × 1.0 × 1.0 = 3.7 MM
AI 차감     = 3.7 × 0.65 = 2.4 MM
→ 중급 1명 × 2.5개월, 주니어 지원 포함
```

---

### Case D — 부분 리모델링, 데이터센터 (Tier III)

- 공사 범위: 서버실 3개 층 (3,000㎡)
- 공종: 구조·공조(정밀)·전기(이중화)·통신 (4공종, 가중치 합 3.5)
- BIM 단계: 실시설계 (MM계수 0.40)
- 건물유형: 데이터센터 Tier III (T = 2.6)
- 공사구분: 부분 리모델링, 범위 명확 (C = 1.3)
- 현황파악: CAD 도면 오래됨 (S = +40%)
- 사용 중 공사: 전체 (×1.40)
- EIR: 엄격 (β = 1.25) / 납기: 촉박 (γ = 1.3)
- AI 절감: 15%

```
기준 공수    = 3 × 3.5 × 0.40 = 4.2 MM
유형·층수    = 4.2 × 2.6 = 10.9 MM
공사구분    = 10.9 × 1.3 = 14.2 MM
현황파악 추가 = 10.9 × 0.40 = +4.4 MM → 소계 18.6 MM
사용 중 공사  = 18.6 × 1.40 = 26.0 MM
EIR·납기    = 26.0 × 1.25 × 1.3 = 42.3 MM
AI 차감     = 42.3 × 0.85 = 36 MM
→ 시니어 2 + 중급 2 + 주니어 1 × 4~5개월
```

> **핵심 인사이트**: 연면적만 보면 소규모(3,000㎡)이지만 데이터센터 + 운영 중 공사 + 오래된 도면 조합으로 업무량이 36 MM까지 증가. 단순 면적 기반 견적은 심각한 과소 산정 위험.

## 6. 견적 산정 시 사전 확인 질문 목록

발주처 또는 RFP에서 반드시 확인해야 할 항목.

```
[기본 정보]
□ 연면적 / 지상층수 / 지하층수
□ 건물 용도 (복수 용도이면 각 비율)
□ 신축 / 증축 / 리모델링 구분

[개보수·증축 추가 확인]
□ 기존 BIM 모델 유무 및 정확도
□ 기존 CAD 도면 유무·최신화 여부
□ 현장 실측 필요 여부
```
