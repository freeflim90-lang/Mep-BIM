## 설비 도면 종류와 읽는 순서 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: mep,drawing,shop-drawing,bim
- Links: [[설비기초]], [[설비장비]], [[설비시공조율]]

설비 도면은 평면도 하나만 보면 판단이 틀리기 쉽다. 일반 질문에 답할 때는 도면 종류를 연결해서 읽는다.

기본 도면:
- 범례/일반사항: 약어, 선종, 기호, 재료, 시공 기준, 특기사항.
- 평면도: 장비, 배관, 덕트, 트레이의 위치와 경로.
- 계통도: 계통 흐름, 장비 연결, 공급/환수, 급수/배수, 전원/신호 관계.
- 상세도: 장비 접속, 관통부, 지지, 점검구, 방화처리, 배관 접합.
- 단면도: 천장 속 높이, 덕트/배관/트레이 상하 관계, 구조 보와의 관계.
- 장비일람표: 장비 번호, 용량, 전원, 풍량, 유량, 압력, 제조 기준.
- 배관/덕트 일람표: 규격, 재질, 단열, 압력, 용도.
- 제어도/포인트 리스트: 자동제어 신호와 운전 시퀀스.

읽는 순서:
1. 일반사항과 범례에서 약어와 선종을 확인한다.
2. 계통도에서 흐름과 장비 연결을 먼저 이해한다.
3. 평면도에서 실제 위치와 경로를 확인한다.
4. 단면도에서 높이와 간섭 가능성을 확인한다.
5. 상세도에서 접속, 지지, 방화, 점검 방법을 확인한다.
6. 일람표에서 성능값과 전원, 규격을 확인한다.
7. BIM 모델과 도면의 장비번호, 계통명, 레벨, 규격이 일치하는지 대조한다.


## 도면에서 자주 묻는 설비 질문 처리법 (2026-05-22)
- Source: LUA BIM LABS internal MEP drawing interpretation baseline
- Tags: mep,drawing,faq
- Links: [[공조덕트]], [[공조배관]], [[위생]], [[소방기계]], [[전기]], [[통신]]

“이 선이 뭐야?”:
- 선종, 색상, 약어, 레이어, 범례를 먼저 확인한다.
- 공조는 SA/RA/EA/OA, 배관은 CHW/HW/CW/CWS/HWS/SAN/V, 전기는 전력/조명/통신/방재를 구분한다.

“이 배관이 왜 기울어져 있어?”:
- 오배수, 우수, 응축수 배관은 자연유하를 위해 구배가 필요하다.
- 급수, 냉온수, 소방 배관은 일반적으로 압송이므로 도면상 구배보다 경로와 공기빼기/드레인이 중요하다.

“천장 안에 들어갈 수 있어?”:
- 덕트 하단고, 배관 외경과 단열, 트레이 높이, 천장 마감고, 구조 보 하단고를 함께 본다.
- 제연, 중력배관, 대구경 덕트, 주 트레이는 우선 배치하고 소구경 배관과 전선관을 조정한다.

“장비 주변에 공간이 부족한가?”:
- 장비 외형만 보지 말고 문 열림, 필터 교체, 코일 인출, 모터 인양, 밸브 조작, 패널 전면 공간을 본다.

“도면과 모델이 다른데 무엇을 믿어야 해?”:
- 계약도서 우선순위, 최신 개정번호, 승인된 Shop Drawing 여부를 확인한다.
- BIM 모델은 검토와 조율 도구이므로, 시공 확정은 승인도면과 현장 조건을 기준으로 판단한다.


## 설비 약어 해석 기준 (2026-05-22)
- Source: LUA BIM LABS internal MEP glossary baseline
- Tags: mep,abbreviation,drawing

공조 덕트:
- SA: Supply Air, 급기
- RA: Return Air, 환기
- EA: Exhaust Air, 배기
- OA: Outside Air, 외기
- MA: Mixed Air, 혼합공기
- SE: Smoke Exhaust, 배연
- PS: Pressurization, 가압

공조 배관:
- CHWS/CHWR: 냉수 공급/환수
- HWS/HWR: 난방 온수 공급/환수 또는 급탕 공급/환수. 문맥 확인 필요.
- CWS/CWR: 냉각수 공급/환수 또는 급수 공급으로 쓰일 수 있어 도면 범례 확인 필요.
- REF: 냉매
- STM/COND: 증기/응축수

위생:
- DCW/CW: 급수
- DHW/HW: 급탕
- SAN: 오배수
- V: 통기
- ST/RW: 우수
- FD: 바닥배수
- CO: 소제구

전기/통신:
- PNL: 분전반
- MCC: 모터제어반
- TR: 변압기
- ATS: 자동절체스위치
- UPS: 무정전전원장치
- MDF/IDF: 주배선반/중간배선반
- CCTV, PA, LAN, TEL: 영상, 방송, 네트워크, 음성통신

주의:
- 같은 약어라도 회사, 국가, 과업 기준에 따라 의미가 달라질 수 있다.
- 답변에는 “일반적으로는”이라고 설명하고, 최종 판단은 해당 도면의 범례와 일반사항을 기준으로 한다.


## 부하계산·공조방식 도면 해석 기준 (2026-05-22)
- Source: 사용자 제공 PDF `부하계산과 공조시스템 선정 - 황동곤 V3.pdf` internalized summary
- Tags: hvac,drawing,load-calculation,system-selection,bim
- Links: [[설비기초]], [[설비장비]], [[공조덕트]], [[공조배관]], [[설비자동제어]]

공조부하와 공조방식은 평면도만으로 판단하지 않는다. 장비일람표, 계통도, 제어도, 부하계산서, 평면도, 단면도를 함께 읽어야 한다.

확인 문서:
- 부하계산서: 실별/존별 냉방부하, 난방부하, 외기량, 피크 조건.
- 장비일람표: AHU, FCU, PAC, VAV, 냉동기, 보일러의 풍량, 유량, 정압, 용량.
- 공조 계통도: OA, RA, SA, EA 흐름과 장비 연결.
- 배관 계통도: 냉수, 온수, 냉각수, 냉매 공급/환수 흐름.
- 자동제어도: CAV/VAV, 코일 밸브, 댐퍼, 센서, BMS 포인트.
- 평면/단면도: 덕트·배관 경로, 천장고, 장비 반입, 점검 공간.

도면 해석 질문 답변 순서:
1. 해당 공간의 용도와 운전 시간을 확인한다.
2. 부하계산서 또는 장비일람표에서 요구 풍량과 장비 용량을 확인한다.
3. 계통도에서 전공기, 전수, 공기-수, 중앙식, 개별식 중 어떤 방식인지 본다.
4. 평면도와 단면도에서 덕트·배관·장비 공간이 가능한지 확인한다.
5. 제어도에서 해당 방식에 필요한 센서와 제어 포인트가 있는지 확인한다.
6. BIM 모델에서는 Space/Zone, Airflow, System Type, 장비 파라미터, 계통 연결 상태를 대조한다.

**설비 도면 종류와 BIM 연계:**
- **계통도 (Schematic)**: 배관·덕트 계통 흐름, 기기 배치 관계 — Revit에서 1D Diagram 또는 별도 2D 뷰로 표현

- **단면도 (Section)**: 층고·클리어런스 확인 — Revit Section View에서 실시간 확인 가능
- **상세도 (Detail)**: 기기 접속부·지지철물·방진 상세 — LOD 400 수준, 현장 제작 도면
- **계획 평면도 vs 시공 상세도**: BIM 모델 → 계획 평면도 추출 자동화 가능, 시공 상세는 2D CAD 병행

**도면 기호 독해 핵심:**
- 소방: SP(스프링클러헤드) △, OS&Y 게이트밸브 (개방형 스템 확인용), 드라이펜던트 △▽
- 공조: ○ (공급 디퓨저 SA), ◎ (환기 그릴 RA), 방화댐퍼 FD(□ + X), 방연댐퍼 SD
- 위생: W(급수), HW(급탕), D(배수), V(통기) — 관경 및 구배 반드시 병기
- 전기: ┤├ (배선), C-T(전류변압기), △(감지기), □(수신기), MCC(전동기제어반)

- 도면 번호 파라미터: Sheet Number, Revision, Drawn by — BIM 도면 발행 이력 관리

## 설비도면해석 마스터급 경험 지식 (2026-05-29)
- Source: claude-code-enhanced 2026-05-29
- Tags: mep,drawing,shop-drawing,field-experience,bim,revit,interpretation

**설비 도면 오독으로 발생하는 시공 오류 상위 5가지:**
1. **공조 계통 방향 오독**: SA(공급)와 RA(환기) 화살표 방향을 반대로 읽어 덕트를 역방향으로 시공. 계통도에서 공기 흐름 방향(AHU OUT → Space → AHU IN)을 먼저 이해하고 평면도 확인
2. **관경 기재 위치 오해**: 배관 관경이 분기점 전·후 중 어느 쪽 관경인지 불명확한 도면에서 대구경을 소구경으로 시공. Revit Pipe Tag에서 전·후 관경을 명시하도록 도면 출력 시 설정
3. **기구 고정 높이 미확인**: 세면기·소화전 호스접속구 설치 높이가 도면에 있는데 무시하고 균일 높이 시공. Revit에서 Element Offset 파라미터가 모든 기구에 입력됐는지 확인
4. **제어 연동 선로 누락**: 공조 제어 도면(자동제어도)를 보지 않고 기계 도면만 참고하여 VAV 댐퍼 구동기 제어선 미설치. 기계+전기+자동제어 도면 세트를 반드시 함께 검토
5. **방화댐퍼 위치 누락**: 평면도에서 벽 관통부를 보고 방화댐퍼(FD)가 누락됐는지 확인하지 않아 소방 준공검사 불합격. 방화구획 벽 관통 덕트 전체에 FD 표시 여부를 클래시 검토와 함께 확인

**Revit에서 도면 추출 품질 기준:**
- View Template 고정: 공종별(공조/위생/소방/전기) View Template을 별도 정의하고 필터·선 굵기·색상 동일하게 유지
- 관경 태그 자동 표시: Pipe Tag에서 `Diameter` + `System Abbreviation` 동시 표시 설정. 분기 전·후 관경 명시
- Sheet 번호 체계: A-기계, B-전기, C-소방, D-위생으로 공종 코드 선행 — Revit Sheet Parameter `Discipline`으로 필터링 가능
- Revision Cloud: 설계 변경 시 Revit Revision 기능으로 클라우드 추가 + 도면 표제란 자동 갱신
