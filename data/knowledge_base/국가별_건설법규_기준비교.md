| 국가 | 건축 기본법 | 구조 기준 | 소방 기준 | 에너지 기준 | BIM 의무화 |

| **한국** | 건축법·건축물관리법 | KDS 41 (내진), KBC 2016 | NFTC (국가화재안전기술기준) | 건축물 에너지절약설계기준 | 공공 500억+ (2025~) |

### 2. 한국 건설 법규 상세 (BIM 적용 기준)

#### 2-1. 주요 법규·기준 체계

```
건축법 (상위)
├── 건축물의 피난·방화구조 등의 기준에 관한 규칙
├── 건축물의 에너지절약설계기준 (국토부 고시)
├── 장애인·노인·임산부 등의 편의증진보장에 관한 법률
└── 주택법 → 주택건설기준 등에 관한 규정

소방 관련법
├── 소방시설 설치 및 관리에 관한 법률
├── NFTC 103 (옥내소화전 설비)
├── NFTC 203 (스프링클러 설비)
├── NFTC 501 (자동화재탐지 설비)
└── NFTC 202 (비상방송 설비)

구조 기준
├── KDS 41 10 15 (건축구조 내진설계기준)
├── KDS 41 20 00 (강구조 설계기준)
└── KDS 41 30 00 (콘크리트구조 설계기준)
```

#### 2-2. BIM 파라미터로 반드시 수용해야 할 한국 기준

| 분야 | 기준 | BIM 파라미터 |
|---|---|---|
| 방화구획 | 건축물 피난방화규칙 제14조 | FireCompartmentArea (1,000m² / 스프링클러 시 3,000m²) |
| 피난 거리 | 건축법 시행령 제34조 | MaxExitDistance_m (30m / 50m) |
| 주차장 기준 | 주차장법 시행령 | ParkingRatio (세대당 대수) |
| 일조권 | 건축법 제61조 | ShadowAngle (북측 인접 대지 일조 침해 자동 체크) |
| 에너지 성능 | 건축물에너지절약설계기준 | U-value (열관류율, 지역별 상이) |
| 내진 등급 | KDS 41 10 15 | SeismicImportanceFactor (I=1.2 특등급) |

### 3. 일본 건설 법규 상세 (BIM 적용 기준)

#### 3-1. 법규 체계

```
建築基準法 (建基法)
├── 建築基準法施行令 (정령)
├── 告示 (각 성청 고시) — 세부 수치 기준 포함
└── 建築確認申請 (건축 허가) — BIM 연동 전자 신청(BIM to 建確) 확산 중

消防法 (소방법)
├── 消防設備設置義務
└── 防火管理者 선임 의무

省エネ法 (에너지절약법)
└── 建築物省エネ法 → BEI (Building Energy Index) 기준
```

#### 3-2. 한국 기준과의 주요 차이점

| 항목 | 한국 | 일본 | BIM 대응 |
|---|---|---|---|
| 방화구획 | 1,000m²/3,000m² | 1,500m²(木造) / 제한 상이 | 국가별 파라미터 분리 |
| 내진 설계 | KDS 41 (동적 해석) | 新耐震基準 (1981~) / 耐震等級 1~3 | SeismicGrade 파라미터 |
| 에너지 기준 | 열관류율 지역별 | BEI 1.0 이하 (ZEB Ready) | 일본어 BEI 파라미터 |
| 건축 확인 | 건축 허가 (지자체) | 建築確認申請 (민간 확인기관) | 일본 BIM to 建確 포맷 지원 |
| 높이 제한 | 용도지역별 일조권 | 高度地区, 日影규制 (그림자 규제) | 일조 시뮬레이션 일본 기준 |

**일본 BIM 특이사항:**
- `建築BIM推進会議` (국토交通省 주도): 2023~2025 BIM 의무화 단계 로드맵
- 건축 허가 전자화(BIM to 建確): 2024~ 일부 지자체 시범, 2026~ 전국 확산 예정
- LUA BIM LABS 기회: 일본 건築確認申請 자동 생성 모듈 → 일본 설계사 핵심 Pain Point

### 4. 싱가포르 건설 법규 상세 (BIM 적용 기준)

#### 4-1. CORENET X — 세계 최선진 BIM 전자 허가 시스템

```
CORENET X (2024~ 전면 시행):
- IFC 4 기반 BIM 모델 직접 제출
- 자동 코드 컴플라이언스 체크 (ACC: Automated Code Checking)
- 검토 기관: BCA (건축), SCDF (소방), NEA (환경), PUB (상하수도)
```

**CORENET X BIM 파라미터 요구:**
| 검토 기관 | 핵심 파라미터 | IFC Property Set |
|---|---|---|
| BCA (건물 구조) | 점유 하중, 방화 등급, 접근성 | Pset_SpaceOccupancy |
| SCDF (소방) | 스프링클러 등급, 피난 거리, 방화구획 | Pset_SpaceFireSafetyRequirements |
| NEA (환경) | 환기 공기량, 화장실 수 | Pset_SpaceOccupancy.NumberOfPeople |
| PUB (상하수도) | 위생기구 수, 우수 유출량 | Pset_SanitaryTerminalTypeWC |

**LUA BIM LABS 싱가포르 전략:**
- CORENET X 자동 제출 Add-in → 싱가포르 설계사 필수 도구
- IFC 파라미터 자동 생성 + 싱가포르 기준 체크리스트 → 수동 작업 80% 절감

### 한국 기업 해외 진출 시 BIM 법규 리스크 Top 4

**1. 싱가포르 IFC 자동 검증 실패**
- CORENET 시스템이 IFC 파일을 자동 검증 → 비정형 부재 IfcBuildingElementProxy → 자동 반려
- 해결: 수출 전 싱가포르 BCA IFC 가이드라인 준수, Entity 매핑 사전 검증

**2. 영국 BIM Level 2 COBie 형식 미준수**
- 영국 공공 발주는 BS 1192/PAS 1192 기반 COBie 제출 필수
- 한국 COBie Export(Revit 플러그인) vs 영국 COBie 2.4 구조 차이 → 검수 반려
- 해결: UK COBie Template 별도 적용, 현지 BIM Coordinator와 협업

**3. UAE 아랍어 문서 병행 요구**
- 두바이 DDA 제출 시 도면·시방서 아랍어 번역본 병행 제출 요구 사례
- BIM 모델 내 파라미터 언어 이슈 (한국어→영어→아랍어 3중 관리)

**4. 미국 주정부별 상이한 에너지 기준**
- 캘리포니아 CEC Title 24 vs 텍사스 IRC 에너지 조항 상이
- BIM 에너지 모델을 주정부 기준에 맞게 재설정 필요 → 추가 공수 10~15%

### Revit API — 국가별 법규 파라미터 자동 전환

```csharp
// 국가 코드에 따른 에너지 기준 파라미터 자동 설정
public void SetEnergyStandard(Document doc, string countryCode)
{
    var projectInfo = doc.ProjectInformation;
    var param = projectInfo.LookupParameter("에너지_기준");
    var standard = countryCode switch
    {
        "KR" => "한국 건축물에너지절약설계기준",
        "GB" => "UK Part L Building Regulations",
        "SG" => "Singapore SS 553 / BCA Green Mark",
        "US" => "ASHRAE 90.1-2022",
        "AE" => "Dubai Green Building Regulations",
        _ => "ISO 52000 (국제 기준)"
    };
    using var tx = new Transaction(doc, "Set Energy Standard");
    tx.Start();
    param?.Set(standard);
    tx.Commit();
}
```
