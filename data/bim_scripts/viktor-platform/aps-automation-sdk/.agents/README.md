# Local Agent Skills

This repository stores local agent skills under `.agents/skills/`.

## Expected layout

Each skill directory should include:

- `SKILL.md` (skill instructions)
- `agents/openai.yaml` (OpenAI/Codex metadata)
- optional `scripts/` helpers used by the skill
