# 7-Day GitHub Proof Plan

## Day 1 - Project Setup
- Create repo structure and environment setup.
- Add `requirements.txt`, `.gitignore`, base README.
- Commit message: `chore: initialize project structure and environment`

## Day 2 - Dataset and Simulation
- Add data loader and synthetic data generator.
- Save first dataset snapshot in `data/energy.csv`.
- Commit message: `feat: add dataset loader with virtual energy simulation`

## Day 3 - Preprocessing Pipeline
- Add cleaning, hourly resampling, missing value handling.
- Save processed trend plot.
- Commit message: `feat: implement preprocessing and trend visualization`

## Day 4 - Feature Engineering + Training
- Add time features, lag features, rolling statistics.
- Train baseline model and save artifact.
- Commit message: `feat: add feature engineering and model training pipeline`

## Day 5 - Evaluation
- Add RMSE/MAE/R2 metrics and export test predictions.
- Commit message: `feat: add evaluation metrics and prediction export`

## Day 6 - Forecasting + API
- Add iterative future forecast logic and forecast plot.
- Add Flask API endpoint for prediction.
- Commit message: `feat: add future forecasting and flask inference api`

## Day 7 - GitHub Finalization
- Polish README with screenshots and architecture.
- Add docs and final output artifacts.
- Commit message: `docs: finalize project documentation and demo assets`
