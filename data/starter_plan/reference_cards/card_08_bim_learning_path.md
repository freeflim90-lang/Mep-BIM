## BIM Career Level Progression

| Level | Title | Core Skills | Typical Timeframe |
|---|---|---|---|
| Level 1 | Starter / Beginner | BIM concepts, LOD, Revit MEP interface, basic modeling, drawing reading, self-review | 0–6 months |
| Level 2 | Modeler | Modeling quality, family management, system parameters, duct/pipe schedules, basic Navisworks | 6–18 months |
| Level 3 | Coordinator | Clash grouping, coordination meetings, issue tracking, data exports, coordination drawing review | 18 months – 3 years |
| Level 4 | BIM Lead | BEP authoring, QA/QC process ownership, delivery milestone management, mentoring, client liaison | 3–6 years |
| Level 5 | Automation / BIM Manager | Dynamo scripting, Python, Revit API, process automation, firm-wide standard setting | 5+ years |

## Tools by Level

| Level | Primary Tools |
|---|---|
| Level 1 (Starter) | Revit MEP (basics), Navisworks Freedom (viewer), PDF viewer for drawings |
| Level 2 (Modeler) | Revit MEP (advanced), Navisworks Manage, BIM 360 / ACC Files |
| Level 3 (Coordinator) | Revit MEP, Navisworks Manage, BIM 360 / ACC Issues, Solibri / BIM Collab |
| Level 4 (Lead) | All Level 3 tools + Dynamo (basics), Excel for reporting, presentation tools |
| Level 5 (Automation) | Dynamo, Python, Revit API, Power BI / data dashboards, CI/CD pipelines for BIM |

## Monthly Learning Habit Structure

**Daily (30 minutes)**
- 10 min: Review one concept — lesson, standard document, or project example
- 15 min: Practice one task in Revit, Navisworks, or Dynamo
- 5 min: Write one observation, question, or lesson learned

**Weekly**
- Monday–Friday: Daily 30-minute practice block
- Saturday: Review the week's topics — consolidate, do not add new content
- Sunday: Plan next week's practice topics

**Monthly**
- Complete one new skill from the next level's skill list
- Review one BIM standard or project document (BEP, LOD Spec, IFC guide)
- Update your personal BIM portfolio with one new item
