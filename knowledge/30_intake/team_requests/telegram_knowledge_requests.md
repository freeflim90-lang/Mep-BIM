## 위생 배관 계통 기본 분류 — 역할·유체 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: plumbing,fluid,classification,water,drainage

위생 배관은 물(급수·급탕)과 오염수(배수)를 운반한다. 절대 혼합 금지.

CWS (Cold Water Supply, 급수):
- 유체: 음용·위생용 냉수
- 경로: 수도 본관 → 저수조 → 가압펌프 → 각 층 위생기구
- 압력: 말단 최소 0.05 MPa (세면기 기준)
- 관재: PVC, 스테인리스, 동관 (음용 가능 재질 필수)
- Revit 계통: Domestic Cold Water

HWS (Hot Water Supply, 급탕):
- 유체: 음용·위생용 온수 (급탕)
- 온도: 가열기 출구 60℃ (레지오넬라균 사멸 기준), 분기점 55℃ 이상 유지
- 경로: 급탕기(보일러·히트펌프) → 각 층 위생기구
- 주의: 공조용 난방 온수(HW)와 완전히 다른 계통 — 절대 혼용 금지
- Revit 계통: Domestic Hot Water Supply

HWR (Hot Water Return, 급탕환수):
- 유체: 순환하며 식은 급탕 (재가열 목적)
- 역할: 분기점에서 온도 유지를 위해 급탕기로 환수 순환
- 없으면: 긴 배관에서 냉수가 나온 뒤 온수 출탕 → 물 낭비
- Revit 계통: Domestic Hot Water Return

Sanitary (오배수):
- 유체: 생활하수 (세면기·양변기·싱크대 등 오수 + 잡배수)
- 특징: 자연유하 방식 → 구배 필수 (구배 없으면 막힘·악취)
- 가스: 황화수소(H₂S) 등 유독 가스 발생 → 트랩(물 막음)으로 차단
- Revit 계통: Sanitary

Vent (통기):
- 유체: 공기 (트랩 봉수 보호용)
- 역할: 배수 흐름 시 발생하는 부압·양압 흡수 → 트랩 봉수 파괴 방지
- 연결: 배수관 → 통기관 → 외기 (지붕 위 600mm 이상 돌출)

Storm (우수):
- 유체: 빗물 (오염도 낮음)
- 경로: 옥상·주차장 우수받이 → 우수관 → 집수정 → 공공 우수관
- 오수와 혼합 금지: 법적 분리 의무 (오수관에 우수 유입 시 하수처리장 부하 급증)
- 재활용: 중수도로 화장실 세정수 등에 활용 가능

배관 색상 관례:
CWS(급수)=하늘색, HWS(급탕공급)=분홍색, HWR(급탕환수)=연주황, 오배수=갈색, 우수=초록색

- 자연유하 배관은 물이 스스로 흐르도록 구배가 필요하다.

- 오배수 배관은 간섭 조율에서 이동 자유도가 낮은 편이다.

집수정과 펌프를 물을 때:
- 지하층처럼 자연유하가 어려운 곳은 집수정에 모은 뒤 펌프로 배출한다.
- 오수, 우수, 배수는 목적에 따라 분리하는 것이 기본이며 악취와 역류 방지를 함께 검토한다.

## 공조덕트 일반 FAQ와 답변 포인트 (2026-05-22)

- Tags: hvac-duct,faq,bim,coordination

덕트 종류를 물을 때:
- 기능 기준으로는 SA, RA, EA, OA, MA, PS, SE로 설명한다.
- 형상 기준으로는 장방형 덕트, 원형 덕트, 스파이럴 덕트, 플렉시블 덕트로 설명한다.
- 용도 기준으로는 일반 공조 덕트, 주방 배기 덕트, 화장실 배기 덕트, 주차장 환기 덕트, 제연 덕트로 설명한다.

덕트 크기를 물을 때:
- 덕트 크기는 풍량, 허용 풍속, 정압 손실, 소음 기준에 따라 정한다.
- 같은 풍량이라도 풍속을 낮추면 덕트가 커지고, 풍속을 높이면 덕트가 작아지지만 소음과 정압 손실이 증가한다.
- BIM 답변에서는 설계 풍량, 풍속, 단면적, 덕트 하단고를 함께 확인해야 한다.

덕트 간섭을 물을 때:
- 제연 덕트와 메인 덕트는 우선 보호하고, 말단 덕트와 플렉시블 덕트는 조정 가능성이 높다.
- 덕트 단면 축소는 풍량 부족과 소음 증가로 이어지므로 단순히 “작게 만들면 된다”고 답하지 않는다.
- 디퓨저, 감지기, 조명, 스프링클러 헤드와의 천장 배치를 함께 확인한다.

덕트 점검을 물을 때:
- 댐퍼, 방화댐퍼, 코일, 필터, VAV 박스 주변은 점검구와 접근 공간이 필요하다.
- 천장 안에 물리 충돌이 없어도 점검구가 없으면 유지관리 이슈로 분류한다.

## 덕트 계통 기본 분류 — 역할·유체 종류 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: hvac-duct,system-type,fluid,classification

덕트는 공기(기체)를 운반한다. 어떤 목적의 공기인지에 따라 계통이 구분된다.

공조(HVAC) 덕트 계통:
- OA (Outside Air, 외기): 외부 신선 공기. 외벽 루버 → AHU 유입. 법정 환기량 확보 목적. 1인당 최소 25m³/h.
- SA (Supply Air, 급기): AHU에서 온도·습도 조정된 공기 → 실내 취출구(디퓨저) 공급. 냉방 시 12~16℃, 난방 시 30~40℃.
- RA (Return Air, 환기/환수): 실내 사용 공기 → AHU로 회수 재처리. 에너지 회수 목적으로 일부 재사용.
- EA (Exhaust Air, 배기): 오염·습기 포함 공기를 외부로 배출. 화장실·주방·주차장에서 주로 발생.
- MA (Mixed Air, 혼합기): OA + RA 혼합. AHU 내부 구간. 별도 덕트 없음.

제연 덕트 계통 (소방법 우선 보호 대상):
- PS (Pressurization, 가압): 신선 외기를 피난 계단실·전실에 공급 → 양압 형성으로 연기 침입 차단. 40~60 Pa.
- SE (Smoke Exhaust, 배연): 화재실 연기를 외부로 강제 배출 → 피난 시간 확보. 고온 연기(고온·부식성).

환기 전용 덕트:
- 주방 배기 (Kitchen Exhaust): 고온(60~80℃) + 유증기 포함. 스테인리스 덕트 필수. 재순환 금지.
- 욕실/화장실 배기: 습기·냄새 포함. 환기횟수 10회/h 이상.
- 주차장 환기: CO·NOx·매연 포함. CO 농도 25ppm 이하 유지 (법정 기준).

덕트 재료:
- 아연도 강판: 일반 공조 SA/RA (가장 일반적)
- 스테인리스: 주방 배기, 제연 (내식·고온 적합)
- 플렉시블: 기기 접속부 진동 차단 (300~600mm)
- FRP: 부식 환경 (수처리실 등)
- 알루미늄: 클린룸, 냉장창고 (경량·내습)

Revit 계통 지정 (System Type):
- Supply Air = 급기(SA), Return Air = 환기(RA), Exhaust Air = 배기(EA), Outside Air = 외기(OA)
- 계통 지정 오류 시 Navisworks 간섭 검토에서 공종 분류 오류 발생 → 반드시 확인

스프링클러 종류를 물을 때:
- 습식은 배관에 물이 차 있어 일반 구역에 많이 쓰인다.

소방 배관 간섭을 물을 때:
- 소방 배관과 제연 계통은 안전 기능이므로 일반 설비보다 우선 보호한다.
- 헤드 주변 살수 장애, 방화구획 관통, 알람밸브 접근성은 별도 검토 항목으로 분리한다.

소방펌프를 물을 때:
- 소방펌프는 필요한 방수압과 방수량을 확보하는 장비다.
- 전원, 예비펌프, 수조, 알람밸브, 압력 스위치, 방재 신호와 함께 검토한다.

## 소방기계 계통 기본 분류 — 역할·유체 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: fire-mechanical,fluid,classification,sprinkler,suppression

소방기계는 화재 시 불을 끄거나 연기를 제어하는 설비다. 유체 종류에 따라 계통이 구분된다.

스프링클러 계통 (물 소화):
- 습식(Wet Pipe): 배관 내 항상 물 충만 → 헤드 개방 즉시 방수. 동결 우려 없는 일반 구역.
- 건식(Dry Pipe): 배관 내 가압 공기 → 화재 시 공기 배출 후 물 충전 → 방수. 동결 우려 구역(주차장·옥외).
- 준비작동식(Pre-action): 감지기 신호 + 헤드 개방 2중 조건 충족 시 방수. 오작동 방지 중요 (전산실·전기실).
- 일제살수식(Deluge): 개방형 헤드 + 감지기 신호로 전 헤드 동시 방수. 위험물 창고·항공기 격납고.

연결송수관 (Standpipe):
- 유체: 소방차에서 가압 공급하는 물
- 역할: 소방대원이 건물 내부에서 직접 호스 연결·방수 (화재 진압용)
- 설치 대상: 11층 이상 또는 연면적 5,000㎡ 이상

옥내소화전:
- 유체: 물
- 역할: 초기 화재 진압용. 재실자가 직접 사용 가능.
- 배관: 스프링클러 배관과 독립 계통 (일부 겸용 가능)

가스계 소화 계통 (비수계 소화):
- CO₂(이산화탄소): 전기실·발전기실. 산소 농도 희석으로 소화. 사람 대피 후 방사.
- HFC-125/HFC-227ea(청정소화약제): 전산실·통신실. 정밀 기기에 무해. 잔존물 없음.
- IG-541(INERGEN, 질소·아르곤·CO₂ 혼합): 미술관·문서보관실. 비도전성·비부식성.
- 주의: 가스계 소화 구역은 방호구역 기밀 필수, 환기 댐퍼 자동 폐쇄 연동.

소방 배관 일반 기준:
- 재료: 배관용 탄소강관(KS D 3507), 동관, 스테인리스
- 색상: 빨간색 (Red) 전체 도장
- 위치: 다른 설비 배관 하부 배치 원칙 (화재 시 충돌 최소화)
- 방화구획 관통: 방화 슬리브 + 내화채움재 필수

## 전기 설비 계통 기본 분류 (2026-05-22)
- Source: LUA BIM LABS internal MEP knowledge baseline
- Tags: electrical,classification,strong-current,weak-current,fire-electrical

전기 설비는 전압 크기에 따라 강전(Strong Current)과 약전(Weak Current/Low Voltage)으로 구분한다.

강전 (Strong Current, 전력 계통):
- 수변전: 한전 수전(특고압 22.9kV) → 변압기 → 배전반 → 각 층 분전반
- 배전: 고압(1kV 초과), 저압(1kV 이하 — 일반 220V/380V)
- 동력: 모터 구동 (펌프, 팬, 엘리베이터, 에스컬레이터)
- 조명: 일반 조명, 비상 조명
- 비상전원: 발전기(Generator), UPS(무정전전원장치)
- 케이블 트레이: 강전 전용 (약전과 최소 300mm 이격 또는 격벽)

약전 (Weak Current, 신호·통신 계통):
- 통신/LAN: Cat6, 광케이블 (MDF-IDF 체계)
- 전화: 일반 전화, 인터폰
- 방송: 공공 방송(PA), 비상방송 (내화 케이블 FR 필수)
- 영상: CCTV, 동축케이블 RG-59 (최대 300m), HD-SDI (100m)
- 보안: 출입통제, 카드리더
- 주의: 약전 케이블과 전력선 최소 150mm 이격 (EMI 방지)

소방전기 (Fire Electrical):
- 화재감지기: 차동식·연기식·열감지식 스포트형
- 수신기/발신기: 화재 신호 수집·발신
- 방재연동: 방화셔터·제연팬·소방펌프 자동 기동
- 배선: 내화 케이블(FR-8) 필수 — 화재 시 30분 이상 기능 유지
- 비상조명·유도등: 예비전원 30분 지속

전기 계통 Revit 카테고리:
- 강전: Conduit(전선관), Cable Tray(케이블 트레이), Electrical Equipment(분전반·배전반)
- 약전: 별도 Conduit 계통 또는 통신 전용 트레이

강전과 약전을 물을 때:
- 강전은 전력을 공급하는 계통이고, 약전은 통신·신호·보안처럼 정보를 전달하는 계통이다.

## Dynamo 카테고리 객체 일괄 선택 빠른 답변 (2026-05-22)
- Source: LUA BIM LABS internal Dynamo FAQ
- Tags: dynamo,revit,category,selection,python,filtered-element-collector
- Links: [[Revit_Addin]], [[설비도면해석]]

원하는 카테고리 객체를 일괄 선택하려면 두 단계로 생각한다.

노드 방식:
1. `Categories` 노드에서 카테고리를 선택한다.
2. `All Elements of Category` 노드에 연결한다.
3. 결과는 해당 카테고리의 인스턴스 요소 리스트다.
4. 조건이 필요하면 `Element.GetParameterValueByName`과 `List.FilterByBoolMask`로 필터링한다.

Revit 화면 선택까지 바꾸는 Python 노드 예시:

```python
import clr
clr.AddReference("RevitServices")
from RevitServices.Persistence import DocumentManager

clr.AddReference("RevitAPI")
from Autodesk.Revit.DB import FilteredElementCollector, BuiltInCategory, ElementId

clr.AddReference("System")
from System.Collections.Generic import List

doc = DocumentManager.Instance.CurrentDBDocument
uidoc = DocumentManager.Instance.CurrentUIApplication.ActiveUIDocument

category_map = {
    "Ducts": BuiltInCategory.OST_DuctCurves,
    "Pipes": BuiltInCategory.OST_PipeCurves,
    "Mechanical Equipment": BuiltInCategory.OST_MechanicalEquipment,
    "Cable Trays": BuiltInCategory.OST_CableTray,
}

names = IN[0] if isinstance(IN[0], list) else [IN[0]]
elements = []
seen = set()

for name in names:
    bic = category_map.get(str(name))
    if bic is None:
        continue
    for element in FilteredElementCollector(doc).OfCategory(bic).WhereElementIsNotElementType():
        if element.Id.IntegerValue not in seen:
            seen.add(element.Id.IntegerValue)
            elements.append(element)

uidoc.Selection.SetElementIds(List[ElementId]([e.Id for e in elements]))
OUT = elements
```

주의:
- `All Elements of Category`는 요소 리스트를 만들지만 Revit UI 선택을 바꾸지는 않는다.
- `uidoc.Selection.SetElementIds()`를 써야 Revit 화면의 선택 상태가 바뀐다.
- 링크 모델 요소는 별도 링크 문서 접근이 필요하다.
- 대형 모델은 현재 뷰, 레벨, 시스템명 등으로 먼저 필터링한다.

방화구획을 MEP가 관통할 때 어떻게 처리하는지 물을 때:
- 배관은 방화 슬리브(Steel Sleeve) + 내화채움재(Firestop)로 처리한다.
- 덕트는 방화댐퍼(Fire Damper)를 방화구획 경계에 설치한다.
- 소방 배관은 방화댐퍼 없이 관통 가능하지만 방화 슬리브와 내화채움재는 필요하다.
- 케이블·전선관은 내화채움재로 처리한다.

천장 안에 MEP가 너무 많아서 천장고가 낮아질 때 어떻게 하는지 물을 때:
- 먼저 우선순위가 낮은 설비(말단 배관, 전선관)를 조정한다.
- 덕트 단면을 넓이보다 높이를 줄이는 방향으로 재검토한다.
- 구조 보를 관통하는 방안을 구조팀과 협의한다.

- 최신 승인된 건축 도면과 모델이 기준이다.
- 건축 링크 모델 개정 이력을 확인하고 최신 버전으로 업데이트한다.
- 차이가 있으면 건축 담당자에게 확인하고 MEP 모델을 수정한다.
