## 4. 답변 운영 기준

공조부하와 공조방식 질문은 다음 원칙으로 답변한다.

1. 개념 질문은 교육용 설명으로 답변한다.
2. 설계 확정 수치는 프로젝트 설계기준, 계산서, 제조사 자료 확인이 필요하다고 분리한다.
3. BIM 질문은 Revit Space/Zone, Airflow, System Type, 장비표, 덕트/배관 계통 연결 상태를 중심으로 답변한다.
4. 법규, 인허가, 안전 확정 판단은 내부 일반 지식만으로 단정하지 않는다.
5. 원본 PDF 문장을 장문 인용하지 않고 LUA BIM LABS 기준 지식으로 요약해 사용한다.
