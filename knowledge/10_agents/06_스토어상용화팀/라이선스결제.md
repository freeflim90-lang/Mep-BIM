## 환불·구독 취소(해지) 정책 (고객 안내 기준)
- Tags: refund,cancellation,환불,취소,해지,policy

고객이 "환불 되나요 / 환불 정책 / 구독 취소·해지" 를 물으면 아래 기준으로 안내한다.
- **직접 판매 채널(Paddle/Stripe)**: 구매 후 **30일 무조건 환불** 보장(무사용·사유 불문). 신뢰도 확보가 목적이므로 별도 증빙을 요구하지 않는다.
- **Autodesk App Store 구매**: 스토어 정책상 **14일 이내 미사용 제품 환불** 가능. 환불 신청·심사는 Autodesk 스토어 정책을 따른다(당사가 직접 처리 불가, 스토어 경로 안내).
- **연간 구독 중도 해지**: 30일 환불 기간 경과 후에는 **잔여 기간 월할(pro-rated) 환불** 정책을 약관에 명기해 분쟁을 예방한다.
- **구독 취소(다음 결제 중지)**: 언제든 가능하며, 취소 시 현재 결제주기 만료일까지는 이용 가능하고 자동 갱신만 중지된다(즉시 환불과 구분).
- 처리 자동화: 환불은 Paddle Dashboard API / Stripe Refunds API 로 처리하며, 자세한 사항은 고객지원(CS)으로 연결한다.

- **직접 판매 채널(Paddle/Stripe)**: 결제 완료 시 결제 영수증(인보이스)이 등록 이메일로 자동 발송된다. Paddle은 Merchant of Record로서 결제·인보이스를 직접 발행하므로, 영수증·VAT 인보이스는 Paddle/Stripe 고객 포털 또는 결제 확인 메일에서 재발급·다운로드할 수 있다.
- **세금계산서(한국 사업자)**: 사업자 고객은 사업자등록번호·이메일을 고객지원(CS)에 전달하면 세금계산서 발행을 안내한다. (직접 판매 채널은 Merchant of Record 인보이스가 기본이며, 한국 전자세금계산서가 별도로 필요한 경우 CS 확인 후 처리.)
- **Autodesk App Store 구매**: 결제·영수증 발행은 Autodesk가 직접 처리하므로 영수증·세금 서류는 Autodesk 계정의 구매 내역에서 확인·발급한다(당사 직접 발행 불가, 스토어 경로 안내).
- 재발급·정정 요청은 고객지원(CS)으로 연결한다.

- Paddle vs Stripe 2025 비교: Paddle은 Merchant of Record(MoR) 방식으로 한국 법인 없이 글로벌 VAT·GST를 자동 처리하며, 수수료는 거래당 5% + $0.50 또는 월정액 플랜 기준이다. Stripe는 한국 법인 필요 없이 Stripe Tax 기능으로 세금 신고를 자동화하며, 수수료는 1.5%(국내 카드) ~ 3.9%(해외 카드)다. 소규모 글로벌 SaaS에는 Paddle이 운영 부담이 적다.
- Autodesk App Store 결제 수수료 구조: Autodesk는 App Store 내 판매 시 수익의 30%를 플랫폼 수수료로 징수한다. 직접 결제(Paddle/Stripe) 연동 시 수수료 절감 가능하나, App Store 외부 유도는 정책 위반이므로 상품 유형별 채널 전략이 필요하다.
- 구독 갱신 자동화 파이프라인: Paddle Webhooks → Python 3.12 FastAPI 수신 → PostgreSQL 라이선스 DB 업데이트 → 자동 이메일 발송(SendGrid API v3) 순으로 구성한다. 결제 실패 시 Dunning 로직으로 3일·7일·14일 재시도 후 라이선스 일시 정지 처리를 자동화한다.
- 환불 정책 표준화: Autodesk App Store 정책상 14일 이내 미사용 제품 환불 가능. 직접 판매 채널에서는 30일 무조건 환불 정책을 적용하여 고객 신뢰도를 높인다. 환불 처리는 Paddle Dashboard API 또는 Stripe Refunds API로 자동화한다.
- 연간/월간 구독 전환 전략: 월간 구독 고객 대상 연간 전환 캠페인 시 20% 할인 제공이 업계 표준이며, LTV(생애 고객 가치) 기준으로 연간 구독 고객이 월간 대비 평균 2.3배 높다. Paddle의 `subscription.update` API로 플랜 업그레이드를 코드 3줄로 처리 가능하다.

### 결제 실패 패턴 4가지

**1. 법인 카드 결제 불가 → 구독 중단**
- 원인: 개인 카드만 지원, 법인 신용카드 거절
- 해결: 세금계산서 발행 + 계좌이체 옵션 병행 제공

**2. Autodesk App Store 결제 수수료 (30%) 미반영 가격 책정**
- 원인: 앱스토어 수수료 30% 고려 없이 수익 계산
- 해결: 목표 수익 × 1.43 = 앱스토어 가격 (역산 공식)

**3. 연간 구독 환불 정책 미명시 → 분쟁**
- 해결: 30일 환불 보장 + 이후 월할 환불 정책 약관 명기

**4. 비정형 기능 체험판 없음 → 구매 전환율 저조**
- 해결: 14일 체험판 (비정형 기능 포함), 샘플 곡면 모델 동봉
