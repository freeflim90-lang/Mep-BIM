NFSC 203 자동화재탐지설비 화재안전기준:
- 열감지기(차동식 스포트형): 설치 간격 9~70㎡/개 (구역 면적에 따라)

**자동화재탐지설비 구성요소 BIM 모델링:**
| 구성요소 | 역할 | BIM LOD 300 요건 |
|---------|------|----------------|
| 감지기 (Detector) | 열·연기 감지 → 신호 발신 | 위치·타입·보호면적 |
| 수신기 (Receiver) | P형/R형 — 감지 신호 수집·표시 | 방재실 위치·함체 크기 |
| 중계기 (Relay) | R형 시스템 — 현장 신호 취합 | 층별 설치 위치 |
| 발신기 (Transmitter) | 수동 경보 버튼 (빨간 버튼) | 복도·계단 벽면 위치 |
| 경종·사이렌 | 경보음 발령 | 층별 설치 위치 |
| 시각경보기 (스트로브) | 청각 장애인 경보 | 객실·화장실 위치 |

**P형 vs R형 수신기 차이점 (AI 즉시 답변):**
```
P형 수신기: 감지기·발신기 배선을 내부에 직접 결선 → 소규모 건물
R형 수신기: 중계기(Relay) 통해 디지털 신호로 취합 → 대형 건물·스마트빌딩
2026년 트렌드: R형 수신기 + BMS 통합 → 화재 발생 위치 BIM 3D 시각화
```

**소방전기 BIM 간섭검토 체크리스트:**
- [ ] 감지기 보호면적: 타입별 기준 면적 초과 여부 확인
- [ ] 방화셔터 연동: 감지기 → 방화셔터 제어 신호 경로 BIM 확인
- [ ] 제연 연동: 감지기 → 제연팬 ON, 제연댐퍼 개방 연동 경로
- [ ] 배선 내화 구조: 내화전선(FR) 또는 내화케이블 사용 구간 표시
- [ ] 비상전원: UPS 또는 예비전원 연결 경로 BIM 확인
- [ ] 복도 감지기 간격: 보행거리 30m 이하마다 1개 이상

**소방전기 BIM LOD 요건:**
| LOD | 감지기 | 수신기/중계기 | 배선 |
|-----|--------|------------|------|
| 200 | 구역별 감지 계통 | 방재실 위치 | 간선 경로 |
| 300 | 위치·타입·보호면적 | 함체 치수 | 층별 배선 경로 |
| 350 | 천장 높이·설치 위치 상세 | 내부 결선 계통 | 배관·내화 표시 |

## 초기 기준
감지기 배치, 방재 신호, 방화셔터/제연 연동, 배선 경로와 유지관리 접근성을 검토한다.

## AI 즉시 답변 패턴 — 감지기·스프링클러·방재 연동 천장 조율 (2026-06-13)
- Source: LUA BIM LABS Telegram QA review
- Tags: fire-electrical,detector,sprinkler,fire-shutter,smoke-control,interlock,bim

감지기와 스프링클러 헤드가 천장에서 가까울 때는 소방전기와 소방기계를 같이 본다.

빠른 체크:
- 감지기: 보, 덕트, 디퓨저 기류, 조명, 천장 장식물이 감지 성능을 방해하지 않는지 확인한다.
- 스프링클러 헤드: 감지기나 조명기구가 살수 방향을 막지 않는지 확인한다.
- 천장 배치: 조명, 감지기, 스프링클러 헤드, 디퓨저, 점검구가 같은 모듈 안에서 겹치지 않게 조율한다.
- 회로/구역: 감지기 회로 구역과 스프링클러 알람밸브 구역, 방화구획이 불일치하지 않는지 확인한다.
- 책임 구분: 감지기 위치와 방재 신호는 소방전기, 헤드 살수 장애와 소방배관은 소방기계가 주관한다.

방화셔터와 제연팬 연동 BIM 확인:
1. 감지기 또는 수신기 신호가 방화셔터, 제연팬, 제연댐퍼로 이어지는 시나리오를 확인한다.
2. 방화셔터 전용 감지기, 셔터 하강 범위, 피난 동선 침해 여부를 함께 본다.
3. 제연팬 기동, 방화댐퍼 폐쇄, 제연댐퍼 개방 신호가 방재반/수신기 포인트에 잡혀 있는지 확인한다.
4. 내화배선, 비상전원, 수신기·중계기 위치와 케이블 경로가 끊기지 않았는지 본다.
5. 소방 연동은 일반 BAS 제어보다 NFTC/NFPC와 인허가 도면이 우선이다.

## 감지기 배치 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: fire-electrical,detector,code

차동식 스포트형: 4m 미만 천장 바닥면적 50㎡/개, 4m 이상 45㎡/개.
연기 감지기(광전식): 복도·통로 보행 15m마다 1개, 계단·경사로는 수직거리 15m마다.
감지기 설치 제외: 천장 높이 20m 이상, 욕실·샤워실, 목욕탕.
Add-in은 평면 감지기 커버리지를 계산하고 미감지 구역, 설치 제외 구역 침입, 최적 추가 위치를 보고서로 제공한다.

## 방재 연동 및 신호 경로 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: fire-electrical,interlock,signal

수신기 → 발신기 → 경보장치 → 방화셔터/제연팬 연동은 간선 전용 내화 케이블(FR-8) 사용.

Add-in은 방재 신호 경로의 단선 가능성 구간(장거리 단일 경로)과 방화셔터 전용 감지기 부재를 경고 항목으로 분류한다.

## 소방전기 적용 근거문서 및 법규 기준 (2026-05-22)
- Source: LUA BIM LABS regulatory reference baseline
- Tags: fire-electrical,regulation,standard,legal-basis

### 국내 법령
- 소방시설 설치 및 관리에 관한 법률 (법률 제18522호, 2022.12.1 시행): 소방전기설비 설치 의무 및 성능 기준 위임
- 소방시설 설치 및 관리에 관한 법률 시행령 [별표 4]: 건물 용도·규모별 경보·피난 설비 설치 의무 목록
- 전기사업법 + 한국전기설비규정(KEC): 소방전기 배선 시공 기준 (내화배선 포함)
- 건축법 시행령 제46조: 방화구획과 감지기·방화셔터 연동 기준

### 화재안전기술기준 (NFTC) — 소방전기 계통
- NFTC 202 (구 NFSC 202) 비상방송설비의 화재안전기술기준: 스피커 배치, 음압 기준, 내화배선 요구
- NFTC 203 (구 NFSC 203) 자동화재탐지설비 및 시각경보장치의 화재안전기술기준: 감지기 종류·배치 기준 전부
- NFTC 204 (구 NFSC 204) 자동화재속보설비의 화재안전기술기준: 소방관서 자동 통보 기준
- NFTC 205 (구 NFSC 205) 누전경보기의 화재안전기술기준
- NFTC 301 (구 NFSC 301) 피난기구의 화재안전기술기준
- NFTC 303 (구 NFSC 303) 유도등 및 유도표지의 화재안전기술기준: 피난구·통로 유도등 설치 위치·크기·조도 기준
- NFTC 304 (구 NFSC 304) 비상조명등의 화재안전기술기준: 바닥 조도 1lx 이상, 예비전원 30분
- NFTC 504 (구 NFSC 504) 비상콘센트설비의 화재안전기술기준: 소방대 전용 비상콘센트 설치 기준
- NFTC 505 (구 NFSC 505) 무선통신보조설비의 화재안전기술기준: 지하 구역 소방 무선통신 커버리지 기준

### 내화·내열 배선 기준 (소방청 고시)
- 소방시설의 내화배선·내열배선 기준: 사용 가능 전선 종류, 공사 방법 (금속관·2종 금속제 가요전선관·내화 케이블)
- 내화 케이블 적용 대상: 수신기→감지기·발신기, 수신기→소방펌프·제연팬·방화셔터 신호선 (30분 내화 이상)

### KS 규격
- KS C IEC 60331-1/2 내화 케이블: 전기적 완전성 시험 (380V, 30분)
- KS C IEC 60332-1 케이블 수직 연소 시험
- KS C 4507 광전식 연기감지기
- KS C 4503 차동식 스포트형 감지기

### BIM/설계 실무 적용 시 우선 확인 법규
1. 감지기 종류별 배치 기준 → NFTC 203 [별표] (천장 높이·구역 면적별 기준)
2. 유도등 상시 점등 의무 → NFTC 303 (소방관서장 승인 없이 차단 불가)
3. 내화배선 공사 방법 → 소방청 고시 내화배선 기준 (공사 방법 3종 중 선택)
4. 방화셔터 연동 감지기 전용 회로 → NFTC 203 + 방화셔터 안전기준 (국토교통부 고시)
5. 비상방송 음압 기준 → NFTC 202 (각 층 임의 장소 1m 기준 65dB 이상 또는 배경소음 +10dB)

## 2026-06-04 KEC와 NFTC/NFPC 경계 기준
- Source: `knowledge/40_curation/updates/daily/2026-06-04_LUA_BIM_LABS_KEC_ELECTRICAL_STANDARD_UPDATE.md`
- Tags: fire-electrical,KEC,NFTC,NFPC,official-source,review-gate

소방전기 BIM 검토는 KEC와 전기설비기술기준만으로 확정하지 않는다. 화재 감지, 비상방송, 유도등, 비상조명, 내화·내열 배선, 방재 연동은 NFTC/NFPC와 소방청 고시를 우선 확인하고, 전기 배선·접지·전원 품질·시공 기준은 KEC 및 KDS/KCS 32와 함께 대조한다.

운영 기준:
- 감지기·유도등·비상방송·비상조명은 NFTC/NFPC 우선 확인
- 전원, 배선, 접지, 전기적 안전은 KEC와 KDS/KCS 32 함께 확인
- 수신기, 제연팬, 소방펌프, 방화셔터 연동 경로는 소방전기와 일반 전기 기준 충돌 여부 표시
- 자동 수집된 조항번호 또는 수치 문장은 공식 전문 확인 전 납품 기준으로 사용하지 않는다.

## 2026-06-04 NFPC/NFTC 공식 기준 게이트
- Source: `knowledge/40_curation/updates/daily/2026-06-04_LUA_BIM_LABS_NFTC_NFPC_FIRE_STANDARD_UPDATE.md`
- Tags: fire-electrical,NFPC,NFTC,official-source,review-gate

소방전기 BIM 검토는 NFPC 성능기준과 NFTC 기술기준을 분리해 확인한다. 자동화재탐지, 시각경보, 비상방송, 유도등, 비상조명, 비상콘센트, 무선통신보조설비, 내화·내열 배선은 최신 행정규칙 전문과 관할 소방서 해석을 기준으로 확인한다.

운영 기준:
- 적용 NFPC/NFTC 코드와 공고 번호, 시행일 확인
- 구 NFSC 인용 문장은 최신 NFPC/NFTC 대응 기준 확인
- 자동 수집된 감지기, 유도등, 비상방송, 내화배선 수치는 공식 전문 확인 전 납품 기준으로 사용하지 않음
- 감지기 사각, 방재 연동 누락, 내화배선 경로, 비상방송 커버리지는 일반 MEP clash와 분리해 소방전기 검수 시트로 관리
- KEC/KDS 32와 충돌 가능성이 있는 전원·배선·접지 항목은 전기 설계자 확인을 추가한다.

## 감지기 종류별 적용 기준 상세 (2026-05-23)
- Source: LUA BIM LABS internal fire-electrical BIM QA baseline
- Tags: fire-electrical,detector,type,placement,bim
- Links: [[소방기계]], [[설비자동제어]], [[설비시공조율]]

감지기 종류와 적용 공간:

차동식 스포트형 (열감지기):
- 온도 상승 속도를 감지. 주위 온도가 급격히 오를 때 동작.
- 적용: 사무실, 복도, 창고 등 일반 구역.
- 설치 기준: 천장 4m 미만 → 바닥면적 50㎡당 1개, 4m 이상 → 45㎡당 1개 (NFTC 203).
- 부적합: 온도 변화가 느린 무풍 환경에서 초기 감지 늦을 수 있음.

정온식 스포트형 (열감지기):
- 일정 온도 이상에서 동작 (60℃, 70℃, 80℃ 등 등급 선택).
- 적용: 주방, 보일러실, 건조실처럼 평상시에도 온도가 높은 구역.
- 차동식 감지기가 오동작하는 고온 환경에서 유리.

광전식 연기감지기 (이온식 포함):
- 연기 입자에 의한 광선 산란 또는 이온전류 변화 감지.
- 적용: 계단실, 복도, 통로, 전산실, 창고, 거실. 국내 11층 이상 또는 특정 용도 건물에 설치 의무.
- 설치 기준 (NFTC 203): 복도·통로 보행 15m마다 1개, 계단·경사로 수직거리 15m마다 1개.
- 부적합: 요리 연기, 담배 연기, 수증기 많은 곳 (오동작 위험) → 화장실, 주방 직근 제외.

불꽃 감지기:
- 적외선 또는 자외선으로 불꽃 발생 즉시 감지.
- 적용: 항공기 격납고, 위험물 저장소, 천장 높은 대형 공간 (열·연기 감지기가 늦는 환경).
- 사각지대: 가시선(Line-of-Sight) 방식이므로 장애물이 있으면 미감지.

감지기 설치 제외 구역 (NFTC 203):
- 천장 높이 20m 이상 구역 (대신 불꽃감지기 검토).
- 욕실, 샤워실, 목욕탕 (습도로 오동작 위험).
- 냉장·냉동 창고 (결빙으로 동작 불가).

BIM 확인 항목:
- 감지기 평면 배치: 바닥 면적 기준으로 커버리지 확인.
- 계통(회로) 구분: 회로별로 구역을 나눠 어느 구역에서 감지됐는지 수신기에서 파악 가능해야 함.
- 천장 구조물 간섭: 보, 덕트, 배관이 감지기 앞을 가리지 않는지 확인.
- 내화배선 연결 확인: 감지기 배선은 내화(FR) 케이블 사용 여부 확인.

## 소방전기 방재 연동 시나리오 (2026-05-23)
- Source: LUA BIM LABS internal fire-electrical controls baseline
- Tags: fire-electrical,interlock,scenario,fire-alarm,smoke-control
- Links: [[소방기계]], [[설비자동제어]], [[공조덕트]]

화재 신호 발생부터 방재 동작까지 일반적인 연동 흐름:

1단계 — 화재 감지:
- 감지기 또는 발신기가 화재 신호를 수신기로 전송.
- 수신기에서 회선 위치 확인 (어느 구역, 어느 층).

2단계 — 경보:
- 경보벨·사이렌 동작 (해당 층 및 전 층 또는 인접 층).
- 비상방송 자동 방송: “X층에서 화재가 발생하였습니다. 대피하십시오.”
- 시각경보장치(청각장애인 대상 구역): 점멸등 동시 동작.

3단계 — 방재 연동:
- 제연팬 기동: 피난 계단실 가압팬 ON, 화재실 배연팬 ON.
- 방화댐퍼 폐쇄: 해당 방화구획 덕트의 방화댐퍼 자동 폐쇄.
- 방화셔터 하강: 방화구획 경계 셔터 하강 (셔터 전용 감지기 신호로 작동).
- 엘리베이터 피난 운전: 전 엘리베이터 1층으로 귀환 후 운전 정지.
- 소방펌프: 스프링클러 알람밸브 유수 → 압력 스위치 → 소방펌프 자동 기동.
- 가스계 소화(해당 구역): 해당 방호구역 내 감지 확인 → 30초 지연 경보 → 소화약제 방출.

4단계 — 방재반 통보:
- 수신기 → 자동화재속보설비 → 소방관서 자동 통보 (NFTC 204).
- 방재실/종합방재실에서 화면으로 상황 모니터링.

BIM 검토 시 방재 연동 확인:
- 방화구획 경계의 덕트에 방화댐퍼가 설치되어 있는지 확인.
- 제연팬 위치와 제연 구역이 제연 설비 도면과 일치하는지 확인.
- 소방전기 내화배선 경로가 방화구획을 올바르게 관통하는지 확인.
- 발신기 세트(발신기, 경보벨, 표시등) 위치가 보행 거리 25m 이내에 있는지 확인.

## 소방전기 시방서 핵심 기준 (2026-05-26)
- Source: KCS 한국건설기준, 국내 표준 시방서 2026-05-26
- Tags: fire-electrical,specification,construction-spec

### 자동화재탐지설비 설치 기준
- 차동식 스포트형: 천장 4m 미만 → 50㎡당 1개, 4m 이상 → 45㎡당 1개 (NFTC 203).
- 정온식 스포트형: 주방·보일러실 등 고온 구역. 천장 4m 미만 → 50㎡당 1개.
- 광전식 연기감지기: 복도·통로 보행거리 15m마다 1개, 계단·경사로 수직거리 15m마다 1개.
- 감지기 상호 간 거리: 차동식·정온식 1종 9m 이하, 2종 10m 이하 (NFTC 203 별표 기준).
- 감지기 설치 제외: 천장 높이 20m 이상, 욕실·샤워실·목욕탕, 냉장·냉동 창고.
- 설치 높이: 감지기 하단이 천장면에서 0.3m 이내 (NFTC 203).

### 비상경보설비 배선 기준
- 내화 배선 적용 대상: 수신기 → 감지기·발신기, 수신기 → 소방펌프·제연팬·방화셔터 신호선.
- 내화 케이블: KS C IEC 60331 기준 380V, 30분 이상 기능 유지. FR-8 또는 이와 동등 이상.
- 내화 배선 공사 방법(소방청 고시 3종 중 선택):
  1. 금속관 공사 (두께 1.2mm 이상 금속관)
  2. 2종 금속제 가요전선관 공사
  3. 합성수지관(두께 2.3mm 이상) + 내화 케이블 사용
- 내열 배선 적용 대상: 비상방송·유도등·비상조명 배선. 380V, 15분 이상 내열 성능.
- 배선 관통부: 방화구획 관통 시 내화 충전재로 밀봉.

### 비상조명등·유도등 설치 기준
- 비상조명등: 바닥면 조도 1lx 이상. 예비전원 방전 완료까지 30분 이상 지속 (NFTC 304).
- 피난구 유도등: 피난구 상단 또는 출입구 상단 0.15m 이내. 상시 점등.
- 통로 유도등: 복도·통로 보행거리 20m마다 1개. 바닥 설치 가능. 조도 1lx 이상.
- 예비전원 용량: 20분 이상 (피난구 유도등 대형), 30분 이상 (비상조명등). 내장 배터리 또는 중앙 집중식.
- 표시 크기: 피난구 유도등 대형(400mm×250mm 이상), 중형(350mm×200mm 이상).

### 소방 전원 공급 기준
- 전용 회로: 소방설비 전원은 일반 설비와 별도의 전용 분기 회로 구성. 분기 차단기 전용 표시.
- 비상 전원 절환 시간: 정전 후 40초 이내 비상 전원(발전기)으로 자동 절환. 소방펌프·제연팬·비상조명 포함.
- 자동 절환 스위치(ATS): 상용 전원 이상 감지 후 자동으로 발전기 전원으로 절환. 역전류 방지 기능 필수.
- 비상 전원 용량: 소방 설비 전체 부하의 1.1배 이상으로 산정. 최소 20분(비상조명), 30분(소방펌프·제연팬) 운전 가능.
- 방재반 전원: 무정전전원장치(UPS) 연결 권장.

### 소방전기 BIM 모델 요구사항
- LOD 기준: 실시설계 단계 LOD 300 이상. 감지기·수신기·발신기 위치, 배선 경로 포함.
- 감지기 파라미터: 감지기 종류(차동식/정온식/연기식), 설치 공간 면적 커버리지, 회로 번호, 설치 높이.
- 수신기·발신기 파라미터: 회선 수, 구역 번호, 설치 높이, 전원 용량.
- 유도등·비상조명 파라미터: 등기구 종류, 예비전원 용량(분), 조도(lx), 설치 높이.
- 배선 파라미터: 배선 종류(내화/내열/일반), 전선 규격, 관로 경로, 회로 번호.
- 방재 연동 검토: Revit/Navisworks에서 방화셔터·제연팬 위치와 감지기 회로 연동 매핑 확인.

## 소방전기 감지기·내화배선 심화 기준 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28

- NFTC 203(자동화재탐지설비 화재안전기술기준) 감지기 설치 기준: 차동식 스포트형 천장 4m 미만 50㎡당 1개, 4m 이상 45㎡당 1개. 광전식 연기감지기 복도·통로 보행 15m마다 1개, 계단·경사로 수직거리 15m마다 1개. 감지기 하단이 천장면에서 0.3m 이내 위치.
- 내화 배선 기준(소방청 고시): KS C IEC 60331 기준 380V, 30분 이상 기능 유지 FR-8 케이블 사용. 수신기→소방펌프·제연팬·방화셔터 신호선 전 구간 내화배선 적용. 공사 방법: 금속관(두께 1.2mm 이상), 2종 금속제 가요전선관, 합성수지관(2.3mm 이상)+내화케이블 3종 중 선택.
- 비상방송(NFTC 202): 각 층 임의 장소 1m 기준 65dB 이상 또는 배경소음 +10dB 이상. 스피커 1개당 수평 거리 25m 이하 커버. 비상방송 배선은 내열케이블(FR-3) 이상.
- 비상전원: 소방설비 전원 정전 후 40초 이내 자동 절환(ATS). 비상조명 예비전원 30분(NFTC 304), 소방펌프·제연팬 30분 이상 운전 용량 비상전원 확보. 방재반 UPS 연결 권장.
- Revit BIM LOD 300 이상: 감지기 파라미터(종류, 면적 커버리지, 회로 번호, 설치 높이), 배선 파라미터(내화/내열/일반 구분, 전선 규격, 회로 번호) 필수 입력. Navisworks에서 방화셔터·제연팬 위치와 감지기 회로 연동 매핑 확인 필수.

## 소방전기 마스터급 경험 지식 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: fire-electrical,detector,R-type,emergency-broadcast,NFTC202,field-experience

감지기 오동작 Top 3 원인과 대처: ① 먼지 축적 — 광전식 연기감지기의 광학 챔버에 누적된 분진이 산란광을 발생시켜 화재 신호를 모의한다. 점검 주기(반기 1회)를 지키고, 창고·공장처럼 분진이 많은 환경은 방진 캡을 갖춘 광전식 선택이 필수다. ② 수증기 — 주방 인근·욕조 주변에 광전식 감지기를 설치한 경우 수증기가 챔버에 유입되어 오동작을 유발한다. NFTC 203 적용 제외 구역 확인 후 이온식 또는 정온식으로 변경한다. ③ 담배 연기 — 비흡연 구역으로 지정되지 않은 공간에서 흡연이 상시 발생하는 경우 광전식 1종 감지기는 오동작 민감도가 높다. 2종 감지기 또는 공기 흡입형(Air Sampling) 감지기로 교체를 검토한다.

R형 수신기 Loop 회로 단선 시 구간 격리 절차: R형 수신기는 어드레스 방식으로 단선 구간의 감지기 어드레스를 자동으로 화면에 표시한다. 점검 시 ① 수신기 화면에서 단선 구간 어드레스 확인 → ② 단선 예상 구간 양단 감지기 베이스 단자 전압 측정(정상 24VDC) → ③ 단선 구간 감지기 회로를 수동으로 격리(Isolator 모듈 조작) → ④ 격리 상태에서 나머지 회로 정상 여부 확인 → ⑤ 단선 구간 배선 교체 후 격리 해제 및 재점검 순서로 처리한다. Isolator 모듈이 없는 구형 P형 수신기에서는 회선 단위로 전체 단선 영향이 생기므로 R형으로의 교체를 검토한다.

비상방송설비 스피커 음압 부족 클레임 대응: NFTC 202 기준은 각 층 임의 장소 1m 기준 65dB 이상(또는 배경소음 +10dB) 이상이다. 현장 음압 측정 결과 미달 시 ① 스피커 입력 전압 확인(정격 4~8W 입력 대비 실측값 비교) → ② 스피커와 앰프 간 배선 저항으로 인한 전력 손실 계산 → ③ 스피커 배치 간격(수평 거리 25m 이하 기준) 재검토 → ④ 배경소음이 높은 구역(기계실 인근 복도)에는 스피커 추가 설치 순서로 대응한다. 배선 길이가 길면 트랜스포머 결선 탭을 높은 입력 전압 탭으로 변경하는 방법도 효과적이다.
