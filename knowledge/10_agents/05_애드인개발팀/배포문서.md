### 비정형 건물 BIM 기능 릴리스 시 배포 문서 특이사항

**비정형 기능 릴리스 노트 필수 포함 항목:**
1. 지원 비정형 수준 명시 (경사/복합경사/자유곡면/전면비정형)
2. 지원 Revit 버전 (비정형 API는 버전별 차이 큼)
3. Dynamo/Grasshopper 연동 버전 요구사항
4. 알려진 제한사항 (특정 곡면 유형에서 오류 발생 등)
5. 마이그레이션 노트 (이전 버전 비정형 프로젝트 호환성)

**릴리스 노트 템플릿:**
```markdown
## v{버전} 릴리스 노트 ({날짜})

### 신규 기능
- [비정형] 자유 곡면 면적 산출 (오차 ±2% 이내)
- [비정형] Grasshopper → Revit 패널 분할 자동화

### 지원 환경
- Revit 2022/2023/2024/2025
- Dynamo 2.17+, Rhino 8+
- 비정형 수준: 경사/복합경사/자유곡면

### 알려진 제한사항
- 완전 폐합곡면(예: 구형 지붕) IFC 수출 시 면 분할 필요

### 마이그레이션
- v{이전버전}에서 업그레이드 시 비정형 패밀리 재로드 필요
```

### 배포 실패 패턴 3가지

**1. 릴리스 노트 없는 배포 → CS 문의 폭발**
- 해결: 릴리스 노트 없으면 배포 금지 원칙

**2. Revit 버전 미기재 → 호환성 오류 CS**
- 해결: 배포 전 체크리스트: Revit 2022/2023/2024/2025 각각 스모크 테스트

**3. 비정형 제한사항 미공개 → 고객 신뢰 손상**
- 해결: 알려진 제한사항은 솔직하게 공개, 다음 버전 로드맵 함께 제시
