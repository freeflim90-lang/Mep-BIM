## 1. 시설 개요 및 BIM 적용 특성

주차장·모빌리티시설은 차량 동선, 램프 경사, 주차 구획, 환기·배연, 전기차 충전, 보행자 안전이 핵심이다. BIM은 자동차 궤적과 구조·설비 간섭을 동시에 확인해야 하며, 운영 단계에서는 주차 관제·EV 충전 자산과 연결된다.

| 구분 | 일반 건물 부속주차 | 독립 모빌리티 시설 |
|---|---|---|
| 목적 | 법정 주차 대수 충족 | 교통 허브, 환승, 충전, 정비 |
| 동선 | 단순 입출차 | 차량·보행자·자전거·배송 동선 분리 |
| 설비 | 기본 조명·환기 | CO 감지, 배연, EV 충전, 관제 |
| 구조 | 일반 슬래브 | 램프, 장스팬, 차량 하중, 진동 |
| 운영 | 수동 관리 | LPR, 주차유도, 충전 과금 |

## 2. BIM 필수 파라미터 목록

### 2.1 주차·램프 파라미터

```
Pset_ParkingMobility
  - Parking_Stall_ID: 주차면 번호
  - Stall_Type: Standard / Compact / Accessible / EV / Loading / Bicycle
  - Stall_Width: 주차면 폭 (mm)
  - Stall_Length: 주차면 길이 (mm)
  - Drive_Aisle_Width: 차로 폭 (mm)
  - Ramp_Slope: 램프 경사 (%)
  - Ramp_Turning_Radius: 램프 회전 반경 (mm)
  - Clear_Height: 유효 높이 (mm)
  - Vehicle_SweptPath_OK: 차량 궤적 검토 통과 여부
  - Pedestrian_Route_ID: 보행자 동선 ID
```

## 6. 실패 사례 Top 5

1. 주차 대수는 맞지만 차량 궤적 검토가 없어 램프 회전부 이용 불가.
2. 유효 높이에 덕트·스프링클러·표지판이 포함되지 않아 대형 차량 충돌.
3. EV 충전기 위치만 표시하고 전력 간선·분전반 용량이 누락됨.
4. 보행자 동선과 차량 동선이 출입구에서 교차해 안전 리스크 발생.
5. 자연환기 가정 후 지하층 CO·배연 검토가 부족해 기계 환기 추가 공사.

## 주차단위구획·차로·기둥 간격(bay) 기준 (2026-06-21)
- Source: 주차장법 시행규칙 제3조(주차단위구획)·제6조(노외주차장 차로) — curated baseline. 정확한 적용은 최신 법령·지자체 조례로 확인한다.
- Tags: parking,unit-stall,driveway,column-grid,bay,dimension

주차단위구획(주차 1대 최소 치수, 2019 개정 기준):
| 구분 | 너비 × 길이 |
|------|------------|
| 경형 | 2.0m × 3.6m |
| 일반형 | 2.5m × 5.0m |
| 확장형 | 2.6m × 5.2m |
| 장애인전용 | 3.3m × 5.0m |
| 평행주차(일반) | 2.0m × 6.0m |
| 평행주차(경형) | 1.7m × 4.5m |

차로 너비(노외주차장, 주차형식별): 직각주차 6.0m, 60도 대향 5.0m, 45도 대향 3.5m, 평행 3.5m(단방향/양방향·교차통행 여부로 상이).

기둥 간격(bay spacing) — **법 규정이 아니라 설계 관행**:
- 주차장법은 기둥 간격을 직접 규정하지 않는다. 주차단위구획 치수와, 기둥이 주차·통행을 방해하지 않는 배치로 결정된다.
- 통상 일반형 2.5m 기준 주차 3대 + 여유로 한 베이(기둥~기둥)를 약 8.0~8.4m(흔히 **8.1m**)로 잡는다. 직각주차 양측 배치 시 차로 6.0m 포함 구조 모듈은 약 15~16m 전후가 일반적이다.
- 기둥은 주차구획 경계선 위나 차량 전면/후면 데드스페이스에 두어 문 개폐·주차 폭(2.5m) 침범을 피한다.
- BIM 적용: 기둥 그리드를 주차단위구획·차로와 함께 배치해 기둥이 구획 폭(2.5m)·차로(6.0m)·장애인구획(3.3m)을 침범하지 않는지 간섭·규정 검토한다.

주차장 경사로(램프) 구배 — 주차장법 시행규칙(노외주차장):
- 출입구 경사로 종단구배: **직선 부분 1/6(약 17%) 이하, 곡선 부분 1/7(약 14%) 이하**.
- 경사 시작·끝에는 종단곡선(완화구간)을 두어 차량 하부 긁힘을 방지한다.
- 경사로 폭·높이(차량 한계높이), 회전반경, 미끄럼방지 마감을 함께 검토한다.
- BIM 적용: 램프 구배(직선 17%/곡선 14%)·종단곡선·유효높이를 모델 파라미터로 관리하고, 천장 설비(덕트·배관·스프링클러)가 차량 한계높이를 침범하지 않는지 간섭 검토한다. (배수 구배(오배수 1/100 등)는 위생 담당 — 경사로 종단구배와 구분.)

## 2026-06-06 EV 충전 의무설치·지하주차장 화재안전 긴급 업데이트
- Source: 대한민국 정책브리핑, 친환경자동차법 시행령, 한국전기차충전서비스(KECA), 소방청 2026
- Tags: ev-charging,parking,fire-safety,mandatory-installation,2026

**전기차 충전시설 의무설치 비율 확대 (2026 기준):**
| 대상 | 의무 설치 비율 | 설치 기한 |
|------|-------------|---------|
| 신축 공동주택 | 주차면수 **5% 이상** | 건축허가 시 반영 |
| 기축 공동주택 (아파트) | **2% 이상** | **2026년 1월 27일**까지 |
| 공공시설 (국가·지자체) | 법 시행 후 1년 내 | 완료 |
| 공중이용시설 | 법 시행 후 2년 내 | 완료 |

**충전기 종류 확대 (2026):**
- 기존: 급속·완속 충전기만 의무 인정
- 변경: **콘센트형 충전기**까지 인정 → 비용 절감으로 의무 이행 용이

**BIM 설계 반영 사항 (EV 충전 급증 대응):**
- 분전반 용량 계획: 주차면수 5% → EV 충전기 용량 합산 → 수변전 설비 재검토 필요
  - 7kW 완속 충전기 기준: 100면 주차장 → 5대 × 7kW = 35kW 추가 부하
  - 50kW 급속 충전기 혼합 시: 분전반·전력 간선 증가 폭 더 큼
- BIM 파라미터 `EV_Charger_Capacity_kW`와 `Circuit_Breaker_Capacity_A` 함께 검토
- 충전 구역 BIM 표시: AC 완속 / DC 급속 / 콘센트형으로 색상 구분

**지하주차장 전기차 화재 안전 강화 (2026 소방 기준 강화):**
- **지하주차장 습식 스프링클러 설치 의무화**: 기존 건식→습식 전환 요구 (지하 EV 화재 대응)
- 화재 감지 시스템 연동 필수: EV 충전 구역 → 조기 경보 시스템
- BIM 체크리스트 추가 항목:
  - [ ] EV 충전 구역 스프링클러 밀도 강화 (습식 기준)
  - [ ] 충전 구역 CO 감지기 + 열감지기 이중 설치
  - [ ] 전기차 화재 대응 배수로(소화수 배수) BIM 반영
  - [ ] 충전기 전기안전관리자 월 1회 점검 기록 → FM COBie 연동
