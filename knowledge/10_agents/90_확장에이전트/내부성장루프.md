## 내부성장루프 실행 메커니즘 심화 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: growth-loop,automation,knowledge-flywheel,kpi,obsidian

**성장 플라이휠(Flywheel) 구조:**
```
[실무 경험·프로젝트] → KB 지식화 → AI 에이전트 품질 향상
        ↑                                         ↓
[고객 문의·피드백]  ←── 제품 품질 향상 ←── 자동화 도구 생산성 증가
```
한 바퀴 돌 때마다 지식의 밀도와 자동화 커버리지가 동시에 증가.

**6개 축별 주간 목표 (2026년 기준):**
| 축 | 주간 목표 | 측정 방법 |
|---|---|---|
| 교육 | 신규 커리큘럼 섹션 1개 | 러닝콘텐츠 파일 업데이트 수 |
| 자동화 | Add-in 기능 1개 개선/추가 | GitHub Commits |
| 전략 | AX 신호 승격 결정 1건+ | AX_전략승격리뷰 항목 |
| 제품화 | App Store 리뷰 응대 전건 | 미응대 리뷰 0건 |
| 품질룰 | KB 품질 점수 +0.5점 이상 | mqa_obsidian_tools 리포트 |
| 지식공백 | 빈약 파일 1개 이상 보강 | 파일 크기 3KB 이상 달성 |

**자동화 파이프라인 상태 점검 (매일 아침):**
- LaunchAgent 상태: `launchctl list | grep luabimlab`
- 어젯밤 실행 로그: `tail -50 logs/daily_knowledge_update.launchd.out.log`
- 오류 확인: `tail -20 logs/daily_knowledge_update.launchd.err.log`
- KB 최신 파일 5개: `ls -lt knowledge/10_agents/90_확장에이전트/*.md | head -5`

**성장 정체 감지 기준:**
- 2주 연속 KB 파일 갱신 0건 → 자동화 파이프라인 점검
- 고아 노드 10개 이상 누적 → 위키링크 보강 우선 작업
- Obsidian 그래프 엣지 증가율 월 5% 미만 → 지식 연결 강화 스프린트
- 관련: [[AX_전략승격리뷰]] · [[인프라_DevOpsObsidian]] · [[지식업데이트]] · [[지식큐레이터]]
