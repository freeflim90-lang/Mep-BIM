**APScheduler 5.x 설정 예시:**
```python
from apscheduler.schedulers.asyncio import AsyncIOScheduler
from apscheduler.triggers.cron import CronTrigger

scheduler = AsyncIOScheduler()
# 매일 자정 KB 업데이트
scheduler.add_job(run_kb_update, CronTrigger(hour=0, minute=0))
# 매시 신호 수집
scheduler.add_job(collect_signals, CronTrigger(minute=0))
scheduler.start()
```
