**AI 즉시 답변 패턴 — "MEP 배관이 구조와 충돌할 때 어떻게 조율하나요?"**
```
MEP-구조 충돌 조율 프로세스:
1. Navisworks Clash Detective로 클래시 자동 감지
2. Hard Clash vs Soft Clash 분류:
   - Hard: 실제 겹침 → 반드시 해결
   - Soft: 이격 불충분 → 유지보수 고려
3. 관통 불가 부재 확인 (기둥·전이보):
   - 기둥: 절대 관통 불가 → 배관 우회 필수
   - 보: 구조기술사 확인 후 LOD 조건 맞으면 가능
4. RFI 발행: BCF 파일로 설계자에게 위치·크기 공식 요청
5. 승인 후 BIM 수정: 경로 변경 → 재검토
```

## AI 즉시 답변 패턴 — 천장 속 설비 배치 순서 (2026-06-13)
- Source: LUA BIM LABS Telegram QA review
- Tags: ceiling,mep-coordination,duct,pipe,cable-tray,kds,kec,nftc

천장 속 설비 배치 순서는 구조 보 하단, 구배 배관, 소방, 제연, 주덕트, 전기 트레이, 통신 트레이, 말단 기구 순서로 잡는다. 프로젝트 BEP, KDS 31, KCS, KEC, NFTC/NFPC, 발주처 특기시방서가 우선이다.

빠른 답변:
1. 구조 보 하단과 천장 마감고를 먼저 확정한다.
2. 오배수·우수 같은 중력배관은 구배가 필요하므로 레벨을 먼저 확보한다.
3. 소방 배관과 스프링클러 헤드는 살수 장애가 생기지 않게 둔다.
4. 제연 덕트는 소방법 우선 설비라 경로 변경을 최소화한다.
5. 주덕트 SA/RA는 단면이 커서 천장고 영향이 크므로 조기 확정한다.
6. 전기 트레이는 포설·교체 공간 200~300mm 내외, 분전반 전면 1,000mm 내외 작업 공간을 확인한다.
7. 통신 트레이는 강전과 이격하거나 격벽을 적용하고, 말단 플렉시블과 소구경 배관은 마지막에 조정한다.

검토 수치 예시:
- 밸브·댐퍼·필터 점검 공간: 전면 600mm 내외를 기준으로 설계도서 확인.
- 강전·약전 트레이 이격: 150~300mm 또는 금속 격벽 적용 여부 확인.
- 방화구획 관통부: 슬리브와 내화채움, 방화댐퍼 점검구 위치를 함께 확인.
- 천장고 영향: 덕트 하단고, 배관 관저고, 트레이 하단고, 마감 천장고를 표로 비교한다.

## 설비 시공 조율 기본 원칙 (2026-05-22)
- Source: LUA BIM LABS internal MEP coordination baseline
- Tags: mep,coordination,construction,bim,clash
- Links: [[설비기초]], [[설비도면해석]], [[설비장비]], [[공조덕트]], [[공조배관]], [[위생]], [[소방기계]], [[전기]], [[통신]]

설비 시공 조율은 “누가 먼저 지나가느냐”가 아니라 “기능, 법규, 유지관리, 시공 가능성”을 동시에 만족시키는 과정이다.

일반 우선순위:
1. 구조체와 방화구획: 보, 슬래브, 기둥, 내력벽, 방화구획은 임의 변경이 어렵다.
2. 피난/방재 계통: 제연, 소방 배관, 방재 신호, 비상전원은 안전 기능이 우선이다.
3. 중력배관: 오배수, 우수, 통기처럼 구배가 필요한 배관은 레벨 조정 여지가 작다.
4. 대구경 덕트: 풍량과 정압, 천장고 영향이 커서 조기 확정이 필요하다.
5. 주배관/주트레이: 계통 간선은 경로 변경 영향이 크다.
6. 소구경 배관/전선관/말단 배선: 상대적으로 조정 가능성이 높다.

조율 판단 기준:
- 성능 유지: 덕트 단면 축소, 배관 급격한 우회, 트레이 과밀은 성능 저하를 만든다.
- 유지관리: 밸브, 댐퍼, 점검구, 필터, 패널 앞 공간은 물리 충돌이 없어도 침해하면 문제다.
- 시공성: 행거 설치, 용접/나사 접합, 보온 작업, 방화채움, 장비 반입이 가능한지 본다.
- 안전성: 누수, 감전, 화재, 질식, 고온, 고압, 회전체 접근 위험을 분리한다.
- 승인성: 법규 또는 인허가 관련 항목은 담당 설계자와 감리 확인을 거친다.


## 천장 속 설비 조율 기준 (2026-05-22)
- Source: LUA BIM LABS internal MEP coordination baseline
- Tags: ceiling,coordination,duct,pipe,cable-tray
- Links: [[공조덕트]], [[공조배관]], [[전기]], [[통신]]

천장 속은 설비 충돌이 가장 많이 발생하는 구간이다.

확인 항목:
- 구조 보 하단고, 슬래브 하단고, 천장 마감고.
- 덕트 하단고, 배관 관저고, 트레이 하단고.
- 단열 포함 외경, 플랜지, 행거, 방진 장치, 점검구.
- 디퓨저, 스프링클러 헤드, 조명, 감지기, 점검구의 천장 배치.

배치 원칙:
- 대구경 덕트는 가능한 직선으로 배치하고, 분기 덕트는 말단에서 조정한다.
- 오배수 배관은 구배를 유지해야 하므로 하단 레벨을 먼저 확보한다.
- 소방 헤드는 살수 장애가 생기지 않도록 덕트와 보 하부 관계를 검토한다.
- 전기 트레이는 점검과 케이블 포설을 위해 상부 여유와 접근성을 확보한다.
- 약전 트레이는 강전 트레이와 이격하거나 격벽을 사용한다.

보고서 문장 예시:
- “해당 구간은 물리 충돌보다 유지관리 공간 침해 가능성이 큽니다.”
- “오배수 구배가 필요한 구간이므로 덕트 우회보다 배관 레벨 보존을 우선 검토해야 합니다.”
- “트레이와 냉매배관 사이 이격 부족은 간섭보다 유지관리/EMI/결로 리스크로 분류하는 것이 적절합니다.”


## 샤프트·기계실·전기실 조율 기준 (2026-05-22)
- Source: LUA BIM LABS internal MEP coordination baseline
- Tags: shaft,machine-room,electrical-room,coordination
- Links: [[설비장비]], [[전기]], [[통신]], [[소방기계]]

샤프트:
- 수직 배관, 덕트, 전기 간선이 집중되므로 층별 관통 위치와 방화처리가 핵심이다.
- 오수, 우수, 통기, 소방, 급수, 냉온수, 전기 트레이가 혼재할 때 유지관리 가능한 배열을 확보한다.
- 방화구획 관통부는 슬리브와 내화채움 계획을 함께 검토한다.

기계실:
- 장비 반입, 교체, 점검, 배수, 환기, 소음, 방진을 동시에 본다.
- 펌프와 배관은 밸브 조작, 스트레이너 청소, 플랜지 해체 공간을 확보한다.
- 냉동기와 보일러는 제조사 유지관리 공간과 배관 접속 방향 확인이 필수다.

전기실/통신실:
- 누수 배관 직상부를 피하고, 전면 작업 공간과 문 개폐 공간을 확보한다.
- 통신실은 냉방, 습도, 케이블 인입, 랙 전면/후면 작업 공간을 검토한다.
- 전기실에 위생 배관이 관통하는 경우 원칙적으로 회피하고, 불가피하면 방수·누수 감지·보호 계획을 확인한다.

## 설비시공조율 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: mep,coordination,clash-detection,bim,navisworks,revit,construction

**BIM 기반 간섭 검토 프로세스 (ISO 19650-2):**
- **Step 1 — 모델 통합**: Navisworks .nwc 파일 링크 (건축+구조+MEP 각 공종)
- **Step 2 — Clash Detective 설정**: Hard Clash (0mm tolerance), Clearance Clash (+50mm MEP·+150mm 전기)
- **Step 3 — 간섭 분류**: Critical(즉시 해결)/Major(2주 내)/Minor(시공 전 해결) 3등급
- **Step 4 — RFI 발행**: Navisworks Clash → BIM 360 RFI 자동 연동 (Forge Webhook 활용)
- **Step 5 — 해결 검증**: 수정 모델 재클래시 → 0 Hard Clash 확인 후 공종 서명 승인

**공종별 시공 순서 (간섭 조율 우선순위):**
- 1. 구조체 완료 확인 (슬래브·보 타설)
- 2. 소방배관 선행 (NFTC 기준 경로 고정)
- 3. 위생 오배수 중력배관 (구배 우선)
- 4. 주배관 (냉온수·급수 DN100 이상)
- 5. 주덕트 (SA/RA 600mm 이상)
- 6. 전기 케이블 트레이·스틸 배관
- 7. 통신 트레이·소구경 배관·플렉시블

**현장 BIM 코디네이션 회의 운영:**
- 주 1회 정기 BIM 조율회의 (공종별 담당자 + BIM 매니저)
- 클래시 현황 대시보드: 총 간섭 건수/해결 건수/잔여 건수 주간 업데이트
- 마감 2주 전: LOD 400 모델 동결, 잔여 Minor 간섭 현장 처리 결정
- 관련: [[설비도면해석]] · [[설비기초]] · [[Navisworks_Addin]] · [[BIM_지침서]]


## 설비시공조율 마스터급 경험 지식 (2026-05-29)
- Source: claude-code-enhanced 2026-05-29
- Tags: mep,coordination,clash-detection,field-experience,navisworks,revit,bim-360

**현장 BIM 조율 실패 상위 5가지 패턴:**
1. **NWC 버전 불일치**: 건축팀이 Revit 2024로 NWC를 내보내고 MEP팀이 Revit 2022 NWC를 사용할 때 Navisworks에서 좌표계가 어긋나거나 객체 색상이 오염됨. 해결: 프로젝트 BIM 실행계획(BEP)에 NWC 내보내기 Revit 버전 단일화 명시
2. **Clash 규칙 미설정으로 잡음 발생**: "All vs All" 클래시 테스트로 실행 시 같은 계통 내부 충돌(소방 배관 vs 소방 배관), 설계 의도 겹침(행거 와 행거 앰블런스)이 수천 건씩 발생해 실제 문제를 덮음. 반드시 Rule 탭에서 `Same System`·`Same Layer`·허용 공차 50mm 설정 후 실행
3. **클래시 해결 재검증 누락**: "해결됨" 처리 후 수정 모델을 재클래시하지 않아 잔여 충돌이 시공 현장에서 발견됨. 수정 건 재검증을 클래시 담당자가 아닌 제3자(BIM 매니저)가 확인하는 더블체크 루틴 필요
4. **공종 우선순위 미합의 상태에서 조율**: 소방·위생·공조 배관이 같은 경로에서 경합할 때 우선순위가 없으면 각 공종이 자기 배관을 고수 → 조율 교착. BEP에 우선순위 명문화 필수: 소방 > 위생 중력배관 > 공조 > 전기 트레이
5. **LOD 수준 불일치 모델 혼재**: 건축 LOD 350, 구조 LOD 200, MEP LOD 300 모델을 합칠 때 구조체 위치가 개략적이어서 MEP 조율 후 구조 상세가 나오면 충돌 재발. LOD 기준 합의 시기를 클래시 검토 착수 전에 고정

**Navisworks Add-in 자동화로 조율 효율화 가능한 영역:**
- 클래시 결과 → Excel 자동 생성: `ClashTest.ResultData.ClashResults` 순회 → 공종 조합별(소방vs구조, MEP vs건축) 시트 분리 + 위치 스냅샷 자동 삽입
- SearchSet 자동화: 공종별 카테고리(Fire Protection / Pipe / Duct / Cable Tray) 기반 SearchSet을 스크립트로 생성 → 클래시 테스트 Selection A/B 자동 할당
