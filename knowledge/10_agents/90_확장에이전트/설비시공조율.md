MEP-구조 충돌 조율 프로세스:
1. Navisworks Clash Detective로 클래시 자동 감지
2. Hard Clash vs Soft Clash 분류:
   - Hard: 실제 겹침 → 반드시 해결
   - Soft: 이격 불충분 → 유지보수 고려
3. 관통 불가 부재 확인 (기둥·전이보):
   - 기둥: 절대 관통 불가 → 배관 우회 필수

4. RFI 발행: BCF 파일로 설계자에게 위치·크기 공식 요청
5. 승인 후 BIM 수정: 경로 변경 → 재검토

## AI 즉시 답변 패턴 — 천장 속 설비 배치 순서 (2026-06-13)
- Source: LUA BIM LABS Telegram QA review
- Tags: ceiling,mep-coordination,duct,pipe,cable-tray,kds,kec,nftc

천장 속 설비 배치 순서는 구조 보 하단, 구배 배관, 소방, 제연, 주덕트, 전기 트레이, 통신 트레이, 말단 기구 순서로 잡는다. 프로젝트 BEP, KDS 31, KCS, KEC, NFTC/NFPC, 발주처 특기시방서가 우선이다. 조건상 천장고가 부족하면 오배수 구배, 스프링클러 살수 장애, 제연 덕트 법규, 주덕트 풍량, 전기 트레이 포설공간 순서로 가능/불가를 나누고, 300mm 이격, 600mm 점검공간, 1m 작업공간, 24시간 조율 회신, 7일 재검토 로그를 함께 남긴다.
다만 천장고 부족이 구조 보, 전이보, 방화구획, 전기실 누수 리스크와 겹치면 일반 높이 순서보다 안전·법규 우선순위를 먼저 적용한다. 반면 말단 플렉시블, 소구경 배관, 통신 트레이처럼 조정 가능한 공종은 예외적으로 후순위 이동 대상으로 두고, 담당자·기한·대안 2개를 BCF/RFI에 기록한다.
검토 표에는 각 공종의 하단고, 관저고, 단열 포함 외경, 행거 공간, 점검구 방향, 천장 마감고를 같은 레벨 기준으로 기록한다. 24시간 안에 충돌 구간 스냅샷과 우선순위 사유를 공유하고, 7일 안에 승인/보류/재검토 상태를 닫아야 조율회의 결정으로 쓸 수 있다. 승인 전 임의 이동은 소방 성능, 배수 구배, 전기 포설공간, 유지관리 책임 리스크를 만든다.
결정 로그에는 우선 보호 공종, 양보 가능한 공종, 이동 거리 mm, 조정 후 천장고, 담당자, 다음 회의일을 남긴다. 같은 문제가 2회 이상 반복되면 BEP 표준 우선순위와 Navisworks Search Set 규칙으로 승격해 다음 프로젝트에서도 같은 판단을 재사용한다.

빠른 답변:
1. 구조 보 하단과 천장 마감고를 먼저 확정한다.
2. 오배수·우수 같은 중력배관은 구배가 필요하므로 레벨을 먼저 확보한다.
3. 소방 배관과 스프링클러 헤드는 살수 장애가 생기지 않게 둔다.
4. 제연 덕트는 소방법 우선 설비라 경로 변경을 최소화한다.
5. 주덕트 SA/RA는 단면이 커서 천장고 영향이 크므로 조기 확정한다.
6. 전기 트레이는 포설·교체 공간 200~300mm 내외, 분전반 전면 1,000mm 내외 작업 공간을 확인한다.
7. 통신 트레이는 강전과 이격하거나 격벽을 적용하고, 말단 플렉시블과 소구경 배관은 마지막에 조정한다.

검토 수치 예시:
- 밸브·댐퍼·필터 점검 공간: 전면 600mm 내외를 기준으로 설계도서 확인.
- 강전·약전 트레이 이격: 150~300mm 또는 금속 격벽 적용 여부 확인.
- 방화구획 관통부: 슬리브와 내화채움, 방화댐퍼 점검구 위치를 함께 확인.
- 천장고 영향: 덕트 하단고, 배관 관저고, 트레이 하단고, 마감 천장고를 표로 비교한다.

## 설비 시공 조율 기본 원칙 (2026-05-22)
- Source: LUA BIM LABS internal MEP coordination baseline
- Tags: mep,coordination,construction,bim,clash
- Links: [[설비기초]], [[설비도면해석]], [[설비장비]], [[공조덕트]], [[공조배관]], [[위생]], [[소방기계]], [[전기]], [[통신]]

설비 시공 조율은 “누가 먼저 지나가느냐”가 아니라 “기능, 법규, 유지관리, 시공 가능성”을 동시에 만족시키는 과정이다.

일반 우선순위:
1. 구조체와 방화구획: 보, 슬래브, 기둥, 내력벽, 방화구획은 임의 변경이 어렵다.
2. 피난/방재 계통: 제연, 소방 배관, 방재 신호, 비상전원은 안전 기능이 우선이다.
3. 중력배관: 오배수, 우수, 통기처럼 구배가 필요한 배관은 레벨 조정 여지가 작다.
4. 대구경 덕트: 풍량과 정압, 천장고 영향이 커서 조기 확정이 필요하다.
5. 주배관/주트레이: 계통 간선은 경로 변경 영향이 크다.
6. 소구경 배관/전선관/말단 배선: 상대적으로 조정 가능성이 높다.

조율 판단 기준:
- 성능 유지: 덕트 단면 축소, 배관 급격한 우회, 트레이 과밀은 성능 저하를 만든다.
- 유지관리: 밸브, 댐퍼, 점검구, 필터, 패널 앞 공간은 물리 충돌이 없어도 침해하면 문제다.
- 시공성: 행거 설치, 용접/나사 접합, 보온 작업, 방화채움, 장비 반입이 가능한지 본다.
- 안전성: 누수, 감전, 화재, 질식, 고온, 고압, 회전체 접근 위험을 분리한다.
- 승인성: 법규 또는 인허가 관련 항목은 담당 설계자와 감리 확인을 거친다.


## 천장 속 설비 조율 기준 (2026-05-22)
- Source: LUA BIM LABS internal MEP coordination baseline
- Tags: ceiling,coordination,duct,pipe,cable-tray
- Links: [[공조덕트]], [[공조배관]], [[전기]], [[통신]]

천장 속은 설비 충돌이 가장 많이 발생하는 구간이다.

확인 항목:
- 구조 보 하단고, 슬래브 하단고, 천장 마감고.
- 덕트 하단고, 배관 관저고, 트레이 하단고.
- 단열 포함 외경, 플랜지, 행거, 방진 장치, 점검구.
- 디퓨저, 스프링클러 헤드, 조명, 감지기, 점검구의 천장 배치.

배치 원칙:
- 대구경 덕트는 가능한 직선으로 배치하고, 분기 덕트는 말단에서 조정한다.
- 오배수 배관은 구배를 유지해야 하므로 하단 레벨을 먼저 확보한다.
- 소방 헤드는 살수 장애가 생기지 않도록 덕트와 보 하부 관계를 검토한다.
- 전기 트레이는 점검과 케이블 포설을 위해 상부 여유와 접근성을 확보한다.
- 약전 트레이는 강전 트레이와 이격하거나 격벽을 사용한다.

보고서 문장 예시:
- “해당 구간은 물리 충돌보다 유지관리 공간 침해 가능성이 큽니다.”
- “오배수 구배가 필요한 구간이므로 덕트 우회보다 배관 레벨 보존을 우선 검토해야 합니다.”
- “트레이와 냉매배관 사이 이격 부족은 간섭보다 유지관리/EMI/결로 리스크로 분류하는 것이 적절합니다.”


## 샤프트·기계실·전기실 조율 기준 (2026-05-22)
- Source: LUA BIM LABS internal MEP coordination baseline
- Tags: shaft,machine-room,electrical-room,coordination
- Links: [[설비장비]], [[전기]], [[통신]], [[소방기계]]

샤프트:
- 수직 배관, 덕트, 전기 간선이 집중되므로 층별 관통 위치와 방화처리가 핵심이다.
- 오수, 우수, 통기, 소방, 급수, 냉온수, 전기 트레이가 혼재할 때 유지관리 가능한 배열을 확보한다.
- 방화구획 관통부는 슬리브와 내화채움 계획을 함께 검토한다.

기계실:
- 장비 반입, 교체, 점검, 배수, 환기, 소음, 방진을 동시에 본다.
- 펌프와 배관은 밸브 조작, 스트레이너 청소, 플랜지 해체 공간을 확보한다.
- 냉동기와 보일러는 제조사 유지관리 공간과 배관 접속 방향 확인이 필수다.

전기실/통신실:
- 누수 배관 직상부를 피하고, 전면 작업 공간과 문 개폐 공간을 확보한다.
- 통신실은 냉방, 습도, 케이블 인입, 랙 전면/후면 작업 공간을 검토한다.
- 전기실에 위생 배관이 관통하는 경우 원칙적으로 회피하고, 불가피하면 방수·누수 감지·보호 계획을 확인한다.

## 설비시공조율 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: mep,coordination,clash-detection,bim,navisworks,revit,construction

**BIM 기반 간섭 검토 프로세스 (ISO 19650-2):**
- **Step 1 — 모델 통합**: Navisworks .nwc 파일 링크 (건축+구조+MEP 각 공종)
- **Step 2 — Clash Detective 설정**: Hard Clash (0mm tolerance), Clearance Clash (+50mm MEP·+150mm 전기)
- **Step 3 — 간섭 분류**: Critical(즉시 해결)/Major(2주 내)/Minor(시공 전 해결) 3등급
- **Step 4 — RFI 발행**: Navisworks Clash → BIM 360 RFI 자동 연동 (Forge Webhook 활용)
- **Step 5 — 해결 검증**: 수정 모델 재클래시 → 0 Hard Clash 확인 후 공종 서명 승인

**공종별 시공 순서 (간섭 조율 우선순위):**
- 1. 구조체 완료 확인 (슬래브·보 타설)
- 2. 소방배관 선행 (NFTC 기준 경로 고정)
- 3. 위생 오배수 중력배관 (구배 우선)
- 4. 주배관 (냉온수·급수 DN100 이상)
- 5. 주덕트 (SA/RA 600mm 이상)
- 6. 전기 케이블 트레이·스틸 배관
- 7. 통신 트레이·소구경 배관·플렉시블

**현장 BIM 코디네이션 회의 운영:**
- 주 1회 정기 BIM 조율회의 (공종별 담당자 + BIM 매니저)
- 클래시 현황 대시보드: 총 간섭 건수/해결 건수/잔여 건수 주간 업데이트
- 마감 2주 전: LOD 400 모델 동결, 잔여 Minor 간섭 현장 처리 결정
- 관련: [[설비도면해석]] · [[설비기초]] · [[Navisworks_Addin]] · [[BIM_지침서]]


## 설비시공조율 마스터급 경험 지식 (2026-05-29)
- Source: claude-code-enhanced 2026-05-29
- Tags: mep,coordination,clash-detection,field-experience,navisworks,revit,bim-360

**현장 BIM 조율 실패 상위 5가지 패턴:**
1. **NWC 버전 불일치**: 건축팀이 Revit 2024로 NWC를 내보내고 MEP팀이 Revit 2022 NWC를 사용할 때 Navisworks에서 좌표계가 어긋나거나 객체 색상이 오염됨. 해결: 프로젝트 BIM 실행계획(BEP)에 NWC 내보내기 Revit 버전 단일화 명시
2. **Clash 규칙 미설정으로 잡음 발생**: "All vs All" 클래시 테스트로 실행 시 같은 계통 내부 충돌(소방 배관 vs 소방 배관), 설계 의도 겹침(행거 와 행거 앰블런스)이 수천 건씩 발생해 실제 문제를 덮음. 반드시 Rule 탭에서 `Same System`·`Same Layer`·허용 공차 50mm 설정 후 실행
3. **클래시 해결 재검증 누락**: "해결됨" 처리 후 수정 모델을 재클래시하지 않아 잔여 충돌이 시공 현장에서 발견됨. 수정 건 재검증을 클래시 담당자가 아닌 제3자(BIM 매니저)가 확인하는 더블체크 루틴 필요
4. **공종 우선순위 미합의 상태에서 조율**: 소방·위생·공조 배관이 같은 경로에서 경합할 때 우선순위가 없으면 각 공종이 자기 배관을 고수 → 조율 교착. BEP에 우선순위 명문화 필수: 소방 > 위생 중력배관 > 공조 > 전기 트레이
5. **LOD 수준 불일치 모델 혼재**: 건축 LOD 350, 구조 LOD 200, MEP LOD 300 모델을 합칠 때 구조체 위치가 개략적이어서 MEP 조율 후 구조 상세가 나오면 충돌 재발. LOD 기준 합의 시기를 클래시 검토 착수 전에 고정

**Navisworks Add-in 자동화로 조율 효율화 가능한 영역:**
- 클래시 결과 → Excel 자동 생성: `ClashTest.ResultData.ClashResults` 순회 → 공종 조합별(소방vs구조, MEP vs건축) 시트 분리 + 위치 스냅샷 자동 삽입
- SearchSet 자동화: 공종별 카테고리(Fire Protection / Pipe / Duct / Cable Tray) 기반 SearchSet을 스크립트로 생성 → 클래시 테스트 Selection A/B 자동 할당
- 주간 클래시 현황 대시보드: COM API로 Navisworks 인스턴스 실행 → 전체 테스트 실행 → 결과 카운트 → Telegram/이메일 자동 발송 (야간 CI 파이프라인)
- 관련: [[Navisworks_Addin]] · [[설비도면해석]] · [[소방기계]] · [[공조덕트]] · [[BIM_지침서]]


## MEP 공종 우선순위 전체 순서 — 현장 조율 기준 (2026-06-06)

**MEP 공종 우선순위 전체 순서 (높을수록 우선 보호):**
1. 제연 덕트 (소방법 최우선 — 변경 불가)
2. 소방 배관 (NFTC 기준 고정 경로, 이동 최소화)
3. 위생 오배수·우수 중력배관 (구배 유지 필수, 레이어 고정)
4. 주급수·냉온수 대구경 배관 (DN100 이상, 단열 포함 외경 기준)
5. 프로젝트 표준 예외 규칙: 예외는 "법규/안전 우선", "BEP 승인 필요", "발주처 승인", "현장 반입 순서", "24시간 임시조치/7일 재검토"로 문서화한다. 프로젝트 표준에는 제연·소방·오배수 우선순위와 예외 승인권자, BCF/RFI 기록 방식을 함께 넣는다. KDS/KCS, NFTC/NFPC, ISO 19650 기준을 연결하고, 예외 판단 예시는 150mm 전기/통신 이격, 300mm 이격, 600mm 점검구, 1m 작업공간, DN100 이상 대구경 배관처럼 수치로 남긴다.
5. 주덕트 SA/RA (600mm 이상, 천장고 최다 점유)
6. 전기 케이블 트레이 (강전 우선, 약전 하부)
7. 통신·약전 트레이 (최하위 — 경로 유연)

**적용 원칙:** 상위 공종이 하위 공종보다 경로 고정 우선권을 가짐. BEP(BIM 실행계획)에 반드시 명문화하고 공종별 담당자 서명 후 시행.
표준화할 때는 예외 코드, 승인권자, 대체 경로, 허용 이격, 다음 검토일을 함께 둔다. 조율회의 후에는 Search Set, Clash Rule, BCF 상태값을 업데이트하고, 같은 예외가 2회 이상 반복되면 프로젝트 표준 규칙으로 승격해 다음 회의에서 재논의하지 않는다.

## 기계실 장비 반입 경로 BIM 검토 방법 (2026-06-06)

**기계실 대형 장비 반입 경로 BIM 검토 체크리스트:**
- 장비 외형 치수(길이×폭×높이) 기준 이동 경로 BIM 시뮬레이션 — Revit 4D 또는 Navisworks TimeLiner 활용
- 기계실 출입문 순 유효 폭: 장비 폭 + 양쪽 여유 최소 300mm 이상 확보 — 반드시 사전 검토 필수
- 기계실 복도·램프: 장비 운반 차량 하중(최대 10t), 최소 회전 반경 4.5m 이상 고려
- 냉동기(200RT 이상) 반입 개구부 최소 2,000×2,500mm — 구조기술사 확인 필수
- 엘리베이터 화물칸 규격(최대 2,000×1,500mm) 초과 장비: 기계실 인접 외벽 임시 개구부 계획 수립
- 반입 경로 클리어런스 불가 경우: 반드시 설계 변경 또는 분할 반입 계획 수립
- KDS 31 00 00 기계설비 기준: 기계실 장비 유지관리 공간 및 반입 경로 확보 규정 준수
- Revit 장비 패밀리에 "반입 이동 박스(Moving Box)" 파라미터 추가 후 경로 클래시 테스트
- 조건상 반입구, 복도, 회전 반경, 임시 개구부 중 하나라도 불가하면 먼저 장비 분할 반입 가능 여부를 보고, 반면 구조 개구부가 필요한 경우에는 구조기술사와 발주처 승인 전 허용하지 않는다. 우선 24시간 내 3D Clear Path 스냅샷을 공유하고 7일 내 KCS 시공 순서, 크레인/리프트 계획, 선반입 후 배관 시공 여부를 확정한다.
- 검토 결과에는 장비 ID, 이동 박스 크기, 문 유효폭, 회전반경, 간섭 공종, 임시 철거 범위, 반입 순서를 표로 남긴다. 승인 후에는 장비 반입 전/후 배관·덕트 선시공 금지 구간을 모델 뷰와 BCF에 고정해 현장 재작업을 막는다.

## NWC 파일 Revit 버전 통일 이유 및 좌표계 정합 (2026-06-06)

**NWC 파일 내보낼 때 Revit 버전 통일이 필요한 이유:**
- Revit 버전이 다른 NWC 파일을 Navisworks에서 합칠 때 좌표계 불일치 발생 → 모델이 어긋남
- 예: 건축팀 Revit 2024 NWC + MEP팀 Revit 2022 NWC → Navisworks에서 좌표 오프셋 발생
- 해결: BEP에 NWC 내보내기 Revit 버전 단일화 명시 (예: 전 공종 Revit 2024 통일) — 반드시 프로젝트 시작 전 합의
- 버전 통일이 불가능한 경우: 공유 좌표(Shared Coordinates) 수동 조정 후 좌표 기준점 일치 확인 — 단계별 검증 필수
- 추가 주의: 공종별 좌표 기준이 동일해야 정합 유지 — 경우에 따라 Survey Point 재설정 필요
- KDS 기준 BIM 성과물 납품 시 공종 간 좌표 정합 확인서 제출 의무
- 버전 통일 후에도 내부 원점(Project Base Point, Survey Point) 확인 필수 — 좌표 기준점 불일치 방지
- 조건상 버전 통일이 불가하면 1. Base Point/Site 좌표, 2. 0,0,0 기준점, 3. 100m 이상 떨어진 기준 그리드, 4. 24시간 내 샘플 NWC 중첩, 5. 7일 내 전체 NWD 재검증 순서로 승인한다. BEP와 ISO 19650 협업 절차에 내보내기 버전, 단위, 좌표 책임자를 기록한다.
- 답변 첫머리에는 "버전 통일은 객체 변환, 좌표 정합, 속성 보존, 책임 추적을 동시에 맞추기 위한 BEP/ISO 19650 통제 항목"이라고 설명한다. 조건상 Revit 2022/2024가 섞이면 좌표 오프셋, 단위 변환, 링크 원점, Shared Coordinates, Project Base Point, Survey Point를 비교하고, KDS/KCS 납품 검수에서는 NWC/NWD 버전, 내보내기 옵션, 좌표 확인서, 담당자 서명을 증빙으로 요구한다.
- 발주처나 CM에게 설명할 때는 "버전 통일을 안 하면 1개 공종만 틀리는 문제가 아니라 전체 간섭 검토, 물량 산출, 책임 추적이 흔들린다"로 정리한다. Revit 2022/2023/2024 혼용, mm/m 단위 불일치, 내부원점·공유좌표 차이, NWC Exporter 옵션 차이를 24시간 내 샘플 3개 파일로 검증하고, 7일 내 BEP, NWD, 좌표 확인서, BCF 이슈 로그를 갱신하지 못하면 Critical 협업 리스크로 올린다.
- 샘플 검증 기준은 10mm 이하 좌표 오차, 100m 기준 그리드 일치, 1m 단위 스케일 오류 없음, 5mm 이하 레벨 차이, 0mm 내부원점 편차 확인으로 둔다. 이 기준을 넘으면 NWC를 다시 내보내고, 예외적으로 버전을 섞어야 할 때만 BEP에 공종별 Revit 버전과 Exporter 옵션을 승인 조건으로 기록한다.

## 천장 속 설비 배치 순서가 어떻게 되나요 — 높이 우선순위 배치 (2026-06-06)

천장 속 설비 배치 순서가 어떻게 되나요? 천장 속 설비 배치 순서를 높이별로 정리합니다.

**천장 속 설비 높이별 배치 순서 (구조 보 하단 ~ 마감 천장 사이):**
1. 구조 보 하단: 최상단 고정 기준선
2. 위생 오배수 배관: 구배 유지를 위해 최우선 레이어 확보 (구조 보 직하)
3. 소방 배관: 오배수 배관 하부, 스프링클러 헤드 위치 확보
4. 제연 덕트: 전용 레이어, 소방법 기준 경로 고정
5. 주덕트 SA/RA (600mm 이상)
6. 전기 케이블 트레이: 덕트 측면 또는 하부
7. 통신 트레이·약전 배관: 전기 트레이 하부 또는 측면
8. 소구경 배관·플렉시블 호스: 최하부 배치
9. 마감 천장: 최하단 기준선

**핵심 원칙**: 덕트·소방 배관은 구획 기준, 트레이는 측면 우선 — 천장고를 최대한 확보한다.

**천장 속 설비 배치 BIM 적용 주의사항:**
- 오배수 배관 구배 확보가 불가능한 경우: 경로 우회 또는 설계 변경 단계 검토 필수
- 제연 덕트 경로 변경이 불가한 경우: 소방 배관과 나머지 공종 경로를 단계적으로 조율
- 천장 속 설비 배치 간섭 발생 시 먼저 오배수 레벨 확보 → 소방 배관 경로 고정 → 덕트 경로 조율 순서
- 트레이 설치 가능 공간이 충분하지 않은 경우: 강전·약전 트레이 분리 허용 (최소 150mm 이격 유지)
- BIM 천장 속 설비 배치 순서를 KDS 31 00 00 기계설비 기준 및 설계도서 일치 여부 확인
- 점검 공간 기준 예시: 밸브·댐퍼·필터 전면 600mm 내외, 전기/통신 트레이 상부 포설 공간 200~300mm 내외, 분전반 전면 작업 공간 1,000mm 내외를 설계도서 기준으로 확인한다.
- 법규·기준 확인 축: KDS 31 기계설비, KCS 시공기준, KEC 전기설비, NFTC/NFPC 소방 기준, 발주처 BEP/특기시방서.

## MEP 공종 우선순위 순서를 알려주세요 — 전체 정리 (2026-06-06)

MEP 공종 우선순위 전체 순서를 알려주세요 — 현장 조율 기준 정리:
제연 덕트 > 소방 배관 > 위생 오배수 중력배관 > 대구경 급수·냉온수 배관 > 주덕트 > 전기 트레이 > 통신·약전 트레이

순서를 정하는 이유: 제연과 소방은 소방법 최우선, 오배수는 구배 고정, 덕트는 단면 크기, 전기·통신은 경로 유연.
천장 속 배치 순서: 구조 보 하단 → 오배수 배관 → 소방 배관 → 제연 덕트 → 주덕트 → 전기 트레이 → 통신 트레이 → 천장 마감
천장 속 설비 배치 순서가 중요한 이유: 천장 속에서 설비 순서를 잘못 정하면 유지관리 불가하거나 천장고를 초과함.
천장 높이 확보: 천장 속 설비 배치 순서를 BIM에서 높이순으로 조율해야 마감 천장고 확보 가능.
조건상 제연, 소방, 오배수, 방화구획, 전기실 안전 중 하나가 걸리면 우선순위를 바꾸지 않는다. 예외는 BEP, KDS/KCS, NFTC/NFPC, 24시간 BCF/RFI 회신, 7일 내 발주처 승인 기록이 있을 때만 허용하며, 점검구 600mm와 전기 트레이 200~300mm 작업 공간을 함께 확인한다.

NWC 버전 통일 중요성: Revit 버전이 달라지면 좌표 오프셋 발생 — 공종 간 모델이 어긋남.
기계실 장비 반입 경로: 장비 외형 치수 기준 반입 경로 BIM 시뮬레이션 필수, 기계실 출입문 폭 확보 필수.
