| AI 1차 자동 응대율 | 70% 이상 | 월간 |
| 재오픈율 (Reopen Rate) | 5% 이하 | 월간 |

**AI 1차 응대 자동화 설계:**
- 트리거: 신규 티켓 생성 즉시
- KB RAG 검색: 문의 내용 → ChromaDB 유사도 검색 → 상위 3개 FAQ 섹션
- 응답 생성: claude-sonnet-4-6 + KB 컨텍스트 → 한국어 초안
- 자동 발송 조건: RAG 유사도 점수 0.85 이상 AND KB 최근 업데이트 30일 이내
- 인간 검토 필요: 유사도 < 0.85 OR "환불"/"법적"/"소송" 키워드 포함

**월간 운영 리뷰 어젠다:**

### 운영 빈도 높은 FAQ 카테고리 우선순위 (BIM 특화)

**1순위 — 설치 및 Add-in 로딩 오류 (전체 문의의 약 40%)**
- Add-in 탭이 Revit 리본에 나타나지 않음 (가장 빈발)
- 다중 Revit 버전 설치 환경에서 특정 버전에만 Add-in 표시
- Anti-virus(Defender, V3, 알약)가 DLL을 격리한 경우
- Revit 2024 설치 후 기존 2023 Add-in이 작동 안 함

**2순위 — 라이선스 및 인증 오류 (전체 문의의 약 25%)**
- Entitlement API 연결 실패 (방화벽, 프록시 환경)
- 오프라인 캐시 만료 후 기능 잠금
- Autodesk ID 변경 후 라이선스 미인식
- 팀 라이선스(5-Pack)에서 사용자 추가 방법

**3순위 — 기능 오작동 (전체 문의의 약 20%)**
- 특정 Revit 버전에서만 기능 미작동
- 특정 프로젝트 파일에서만 오류 발생 (파일 특수성)
- MEP 자동 배치 결과가 의도와 다름
- 일람표 데이터 누락 또는 계산 오류

**4순위 — 결제 / 환불 / 계정 (전체 문의의 약 10%)**
- Autodesk App Store 환불 절차
- 구독 갱신 방법
- 청구서 발행 요청

**5순위 — 기능 요청 / 피드백 (전체 문의의 약 5%)**
- 신규 기능 제안
- 특정 Revit 버전 지원 요청
