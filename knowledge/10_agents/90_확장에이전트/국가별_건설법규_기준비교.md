### 한국 기업 해외 진출 시 BIM 법규 리스크 Top 4

**1. 싱가포르 IFC 자동 검증 실패**
- CORENET 시스템이 IFC 파일을 자동 검증 → 비정형 부재 IfcBuildingElementProxy → 자동 반려
- 해결: 수출 전 싱가포르 BCA IFC 가이드라인 준수, Entity 매핑 사전 검증

**2. 영국 BIM Level 2 COBie 형식 미준수**
- 영국 공공 발주는 BS 1192/PAS 1192 기반 COBie 제출 필수
- 한국 COBie Export(Revit 플러그인) vs 영국 COBie 2.4 구조 차이 → 검수 반려
- 해결: UK COBie Template 별도 적용, 현지 BIM Coordinator와 협업

**3. UAE 아랍어 문서 병행 요구**
- 두바이 DDA 제출 시 도면·시방서 아랍어 번역본 병행 제출 요구 사례
- BIM 모델 내 파라미터 언어 이슈 (한국어→영어→아랍어 3중 관리)

**4. 미국 주정부별 상이한 에너지 기준**
- 캘리포니아 CEC Title 24 vs 텍사스 IRC 에너지 조항 상이
- BIM 에너지 모델을 주정부 기준에 맞게 재설정 필요 → 추가 공수 10~15%

### Revit API — 국가별 법규 파라미터 자동 전환

```csharp
// 국가 코드에 따른 에너지 기준 파라미터 자동 설정
public void SetEnergyStandard(Document doc, string countryCode)
{
    var projectInfo = doc.ProjectInformation;
    var param = projectInfo.LookupParameter("에너지_기준");
    var standard = countryCode switch
    {
        "KR" => "한국 건축물에너지절약설계기준",
        "GB" => "UK Part L Building Regulations",
        "SG" => "Singapore SS 553 / BCA Green Mark",
        "US" => "ASHRAE 90.1-2022",
        "AE" => "Dubai Green Building Regulations",
        _ => "ISO 52000 (국제 기준)"
    };
    using var tx = new Transaction(doc, "Set Energy Standard");
    tx.Start();
    param?.Set(standard);
    tx.Commit();
}
```

## 2026-06-06 싱가포르 CORENET X IFC+SG 의무화·글로벌 BIM 인허가 최신 현황 보강
- Source: CORENET X 공식 사이트 (info.corenet.gov.sg), URA 원형 dc25-07 (2025.08), BCA 원형 APPBCA-2025-20, BIM Mandate 글로벌 보고서 2026
- Tags: CORENET-X,IFC+SG,싱가포르BIM,글로벌BIM인허가,2025,2026,permit

**싱가포르 CORENET X — IFC+SG 전자 인허가 (2025~2026 완전 의무화):**
- 기관: BCA(건설청) + URA(도시재개발청) + SCDF(소방) + LTA(육상교통) + PUB(수도) 멀티에이전시 통합 플랫폼
- **의무화 타임라인**:
  | 시기 | 적용 범위 |
  |------|---------|
  | 2025년 10월 1일 | GFA **30,000m² 이상** 신규 프로젝트 → CORENET X 의무 사용 시작 |
  | **2026년 10월 1일** | **모든 신규 프로젝트** (GFA 무관) → CORENET X 전면 의무화 |
- **IFC+SG 표준**: IFC4 국제 표준 + 싱가포르 로컬 파라미터 추가 확장
  - 싱가포르 규제 기관별 필수 파라미터를 IFC 속성으로 요구 (BCA·URA·SCDF 각각 다름)
  - 누락·오류 파라미터 → 서면 지시(Written Direction) → 공사 지연 직결
- **CORENET X Model Checker**: IFC+SG 규정 준수 자동 검증 도구 (IDS 방식과 유사)
  - 제출 전 자체 품질 검사(QC) → 이기종 심사 거절 방지
- **LUA BIM LABS 싱가포르 진출 핵심 요건**:
  ```
  Revit 모델 → IFC4 내보내기 → IFC+SG 파라미터 매핑
    → CORENET X Model Checker 사전 검증
    → CORENET X 포털 제출 → 멀티에이전시 동시 심사
  ```
  - IFC+SG 파라미터: 한국 국토부 BIM 파라미터와 구조는 유사하나 필드명·값 형식이 다름
  - 싱가포르 파트너사와 Qualified Person(QP) 확보 필수 (싱가포르 법적 서명 요건)

**글로벌 BIM 인허가 의무화 현황 업데이트 (2026 기준):**
| 국가 | 플랫폼/표준 | 2025 현황 | 2026 현황 | BIM 파일 형식 |
|------|----------|---------|---------|------------|
| 싱가포르 | CORENET X + IFC+SG | 30,000m²+ 의무 | **전면 의무** | IFC4+SG 확장 |
| 영국 | UK BIM Level 2 | ISO 19650 전면 적용 | Level 3 전환 진행 | IFC4 / COBie |

| 한국 | 국토부 BIM 시행지침 | 500억+ 공공 의무 | 철도 500억+ 추가 | IFC4 / 국토부 표준 |
