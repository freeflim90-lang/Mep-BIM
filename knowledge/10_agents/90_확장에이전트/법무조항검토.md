## 2026-06-05 외부 AI disclosure rule 초안
- Source: `knowledge/10_agents/conflict_resolution/EXTERNAL_AI_DISCLOSURE_RULE_DRAFT_20260605.md`
- Tags: legal,privacy,external-ai,disclosure,local-only,draft

CEO/법무/보안관 승인 전까지 고객 모델 원본, 고객 로그, 프로젝트명, 담당자 정보, 계약/가격 정보, 개인정보는 외부 AI에 전송하지 않는다. 개인정보 처리방침의 "마스킹 처리" 문구는 외부 AI 전송 허용이 아니라 보호조치 후보로 해석한다.

법무 검토 기준:
- 외부 AI 제공자명, 약관, 보관 기간, 삭제 가능 여부, 학습 사용 여부를 확인한다.
- 처리위탁, 제3자 제공, 국외 이전, 고객 동의 필요성을 구분한다.
- 고객에게 "마스킹하면 외부 AI 사용 가능"이라고 일반화하지 않는다.
- 승인 전 고객 응대 문구는 Local-only 원칙으로 통일한다.

## BIM 용역 계약 지식재산권 및 EULA 전략 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: 법무,계약,IP,EULA,BIM용역,소프트웨어라이선스

- BIM 용역 계약의 핵심 IP 조항: BIM 모델 저작권은 계약 체결 시 명시적으로 귀속 주체를 지정해야 한다. 공공 발주의 경우 국가계약법에 따라 발주처가 2차 활용 권한을 요구하는 경우가 많으므로, 모델 원본(Native File) 납품 의무와 사용 목적 제한 조항을 반드시 구분해 계약서에 포함해야 한다.
- Autodesk EULA 핵심 제약 사항: Revit 파일을 자동 생성·처리하는 Add-in은 Autodesk API 이용 약관 Section 4를 준수해야 하며, 생성된 데이터를 Autodesk 외부 서버로 전송할 경우 사전 동의 및 암호화 처리가 필수다. EULA 위반 시 App Store 제품 삭제 및 파트너십 취소 리스크가 있다.
- 소프트웨어 라이선스 유형별 법적 구분: 영구 라이선스(Perpetual)는 자산 처리, SaaS 구독은 비용 처리가 원칙이며 이는 발주처 예산 편성에 영향을 미친다. Add-in 판매 시 구독 모델 선택이 발주처 입장에서 예산 확보가 용이한 구조임을 영업 과정에서 강조해야 한다.
- 파견 인력 관련 BIM 계약: 「파견근로자보호법」상 건설 현장 BIM 전문가 파견은 허용 업종 해당 여부를 사전 검토해야 한다. BIM 컨설팅은 용역 계약, 상주 지원은 파견 계약으로 구분해 계약 형태를 결정하면 법적 리스크를 줄일 수 있다.
- 하자 책임 기간: 소프트웨어 용역 결과물의 하자 담보 기간은 통상 납품 후 1년이며, BIM 모델 오류로 인한 시공 피해 발생 시 손해배상 상한을 계약 금액의 10% 이내로 제한하는 조항을 반드시 삽입해야 한다.
