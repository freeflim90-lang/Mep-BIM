## 초기 기준
설계 시방서는 설계 의도를 재료·공법·품질 기준으로 구체화한 문서이며, BIM 파라미터와 1:1 연계되어야 한다.


## 설계 시방서 개요 및 체계 (2026-05-26)
- Source: KCS(한국건설기준) 표준 시방서, 국토교통부 설계 지침, LUA BIM LABS curated baseline 2026-05-26
- Tags: specification,design,standard,KCS

설계 시방서는 설계 결과물에 적용될 재료, 공법, 품질 기준을 규정한 기술 문서다.

표준 시방서 vs 특기 시방서 비교:

| 구분 | 표준 시방서 | 특기 시방서 |
|------|-------------|-------------|
| 정의 | 국가 또는 발주처가 제정한 일반 기준 | 해당 프로젝트 전용 보완·변경 기준 |
| 근거 | KCS(한국건설기준), 국토교통부 고시 | 설계사 작성, 발주처 승인 |
| 우선순위 | 하위 | 상위 (특기가 표준 대체·보완) |
| 적용 범위 | 전 공종 기본 기준 | 프로젝트 특수 조건, 특수 재료, 성능 향상 요구 |
| 예시 | KCS 14 20 10 콘크리트 표준 | 고강도 콘크리트 fck=40MPa 적용 특기 사항 |

적용 우선순위 원칙: 특기 시방서 → 표준 시방서 → 제조사 시방 → 관련 KS 기준.

KCS 체계 분류:
- KCS 10: 가설공사
- KCS 11: 토공사·기초공사
- KCS 14: 콘크리트 공사
- KCS 21: 강구조 공사
- KCS 31: 건축 설비 (기계, 전기 포함)
- KCS 41: 건축 마감 공사


## 건축 설계 시방서 핵심 항목 (2026-05-26)
- Source: KCS 41 시리즈, 국토교통부 건축공사 표준 시방서, LUA BIM LABS curated baseline 2026-05-26
- Tags: specification,architecture,material,quality

재료 기준:
- 콘크리트 벽체·바닥: 설계 기준 강도 fck 명시, 골재 최대 치수(25mm 이하 일반), 물-시멘트 비(W/C ≤ 0.55)
- 단열재: 열관류율(U값) 부위별 기준, KS M 3808(그라스울), KS M ISO 4898(압출법 보온판)
- 창호: 단열 성능 등급(1~5등급), 기밀 성능(1~5등급), 차음 성능(25~45dB) 명시
- 방수 재료: 도막 방수(두께 2.0mm 이상), 시트 방수(1.5mm 이상), 복합 방수 층수
- 마감재: VOC 방출량 기준(친환경 마감재 인증 여부), 불연·준불연·난연 등급 구분

공법 기준:
- 철근 피복 두께: 외벽·바닥 40mm, 내벽·천장 20mm, 기초 70mm (KCS 14 20 11)
- 이음 길이: 직경의 40~50배(겹침 이음 일반), 가스압접 이음 시 KS B 0882 준수
- 거푸집 존치 기간: 보 측면 3일, 슬래브 하면 21일(압축강도 70% 이상 확인 후 해체)
- 방수층 보호 모르타르: 두께 50mm 이상, 격자 눈금 방수층 보호

품질 기준:
- 바닥 수평도: 3m당 ±5mm 이내 (마감 후 기준)
- 벽체 수직도: 1m당 ±3mm, 전체 높이 ±10mm
- 창호 설치 틈새: 10mm 이하, 이중 실런트 처리
- 도장 도막 두께: 외부용 에폭시 150μm 이상, 내부 수성 40μm 이상


## MEP 설계 시방서 핵심 항목 (2026-05-26)
- Source: KCS 31 시리즈, 건축기계설비 설계기준, 내선규정, LUA BIM LABS curated baseline 2026-05-26
- Tags: specification,mep,hvac,plumbing,electrical,fire,communication

공조 (HVAC) 시방 핵심:
- 덕트 재질: 아연도금 강판 KS D 3506, 두께 0.5~1.6mm (단변 길이에 따른 등급 구분)
- 덕트 기밀 등급: 저압(≤500Pa), 중압(500~1,000Pa), 고압(>1,000Pa) 구분, 기밀 시험 필수
- 단열 두께: 공조 덕트 25mm 이상(기계실 내 50mm 이상), 냉수 배관 20~50mm(관경 별도)
- 소음 기준: 거실 NC-35 이하, 사무실 NC-40 이하, 기계실 인접 공간 NC-45 이하
- 외기량: 1인당 25~30m³/h 이상(KS F 2603 환기 기준)

위생 (Plumbing) 시방 핵심:
- 급수관: 스테인리스강 STS304 또는 CPVC, 수압 0.7MPa 이상에서 3분간 누수 없음
- 배수 구배: 횡주관 1/100 이상, 지관 1/50 이상
- 오수관·통기관: 경질 염화비닐관(PVC), 주철관 사용 가능(옥내 매립 부위)
- 온수 순환: 60°C 유지(레지오넬라 억제 기준), 순환 배관 단열 두께 20~30mm
- 사용 수압: 0.05MPa 이상 ~ 0.3MPa 이하(말단 기구 기준)

전기 시방 핵심:
- 전선관: 후강 전선관(C관) 또는 박강 전선관(E관), 점유율 40% 이하
- 케이블 트레이: 강전/약전 분리 또는 격벽 설치(300mm 이격 권장)
- 접지: 제3종 접지(100Ω 이하), 특수 장소 제2종(10Ω 이하)
- 변압기 용량 여유율: 설계 부하의 120% 이상 확보
- 비상전원: 발전기 기동 시간 30초 이내, 소방용 30분 이상, 일반 비상 최소 1시간

소방 시방 핵심:
- 스프링클러 헤드 간격: 정방형 배치 최대 3.7m×3.7m, 벽체에서 1.7m 이내
- 소화 배관: KS D 3507 배관(흑관), 작동 수압 0.1~1.2MPa
- 방화댐퍼: 72°C 퓨즈 용융식, 덕트 방화구획 관통 부위 전수 설치
- 소화 펌프 유량: 스프링클러 80L/min × 헤드 수(최대 30개 기준)
- 감지기 설치 간격: 연기식 1종 90m² 이하, 차동식 스포트형 50m² 이하

통신 시방 핵심:
- 구내 통신: MDF → IDF → TO(Telecommunication Outlet) 체계, Cat6 UTP(10GBase-T)
- 광케이블: OS2(단일 모드) 건물 간, OM3/OM4 건물 내 수직 계통
- CCTV: Full HD 이상, 영상 저장 30일 이상(법정 기준), 야간 적외선 30m 이상
- 방송(PA): 비상방송 내화 케이블(FR-8), 실 내 음압 70dB 이상


## 구조 설계 시방서 핵심 항목 (2026-05-26)
- Source: KCS 14 시리즈, KDS(한국설계기준), 건축구조기준 KBC, LUA BIM LABS curated baseline 2026-05-26
- Tags: specification,structure,concrete,steel,foundation

콘크리트 시방 핵심:
- 설계 기준 강도(fck): 일반 건축 21~30MPa, 고층·특수구조 30~50MPa
- 물-결합재비: fck 21MPa: W/B ≤ 0.60, fck 30MPa: W/B ≤ 0.50, 고내구성 ≤ 0.45
- 혼화제: AE 감수제(공기량 3~6%), 고성능 감수제(슬럼프 플로우 550±75mm)
- 타설 온도: 5°C 이상~35°C 이하 (동절기·서중 콘크리트 별도 관리)
- 양생 기간: 보통 포틀랜드 시멘트 7일 이상, 압축강도 70% 달성 시까지 습윤 양생

철골 시방 핵심:
- 강재 등급: SS400(일반), SM490(중요 구조), HSB600(고강도 특수)
- 용접 품질: KS B ISO 5817 품질 등급 B(중요), C(일반)
- 고력볼트: F10T 마찰이음(미끄럼계수 μ ≥ 0.45), 도입 축력 확인
- 내화피복: 1시간(일반 건물 7층 이하), 2시간(7~15층), 3시간(16층 이상) — 뿜칠·보드·도막 선택
- 현장 용접 기온: 0°C 이하 예열 필수(강재 두께 25mm 이상 시 예열 60~80°C)

기초 시방 핵심:
- 직접 기초: 허용 지내력 100kN/m² 이상 확인(시험 말뚝 또는 평판재하 시험)
- 말뚝 기초: PHC 말뚝(KS F 4306), 현장 타설 콘크리트 말뚝(설계 강도 fck 24~27MPa)
- 지정공사: 터파기 여굴 최소화, 바닥 면 다짐 후 버림 콘크리트(60mm 이상) 타설
- 방수·방습: 기초 외벽 방수층 + 배수판 + 보호 콘크리트 체계
- 침하 기준: 최대 침하 25mm 이내, 부등 침하 각도 1/300 이내


## 설계 시방서 BIM 연계 (2026-05-26)
- Source: 국토교통부 BIM 적용 지침, LUA BIM LABS curated baseline 2026-05-26
- Tags: specification,bim,parameter,revit,ifc

시방서 정보를 BIM 파라미터로 입력하는 기준:

공유 파라미터 필수 항목:
- Spec_Code: 해당 KCS 코드 (예: KCS 14 20 10)
- Spec_Material: 재료 규격 및 등급
- Spec_Thickness: 설계 두께(mm)
- Spec_Strength: 설계 강도(MPa 또는 등급)
- Spec_FireRating: 내화 등급(시간)
- Spec_Warranty: 하자 보증 기간(년)
- Spec_Standard: 적용 표준 규격(KS 번호)

BIM-시방서 연계 원칙:
- 모든 구조 부재(보, 기둥, 슬래브): fck, 피복 두께, 내화 등급 파라미터 입력 필수
- 마감재 패밀리: 두께, 재료 코드, VOC 등급, 불연 여부 파라미터 포함
- 설비 장비: 제조사 모델 번호, 성능 사양(용량, 효율), 보증 기간 연동
- IFC 출력 시: Pset_SpecificationInformation 속성 세트에 시방 코드 매핑

Revit 패밀리 파라미터 입력 기준:
- 인스턴스 파라미터: 현장 고유 정보(설치 날짜, 검사 결과)
- 타입 파라미터: 시방 코드, 재료 규격, 성능 기준 (공통 적용)
- 공유 파라미터 파일: 프로젝트 전체 통일 사용, 버전 관리 필수

## 설계 시방서 Claude Code 심화 업데이트 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28

국가건설기준센터(KCSC)에서 관리하는 KCS(Korean Construction Specification) 체계는 건설공사 표준시방서와 전문시방서로 구성된다. 설계 시방서는 KCS 41(건축공사), KCS 35(기계설비), KCS 31(구조)을 기반으로 프로젝트 특성에 맞는 특기시방서를 작성한다.
- BIM 파라미터 연계: 시방서의 재료 규격과 품질 기준을 Revit 공유파라미터로 맵핑한다. 예: 콘크리트 강도(KCS 41 30 00)→Revit Material Asset의 Compressive Strength 파라미터, 철골 등급(KCS 41 31 00)→Material Grade 파라미터. 파라미터명은 영문으로 통일하되 한국어 설명을 주석에 포함한다.
- 특기시방서 작성 기준: ① 공종별 재료 규격 번호(KS 인증번호) 명시, ② 시공 허용오차(예: 콘크리트 피복 두께 ±5mm), ③ 검사·시험 빈도(예: 레미콘 슬럼프 시험 150㎥당 1회), ④ 하자담보 기간(시설물안전법 기준)
- MEP 설계 시방서: 냉난방 설비는 KCS 35 50 00(공기조화설비), 급배수는 KCS 35 30 00(위생설비), 전기는 KCS 45 시리즈를 적용한다. Revit MEP 모델의 System Classification과 시방서 공종 코드를 1:1 매핑하여 물량산출 자동화를 구현한다.

### 설계 시방서 현장 실패 패턴 5가지

**1. 도면(BIM)과 시방서 불일치 → 시공 혼란**
- 원인: 설계 변경 시 BIM 모델만 갱신, 시방서 미동기화 (또는 반대)
- 증상: 현장에서 "도면이냐 시방서냐" 분쟁 → 공기 지연
- 해결: Revit 파라미터에 시방서 참조 코드(KCS 번호) 삽입, 변경 시 양쪽 동시 업데이트 체크리스트

**2. KCS 조항 단순 복붙 → 프로젝트 특수성 미반영**
- 원인: 표준 시방서 붙여넣기, 특수 재료/공법 수정 누락
- 증상: 실제 사용 자재 규격과 시방서 불일치 → 검수 거부
- 해결: 표준 KCS 조항 + 프로젝트 특기사항(붉은 글자 표시) 분리 관리

**3. MEP 시방서 공종 코드-BIM 시스템 불일치**
- 원인: KCS 35 시리즈(기계)와 Revit MEP 시스템 분류명 상이
- 증상: 물량 산출 시 공종 혼재, 엑셀 집계 오류
- 해결: Revit System Type 파라미터에 KCS 코드 직접 기입 (예: `KCS_35_50_00`)

**4. 재료 대체 승인 절차 미명시 → 무단 대체**
- 원인: "동등 이상 재료 대체 가능" 조항만 있고 승인 절차 없음
- 해결: 시방서에 대체 자재 승인 양식 번호 및 제출 기한 명기

**5. 설계 VE 단계 시방서 업데이트 누락**
- 원인: VE(Value Engineering)로 공법 변경 후 시방서 갱신 안 됨
- 해결: VE 결정 사항을 시방서 변경 이력 표에 즉시 등록, BIM 모델 수정과 동시 처리
