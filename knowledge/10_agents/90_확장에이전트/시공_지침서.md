## 시공 BIM 활용 기준 — 시공도·Shop Drawing BIM 연계 (2026-05-26)
- Source: LUA BIM LABS curated baseline, 국토교통부 시공단계 BIM 적용 지침 2026-05-26
- Tags: construction,bim,shop-drawing,coordination,as-built

시공 BIM의 역할 구분:
- 설계 BIM(Design BIM): 설계 의도 표현, LOD 350 이하 수준.
- 시공 BIM(Construction BIM): 실제 시공에 쓰이는 상세 모델, LOD 350~400.
- 준공 BIM(As-Built BIM): 실제 시공 결과 반영, LOD 400~500.

시공도(Shop Drawing) BIM 연계 절차:
1단계 — 설계 BIM 인수: 설계 BIM 모델을 기반으로 시공 BIM 작업 시작(재작성 금지).
2단계 — 시공 상세 추가: 설계 BIM에 시공 접합 상세, 슬리브, 인서트, 앵커 위치 추가 → LOD 400 달성.
3단계 — Shop Drawing 추출: BIM 뷰에서 시공도 직접 추출, BIM 모델과 도면 자동 일치.
4단계 — 발주처/설계사 승인: 제출(Submittal) → 검토(Review) → 승인(Approved) 절차.
5단계 — 현장 적용: 승인된 Shop Drawing 기반 시공, BIM 모델 현장 태블릿 열람.
6단계 — 시공 후 BIM 업데이트: 변경 사항(RFI 처리, 현장 조정) 즉시 BIM 반영.

Shop Drawing 종류별 BIM 연계 기준:
- 배근도(철근 상세): 구조 BIM 기반, 주철근·스터럽·이음 위치 상세 표현(LOD 400).
- MEP 시공도: MEP BIM에서 직접 추출, 행거·슬리브·개구부 위치 포함.
- 철골 상세도: 구조 BIM에서 접합부 상세 추출, 용접 기호·볼트 규격 파라미터.
- 외장 패널 제작도: 건축 BIM에서 패널 분할선·취부 상세 추출.

BIM 모델 버전 관리:
- 버전 명칭: [프로젝트코드]-[공종]-[단계]-Rev[번호] (예: LUA-M-CD-Rev03)
- 승인 전 모델: "DRAFT" 워터마크 뷰 파라미터 설정.
- 승인 후 모델: "FOR CONSTRUCTION" 뷰 파라미터 설정.
- 변경 이력: 매 Rev마다 변경 사유, 변경 내용, 담당자, 날짜 기록 필수.


## 공종별 시공 순서 및 선행·후행 작업 기준 (2026-05-26)
- Source: LUA BIM LABS internal construction sequence baseline, 건설기술진흥법 기준 2026-05-26
- Tags: construction,sequence,trade,schedule,predecessor,successor

전체 공사 대분류 순서 (일반 건축물 기준):
1. 가설공사 (Temporary Works): 규준틀, 비계, 안전시설, 가설전기·용수
2. 토공사 (Earthwork): 터파기, 흙막이, 지반 개량
3. 기초공사 (Foundation): 말뚝, 기초 콘크리트, 방수
4. 골조공사 (Structure): 철근 배근 → 거푸집 → 콘크리트 타설 → 양생 → 탈형
5. MEP 선행 작업 (Underground/Embedded MEP): 슬래브 내 매립 배관, 피트 배관
6. 외장공사 (Envelope): 외벽, 커튼월, 지붕
7. 내장공사 (Interior): 경량 칸막이, 천장틀, 바닥
8. MEP 공사 (Systems): 기계, 전기, 소방, 통신 설비 설치
9. 마감공사 (Finishes): 타일, 도장, 바닥재
10. 시운전 및 검사 (Commissioning): 설비 시운전, 관할 기관 검사, 사용승인

공종별 선행·후행 관계 핵심 기준:

구조 골조:
- 선행 조건: 지반 조사 완료, 기초 승인 완료.
- 후행 작업 대기: 골조 완성 후 외장 착수 가능 — 구조 안전성 확인(층별 콘크리트 압축강도 21MPa 이상 확인 후).
- 타설 후 최소 양생: 거푸집 탈형 기준 — 슬래브 하부 거푸집 설계강도의 100% 도달 후 탈형(일반 콘크리트 28일 양생).

MEP 공사 선행 조건:
- 골조 완성(해당 층 이상): 슬래브 타설 완료 후 MEP 설치 착수.
- 슬리브/인서트: 골조 공사 중 사전 설치 — 슬래브 타설 전 슬리브 위치 BIM 확인 필수.
- 기계실·전기실 바닥: 방수 완료 후 기기 반입.
- 외장 기밀 완료 후: 공조 시운전 가능 (기밀 완료 전 시운전 금지).

내장·마감 공종 순서 (층별 표준):
1. 경량 칸막이 뼈대 설치
2. MEP 배관·덕트·트레이 설치 (천장 내부)
3. 소방 스프링클러 헤드 위치 선행 확인 (천장 패턴 맞춤)
4. 석고보드 설치 (경량 칸막이, 천장)
5. 도장 (퍼티 → 페인트)
6. 바닥 마감 (에폭시, 타일, 목재 순서)
7. 소방·전기·통신 기구 설치 (면 기구: 콘센트, 스위치, 스프링클러 헤드)
8. 도어·유리 설치
9. 가구·기기 반입

BIM 기반 시공 순서 시뮬레이션 활용:
- 공종별 BIM 요소에 착수일·완료일 파라미터 입력 → 4D 시뮬레이션 연동.
- 선행 작업 미완료 시 후행 작업 요소 시각적으로 차단(색상 코딩).
- 공종 간 작업 공간 충돌(Space Conflict): 동일 구역 동시 작업 불가 구간 BIM으로 사전 확인.


## BIM 기반 공정 관리 — 4D 시뮬레이션, 진도 관리 (2026-05-26)
- Source: LUA BIM LABS internal 4D BIM baseline, PMIS 연계 기준 2026-05-26
- Tags: construction,4d,schedule,progress,simulation,bim

4D BIM 구성 요소:
- 3D BIM 모델: Revit 또는 IFC 기반 모델.
- 공정표(Schedule): MS Project, Primavera P6, 또는 BIM 소프트웨어 내장 스케줄러.
- 연동 도구: Navisworks TimeLiner, Synchro Pro, Autodesk Build Schedule.
- BIM 요소와 공정 Task 연결: 요소별 공정 코드(Activity ID) 파라미터 입력.

4D 시뮬레이션 설정 기준:
- 색상 코딩 표준:
  - 계획 공정(미착수): 회색(Gray)
  - 진행 중(In Progress): 노란색(Yellow)
  - 완료(Completed): 초록색(Green)
  - 지연(Delayed): 빨간색(Red)
  - 철거 예정: 보라색(Purple)
- 시뮬레이션 단위: 일별(Daily) 또는 주별(Weekly) 타임 스텝.
- 기준 공정 vs. 실적 공정: 계획선(Baseline)과 실적선(Actual) 동시 표시.

진도 관리 BIM 활용 방법:
- 주 1회 BIM 모델 진도 업데이트: 완료 요소에 "완료" 파라미터 값 입력.
- 진도율 산출: (완료 요소 물량 / 전체 물량) × 100 → 자동 집계.
- 진도 초과/지연 구간: Navisworks 또는 Autodesk Build에서 히트맵으로 시각화.
- 주간 공정 회의: BIM 4D 화면 공유 기반 회의 진행, 캡처 이미지 회의록 첨부.

공정 지연 경보 기준:
- 7일 이상 지연: 담당 공종 팀장 보고 + 대책 수립.
- 14일 이상 지연: 공정 만회 계획서 제출 + 발주처 보고.
- 21일 이상 지연: 공기 연장 검토 또는 투입 인력·장비 증원.


## 시공 품질 관리 BIM 활용 — NCR, RFI, 이슈 관리 (2026-05-26)
- Source: LUA BIM LABS internal QA/QC BIM baseline, ISO 19650 Part 2 기준 2026-05-26
- Tags: construction,quality,ncr,rfi,issue,bim,qc

NCR(Non-Conformance Report, 부적합 보고서) BIM 연계:
- NCR 발행 조건: 시방서·도면 기준 미달, 재료 불합격, 치수 오류, 시공 순서 위반.
- BIM 연계 방법: 해당 BIM 요소에 NCR 번호 파라미터 입력 → 위치 추적 가능.
- NCR 상태 분류: Open(미해결), In Review(검토 중), Closed(해결 완료).
- 폐쇄(Close) 조건: 재시공 완료 또는 설계 변경 승인 후 감리 서명.
- NCR 현황: 주간 보고서에 공종별 Open/Closed 건수 표기.

RFI(Request for Information, 설계 질의) BIM 연계:
- RFI 발행 조건: 도면 불명확, 공종 간 충돌, 현장 조건과 도면 불일치.
- BIM 연계 방법: 질의 위치를 BIM 모델에 마크업(Markup)으로 표시 → RFI 번호 부여.
- 응답 기한: 일반 RFI 7영업일, 긴급 RFI 48시간 이내.
- 설계 변경 유발 RFI: 설계 변경 지시서(Design Change Order) 발행 → BIM 모델 업데이트.
- RFI 상태: Open → Answered → Closed 순서.

BIM 이슈 관리 체계 (Autodesk Build / BIM 360 기준):
- 이슈 유형: 간섭(Clash), 품질(Quality), 안전(Safety), 설계 오류(Design Issue).
- 이슈 우선순위: Critical(1) → High(2) → Medium(3) → Low(4).
- 담당자 지정: 이슈 생성 시 담당 공종, 담당자, 기한 자동 지정.
- 이슈 해결 확인: 현장 사진 첨부 + 담당자 Closed 처리.
- 미결 이슈 보고: 주간 보고서에 공종별 미결 이슈 수 및 기한 초과 현황 표기.

시공 단계 품질 체크포인트 (Hold Point / Witness Point):
- Hold Point(작업 중단 후 확인): 기초 터파기 바닥 확인, 콘크리트 타설 전 철근 검사, 방수층 시공 전 바탕 처리 확인.
- Witness Point(입회 확인): 콘크리트 타설 슬럼프 시험, 절연저항 측정, 소방설비 수압 시험.
- BIM에서 Hold Point/Witness Point 위치를 색상으로 표시하여 감리원 입회 전 알림.


## 현장 좌표계 설정 및 측량 기준 (2026-05-26)
- Source: LUA BIM LABS internal site coordinate baseline, 국토지리정보원 측량 기준 2026-05-26
- Tags: construction,coordinate,survey,bim,shared-coordinate,project-basepoint

현장 좌표계 기본 원칙:
- BIM 모델의 프로젝트 기준점(Project Base Point)과 측량 기준점(Survey Point)은 반드시 구분하여 사용한다.
- 프로젝트 기준점: BIM 모델 내부 상대 좌표계 원점 (임의 설정 가능).
- 측량 기준점(Survey Point): 실제 현장 측량 성과와 연결된 절대 좌표 기준점.
- 공유 좌표(Shared Coordinates): 전 공종 BIM 모델이 동일한 공유 좌표 기준으로 링크되어야 한다.

현장 기준점 설정 절차:
1단계 — 측량 성과 수령: 측량 회사로부터 현장 기준점 좌표(X, Y, Z), 방위각(Bearing) 수령.
2단계 — Survey Point 설정: Revit에서 Survey Point에 측량 성과 좌표 입력.
3단계 — 프로젝트 기준점 설정: Survey Point 기준으로 건물 원점(주요 그리드 교점) 지정.
4단계 — 공유 좌표 발행: 건축 BIM 모델을 기준 파일로 지정, "Publish Coordinates" 실행.
5단계 — 전 공종 좌표 수신: 구조, MEP 등 모든 공종 BIM 모델이 건축 모델에서 "Acquire Coordinates" 실행.
6단계 — 좌표 일치 확인: 전 공종 모델 링크 후 기준점·그리드 일치 여부 시각 확인.

현장 측량 기준:
- 기준점 망 설치: 건물 외부 4개소 이상 영구 기준점(CP: Control Point) 설치.
- 레벨 기준: 수준 측량으로 각 층 FL 레벨 확인 (설계값 대비 ±3mm 이내).
- 평면 위치 기준: 기둥·벽 위치 설계값 대비 ±5mm 이내.
- 수직도 기준: 층당 H/500 이하 (10m 층고 기준 20mm 이하).
- 측량 성과 BIM 반영: 준공 BIM(As-Built)에 실측값 반영 필수.

BIM 좌표계 오류 방지 주의 사항:
- IFC 내보내기 시 좌표계: "Shared Coordinates" 기준으로 내보내기 설정.
- 링크 모델 삽입 옵션: "Auto - By Shared Coordinates" 선택 (Manual 선택 금지).
- 모델 이동(Move) 금지: BIM 모델 완성 후 Survey Point 및 Project Base Point 이동 금지.
- 좌표 변경 발생 시: 전 공종 팀장 사전 협의 후 변경, 변경 이력 기록 필수.
