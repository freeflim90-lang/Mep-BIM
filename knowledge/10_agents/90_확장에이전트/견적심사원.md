## 2026-06-05 범위 외 추가 작업 무상/유상 선례
- Source: `knowledge/10_agents/conflict_resolution/cases/AITEST_20260605_004.md`, `knowledge/10_agents/conflict_resolution/cases/AITEST_20260605_008.md`
- Tags: estimate,precedent,scope-change,free-support,cfo,customer-support

고객이 계약 또는 SOW 범위를 넘어선 추가 모델링, 추가 리뷰, clash report 재검토, 납품 후 보완 작업을 "이번만 무료"로 요청하면 견적심사원은 범위 외 작업으로 분류하고 유상 견적 원칙을 우선 적용한다.

운영 기준:
- 계약서, SOW, 기존 견적서에 포함된 산출물, 검토 횟수, 납품 후 지원 범위를 먼저 확인한다.
- 범위 외 추가 작업은 유상 옵션 또는 Change Order 대상으로 산정한다.
- 고객 신뢰 회복을 위한 1회성 무상 예외는 CFO 승인과 공수 상한이 있을 때만 허용한다.
- 고객지원CS가 CFO 승인 전 "무료로 해드리겠다"는 확정 문구를 쓰지 않도록 안내한다.
- 예외 승인 기록에는 고객 요청 원문, 예상 공수, 무료 처리 사유, 승인자, 1회성 조건을 남긴다.

## 2026-06-04 BF/편의시설 기준 확인 공수 견적 반영
- Source: `knowledge/40_curation/updates/daily/2026-06-04_LUA_BIM_LABS_BF_ACCESSIBILITY_STANDARD_UPDATE.md`
- Tags: estimate,BF,accessibility,architecture,review

BF 또는 편의시설 대응 BIM 용역은 일반 건축 모델링 공수 외에 대상시설 판정, 의무/권장 편의시설 검토, 예비인증/본인증 대응, 지자체 사전검사 대응, 재검수 공수를 별도로 산정한다.

견적 전 확인 항목:
- BF 인증 대상 여부와 인증 단계
- 장애인등편의법 시행령 별표상 대상시설 여부
- 시행규칙 별표 세부기준 확인 범위
- 발주처 특기시방서와 BF 인증 요구 수준
- 지자체 조례 또는 편의시설 사전검사 적용 여부
- 자동검수 개발 범위와 수동 검토 범위
- 인증기관 또는 설계자 질의응답 필요 여부

운영 기준:
- BF 적용 범위가 미확정이면 견적서에 `BF 기준 미확정 리스크`를 명시한다.
- 자동 수집된 편의시설 치수만으로 모델링 책임, 인증 책임, 납품 합격 책임을 확정하지 않는다.
- BF 체크리스트, 도면 대조, RFI, 재검수, 인증 심사 대응 회의는 별도 공수로 본다.

## 2026-06-04 BF 기준 미확정 리스크 메모 샘플 연결

- Tags: estimate,BF,scope-risk,KST,training

견적심사원은 BF/편의시설 적용 범위가 미확정인 프로젝트에서 `BF 기준 미확정 리스크`를 견적서 또는 검토 메모에 명시한다. 대상성, 의무/권장, 예비인증/본인증, 지자체 사전검사, 특기시방서가 확인되지 않으면 가격과 일정, 책임 범위를 확정하지 않는다.

분리 기준:
- `KST01`: 공식 법령, 시행령 별표, 시행규칙 별표, BF 인증제도 존재
- `KST03`: 프로젝트 대상성, 인증 단계, 지자체 조건, 발주처 요구
- `KST04`: 자동 수집된 편의시설 치수와 항목명

운영 기준:
- BF 대상성 검토, 체크리스트 작성, 도면·모델 대조, RFI/회의 대응, 예비/본인증 보완, 재검수는 별도 공수로 산정한다.
- 인증 통과 책임을 BIM 모델링 또는 자동검수 결과에 포함하지 않는다.
- BF 적용 조건이 확정될 때까지 견적서에는 별도 공수 또는 변경 견적 조건을 유지한다.

## 견적심사원 마스터급 경험 지식 (2026-05-29)
- Source: claude-code-enhanced 2026-05-29
- Tags: 견적심사, 비정형건물가산, 덤핑견적감지, 원가검증, 이상견적

### 비정형 건물 견적 심사 특별 기준

정형 건물 기준 견적을 비정형 건물에 그대로 적용한 견적은 **즉시 반려** 기준:

| 심사 항목 | 정형 건물 기준 | 비정형 건물 추가 확인 |
|---------|-------------|-------------------|
| 모델링 공수 | 표준 단가표 비교 | 곡면/파라메트릭 가산율 (10~70%) 반영 여부 |
| IFC 납품 공수 | 일반 Export 시간 | Entity 매핑, 납품 검증 공수 별도 명기 여부 |
| Dynamo/Grasshopper | 불필요 | 전문 인력 투입 계획 확인 |
| 납품물 목록 | 표준 납품물 | 비정형 특수 납품물 (패널화 도면, 곡면 수량 등) |

### 이상 견적 감지 패턴 5가지

**1. 비정형 건물에 정형 단가 적용 → 수행 불능 리스크**
- 징후: 경쟁사 견적 대비 30%+ 저가이면서 비정형 건물
- 확인: 공수 세부 내역서 요청 → 가산율 0% 확인 시 반려

**2. 소프트웨어 라이선스 미산정**
- 징후: "소프트웨어 비용 없음" 또는 월 1~2만원 산정
- 실제: Revit 월 15만원+, Navisworks 월 8만원+, ACC 사용자당 월 3~5만원
- 확인: 프로젝트 기간 × 인원 × 단가 직접 계산하여 비교

**3. 재작업 버퍼 0% 견적**
- 징후: 공수 세부 내역에 변경 대응 항목 없음
- 기준: 설계 단계 15~20%, 시공 단계 10% 이상 버퍼 여부 확인

**4. 핵심 인력 자격 미기재**
- 징후: "BIM 모델러 3인" 기재, 개인별 자격·경력 없음
- 확인: 담당자별 이름·자격증·유사 프로젝트 경험 제출 요구

**5. 현장 파견 비용 일괄 처리**
- 징후: "현장 파견비 포함" 단일 금액 표기
- 확인: 방문 횟수 × 거리 × 단가 세분화 요구

### 공정 견적 검증 공식 (심사원 내부 기준)

```
정상 견적 범위 = 면적(㎡) × 기본단가(원/㎡) × 복잡도계수 × 비정형가산 × (1 + 버퍼율)

복잡도계수: 의료 1.30 / 데이터센터 1.25 / 사무소 1.00 / 주거 0.95
비정형가산: 1.10~1.70 (경사~전체비정형)
버퍼율: 0.10~0.20 (시공~설계 단계)

하한선 경고 기준: 정상 견적의 70% 미만 → 원가 세부 내역 제출 요구
```
