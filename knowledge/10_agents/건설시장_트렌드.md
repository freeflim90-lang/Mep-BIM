# 건설시장 트렌드 지식 베이스


## 2026-06-13 뉴스 트렌드 자동 심화 반영
- Source: `obsidian_vaults/lua_bim_lab_global_map/NAS_Knowledge/2026-06-13 뉴스 트렌드 18항목 지식 업데이트.md`
- Tags: news-trend,auto-deepening,openbim,ai-aec,construction-robotics

이번 업데이트는 일일 뉴스 신호를 자동 분류해 시장 트렌드 KB로 승격한 결과다.

OpenBIM은 파일 교환을 넘어 IDS 요구사항, BCF 이슈 추적, 검증 서비스, 소프트웨어 인증까지 포함하는 납품 계약 언어로 이동한다. [[BIM_납품검수]]와 [[Model Quality Auditor]]에 IFC/IDS/BCF 기반 납품 반려 리스크 진단 후보를 연결한다.

AEC 자동화는 전문 개발자만의 영역이 아니라 실무자가 반복 업무를 줄이는 단계형 역량으로 바뀌고 있다. [[교육컨설팅]]에 Python 업무 자동화, Dynamo/Grasshopper 모델 자동화, Revit API/Add-in 제품화의 3단계 교육 사다리를 연결한다.

국내 건설 AI는 설계 자동화, 원가·견적·공정관리, 응용솔루션으로 분화되는 단계에 들어섰다. [[AX_전략승격리뷰]]에서 설계/BIM 자동화, 견적·공정 데이터 자동화, 운영/CS 응용솔루션으로 제품 후보를 분리한다.

Autodesk가 Forma 중심 AECO 클라우드와 Revit 연결을 강화할수록 Add-in은 단독 버튼 기능보다 데이터 내보내기와 협업 흐름이 중요해진다. [[Revit_Addin]] 로드맵에서 CSV/JSON/BCF/IFC/Excel export와 협업 연동 가능성을 별도 점검한다.

현장 로봇은 BIM의 대체재가 아니라 as-built 현장 상태를 BIM/디지털트윈과 정렬하는 데이터 수집 계층으로 보는 것이 적합하다. [[시공_지침서]], [[Navisworks_Addin]], [[FM_자산관리]]에 현장 검측 데이터 입력값 후보로 연결한다.
