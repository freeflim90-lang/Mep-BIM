```
구조체 배관 관통(Penetration) 기준:
- 보(Beam) 관통: 보 깊이의 1/3 이하, 보 끝에서 1/3 구간 관통 금지
  (전단력 집중 구간 — 구조기술사 확인 필수)
- 슬래브 관통: 직경 300mm 이하 통상 허용, 이상은 개구부 보강근 추가
- 전이보: 하중 집중 부재 — 관통 원칙적 금지, 우회 배관 필수
- 기둥: 모든 관통 금지 (구조 안전성 최우선)
- 내력벽: 개구부 위치·크기 구조계산 후 결정
```

| 350 | 철근 배근 방향 (LOD 350+) | 볼트·플레이트 상세 | 지중보·파일 |
| 400 | 시공용 철근 상세도 수준 | 제작 도면·조립 순서 | 기초 완성 |

**구조-MEP 간섭 시 처리 프로세스 (실무 기준):**
1. **자동 간섭 감지**: Navisworks Clash Detective로 구조-MEP 간섭 리스트 추출
2. **간섭 분류**: Hard Clash(실제 겹침) vs Soft Clash(이격 부족)
3. **구조 검토 요청**: 관통 불가 위치는 구조기술사에게 RFI 발행
4. **우회 설계**: 배관·덕트 경로 변경 → BIM 재모델링
5. **개구부 확정**: 구조 승인 후 도면에 반영, BIM 파라미터 업데이트

**MEP 관통 가능 여부 판단 기준표:**
| 구조 부재 | 관통 가능 여부 | 조건 |

| 보 | 제한적 | 보 깊이 1/3 이하, 끝단 1/3 구간 회피 |
| 기둥 | 불가 | 절대 금지 |
| 전이보 | 불가 | 원칙적 금지 |
| 내력벽 | 제한적 | 구조계산 후 결정 |
| 비내력 칸막이 | 가능 | 규격 제한 없음 (방화 고려) |

## 보·기둥 관통 및 개구부 보강 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,penetration,opening

일반 RC보: 높이의 1/3 이하, 보 단부에서 보 높이 2배 이상 이격한 위치에만 관통 허용.

Add-in은 관통 요청을 위치·크기·이격 기준으로 자동 분류하고 구조 검토 필요/불필요를 초기 판단값으로 표시한다.

## 전이보·전이 슬래브 처리 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,transfer-beam,slab

전이보는 상부 하중을 받는 핵심 부재로 절대 관통 금지. Add-in 태그로 TRANSFER_BEAM 분류 필수.
전이 슬래브 두께는 일반 슬래브 대비 1.5~2배 이상이므로 레벨 계산 시 실제 두께 데이터 활용.
Add-in 보고서에는 '전이 부재 여부' 필드를 별도 컬럼으로 표기하고 담당 구조기술사 확인 요청 플래그를 붙인다.

## 구조 모델 파라미터 및 데이터 기준 (2026-05-19 17:26:40)
- Source: LUA BIM LABS domain knowledge baseline 2026-05-19
- Tags: structure,parameter,bim-data

공유 파라미터 필수: 부재유형(보/기둥/슬래브/벽), 설계강도(fck), 관통허용여부, 보강필요여부.

개구부 보강 정보(보강근 규격, 길이)는 모델 파라미터로 직접 저장하거나 연결 문서로 링크한다.

## 구조 관통 요청 처리 절차 (2026-05-23)
- Source: LUA BIM LABS internal structural coordination baseline
- Tags: structure,penetration,opening,bim,rfi,coordination
- Links: [[설비시공조율]], [[건축]], [[설비도면해석]]

MEP에서 구조체 관통이 필요할 때 처리 순서:

1단계 — 필요성 확인:
- 관통이 꼭 필요한지 우회 경로 먼저 검토한다. 보를 관통하는 것보다 보 아래로 우회하는 것이 구조적으로 안전하다.
- 관통 크기, 위치(보에서의 거리), 이격 거리, 관통 개수를 먼저 결정한다.

2단계 — 관통 가능 여부 초기 판단 (BIM에서 자동 확인 가능):
- RC보 관통 허용 범위: 관통 위치가 보 단부에서 보 높이 2배 이상 이격, 관통 직경이 보 높이 1/3 이하, 1/4 이하가 안전.
- 슬래브 관통: 단변 1/6, 장변 1/10 이하면 보강 없이 가능 (설계자 확인 필수).
- 전이보·내력벽: 원칙적으로 관통 금지. 무조건 구조기술사 확인.

3단계 — 구조 설계자 협의 (설계 질의, RFI):
- 관통 위치 좌표, 크기, 목적 공종, 관통 방향을 RFI로 제출한다.
- 허용 가능하면 보강 방법(보강근 규격, 배근 방법)을 받는다.
- 불가 판정이면 대체 경로를 협의한다.

4단계 — BIM 모델 반영:
- 승인된 관통 위치에 Opening 요소 또는 Sleeve를 삽입한다.
- 보강 정보를 구조 모델 파라미터에 기록하거나 연결 문서로 남긴다.
- Navisworks 간섭 검토에서 구조 관통 승인 여부를 이슈 상태(Resolved/Pending)로 관리한다.

자주 발생하는 구조 관통 이슈:
- 보 단부(기둥 접합부 근처) 관통: 구조적으로 가장 위험한 위치 → 기본적으로 불가.
- 대구경 배관이 보를 직각이 아닌 사선으로 관통: 개구부가 타원형이 되어 보강이 복잡해짐.
- 여러 공종 배관이 같은 위치 요청: 관통 간격 기준(관통경의 3배 이상)을 만족하도록 조정.
- 이미 시공된 구조체에 관통 추가: 코어 드릴링 전 구조기술사 현장 확인 필수.

## 구조 시방서 핵심 기준 (2026-05-26)
- Source: KCS 한국건설기준, 국내 표준 시방서 2026-05-26
- Tags: structure,specification,construction-spec

### 철근 공사 기준 — 피복 두께
- 기초(직접 흙에 접하는 부분): 80mm 이상.
- 외부 노출 구조(슬래브·보·기둥 외부): 40mm 이상.
- 실내 구조(내부 보·기둥·슬래브): 20mm(슬래브·벽체), 40mm(보·기둥) 이상.
- 해안·염해 환경: 피복 두께 10mm 추가 가산 적용.
- 철근 간격: 철근 최소 순간격 — 굵은 골재 최대 치수 4/3배 이상, 25mm 이상, 철근 공칭 직경 이상 중 최대값.
- 이음 길이: 인장 겹침 이음 — D19 이하 40d, D22 이상 50d 기준(fck 21MPa 기준). 기계적 이음(커플러)은 구조 검토 필수.

### 거푸집 공사 기준
- 측압 설계: 콘크리트 타설 속도, 온도, 배합에 따라 산정. 일반적으로 60~120kN/m² 범위.
- 처짐 한계: 거푸집 스팬의 1/400 이하(마감 영향 부재), 1/250 이하(비마감 부재).
- 탈형 강도: 압축강도 5MPa 이상 도달 후 수직 거푸집 탈형. 슬래브·보 하부 동바리는 설계 강도의 2/3 이상 도달 후 탈형.
- 존치 기간: 평균기온 10~20℃ 기준 — 기초·보 측면 2일, 슬래브 하면(스팬 3m 이하) 17일, 스팬 6m 이상 28일.
- 재사용: 거푸집 변형·손상 검사 후 재사용 가능. 결함 부위 수리 또는 교체.

### 콘크리트 타설 기준
- 낙하 높이: 분리(재료분리) 방지를 위해 타설 낙하 높이 1.5m 이하. 슬러이드 파이프 사용 시 3m 이하.
- 진동기(바이브레이터) 간격: 진동 유효 반경 × 1.5배 이하(일반 45cm 진동기 기준 — 삽입 간격 60cm 이하).
- 진동기 삽입 깊이: 하층 타설면 아래 100mm까지 삽입. 철근·거푸집에 직접 접촉 금지.
- 이어치기: 수평 이어치기 — 상부 타설은 하부 콘크리트 강도 1.5MPa 이상 발현 후. 이음면 레이턴스 제거 + 습윤 처리.
- 타설 온도: 콘크리트 온도 5~35℃ 범위. 한중(5℃ 미만) 또는 서중(35℃ 초과) 특별 관리.

### 철골 공사 기준
- 밀착 정밀도: 기둥 베이스 플레이트 수평도 ±1mm/m 이하, 기둥 기울기 1/1,000 이하.
- 용접 검사: 육안 검사(전수), 초음파탐상검사(UT) — 완전용입 용접부 100%. 자분탐상(MT) 또는 침투탐상(PT) — 필요 부위.
- 볼트 조임: 고장력볼트(F10T) 표준 조임 토크 적용. 너트 회전법 또는 토크 관리법. 마킹 후 확인.
- 용접 예열: 강재 두께 25mm 이상 또는 탄소당량(Ceq) 0.4 이상 시 예열 필요(최소 50℃).
- 도장: 현장 용접부 및 볼트 조임부 — 조립 후 보수 도장 필수. 녹 발생 부위 블라스팅 처리 후 도장.

### 프리스트레스 공사 기준 (PS 콘크리트)
- 긴장력: 설계 긴장력 Pp의 80% 이하(초긴장 허용). 즉시 손실(탄성 수축, 활동) 및 장기 손실(크리프, 건조수축, 릴랙세이션) 고려.
- 긴장 작업: PS 강재 인장 시 콘크리트 압축강도 설계기준강도의 75% 이상(최소 28MPa 이상) 확인 후 시행.
- 그라우팅: 긴장 완료 후 덕트 내 시멘트 그라우트 주입. 물-시멘트비 0.35~0.45, 팽창재 혼입 권장. 주입압 0.3~0.5MPa.

## 구조 마스터급 경험 지식 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: structure,MEP-penetration,rebar-BIM,transfer-structure,structural-analysis,field-experience

구조 검토 없이 MEP 관통이 시공된 사례 대응: 코어 드릴링 후 구조기술사에게 현황을 통보하고 긴급 보강 설계를 요청하는 것이 최우선이다. 보강 방법은 관통 크기와 위치에 따라 ① 관통 주변 철근 추가 용접(Carbon Fiber 보강) ② 강판 보강(Steel Plate Epoxy Anchoring) ③ 구조 부재 전체 보강 중 구조기술사가 선택한다. 긴급 보강 설계 없이 마감 처리하거나 은폐하는 것은 법적 책임 문제로 직결되므로 반드시 서면 확인을 받아야 한다.

철근 3D BIM 모델과 실제 배근도 불일치 시 현장 처리 기준: 불일치가 주근 위치 오차(±25mm 이내)이면 시공 허용 오차 범위로 처리하고 준공 AS-BUILT 모델을 현장 기준으로 수정한다. 주근 직경이나 간격이 다른 경우에는 구조 검토서 상 해당 부재의 설계 강도 확인이 필요하며, 강도 미달 여부를 계산하여 보강 여부를 결정한다. LOD 350 이상 철근 BIM 모델에서 이러한 불일치를 시공 전에 발견하는 것이 가장 이상적이며, 현장 배근 QC 체크리스트에 BIM 모델 기준값을 출력해 비교하는 것이 효과적이다.

전이구조 설계 변경 시 BIM 모델 업데이트 우선순위: 전이보·전이 슬래브는 상부 하중 경로가 바뀌기 때문에 전이 부재 변경이 확정되면 ① 구조해석 모델 재해석(상부 수직 부재 하중 재분배) → ② 영향받는 하부 기둥·벽체 단면 검토 → ③ BIM LOD 300 이상 부재 규격 업데이트 → ④ 연관 MEP 관통 및 배관 경로 재검토 순서로 업데이트한다. 전이 부재 변경을 BIM 모델에 반영하지 않고 MEP 작업을 진행하면 관통 허용 여부 판단이 완전히 틀릴 수 있다.

구조해석 결과와 BIM 부재 규격 불일치 조정: 해석 모델(MIDAS, ETABS)과 Revit BIM 모델 간에 기둥 단면이 다를 때 가장 먼저 확인할 것은 해석 모델의 마지막 업데이트 날짜와 BIM 모델의 마지막 업데이트 날짜다. 어느 쪽이 최신인지를 확인한 후 최신 버전을 기준으로 다른 쪽을 동기화한다. Revit과 해석 소프트웨어 간 양방향 동기화 플러그인(RAM Structural System Link, Robot Structural Analysis Link)을 사용하면 반복 작업 오류를 줄일 수 있다.
