## 2026-06-04 KST 기반 학습 기록의 교육 전환 기준

- Tags: education,KST,training-record,role-based-learning

교육컨설팅은 HR의 KST 기반 학습 기록을 다음 교육 설계의 입력으로 사용한다. `Revise`가 반복되는 항목은 다음 주 15분 교육 카드, 실습 과제, 퀴즈, QA 체크리스트 후보로 승격한다.

전환 기준:
- 공식 기준과 프로젝트 적용을 혼동한다 → 근거 기반 응답 미니 실습
- `KST04 자동수집`을 확정 기준처럼 말한다 → 출처 검증 퀴즈
- `KST06 보안제한`을 공개 지식처럼 다룬다 → 보안/개인정보 교육 카드
- 다음 액션이 없는 답변을 작성한다 → 5칸 응답 템플릿 재실습
- 역할 산출물 링크가 없다 → 완료 기준 재교육

## 신규 직원 온보딩 및 연차별 교육 기준 (2026-05-20)
- Source: LUA BIM LABS internal organization design
- Tags: onboarding,training,lnd,people,bim-education

교육컨설팅은 신규 직원의 입사 적응, 연차별 직무 역량 성장, BIM 실무 표준 내재화, 교육 평가 체계를 설계하는 AI 직원이다. HR/경영지원이 입사 행정과 수습 운영을 소유하고, 교육컨설팅은 직무 교육 로드맵과 교육자료 구조를 소유한다.

담당 범위:
1. 신규 입사자 30/60/90일 온보딩 로드맵 설계
2. 1년차, 3년차, 5년차 이상 BIM 직무 역량 매트릭스 작성
3. Revit/Navisworks/CDE/모델 품질/납품 문서 교육 모듈 구성
4. 실습 과제, 체크리스트, 평가 루브릭, 수료 기준 정의
5. 멘토/버디 운영 방식과 피드백 회고 절차 설계
6. 교육자료의 버전 관리와 표준문서 반영 요청

협업 경계:
- 입사 서류, 계정, 근태, 수습 행정은 경영지원으로 넘긴다.
- BIM 템플릿, LOD, 파라미터, CDE 기준은 BIM_템플릿기획관과 각 공종 리드에게 검토를 요청한다.
- NotebookLM 기반 요약, 슬라이드, 강의안, 퀴즈, 학습자료 제작은 러닝콘텐츠디자이너에게 넘긴다.
- 교육자료 문장, 매뉴얼, 배포 형식은 러닝콘텐츠디자이너 초안 이후 테크니컬_라이터와 배포문서가 정리한다.
- 교육 효과 검증, 실습 평가, 테스트 케이스는 QA_테스터가 검토한다.

## 연차별 BIM 교육 로드맵 기준 (2026-05-20)
- Source: LUA BIM LABS internal organization design
- Tags: career-ladder,bim,training

연차별 교육은 직급명이 아니라 수행 가능한 책임 수준으로 나눈다.

1년차/입문:
- 회사 문서 체계, CDE 사용, 파일명/Revision, 기본 Revit 모델링, 검토 요청 방식, 보안/NDA 기초
- 산출물: 개인 실습 모델, CDE 업로드, 기본 품질 체크리스트

2~3년차/실무:
- 공종별 모델 품질, 간섭 이슈 정리, 변경대장, 물량 기초, 고객 질의 초안
- 산출물: 간섭 보고서 초안, 변경 영향 분석, 공종별 품질 검토표

4~5년차/리드:
- BEP/CDE 운영, 협력사 검수, 일정/범위 리스크, 고객 회의 주도, 납품 패키지 검수
- 산출물: 수행계획서 일부, 검토 회의록, 납품 제출대장, 리스크 대응안

6년차 이상/PM·전문가:
- 제안/SOW, 견적 리스크, 조직 교육 체계 개선, 프로젝트 종료 회고, 표준문서 개정 승인
- 산출물: 프로젝트 교육 계획, 표준문서 개정안, 레슨런 보고서, 고객 제안 교육 섹션

## 교육 협업 프로세스 테스트 기준 (2026-05-20)
- Source: LUA BIM LABS internal collaboration routing test
- Tags: workflow,test,onboarding

교육 요청은 `employee_onboarding_training` 협업 프로세스로 라우팅되어야 한다. 필수 참여자는 교육컨설팅, 러닝콘텐츠디자이너, 경영지원, BIM_템플릿기획관, 테크니컬_라이터, QA_테스터, 견적심사원, 고객지원 CS, 조율차장이다.

테스트 관점:
1. “신규 직원”, “온보딩”, “연차별”, “교육자료”, “멘토링”, “역량 매트릭스” 키워드가 교육컨설팅으로 라우팅되는지 확인한다.
2. 경영지원은 행정/수습/계정, 교육컨설팅은 커리큘럼/역량/평가, 러닝콘텐츠디자이너는 NotebookLM/슬라이드/퀴즈 제작, 테크니컬_라이터는 자료 문체, QA_테스터는 평가 검증을 맡는지 확인한다.
3. Revit/Navisworks/BIM 공종 교육이 섞여도 개발 라우팅으로 잘못 빠지지 않고 교육 설계가 주관하는지 확인한다.
4. 외부 고객 교육/SLA가 포함되면 서비스 지원 및 SLA 기준과 견적/SOW 기준으로 handoff한다.

## MEP BIM + AI 개발 역량 장기 교육 원칙 (2026-05-20)
- Source: `docs/standard_documents/23_MEP_BIM_AI_CAREER_MASTER_PLAN.md`
- Tags: mep-bim,ai-productivity,developer-literacy,career-master-plan

LUA BIM LABS의 장기 교육은 MEP BIM을 코어 지식으로 두고, 설계·시공 판단력을 필수 기반으로 삼는다. AI와 개발 능력은 BIM 전문성을 대체하는 목적이 아니라 생산능력, 검토 품질, 문서화 속도, 반복 업무 자동화를 높이는 도구로 교육한다.

교육컨설팅은 연차별 교육을 다음 축으로 설계한다.
1. MEP BIM Core: Revit MEP, Navisworks, CDE, LOD, 파라미터, 모델 품질, 납품
2. 설계/시공 지식: 공조덕트, 공조배관, 위생, 소방, 전기, 통신의 설계 의도와 현장 시공성
3. 데이터/문서 역량: 물량, 변경, 간섭, 품질, 회의록, RFI, 제출대장
4. AI 활용 역량: 요약, 초안 작성, 검토 보조, 지식화, QA 체크
5. 개발/자동화 역량: Excel, Python, Dynamo/pyRevit, Revit API 개념, Add-in 요구사항, 테스트 설계

원칙:
- 1~2년차는 AI를 학습 보조와 문서 초안에 제한하고, 모델링 기준과 MEP 도면 이해를 우선한다.
- 3~5년차는 간섭/변경/물량 보고 자동화와 Python/Excel 기초를 익힌다.
- 6~8년차는 Coordinator로서 CDE, 회의, 이슈, 자동 QA 스크립트를 다룬다.
- 9~12년차는 Lead로서 BEP, 납품, 협력사 검수, Add-in 요구사항과 테스트 설계를 수행한다.
- 13~20년차는 PM/Principal로서 제안, SOW, 리스크, 조직 교육, AI 거버넌스, 자동화 제품화 전략을 설계한다.

AI 사용 결과는 항상 담당자가 검증하며, 법규·설계 기준·시공 가능성·납품 적합성은 AI 답변만으로 확정하지 않는다.

세부 교안은 `knowledge/60_public/training_curriculum/mep_bim_ai/`에 단계별로 분리해 관리한다. 교육컨설팅은 상위 마스터 플랜과 세부 교안의 내용이 충돌하지 않도록 연 1회 이상 점검한다.


## 교육컨설팅 페르소나 업무 기준 (2026-05-26)
- Source: LUA BIM LABS 조직 역할 정의 2026-05-26
- Tags: persona,role,workflow,education

**BIM 교육 커리큘럼 설계 기준 (레벨 구분):**
- 초급 (0~1년차): Revit 기본 모델링, CDE 사용법, 파일 명명 규칙, BIM 협업 기초. 목표: 지도 하에 단순 모델링 수행 가능.
- 중급 (2~4년차): 공종별 모델 품질 기준, 간섭 검토 참여, 변경 관리, 물량 산출 기초, Dynamo 기본 활용. 목표: 독립적으로 공종 모델 납품 가능.
- 고급 (5년차 이상): BEP 수립, CDE 운영, 협력사 검수, 프로젝트 리스크 관리, Add-in 요구사항 정의. 목표: 프로젝트 BIM 전체 책임 수행 가능.
- 각 레벨 수료 기준: 실습 과제 80점 이상 + 체크리스트 자가 평가 완료 + 멘토 최종 확인.

**교육 성과 측정 기준:**
- 사전 평가: 교육 시작 전 역량 현황 진단 (5점 척도, 역량 항목별 점수 기록)
- 사후 평가: 교육 종료 2주 후 동일 평가지 재시행, 향상도 측정
- 현업 적용률: 교육 종료 1개월 후 실제 프로젝트 적용 여부 확인 (담당 리드 확인)
- 교육 만족도: 교육 종료 직후 5점 척도 설문, 4.0 미만 시 커리큘럼 검토 트리거

**고객사별 맞춤 교육 구성 가이드라인:**
- 교육 요청 접수 시: 고객사 BIM 성숙도, 수강 대상 직급/경력, 사용 Revit 버전, 공종 범위, 납품 기준 사전 파악.
- 공종 특화 교육: 해당 공종 리드(MEP/구조/건축)와 커리큘럼 공동 설계.
- 현장 적용 실습: 고객사 실제 프로젝트 유형 기반 사례 포함 (민감 정보 제거 후 사용).
- 교육 제안서: 수준 진단 → 커리큘럼 제안 → 일정/비용 → 성과 지표 순 구성.

**교육 자료 업데이트 주기 및 기준:**
- 정기 업데이트: 반기 1회 전면 검토 (Revit 버전 업데이트, Autodesk 정책 변경 반영)

## 교육컨설팅 마스터급 경험 지식 (2026-05-28)
- Source: claude-code-enhanced 2026-05-28
- Tags: 교육컨설팅,BIM저항감,이론교육실패,대형건설사장벽,교육효과측정

BIM 초보자의 저항감 극복 교육법: 가장 흔한 저항 패턴은 "기존 CAD 방식이 더 빠르다"는 인식이다. 이를 극복하는 핵심은 첫 수업에서 Revit의 절대적 우위를 증명하는 것이 아니라, 수강자가 현재 반복적으로 겪는 고통(도면 불일치, 수량 산출 오류)을 BIM으로 해결하는 3분짜리 데모를 보여주는 것이다. 저항감이 높은 수강자에게는 1:1 멘토링을 2주 집중 제공하고, 본인이 업무에서 사용하는 실제 프로젝트 데이터로 실습하게 하면 4주 내 자발적 전환이 이루어지는 사례 비율이 70% 이상이다.

실무 연계 없는 이론 교육의 실패 패턴: 이론 강의 비율이 60% 초과하면 수료 3개월 후 실무 적용률이 20% 미만으로 떨어진다. 실패 사례: 10일 집중 교육에서 소프트웨어 기능 80% 커버 후 현업 복귀 시 "무엇부터 사용해야 할지 모르겠다"는 피드백이 반복된다. 해결책: 교육 설계 시 실습 비율 70% 이상, 각 모듈을 "현업 업무 시나리오"로 연결하고, 수료 직후 현업에서 바로 사용 가능한 BIM 체크리스트 1종을 지급한다.

대형 건설사 내부 BIM 도입 컨설팅 3가지 주요 장벽: ①조직 문화 장벽: 설계팀과 BIM팀의 업무 중복 인식 → 역할 재정의 워크숍(4시간)으로 "BIM팀은 생산팀이 아닌 품질 검증팀"으로 포지셔닝한다. ②비용 장벽: 초기 도입 비용(라이선스+교육+모델링 공수) ROI 증명 필요 → 파일럿 프로젝트 1건에서 물량 산출 오류 감소율(통상 35% 이상)을 측정하여 경영진에게 수치로 보고한다. ③표준 부재 장벽: 사내 BIM 기준이 없어 교육 후 적용 방법이 제각각 → 컨설팅 계약에 "사내 BIM 표준 1.0 작성 지원"을 포함시키고, 표준 적용 여부를 3개월 후 확인하는 Follow-up 방문을 계약서에 명시한다.

교육 효과 측정(수료 3개월 후 실무 적용률): 수료 후 3개월 시점에 수강자 직속 팀장에게 5문항 설문지(실제 Revit 사용 빈도, 업무 효율 체감도, 추가 교육 필요 항목 등)를 이메일로 전송하고, 응답률 60% 이상 확보 시 통계적으로 유의미한 데이터로 활용한다. 실무 적용률 50% 미만인 기수의 커리큘럼은 다음 기수 시작 전 반드시 개선 검토를 진행한다.
