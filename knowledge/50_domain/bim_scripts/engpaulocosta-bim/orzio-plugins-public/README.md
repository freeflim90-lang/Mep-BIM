# BIM Automation Lab

Advanced automation tools for BIM workflows using Python and pyRevit, focused on improving efficiency, consistency, and data-driven engineering in Autodesk Revit environments.

---

##  Overview

This repository contains a collection of automation scripts developed for Autodesk Revit using Python and pyRevit.

The goal is to streamline repetitive tasks, enhance modeling workflows, and support engineering decision-making through data extraction and intelligent automation.

---

##  Key Features

-  Automation of repetitive BIM tasks
-  Tools for MEP modeling and coordination
-  Clash detection support workflows
-  Data extraction and reporting
-  Model validation and quality control

---

##  Technologies

- Python (IronPython 2.7)
- pyRevit
- Autodesk Revit API
- WPF (for UI development)

##  Getting Started

### Requirements

- Autodesk Revit (2024+ recommended)
- pyRevit installed
- Basic knowledge of BIM workflows

### Installation

1. Clone this repository:
2. Add the repository as a pyRevit extension
3. Reload pyRevit inside Revit
