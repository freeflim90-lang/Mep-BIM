📚 [BIM 일일 교육] Day 141 — 2년차 과정

🏗 오늘의 주제: Python — 파라미터 읽기/쓰기

💡 핵심 개념
Python을 통해 Revit 요소의 파라미터에 접근하고 값을 읽거나 수정하는 것은 BIM 자동화의 핵심입니다. 파라미터 읽기/쓰기를 마스터하면 설계 변수를 일괄 처리할 수 있어 수작업을 획기적으로 줄일 수 있습니다.

🔧 실무 예시
건축 프로젝트에서 모든 문의 높이를 한 번에 2,100mm로 변경해야 할 때, Python 스크립트로 모든 Door 요소의 'Height' 파라미터를 일괄 쓰기할 수 있습니다. 또 다른 예로, 구조 설계 단계에서 기둥의 'Mark' 파라미터를 읽어 스프레드시트로 자동 내보내기 하여 시공 리스트를 생성할 수 있습니다.

💼 오늘의 팁
파라미터 접근 시 요소의 Parameter 컬렉션을 먼저 확인하고, 각 파라미터의 StorageType(텍스트, 숫자, 길이 등)을 파악한 후 적절한 메서드(AsString, AsDouble 등)로 값을 가져오세요.
