## 1. 과정 목표

3~5년차는 지시받은 모델링을 넘어 공종별 품질 문제를 발견하고, 간섭/변경/물량 보고의 초안을 만들 수 있어야 한다.

## 2. 핵심 역량

| 역량 | 목표 |
|---|---|
| 공종 모델 품질 | 단열, 구배, 점검공간, 시스템명, 관경/덕트 크기 검토 |
| 간섭 검토 | Clash 분류, 우선순위, 책임자, 조치기한 정리 |
| 변경 관리 | 변경 전/후 모델 비교, 영향 범위 작성 |
| 물량 기초 | 모델 객체와 내역 항목의 관계 이해 |
| AI/개발 | Python/Excel 자동화 기초, Dynamo/pyRevit 입문 |

## 3. 교육 모듈

| 모듈 | 내용 | 산출물 |
|---|---|---|
| M01 공종별 LOD/파라미터 | MEP 필수 정보, 시스템명, ItemCode | 파라미터 점검표 |
| M02 Navisworks Clash | 규칙, 그룹핑, 상태, 우선순위 | 간섭 보고서 |
| M03 변경 영향 분석 | 변경 전후 비교, 물량/일정 영향 | 변경 영향표 |
| M04 물량 산출 기초 | Revit Schedule, Excel 정리 | 물량 산출서 |
| M05 현장 시공성 | 점검공간, 장비 반입, 단열, 지지 | 시공성 검토표 |
| M06 RFI/회의록 | 질문 구조, 근거 파일, 조치사항 | RFI 초안 |
| M07 Python/Excel | pandas, openpyxl, CSV 정리 | 자동 정리 스크립트 |
| M08 Dynamo/pyRevit | 반복 선택, 파라미터 점검 | 간단한 자동화 |

## 4. 공종별 심화 지식

| 공종 | 필수 지식 | BIM 검토 포인트 |
|---|---|---|
| 공조덕트 | 풍량, 정압, 제연, 점검구 | 덕트 크기, 천장고, 댐퍼 접근 |
| 공조배관 | 냉온수, 냉매, 밸브, 단열 | 단열 포함 외경, 밸브 조작 공간 |
| 위생 | 급수/배수, 통기, 구배 | 배수 구배, 슬리브, 점검구 |
| 소방기계 | 헤드 배치, 알람밸브, 펌프 | 살수 장애, 법규 민감 구간 |
| 전기 | 트레이, 분전반, 강전 이격 | 트레이 경로, 점검/증설 공간 |
| 통신/보안 | 랙, 단말, CCTV, 출입통제 | 전원/통신 경로, 보안 구역 |

## 5. AI/개발 교육

| 항목 | 교육 내용 | 실습 |
|---|---|---|
| 프롬프트 | 간섭 이슈 요약, RFI 초안, 보고서 문장 | Clash 20건 요약 |
| Excel 자동화 | 필터, 피벗, 조건부 서식, Power Query 기초 | 이슈대장 정리 |

| openpyxl | 보고서 시트 생성, 스타일, 필터 | 물량표 자동 생성 |
| Dynamo/pyRevit | 선택 요소 파라미터 점검 | 누락 파라미터 리스트 |
