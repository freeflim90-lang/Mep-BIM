## 가격-가치 정렬 원칙

**고객이 지불하는 금액은 커리큘럼 깊이 + 맞춤화 수준 + 실무 전달 가치에 비례해야 한다.**

가격 인상은 "가격을 올리고 싶다"는 이유가 아니라 **콘텐츠 품질, 서비스 깊이, 납품 역량이 실제로 업그레이드되었을 때**만 정당화된다.

## 가격 인상 트리거 기준

가격 인상은 아래 기준 중 **2가지 이상** 충족 시에만 집행한다.

| 트리거 | 기준 |
|---|---|
| 콘텐츠 업그레이드 | 레슨 수 +20% 이상 추가 또는 새 트랙 추가 |
| 맞춤화 깊이 향상 | 약점 추적, 공종별 특화, 실무 시나리오 리뷰 추가 |
| 수요 검증 | 해당 상품 구독자 50명 이상, 평균 만족도 4.5/5 이상 |
| 납품 채널 확장 | Telegram 전용 → 포털/앱 연동, 영상 자료 추가 |
| 시장 가격 변화 | 유사 글로벌 서비스 대비 가격 경쟁력 재검토 |

## 가격 인상 시 필수 포함 내용

가격 인상 공지에는 다음을 반드시 포함한다:

1. **무엇이 추가되었는가** — 신규 트랙, 레슨 수, 기능 명시
2. **기존 구독자 혜택** — 유예 기간 및 적용 방식
3. **인상 일시** — 최소 30일 전 공지
4. **환불/취소 정책 재확인** — 변경 없음 명시 또는 변경 내용 공지

## 가격-콘텐츠 품질 체크리스트

가격 인상 전 아래 기준을 모두 충족해야 한다:

**콘텐츠 품질:**
- [ ] 신규 레슨의 품질 기준(Why / Core / Real Project / BIM Check / Mistake / Action) 충족
- [ ] 영어 원어민 수준 문장 검토 완료 (또는 프리랜서 검수)
- [ ] 최소 5명 테스트 수강자 피드백 반영

**운영 역량:**
- [ ] 신규 커리큘럼 30일분 이상 레슨 스톡 확보
- [ ] 약점 추적 시스템 (Obsidian 노트) 운영 실적 3개월 이상
- [ ] Q&A 응답 평균 시간 24시간 이내 유지

**시장 검증:**
- [ ] 현재 가격대 만족도 4.5/5 이상 (최소 20명 응답)
- [ ] 유사 글로벌 서비스 대비 가격 포지션 재검토 완료
