## 1. 3줄 판단

1. BF/편의시설 대응은 `품질진단 + 교육 + 컨설팅` 묶음 상품 후보로 승격한다.
2. 단, 자동검수 결과를 BF 적합 또는 인증 통과로 판매하지 않고, `대상성·의무/권장·인증 단계·지자체 조건 확인 리스크를 줄이는 진단 서비스`로 포지셔닝한다.
3. 첫 MVP는 공공건축 또는 BF 요구가 명시된 건축 BIM 프로젝트를 대상으로 2주짜리 파일럿 진단 패키지로 운영한다.

## 2. 근거

| 근거 | 판단 |
|---|---|
| 공식 기준 존재 | 장애인등편의법, 시행령 별표, 시행규칙 별표, BF 인증 규칙, 한국장애인개발원 BF 인증제도 페이지가 공식 출처다. |
| 실무 반복성 | 모델러, QA, 견적심사원, PM, CS가 모두 같은 BF 기준 경계에 걸린다. |
| 고객 가치 | 고객은 인증 통과 보장보다 누락·증빙·재검수·RFI 리스크를 조기에 알고 싶어 한다. |
| 자동화 적합성 | 객체 존재, 파라미터 입력, 증빙 링크, 장애물 후보는 자동화 후보가 된다. |
| 주의 | 대상시설, 의무/권장, 예비인증/본인증, 지자체 조건은 `KST03 적용주의`로 남긴다. |

## 3. 상품 후보

| 후보 | 설명 | 우선순위 | 이유 |
|---|---|---:|---|
| BF BIM 품질진단 라이트 | 모델 내 편의시설 객체, 파라미터, 증빙 링크, 장애물 후보 리포트 | 1 | 현재 QA/모델러 지식과 바로 연결 가능 |
| BF 기준 미확정 리스크 견적 검토 | 대상성·인증 단계·지자체 조건 미확정에 따른 공수/변경 견적 메모 | 2 | 견적심사원 산출물과 연결 가능 |
| BF 실무 15분 교육 카드 | PM/모델러/QA/CS 대상 KST03/KST04 경계 교육 | 3 | 내부 교육 후 외부 교육 상품으로 전환 가능 |

선정 기준:
- 1순위 상품은 `BF BIM 품질진단 라이트`다.
- 인증 컨설팅 전체 대행이 아니라 BIM 모델 품질진단과 리스크 발견으로 범위를 제한한다.
- 자동검수 룰 후보는 제품화 가능성이 있지만, 초기에는 보고서형 컨설팅으로 검증한다.

## 4. MVP 범위

포함:
- 모델 BF 항목 존재/누락 후보 점검
- 파라미터 입력 누락, 더미 값, 증빙 링크 누락 점검
- BF 체크리스트와 도면/뷰/RFI/BCF 연결 상태 확인
- 자동검수 후보와 수동 검토 필요 항목 분리
- `BF 기준 미확정 리스크` 요약 메모 작성

제외:
- BF 인증 통과 보장
- 법률 자문
- 인증기관 최종 판정 대행
- 지자체 해석 확정
- 설계 변경 책임

## 5. 리스크 평가

| 리스크 | 등급 | 대응 |
|---|---|---|
| 자동검수 결과를 인증 통과로 오해할 가능성 | HIGH | 보고서 첫 페이지에 `인증 통과 판정 아님` 문구 고정 |
| 지자체/인증기관 해석 차이 | HIGH | KST03 항목은 RFI 또는 전문가 확인으로 분리 |
| 고객이 설계 책임을 전가할 가능성 | MEDIUM | SOW에 설계 변경 책임 제외 명시 |
| 내부 공수 과소산정 | MEDIUM | 견적심사원 리스크 메모와 재검수 공수 연결 |
| 시장 수요 검증 부족 | MEDIUM | 2주 파일럿 3건으로 반복 질문과 지불 의사 확인 |

## 6. 가격·운영 가설

| 항목 | 가설 |
|---|---|
| 파일럿 기간 | 2주 |
| 대상 고객 | BF 요구가 있는 공공건축, 교육시설, 문화집회시설, 오피스 프로젝트 |
| 산출물 | BF BIM 품질진단 리포트, 리스크 메모, RFI 후보, 자동검수 후보 목록 |
| 내부 투입 | PM 0.5일, BIM 모델러 1일, QA 0.5일, 견적심사원 0.25일 |
| 성공 기준 | 고객이 재검수/보완/RFI 의사결정에 리포트를 사용했다는 피드백 2건 이상 |

## 7. CEO/CSO 결정

결정:
- `BF BIM 품질진단 라이트`를 파일럿 상품 후보로 등록한다.
- 파일럿 전까지 대외 문구에서 `인증 통과`, `법규 적합 확정`, `BF 합격 보장` 표현을 금지한다.
- 내부 산출물은 QA 자동검수 룰 후보, 견적 리스크 메모, 교육 카드와 연결한다.

보류:
- 인증 컨설팅 대행 상품화는 보류한다.
- 자동검수 Add-in 단독 판매는 파일럿 리포트 수요 확인 후 판단한다.
