## 1. 샘플 상황

고객이 다음과 같이 문의했다.

> Revit 모델에서 장애인 화장실 문폭과 접근로 경사가 자동검수상 기준에 안 맞는 것 같습니다. 이 모델은 BF 기준상 불합격이라고 보면 되나요? 바로 수정 지시를 내려도 될까요?

## 2. 근거기반 CS 답변 샘플

```text
안녕하세요, [고객명]님.

문의하신 장애인 화장실 문폭과 접근로 경사 항목에 대한 현재 확인 가능 범위와 추가 확인이 필요한 범위를 분리해 안내드립니다.

1. 결론
현재 자동검수 결과만으로 해당 모델을 “BF 기준상 불합격”이라고 확정해서 안내드리기는 어렵습니다.
다만 문폭, 접근로 경사, 단차, 화장실 주변 장애물 후보는 BF/편의시설 검토에서 중요한 확인 대상이므로, 모델 누락 또는 보완 후보로 우선 표시하고 관련 도면·특기시방서·인증 단계와 함께 대조해야 합니다.

2. 근거
- 장애인등편의법, 시행령 별표, 시행규칙 별표, BF 인증 규칙은 공식 출처로 확인되는 기준입니다.
- LUA BIM LABS의 `2026-06-04_LUA_BIM_LABS_BF_ACCESSIBILITY_STANDARD_UPDATE.md`에서는 BF 검토 시 대상시설 여부, 의무/권장 구분, 예비인증/본인증 단계, 지자체 조건, 증빙 문서를 함께 확인하도록 정리했습니다.
- 내부 QA 기준상 자동검수 결과는 객체 존재, 파라미터 입력, 증빙 링크, 장애물 후보 탐지에 사용하며, 인증 통과 또는 불합격 판정을 단독으로 확정하지 않습니다.

3. 적용 범위
이 답변은 BIM 모델의 BF/편의시설 검토 후보를 정리하는 1차 기술지원 답변입니다.
실제 BF 적용 여부와 최종 적합성은 시설 용도, 규모, 공공/민간 여부, 발주처 요구, 예비인증/본인증 단계, 지자체 사전검사 조건, 인증기관 또는 설계자 검토 결과에 따라 달라질 수 있습니다.

4. 주의
자동검수에서 발견된 문폭, 경사, 단차, 회전반경 등 수치 후보는 `KST04 자동수집` 또는 자동검수 후보로 보고, 공식 전문과 프로젝트 조건 확인 전에는 확정 불합격 근거로 사용하지 않습니다.
프로젝트 적용 여부는 `KST03 적용주의`로 안내해야 하며, CS가 인증 통과·불합격·설계 변경 책임을 단독으로 확정하지 않습니다.

5. 다음 액션
정확한 검토를 위해 아래 자료를 공유해 주세요.
- BF 인증 대상 여부 또는 발주처 BF 요구 문서
- 예비인증/본인증 등 현재 인증 단계
- 관련 도면 번호와 특기시방서
- 문제가 표시된 Revit 뷰, 요소 ID, 자동검수 리포트
- 지자체 사전검사 또는 인증기관 질의응답이 있다면 해당 문서

자료가 확인되면 LUA BIM LABS에서 자동검수 후보와 수동 검토 필요 항목을 분리해 리포트하고, 확정 판단이 필요한 항목은 PM/RFI 또는 설계자·인증기관 확인 절차로 전환하겠습니다.

LUA BIM LABS | [이메일]
```

## 3. CS 판단 체크리스트

| 확인 항목 | CS 응답 기준 |
|---|---|
| 고객이 “불합격인가요?”라고 묻는가 | 자동검수만으로 불합격 확정 금지 |
| 공식 기준이 있는가 | `KST01 공식확인`으로 근거를 제시하되 프로젝트 적용은 분리 |
| 대상시설·인증 단계가 확인됐는가 | 미확정이면 `KST03 적용주의`로 안내 |
| 자동검수 수치를 근거로 삼는가 | `KST04 자동수집` 또는 자동검수 후보로 표현 |
| 설계 변경 지시가 필요한가 | CS 단독 지시 금지, PM/RFI/설계자 확인으로 전환 |
| 고객 자료에 개인정보/프로젝트명이 있는가 | Obsidian FAQ 등록 전 익명화 |

## 4. 좋은 답변의 조건

- 결론에서 확정 가능 범위와 확정 불가 범위를 먼저 분리한다.
- 공식 출처와 내부 QA 기준을 함께 제시한다.
- 대상시설, 의무/권장, 예비인증/본인증, 지자체 조건을 반드시 확인 항목으로 남긴다.
- 고객이 다음에 제공해야 할 자료를 구체적으로 요청한다.
- 인증 통과, 불합격, 설계 변경 책임을 CS 답변에서 단정하지 않는다.

## 5. 피해야 할 답변

- “자동검수에서 빨간색이면 BF 불합격입니다.”
- “문폭 수치가 맞지 않으니 바로 수정 지시하세요.”
- “BF 기준은 법령에 있으니 모든 프로젝트에 동일하게 적용됩니다.”
- “지자체나 인증기관 확인 없이 내부 기준으로 확정해도 됩니다.”
- “이 모델은 LUA BIM LABS 기준상 인증 통과가 어렵습니다.”
