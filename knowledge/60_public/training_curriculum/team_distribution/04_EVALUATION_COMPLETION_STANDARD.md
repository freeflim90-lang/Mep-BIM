# 평가 및 수료 기준

문서상태: 팀 배포용 초안  
작성일: 2026-05-20

---

## 1. 평가 원칙

평가는 단순 출석이 아니라 실무 산출물의 품질, 기준 준수, 설명 가능성, 검증 태도를 기준으로 한다.

---

## 2. 평가 비중

| 평가 축 | 비중 | 설명 |
|---|---:|---|
| MEP BIM 정확도 | 30% | 모델, 파라미터, 좌표, LOD, 납품 형식 |
| 설계/시공 이해 | 20% | 공종별 시스템, 시공성, 점검공간, 변경 영향 |
| 데이터/문서 운영 | 15% | 이슈, 변경, 물량, 보고서, 제출대장 |
| 품질/리스크 판단 | 15% | 간섭, 오류, 우선순위, 리스크 설명 |
| AI/자동화 활용 | 10% | 초안 생성, 반복 업무 개선, 검증 기록 |
| 협업/커뮤니케이션 | 10% | 회의, RFI, 피드백, 후배 교육 |

---

## 3. 수료 기준

| 구분 | 기준 |
|---|---|
| 출석 | 교육 시간 80% 이상 참여 |
| 산출물 | 필수 산출물 100% 제출 |
| 품질 | 핵심 평가 항목 평균 70점 이상 |
| 보완 | Critical 지적사항은 보완 후 재제출 |
| 회고 | 개인 회고 및 다음 개선 과제 제출 |

---

## 4. 공통 평가표

| 항목 | 점수 | 평가 메모 |
|---|---:|---|
| 모델 기준 준수 | /20 | |
| 도면/설계 의도 이해 | /15 | |
| 간섭/품질 이슈 판단 | /15 | |
| 산출물 문서화 | /15 | |
| CDE/파일 관리 | /10 | |
| 설명 및 커뮤니케이션 | /10 | |
| AI/자동화 결과 검증 | /10 | |
| 회고 및 개선 태도 | /5 | |
| 합계 | /100 | |

---

## 5. 수료 산출물 예시

| 단계 | 필수 산출물 |
|---|---|
| Foundation | 기본 모델, 품질 체크리스트, 파일 제출대장 |
| Practitioner | 공종별 품질 검토표, 간섭 보고서, 변경 영향표 |
| Coordinator | 회의록, 조율 이슈 대장, CDE 배포 이력표 |
| Lead | BEP 섹션, 납품 검수표, 협력사 검수 의견서 |
| PM/Consultant | 제안서, SOW, 리스크/ROI 분석 |
| Principal | 교육 체계 개정안, 방법론 문서 |
