# CEO Q&A 지식

텔레그램 질문·자동 수집 응답 저장소 — 기준 가이드라인(CEO.md)과 분리 운영.

## 2026-06-09 CEO 전략 판단 실전 Q&A
- Source: LUA BIM LABS CEO 운영 기준, KST 태그 체계
- Tags: ceo,strategy,product,revenue,qa,2026

**Q: BIM Command Center App Store 제출 준비가 됐는지 CEO는 어떻게 판단하나요?**
A: CEO 판단 기준은 3가지다. ① 스토어심사 에이전트가 차단 이슈 0개를 확인했는가 ② 라이선스/결제/개인정보 처리방침이 완비됐는가 ③ QA 테스터가 지원 버전(Revit 2023~2026) smoke test를 통과했는가. 세 가지 모두 통과하면 CEO는 제출 승인을 내린다. 하나라도 미통과면 제출 보류다. *(KST01 — Autodesk 심사 기준)*

**Q: Starter Plan 가격은 어떻게 결정하나요?**
A: 가격 결정은 CFO가 MRR/유료전환율/지원비용을 계산한 뒤 CEO가 최종 승인하는 구조다. 현재 기준은 월 구독 기준 시장 중간 포지셔닝이며, 첫 출시 후 90일 피드백 기간에 조정한다. 가격 확정 전에는 내부 기준값을 고객에게 안내하지 않는다. *(KST03 — 검증 전 수치 확정 금지)*

**Q: R&D 예산이 부족할 때 CEO는 어떤 기준으로 우선순위를 정하나요?**
A: CEO의 우선순위 기준: ① 고객 반복 고통(pain) 크기 ② 3개월 내 수익 기여 가능성 ③ 기술 의존성(외부 API 없이 구현 가능한지). 이 세 가지 중 2개 이상을 충족하는 기능이 최우선이다. 전략기획이 후보 목록을 정리하면 CEO가 최종 cut 결정을 내린다.

**Q: 신규 파트너십 제안이 왔을 때 CEO 검토 기준은?**
A: CEO가 먼저 보는 항목은 ① 파트너가 실사용자인가(빌더 vs 바이어) ② LUA BIM LABS 제품이 파트너 워크플로우에서 반복 사용되는가 ③ 독점 조항·데이터 귀속 조항이 없는가. 세 가지 모두 OK이면 COO에게 계약 검토를 위임한다. 하나라도 애매하면 법무조항검토 에이전트에 먼저 넘긴다.

**Q: CEO가 "이 결정이 최종인가"라고 물을 때 AI는 어떻게 답변해야 하나요?**
A: AI는 "CEO가 확인하면 확정입니다"라고 답변하고 결정 내용, 근거, 실행 담당, 기한을 정리해 제시한다. AI 단독으로 "확정"을 선언하지 않는다. 대표 또는 소유 AI의 명시적 첨언이 없으면 가설 상태다. *(AI 협업 운영 기준)*

**Q: AX 원스톱 바우처 신청 타당성은 어떻게 검토하나요?**
A: CEO 검토 기준: ① 과제 요건(기업 규모, 매출, 업종)이 충족되는가 ② 신청 가능한 과제 범위(소프트웨어 개발, BIM, AI)에 해당하는가 ③ 공수 대비 지원금 ROI가 2배 이상인가. 타당하면 COO에게 신청 서류 준비를 지시하고 CFO에게 예산 영향 검토를 요청한다.
