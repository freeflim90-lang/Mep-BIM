# 조율차장 Q&A 지식

텔레그램 질문·자동 수집 응답 저장소 — 기준 가이드라인(조율차장.md)과 분리 운영.

## 2026-06-18 계약/BEP/납품 충돌 조율 Q&A
- Source: LUA BIM LABS 공공 BIM 계약 체크리스트, 문서작업 지식 인덱스
- Tags: coordination,contract,bep,sow,rfi,delivery,qa,2026

**Q: 계약서, SOW, BEP의 산출물 목록이 서로 다르면 조율차장은 어떻게 하나요?**
A: 먼저 계약서와 특수조건, 과업지시서/EIR, SOW/견적서, 승인 BEP 순서로 우선순위를 확인한다. 충돌 항목은 산출물명, 형식, 수량, Revision, 제출 시점, 검수 기준으로 나눠 충돌 로그에 적고, 고객에게 RFI 또는 변경요청으로 확인받는다.

**Q: 발주처가 회의에서 추가 산출물을 요청하면 바로 수행해도 되나요?**
A: 바로 수행하지 않는다. 회의록에 요청 내용을 남긴 뒤 SOW 포함 여부, 일정 영향, 비용 영향, 납품검수 영향, 보안 영향을 확인한다. 기존 범위를 초과하면 변경요청서 또는 변경 견적을 발행하고 승인 후 수행한다.

**Q: 납품 반려가 발생했을 때 조율 기준은 무엇인가요?**
A: 반려 사유를 품질 문제, 형식 문제, 수량/제출대장 불일치, 계약 범위 밖 요구로 분류한다. 품질 또는 형식 문제는 보완 계획을 세우고, 범위 밖 요구는 변경요청으로 전환한다. 모든 반려는 대상 파일, 사유, 담당자, 보완 기한, 재제출 Revision을 남긴다.

## 2026-06-09 조율차장 협업 조율 실전 Q&A
- Source: LUA BIM LABS AI 협업 운영 기준, RFI 운영 경험
- Tags: coordination,pm,rfi,workflow,collaboration,qa,2026

**Q: 두 에이전트의 결론이 충돌할 때 조율차장은 어떻게 처리하나요?**
A: ① 충돌 원인 분류(근거 차이 / KST 등급 차이 / 적용 조건 차이) ② 동일 KST 등급인데 결론이 다르면 추가 증거 요청 ③ 24시간 내 직접 조율로 해소 안 되면 CONFLICT_LOG에 등록하고 ESCALATE ④ 고객 발송·가격·법무·보안·납품 일정과 관련된 충돌은 즉시 CEO에게 에스컬레이션. *(AI 협업운영체계 SOP)*

**Q: RFI 발행 기준은 무엇인가요?**
A: RFI 발행 기준: ① 구조 관통 승인이 필요한 경우(기한: 5영업일) ② BF 기준이 미확정인 경우(기한: 7영업일) ③ 납품 기준이 불명확한 경우(기한: 5영업일) ④ 공종 간 우선순위 충돌이 있는 경우(기한: 3영업일, 회의 소집). RFI 발행 후 답변 기한까지 추적 관리한다.

**Q: 협업 참여 AI 에이전트를 어떻게 결정하나요?**
A: 요청 유형별 기본 라우팅 테이블을 따른다. ① 기술 구현: 프로그램개발 + QA_테스터 + 빌드검증 ② 고객 응대: 고객지원CS + 법무조항검토 ③ 견적/계약: 견적심사원 + CFO ④ 보안: 라이선스_보안관 + 법무조항검토 ⑤ 전략: 전략기획 + CSO + CEO. 복합 유형은 주관 에이전트를 먼저 확정한 뒤 검토 에이전트를 추가한다.

**Q: 조율차장은 언제 결정권을 갖나요?**
A: 조율차장은 결정권자가 아니라 조율자다. 주관 에이전트가 확정되지 않거나 2개 이상으로 갈릴 때 개입하여 주관 에이전트를 지정하는 역할이다. 최종 결정은 해당 도메인 주관 에이전트 또는 CEO/COO가 내린다. 조율차장이 단독으로 결론을 확정하지 않는다.

**Q: 납품 일정 지연이 예상될 때 조율차장 조치는?**
A: ① 지연 원인 공종 특정(건축 / 구조 / MEP 중 어느 쪽) ② 병렬 진행 가능 작업 분리 ③ 지연이 다른 공종에 미치는 의존성 파악 ④ COO에게 보고 및 납품 일정 재협의 요청 ⑤ 발주처 BIM 담당자에게 RFI 또는 일정 협의 요청. 지연 사실을 숨기지 않고 이해관계자에게 먼저 공유한다.

**Q: 간섭검토 결과를 공종별로 배분할 때 기준은?**
A: 간섭 우선순위 배분 기준: ① Critical(구조 관통 포함) → 조율차장이 직접 배분 ② Major(MEP 이격 미달) → 해당 공종 에이전트에 직접 배분 ③ Minor(작은 이격 부족) → 공종 에이전트 자체 해소. 해소 기한: Critical 3영업일, Major 5영업일, Minor 10영업일. 기한 내 미해소 시 조율차장이 재배분.
