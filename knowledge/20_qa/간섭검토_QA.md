**Q: Revit Interference Check와 Navisworks Clash Detective 중 어떤 걸 써야 하나요?**
A: 사용 목적이 다릅니다.
- **Revit Interference Check**: 설계 중 즉시 자체 점검용. Hard Clash만 잡히고 규칙 설정 불가. 공식 조율 보고서로는 부적합.
- **Navisworks Clash Detective**: 공종 통합 후 공식 간섭 보고. Hard/Soft/Clearance Clash 모두 가능, Rule 설정으로 같은 계통 내부 충돌 필터링, BCF→RFI 연동, 결과 보고서 자동 출력.
→ **설계 자체 점검은 Revit, 공식 다공종 조율은 Navisworks.**

**Q: 구조 보에 배관을 관통할 수 있나요?**
A: 조건부 허용입니다.
- **기둥·전이보: 절대 불가** — 우회 필수.
- **일반 보**: 보 높이 중앙 1/3 구간, 관통경 ≤ 보 춤의 1/3 이하일 때 구조기술사 검토 후 허용 가능.
- **PC보**: 공장 제작 전 슬리브 위치 확정 필수. 현장 천공 불가.
- 승인 없이 관통한 경우 구조 안전성 문제로 책임 소지 발생.

**Q: 오배수 배관이 덕트와 충돌하면 어느 쪽을 이동시켜야 하나요?**
A: 오배수 배관을 보존하고 덕트를 우회시킵니다.
오배수는 자연유하 방식이라 구배(DN100 기준 1/100)가 필수입니다. 구배 방향으로 레벨이 낮아지므로 경로 조정 여지가 없습니다. 덕트는 상향 우회 또는 경로 변경이 상대적으로 쉽습니다. 보고서 문구: "오배수 구배 구간(DN100, 1/100)은 레벨 보존 우선 — 공조 덕트 상향 우회 검토."

**Q: 스프링클러 헤드 위에 덕트가 있어도 되나요?**
A: 살수 장애 여부를 반드시 확인해야 합니다.
NFTC 103 기준으로 헤드 아래 수평 1.2m 이상 폭의 장애물은 살수 장애 구간입니다. 덕트 폭이 1.2m 이상이거나 헤드와 덕트의 위치 관계에 따라 측벽형 헤드 변경 또는 헤드 위치 조정이 필요합니다. Navisworks에서 Clearance Clash +1200mm 설정으로 자동 탐지 가능합니다.

**Q: 강전 케이블 트레이와 약전 트레이는 얼마나 이격해야 하나요?**
A: **300mm 이상** 이격이 원칙입니다 (KEC·내선규정 기준).
이격이 불가능할 경우 **금속 격벽(차폐판)** 설치로 대체할 수 있습니다. 통신 케이블(Cat6)과 일반 전력선 간은 TIA-569 기준 150mm 이상입니다. BIM 간섭 검토에서 이격 부족은 Hard Clash가 아닌 Clearance Clash로 분류하되, 전기실·데이터룸에서는 Critical 등급으로 처리합니다.

**Q: Navisworks에서 All vs All 클래시 테스트를 돌렸더니 수천 건이 나왔는데 어떻게 해야 하나요?**
A: Rule 설정 없이 All vs All을 돌리면 같은 계통 내부 충돌(소방 배관 vs 소방 배관), 설계 의도적 겹침(행거 브라켓 중복)이 수천 건 발생해 실제 문제가 묻힙니다.
Rule 탭에서 `Same System`, `Same Layer` 제외를 설정하고, 공차 50mm를 적용한 뒤 공종 조합별로 Selection A/B를 분리해서 테스트를 재실행하세요. 예: Selection A = 소방배관 / Selection B = 구조체.

**Q: 방화댐퍼 설치 위치 기준이 어떻게 되나요?**
A: 방화구획 경계 벽체·슬래브에 **밀착 설치**가 원칙입니다.
벽체에서 300mm 이상 이격하면 소방 허가 반려 사유가 됩니다. 댐퍼 전후 300mm 이내 구간은 불연재 덕트를 사용하고, 댐퍼에서 600mm 이내에 점검구 설치가 의무입니다(KCS 시방). 퓨즈 작동 온도는 일반 구역 72°C, 주방 배기 182°C를 적용합니다.

**Q: 천장 속에 설비를 배치하는 순서가 어떻게 되나요?**
A: 아래 순서로 레벨을 확정해가며 배치합니다.
1. **구조 보 하단고 확인** — 설비 공간 최대 높이 결정
2. **오배수 배관 레벨 확보** — 구배 방향으로 하단고 먼저 결정
3. **제연 덕트 경로 고정** — 소방법 최우선
4. **소화 배관 배치** — 헤드 위치·살수 반경 확인
5. **공조 주 덕트(SA/RA 600mm 이상) 배치** — 직선 경로 확보
6. **전기 케이블 트레이(강전) 배치** — 배관 직상부 금지
7. **통신 트레이(약전) 배치** — 강전 트레이에서 300mm 이격
8. **공조 소구경 배관·분기 덕트·플렉시블·말단 배선** — 나머지 공간 활용
9. **디퓨저·스프링클러 헤드·조명·감지기·점검구** 천장 면 최종 확정

**Q: 간섭 보고서에는 어떤 항목이 들어가야 하나요?**
A: 공식 간섭 보고서 필수 항목:
클래시ID / 등급(Critical·Major·Minor) / 공종A / 공종B / 레벨 / 위치(X/Y/Z) / 이격거리(mm) / 상태(Active·Resolved 등) / 해결방안 / 담당자.
Navisworks에서 HTML/XML로 자동 출력하고, 위치 뷰포인트(스냅샷)를 포함시킵니다. BCF 파일로 내보내면 BIM 360/ACC RFI와 자동 연동됩니다.
