## Summary

`Model Quality Auditor`는 Autodesk App Store 판매를 목표로 하는 모델 품질진단 Add-in 제품이다. - Revit 모델 품질 상태를 로컬에서 진단한다. - 외부 AI/클라우드 전송 없이 리포트 초안을 생성한다. - 품질 점수, 이슈 목록, 우선순위, 수정 권장사항을 제공한다. - 추후 Revit API 가능한 환경에서 빌드/테스트 후 Addin Dashboard 패키지에 합류한다.
