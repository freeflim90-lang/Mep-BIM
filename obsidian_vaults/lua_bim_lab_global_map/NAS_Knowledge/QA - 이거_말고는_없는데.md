# QA - 이거 말고는 없는데?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

이거 말고는 없는데?

## 답변

라이브러리에서 먼저 **Custom Search JSON API**를 활성화해야 목록에 나타납니다.

왼쪽 사이드바 **"라이브러리"** 클릭 → 검색창에 `Custom Search JSON API` 입력 → **사용 설정** 클릭해주세요.
