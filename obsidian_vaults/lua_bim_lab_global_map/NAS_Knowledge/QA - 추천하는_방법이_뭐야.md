# QA - 추천하는 방법이 뭐야?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

추천하는 방법이 뭐야?

## 답변

**CLI (wrangler) 방식 추천합니다.**

이유:
- 코드 수정 후 터미널 명령어 한 줄로 재배포 가능
- 나중에 자동화(LaunchAgent, GitHub Actions) 연결하기 쉬움
- 대시보드 업로드는 매번 파일 찾아서 드래그해야 함

**API 토큰 만드는 방법 (2분):**

1. [dash.cloudflare.com/profile/api-tokens](https://dash.cloudflare.com/profile/api-tokens) 접속
2. **Create Token** → **"Edit Cloudflare Workers" 템플릿** 선택
3. 생성 후 토큰 복사

토큰 받으시면 바로 배포해드릴게요.
