# QA - 1~10년차 순차적으로 생성하되 개별 팀원에 따라 만들 필요는 없어

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

1~10년차 순차적으로 생성하되 개별 팀원에 따라 만들 필요는 없어

## 답변

맞습니다 - 현재 구조 그대로입니다. 트랙당 1세트(`1yr/`, `2yr/` ... `10yr/`)만 생성하고, 팀원들은 그 트랙을 공유합니다.

현재 돌고 있는 백그라운드 작업은 1~2년차만 포함된 구버전이라, 완료 후 3~10년차를 이어서 실행해야 합니다. 지금 상태 확인하고 큐 잡겠습니다.
