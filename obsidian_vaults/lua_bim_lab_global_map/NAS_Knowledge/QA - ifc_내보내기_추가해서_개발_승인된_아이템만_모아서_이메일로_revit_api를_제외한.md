# QA - ifc 내보내기 추가해서 개발 승인된 아이템만 모아서 이메일로 revit api를 제외한

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-31
> **출처:** Claude Code 세션

ifc 내보내기 추가해서 개발 승인된 아이템만 모아서 이메일로 revit api를 제외한 나머지 개발 완료하여 발송해

## 답변

채택 확정 목록 + IFC 승인 확인했습니다. 6차 러너 작성 후 즉시 실행합니다.

**이번에 개발할 것 (Revit API 제외):**
- 수정 항목: 링크 3버튼 / QA 삭제기능 / 인쇄 셋팅 / N-01 우선순위 룰 / N-02 좌표 그룹화 / N-03 SelectionSet 자동 연결
- 신규: N-05 IFC 내보내기 (승인)
- Revit API 호출부는 `// REVIT_API:` 마커로 표시, 개발 PC에서 완성
