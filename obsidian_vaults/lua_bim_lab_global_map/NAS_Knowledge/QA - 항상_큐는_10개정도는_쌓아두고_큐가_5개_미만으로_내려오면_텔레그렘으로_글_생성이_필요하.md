# QA - 항상 큐는 10개정도는 쌓아두고 큐가 5개 미만으로 내려오면 텔레그렘으로 글 생성이 필요하

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-06-09
> **출처:** Claude Code 세션

항상 큐는 10개정도는 쌓아두고 큐가 5개 미만으로 내려오면 텔레그렘으로 글 생성이 필요하다고 알림을 주면 내가 claude code를 통해서 글 생성할께

## 답변

두 가지 수정이 필요하네요:
1. 발행 후 큐 개수 확인 → 5개 미만이면 텔레그램 알림
2. plist에서 `--generate-when-empty` 제거 (더 이상 자동 생성 안 함)
