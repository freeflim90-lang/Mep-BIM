# QA - 위생 workset에 system type이 sa로 덕트 관련된 공종이 포함되면 몇개의 객

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-06-06
> **출처:** Claude Code 세션

위생 workset에 system type이 sa로 덕트 관련된 공종이 포함되면 몇개의 객체가 해당 공종 workset으로 잘못들어왔는지 확인도 가능하면 좋겠어
