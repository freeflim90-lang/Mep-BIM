# QA - 무료로 사용 가능한 검색 엔진이 또 있을까?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

무료로 사용 가능한 검색 엔진이 또 있을까?

## 답변

현재 Tavily + DuckDuckGo 두 개가 이미 작동 중입니다. 추가 무료 옵션으로:

**한국어에 특히 유용한 것:**
- **Naver 검색 API** — 완전 무료, 한국어 콘텐츠 강함, developers.naver.com 에서 앱 등록만 하면 됨. 건축법규, 면적 같은 한국 정보는 네이버가 Tavily/DDG보다 훨씬 정확함

**범용 구글 검색 대체:**
- **Serper.dev** — 구글 검색 결과를 API로 제공, 가입 시 2500건 무료
- **Brave Search API** — 월 2000건 무료, 키 발급 쉬움

한국어 질문이 많은 이 시스템에는 **Naver API 추가**가 가장 효과적일 것 같습니다. 추가해볼까요?
