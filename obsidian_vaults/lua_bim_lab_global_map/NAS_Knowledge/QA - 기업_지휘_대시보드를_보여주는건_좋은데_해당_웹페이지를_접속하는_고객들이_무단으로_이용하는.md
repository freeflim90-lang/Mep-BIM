# QA - 기업 지휘 대시보드를 보여주는건 좋은데 해당 웹페이지를 접속하는 고객들이 무단으로 이용하는

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

기업 지휘 대시보드를 보여주는건 좋은데 해당 웹페이지를 접속하는 고객들이 무단으로 이용하는 건 못하게 해야되

## 답변

두 가지를 동시에 해야 합니다:
1. **공개 홈페이지에서 대시보드 iframe 제거** (지금 누구나 볼 수 있는 상태)
2. **Cloudflare Access로 대시보드 접근 인증** 추가

먼저 홈페이지 iframe부터 제거합니다.
