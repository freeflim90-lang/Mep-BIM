# QA - 내 생각에는 질문을 검색을 기본적으로 하고 그에 따른 검색답변을 기준으로하여 지식저장된 내

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

내 생각에는 질문을 검색을 기본적으로 하고 그에 따른 검색답변을 기준으로하여 지식저장된 내용과 합쳐서 요약하여 답변주는게 좋을거 같아 지식내에서 분류하여 답변하는게 어려워보이는데|?

## 답변

좋은 방향입니다. 현재 구조(로컬 우선 → 품질 판단 → 실패 시 웹)가 분류 오류, 품질 게이트 오작동 등 복잡성을 만들고 있습니다.

제안하신 구조로 단순화합니다:

**새 흐름**: 웹 검색(항상) + 로컬 지식(있으면 참고) → 합쳐서 요약 → 답변 1개
