# QA - 재시작 한번해보고 안되면 나중에 내가 다시 요청할께

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-31
> **출처:** Claude Code 세션

재시작 한번해보고 안되면 나중에 내가 다시 요청할께

## 답변

이미 다른 프로세스(PID 8836)가 실행 중이었고 제가 시작한 건 lock 감지 후 종료됐습니다. 166개 파일이 큐에 남아있고 기존 프로세스가 계속 작업 중입니다.
