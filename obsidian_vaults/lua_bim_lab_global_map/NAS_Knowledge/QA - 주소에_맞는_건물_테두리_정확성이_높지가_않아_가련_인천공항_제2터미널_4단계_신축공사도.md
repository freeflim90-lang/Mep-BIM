# QA - 주소에 맞는 건물 테두리 정확성이 높지가 않아 가련 인천공항 제2터미널 4단계 신축공사도

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-25
> **출처:** Claude Code 세션

주소에 맞는 건물 테두리 정확성이 높지가 않아 가련 인천공항 제2터미널 4단계 신축공사도 신설된 구역에 대한 표시가 제대로 폴리라인으로 표현이 안되고 엉뚱한 위치에 표현되어있어

## 답변

OSM 자동 조회를 제거하고, **관리자가 지도에서 직접 건물 외곽선을 클릭해 그리는 방식**으로 전환합니다. 저장된 폴리곤이 있으면 그것을 우선 표시합니다.
