# QA - 사용자는 약어 rgb 값만 입력해도 되는지해서

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-27
> **출처:** Claude Code 세션

사용자는 약어 rgb 값만 입력해도 되는지해서

## 답변

맞습니다. 사용자 입장에서는 약어와 RGB 값만 입력하면 됩니다.

실제 사용 흐름은:

1. **내보내기** → 시스템명은 Revit에서 자동으로 채워진 채로 CSV 생성
2. **사용자 편집** → 약어, R, G, B 4개 열만 채움 (시스템명은 건드리지 않음)
3. **가져오기** → 시스템명으로 자동 매칭 후 적용

시스템명은 매칭 키 역할을 하는 거라 사용자가 직접 입력할 필요는 없고, 내보내기 할 때 자동으로 생성됩니다. 현재 구현이 이 흐름과 정확히 맞습니다.
