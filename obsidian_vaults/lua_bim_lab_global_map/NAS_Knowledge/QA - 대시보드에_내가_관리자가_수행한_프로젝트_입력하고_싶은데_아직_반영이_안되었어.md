# QA - 대시보드에 내가 관리자가 수행한 프로젝트 입력하고 싶은데 아직 반영이 안되었어

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

대시보드에 내가 관리자가 수행한 프로젝트 입력하고 싶은데 아직 반영이 안되었어

## 답변

백엔드와 API는 정상 동작하는데, 문제를 파악했습니다. 등록한 프로젝트가 지도에 바로 표시되지 않아서 "반영 안 됨"으로 느끼는 것입니다. 현재는 실제 sync 데이터가 있어야만 지도에 나타나는데, **등록 즉시 지도에 핀으로 표시**되도록 수정하겠습니다. 테스트 데이터도 정리합니다.
