# QA - docker 비용이 들어?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

docker 비용이 들어?

## 답변

**Docker Engine (서버용) — 완전 무료입니다.**

서버에 설치하는 Docker Engine은 오픈소스라 비용 없어요.

**유료인 경우:**
- **Docker Desktop** (Mac/Windows GUI 앱) — 대기업(직원 250명+ 또는 연매출 $10M+)은 유료, 개인/소규모는 무료
- **Docker Hub** — 이미지 저장소, 무료 플랜으로 개인 용도 충분

**개인 서버 용도라면 드는 비용은 0원입니다.** 서버 OS(Linux)에 Docker Engine 설치해서 쓰는 건 완전 무료예요.

서버 OS가 뭔가요? Ubuntu/Debian이면 설치 명령어 바로 드릴게요.
