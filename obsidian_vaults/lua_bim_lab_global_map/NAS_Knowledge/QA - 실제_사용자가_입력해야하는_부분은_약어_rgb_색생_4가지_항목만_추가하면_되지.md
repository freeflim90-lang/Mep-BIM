# QA - 실제 사용자가 입력해야하는 부분은 약어 rgb 색생 4가지 항목만 추가하면 되지?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-27
> **출처:** Claude Code 세션

실제 사용자가 입력해야하는 부분은 약어 rgb 색생 4가지 항목만 추가하면 되지?

## 답변

맞습니다. 시스템명(`SystemName`)은 Revit에서 자동 수집되므로 사용자가 직접 입력해야 하는 항목은 4가지뿐입니다:

| 항목 | 역할 |
|---|---|
| **약어** | 필터 레이블용 줄임말 |
| **R** | 색상 Red (0~255) |
| **G** | 색상 Green (0~255) |
| **B** | 색상 Blue (0~255) |

`시스템명`은 Revit 프로젝트에서 읽어온 키값으로 Excel과 매칭하는 데만 사용되고, `PIN잠금`은 내보내기엔 포함되지만 사용자가 필수로 입력해야 하는 값은 아닙니다.

그렇다면 현재 CSV 내보내기 형식도 약어와 RGB만 채우면 되는 형태로 좀 더 단순화할까요? 예를 들어 PIN잠금 열을 제외하고, Excel에서 편집 시 사용자가 채워야 할 열(약어, R, G, B)에 열 색상 강조라도 안내 행으로 넣어주는 방식으로 개선할 수 있습니다.
