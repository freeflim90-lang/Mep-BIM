# QA - 반지름이 아니라 건물 테두리를 기반으로 표현되기를 바래

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-25
> **출처:** Claude Code 세션

반지름이 아니라 건물 테두리를 기반으로 표현되기를 바래

## 답변

`L.circleMarker`(픽셀)도 `L.circle`(반지름)도 아니라 **GFA 기반 정사각형 폴리곤**으로 즉시 그리고, 백그라운드에서 **OSM Overpass API의 실제 건물 외곽선**으로 자동 교체하는 방식으로 구현합니다.
