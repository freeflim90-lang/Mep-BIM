# QA - 새로 1~5차 추가된 기능에 대해서 리스트와 설명을 주면 내가 보고 판단을 해서 필요한지

## 1~5차 신규 개발 기능 전체 목록

### Revit Add-in (19개)

---

**[도면·주석]**

| # | 폴더 | 버튼명(안) | 기능 설명 |
|---|---|---|---|
| R-01 | `TagTextAligner` | **태그/문자 정렬** | 선택한 IndependentTag·TextNote를 좌/우/상/하 정렬 또는 균등 배분. 핀된 주석 자동 스킵. |
| R-06 | `LineCleanupLite` | **선 정리** | 현재 뷰 내 중복 선·단선(10mm 이하) 감지 후 삭제. CAD Import 잔재 정리에 특화. |

---

**[뷰 관리]**

| # | 폴더 | 버튼명(안) | 기능 설명 |
|---|---|---|---|
| R-02 | `ViewTemplateCopier` | **뷰 템플릿 복사** | 열린 다른 문서의 뷰 템플릿을 현재 프로젝트로 복사. 충돌 정책(스킵/이름변경/덮어쓰기) 선택 가능. |
| R-07 | `SmartSelectorLite` | **Smart 선택** | 카테고리·패밀리명·파라미터 조건으로 요소를 필터링 후 선택. 규칙 저장 지원. |
| R-08 | `WorksetInspectorLite` | **Workset 검사** | 워크셋별 요소 수 집계. 선택 워크셋의 요소를 뷰포트에서 즉시 선택. |
| R-14 | `SheetViewDuplicator` | **시트 복제** | 선택 시트와 뷰를 일괄 복제. 번호·이름 접미사 지정, 상세 포함/종속뷰 모드 선택. |

---

**[모델 처리]**

| # | 폴더 | 버튼명(안) | 기능 설명 |
|---|---|---|---|
| R-03 | `TypeBatchDefiner` | **타입 일괄 정의** | JSON 파일로 패밀리 타입을 일괄 생성·업데이트. Dry-Run으로 예상 결과 미리 확인. |
| R-04 | `ElementRenumbering` | **번호 재배정** | 룸·도어·스페이스 등의 번호를 접두사+순번 규칙으로 재배정. X/Y/레벨 정렬 기준 선택. |
| R-16 | `RoomFinishingPro` | **룸 마감 배정** | 룸 이름 키워드 기반으로 바닥·벽·천장·걸레받이 마감 파라미터 자동 설정. 한국 KCS 기본 룰셋 내장. |
| R-18 | `MultiMaterialTagger` | **다중 재료 태그** | 선택 요소의 재료 레이어 수만큼 재료 태그를 자동으로 현재 뷰에 배치. |
| R-19 | `FamilyPackageTransfer` | **패밀리 전송** | 현재 문서의 패밀리를 폴더로 일괄 내보내기(RFA), 또는 폴더의 RFA 파일을 일괄 로드. |

---

**[선택·검사]**

| # | 폴더 | 버튼명(안) | 기능 설명 |
|---|---|---|---|
| R-09 | `LinkHealthReload` | **링크 상태** | Revit·CAD 링크의 로드/미로드/경로오류 상태 목록 표시. 선택 링크 일괄 재로드. |
| R-12 | `WarningManager` | **경고 격리** | 경고를 유형별로 집계. 선택 유형의 해당 요소를 뷰포트에서 즉시 선택·격리. |

---

**[QA·감사]**

| # | 폴더 | 버튼명(안) | 기능 설명 |

| R-17 | `IFCDeliveryValidator` | **IFC 납품 검증** | 국토부·LH BIM 납품 기준 룰셋으로 모델 자동 검증. 오류/경고 목록 + HTML 리포트 생성. |

**[데이터]**

| # | 폴더 | 버튼명(안) | 기능 설명 |
|---|---|---|---|
| R-10 | `ScheduleExcelExport` | **일람표 Excel 저장** | 현재 활성 일람표를 xlsx로 내보내기. 헤더·행 순서·한글 파라미터 그대로 유지. |
| R-15 | `ScheduleExcelSync` | **일람표 Excel 동기화** | Excel 수정본을 Revit 파라미터에 역반영. ElementId 기준 매핑, 읽기전용 파라미터 자동 스킵. |
| R-11 | `MEPLengthCalculator` | **MEP 길이 계산** | 배관·덕트·전선관을 시스템 종류별로 집계 (단위: m). Excel 저장 지원. |

### Navisworks Add-in (4개)

| # | 폴더 | 플러그인명 | 기능 설명 |
|---|---|---|---|
| N-01 | `ClashResponsibilityBoard` | **간섭 책임자 배정** | Clash Detective 결과를 파일명·레이어 키워드로 공종 분류 → 담당자 자동 배정 → 한글 Excel 보고서 생성. JSON 룰셋 편집 가능. |
| N-02 | `ClashGroupEngine` | **간섭 그룹화** | 클래시 이름·상태 패턴으로 자동 그룹화. 그룹별 건수 집계. JSON 규칙 저장/불러오기. |
| N-03 | `ClashTestDefiner` | **간섭 테스트 일괄 정의** | JSON으로 ClashTest 조건(공종A vs 공종B, 허용오차, 타입)을 일괄 생성. Dry-Run 지원. |
| N-05 | `IFCExportHelper` | **IFC 내보내기** | 모델별 분리 또는 통합 IFC 내보내기. IFC 버전·프로젝트 정보 설정. 한국 납품 메타데이터 포함. *(Navisworks SDK API 바인딩은 개발 PC에서 완성 필요)* |

---

### 판단 기준 참고

- **즉시 가치** (빠른 빌드·테스트 가능): R-01, R-04, R-05, R-07, R-10, R-12
- **한국 특화 고유 기능** (경쟁앱 없음): R-16(룸마감), R-17(IFC검증), N-01(한글 책임자 배정)
- **기존 기능 완성** (현재 dry-run → 실행): R-02, R-03, R-04, R-10
- **Navisworks 연동 필요**: N-01~N-03 (Revit 간섭 마커와 함께 써야 가치 극대화)
