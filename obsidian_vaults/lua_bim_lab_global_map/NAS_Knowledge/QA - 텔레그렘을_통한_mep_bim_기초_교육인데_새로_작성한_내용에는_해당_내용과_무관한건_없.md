# QA - 텔레그렘을 통한 mep bim 기초 교육인데 새로 작성한 내용에는 해당 내용과 무관한건 없

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-29
> **출처:** Claude Code 세션

텔레그렘을 통한 mep bim 기초 교육인데 새로 작성한 내용에는 해당 내용과 무관한건 없는거지?

## 답변

네, 내용 자체는 전부 Starter Plan과 관련된 것들입니다. 다만 한 가지 실용적인 문제가 있습니다.

---

## 내용은 맞는데, 아직 준비 안 된 항목이 있습니다

폼에 "Included"로 적은 항목 중 현재 **실제 전달 가능한 것**과 **아직 파일이 없는 것**이 섞여 있습니다.

| 항목 | 상태 |
|---|---|
| 매일 레슨 (Telegram) | ✅ Day 1–30 파일 존재 |
| BIM Check Friday 퀴즈 | ✅ 13주분 파일 생성 완료 |
| Day 1–60 커리큘럼 구조 설명 | ✅ 정확함 |
| Quick Reference Card 8장 | ❌ PDF 아직 미제작 |
| Day 60 수료 인증서 | ❌ PDF 아직 미제작 |
| Day 90 수료 인증서 | ❌ PDF 아직 미제작 |
| Days 31–60 레슨 파일 | ❌ `.txt` 변환 미완 |
| Days 61–90 공종별 레슨 | ❌ `.txt` 변환 미완 |

---

## 권고

지금 당장 폼에 넣을 "Included" 문구는 **실제 전달 가능한 것만** 적는 것이 안전합니다.

```
Included:
- One practical MEP BIM lesson per day through Telegram
- Beginner-friendly explanation with checklist or action item
- Weekly BIM Check Friday quiz (every Friday)
- 90-Day structured MEP BIM curriculum
  (Days 1–60: Foundation / Days 61–90: Discipline deep-dive)
```

Reference Card와 인증서는 **PDF 제작이 완료된 시점에 추가**하는 게 좋습니다. 먼저 약속하고 못 보내는 것보다 나중에 "보너스로 드립니다" 형태가 훨씬 낫습니다.
