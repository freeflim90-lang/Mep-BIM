# LUA BIM LABS Starter Plan - Application and Telegram Onboarding

## Purpose

The application form must collect payment verification, Telegram connection data, learning profile, and consent before daily MEP BIM lessons are delivered.

Use a semi-integrated payment flow:

1. Show the PayPal payment link inside the Google Form.
2. Ask the client to pay first.
3. Ask for the email used for PayPal payment.
4. Verify payment manually before onboarding.

## Required Client Data

- full name
- email
- country
- time zone
- preferred language
- PayPal email
- PayPal transaction ID, optional
- Telegram username
- Telegram display name
- current BIM level
- Revit MEP experience
- Navisworks experience
- main MEP discipline
- learning goal
- biggest difficulty

## Required Consents

- Starter is daily MEP BIM lessons through Telegram.
- Starter does not include project file review.
- Starter does not include Revit model QA.
- Starter does not include clash report review.
- Starter is educational content only and does not provide engineering design verification, code compliance confirmation, construction approval, legal advice, or professional certification.
- LUA BIM LABS does not guarantee employment, certification, project approval, construction acceptance, or business results.
- Coming Soon services are not included.
- Client agrees to receive Telegram educational messages.
- Client agrees to Starter Plan terms, disclaimers, refund policy, and cancellation policy.
- Client agrees not to send confidential files.
- General questions may be anonymized and reused for education.
- Learning profile may be stored for service delivery.

## Telegram Delivery Account

The Telegram delivery account should be clearly presented as the official LUA BIM LABS education delivery channel.

Recommended display name:

LUA BIM LABS Starter

Recommended description:

Daily MEP BIM lessons for LUA BIM LABS Starter Plan clients.

## Manual Onboarding Steps

1. Confirm form submission.
2. Verify PayPal payment.
3. Check Telegram username.
4. Create client profile.
5. Create Obsidian client note.
6. Send Telegram welcome message.
7. Add client to daily lesson delivery.
