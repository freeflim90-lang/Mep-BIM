# QA - 다른 pc에서 텔레그렘 접속을 하려고 하니까 기존에 교육용 상품으로 만든 lua bim l

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

다른 pc에서 텔레그렘 접속을 하려고 하니까 기존에 교육용 상품으로 만든 lua bim labs starter 쪽으로 연결이 되다보니 연락이 불가하여 내생각에는 이메일로 변경하고 연락온 이메일을 감지하여 알림을 나에게 주는 방식은 어때?

## 답변

좋은 방법입니다. 구체적으로 제안하자면:

1. **웹사이트**: 이메일 링크로 복원
2. **백엔드 스크립트**: Gmail 신규 메일 감지 → 텔레그램 알림 전송
3. **LaunchAgent**: 주기적으로 체크 (예: 5~10분마다)

기존 Gmail/Telegram 인프라를 활용하면 추가 세팅 없이 구현 가능합니다. 먼저 기존 Gmail 연동 방식을 확인합니다.
