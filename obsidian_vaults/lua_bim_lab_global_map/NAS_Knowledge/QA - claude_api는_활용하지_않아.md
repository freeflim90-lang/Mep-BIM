# QA - claude api는 활용하지 않아

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-28
> **출처:** Claude Code 세션

claude api는 활용하지 않아

## 답변

확인. 현재 설정 그대로 유지한다.

현재 KB 자동 성장 파이프라인:
- **AI 생성**: Ollama `qwen2.5:7b` (로컬, 무료) 단독 사용
- **Claude API 폴백**: 코드에는 있지만 실질적으로 미사용

Anthropic API 키가 없거나 크레딧이 없어도 Ollama만으로 매일 자정 75개 파일이 정상 업데이트된다.

나중에 생성 품질을 높이고 싶을 때 고려할 수 있는 무료/저비용 대안:
- **더 큰 Ollama 모델** — `qwen2.5:14b`, `qwen2.5:32b` (로컬, 무료, 메모리 필요)
- **DeepSeek API** — 이미 채팅 파이프라인에 연결되어 있음, 비용 매우 저렴

필요하면 언제든지 말해줘.
