# QA - 나머지 상품은 별도의 링크로 결제를 해야하므로 추후에 하고 starter 상품으로만 런칭하

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-29
> **출처:** Claude Code 세션

나머지 상품은 별도의 링크로 결제를 해야하므로 추후에 하고 starter 상품으로만 런칭하였으며 기존 런칭시기의 커리큘럼에 대한 설명을 구글 폼에 변경해야해?

## 답변

네, 업데이트가 필요합니다. 커리큘럼이 업그레이드되면서 달라진 부분이 있기 때문입니다.

## Google Form에서 변경해야 할 항목

### 1. 폼 소개 텍스트 / 프로그램 설명

| 기존 | 변경 후 |
|---|---|
| "60-day MEP BIM program" | **"90-Day MEP BIM Starter Program"** |
| (공종 선택 목적 불명확) | "Day 61부터 선택한 공종 집중 Deep-Dive 트랙으로 전환됩니다" |

### 2. 공종 선택 질문 (Discipline Track)

기존에도 공종 선택이 있었다면, 이제는 그 선택이 **Day 61~90 커리큘럼을 결정**한다는 것을 명시해야 합니다.

```
Which MEP discipline would you like to focus on? (Days 61–90)
- HVAC (Air Conditioning & Ventilation)
- Piping / Mechanical
- Plumbing / Sanitary
- Fire Protection
- Electrical
```

### 3. 서비스 이해 체크박스 (포함 항목)

기존 체크박스가 60일 기준이었다면, 아래 항목 추가/수정:

```
✓ 90일 커리큘럼 (Day 1–60: Foundation, Day 61–90: 공종 Deep-Dive)
✓ 매주 금요일 BIM Check Friday 퀴즈 발송
✓ 각 트랙 완료 시 Quick Reference Card 발송 (총 8장)
✓ Day 60 Foundation 수료 인증서 발행
✓ Day 90 공종 전문가 수료 인증서 발행
```

---

현재 Google Form의 실제 내용을 공유해 주시면 구체적으로 어떤 문구를 어떻게 바꿔야 하는지 정확히 알려드릴 수 있습니다. 스크린샷이나 텍스트 복사 모두 가능합니다.
