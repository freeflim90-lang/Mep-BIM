# 지식맵 — BIM 실무 표준

LUA BIM LABS의 공종별 BIM 실무 기준. 간섭 검토, 공간 기준, 코드 기준 통합.

---

## 1. 간섭 우선순위 체계

| 우선순위 | 기준 | 색상 코드 |
|---|---|---|
| 긴급 | >50mm 관통 / 제연덕트·소방 충돌 / 피난동선 침해 | #FF4444 (빨강) |
| 보통 | 10~50mm 간섭 / 유지보수 공간 부족 | #FFDD44 (노랑) |
| 낮음 | 10mm 미만 / 마감 내 해결 가능 | #44CC66 (초록) |

---

## 2. 공종별 핵심 기준

### 2-1. 건축

| 항목 | 기준 |
|---|---|
| 방화구획 | 바닥면적 1,000㎡마다 (스프링클러 3,000㎡) |
| 관통부 | 내화채움재·방화댐퍼 필수 |
| 피난동선 폭 | 최소 1.2m |
| 보행거리 | 30m 이하 (스프링클러 50m) |
| 거실 천장고 | 최소 2.1m |
| 주차장 천장고 | 최소 2.3m |

### 2-2. 구조

| 항목 | 기준 |
|---|---|
| 슬래브 관통 | 구조검토 필수, 보강근 계획 포함 |
| 보 관통 (직사각형 개구) | 보 춤의 1/3 이하, 중앙 1/3 구간 위치 |
| 보 관통 (원형) | 직경 보 춤의 1/4 이하 |
| 기둥 관통 | 금지 (구조검토 후 예외 인정) |

### 2-3. 공조 덕트 (HVAC)

| 항목 | 기준 |
|---|---|
| 주 덕트 풍속 | 6~10m/s |
| 분기 덕트 풍속 | 4~6m/s |
| 장방형 종횡비 | 4:1 이하 (최대 8:1) |
| 제연덕트 이격 | 외면에서 최소 100mm |
| 제연덕트 방화구획 관통 | 300mm 이격 |
| 점검구 간격 | 분기·댐퍼·코일 전후, 직선 10m마다 |
| 점검구 전면 공간 | 최소 600mm |

### 2-4. 위생 (Plumbing)

| 항목 | 기준 |
|---|---|
| 오배수 구배 | DN50=1/50, DN100=1/100, DN150=1/150 |
| 최소 유속 | 0.6m/s 이상 |
| 수평 길이 12m 이상 | 소제구 설치 |
| 트랩 깊이 | 50~100mm |
| 통기관 지붕 돌출 | 600mm 이상 |

### 2-5. 배관 (Piping)

| 항목 | 기준 |
|---|---|
| 벽관통 이격 | 외면에서 최소 25mm |
| 슬래브 관통 시공 여유 | 50mm |
| 행거 간격 | DN50 이하 2.0m, DN100 이상 3.0m |
| 냉온수관 보온재 포함 | 관경+보온재 두께 합산으로 간섭 검토 |

### 2-6. 전기 (Electrical)

| 항목 | 기준 |
|---|---|
| 강전/약전 트레이 이격 | 최소 300mm (또는 격벽) |
| 분전반 전면 공간 | 최소 1,000mm (고압 1,200mm) |
| 분전반 상단 인입 여유 | 최소 300mm |
| 케이블 트레이 충전율 | 50% 이하 |

### 2-7. 소방

| 항목 | 기준 |
|---|---|
| 스프링클러 헤드 | 설치 범위 내 차폐체 없어야 함 |
| 소화배관 | 다른 배관 하부 원칙 |
| 소방 분전반 | 별도 구획 |
| 소방 간섭 분류 | 최우선 경고 (건물 안전 관련) |

## 3. BIM 프로젝트 표준 설정

### 3-1. Workset 구성 (기본)

```
WS-공유          (공통 요소: 레벨, 그리드)
WS-건축
WS-구조
WS-기계
WS-전기
WS-소방
WS-외부참조 (링크 파일)
```

### 3-2. 공유 파라미터 (필수)

| 파라미터명 | 적용 | 용도 |
|---|---|---|
| LBL_ClashStatus | 모든 요소 | 간섭 상태 추적 |
| LBL_AssignedTo | 모든 요소 | 담당자 매핑 |
| LBL_DueDate | 모든 요소 | 조치 기한 |
| LBL_FireCompartmentId | 건축 요소 | 방화구획 ID |
| LBL_FinishThickness | 건축 요소 | 마감 두께 |

### 3-3. 파일 명명 규칙

```
PROJ_YYYY-MM-DD_[공종]_[용도]_[버전].rvt
```

예시: `LBL_2026-05-22_MEP_협의도_v2.rvt`

### 3-4. 레벨 명명 규칙

```
FL_B2, FL_B1, FL_1F, FL_2F, ... FL_RF, FL_PH
```

## 4. 간섭 검토 프로세스

```
Revit → NWC 내보내기 (각 공종별)
    ↓
Navisworks → 파일 병합 (Append)
    ↓
Clash Detective → 공종 조합별 ClashTest 실행
    ↓
결과 XML 내보내기
    ↓
Python → Excel 리포트 (엑셀자동화 - BIM 리포트 자동화)
    ↓
담당자 배분 → 조치 → 재검토
```

---

## 5. LOD (Level of Development) 기준

| 단계 | LOD | 내용 |
|---|---|---|
| 기본계획 | 100~200 | 개략 형상, 위치 |
| 실시설계 | 300 | 정확한 형상, 크기, 위치 |
| 시공도 | 350 | 연결부, 상세 포함 |
| 시공 | 400 | 제작 정보 포함 |
| 준공 | 500 | 실측 반영 |

간섭 검토는 **LOD 300 이상** 모델에서 수행한다.
