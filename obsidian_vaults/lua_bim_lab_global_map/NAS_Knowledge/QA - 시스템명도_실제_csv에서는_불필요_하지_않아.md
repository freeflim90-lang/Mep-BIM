# QA - 시스템명도 실제 csv에서는 불필요 하지 않아?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-27
> **출처:** Claude Code 세션

시스템명도 실제 csv에서는 불필요 하지 않아?

## 답변

시스템명 없이 내보내면 **가져올 때 어느 행이 어느 시스템인지 알 방법이 없어서** 행 순서로 매칭해야 합니다. 이 경우 Revit 프로젝트에 시스템이 추가/삭제되면 행이 밀려서 잘못된 시스템에 적용되는 문제가 생깁니다.

두 가지 방향이 있습니다:

| 방식 | 장점 | 단점 |
|---|---|---|
| **시스템명 유지** (현재) | 어떤 순서든 안전하게 매칭 | 사용자가 시스템명 열을 직접 채울 필요는 없음 |
| **시스템명 제거** | CSV가 더 단순해짐 | Revit 시스템 목록 순서와 CSV 행 순서가 정확히 일치해야 함 |

실제 사용 방식이 어떻게 되시나요?
- 내보낸 CSV에 바로 값을 채워서 다시 가져오는 용도라면 → 시스템명 있는 게 훨씬 안전합니다
- 완전히 새 Excel에 약어·RGB만 적어서 붙여넣는 용도라면 → 행 순서 기반으로 바꿀 수 있습니다
