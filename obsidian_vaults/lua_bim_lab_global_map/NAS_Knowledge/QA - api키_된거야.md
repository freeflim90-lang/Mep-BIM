# QA - api키 된거야?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

api키 된거야?

## 답변

아직 안 됩니다. 두 가지가 남았습니다.

**지금 상태**: API 키가 하나 있는데 **Google Sheets API 전용**으로 제한되어 있습니다. Custom Search에는 못 씁니다.

---

**순서:**

**① 먼저 Custom Search API 활성화 확인**

왼쪽 메뉴 → **라이브러리** → 검색창에 `Custom Search API` → 클릭 → **사용 설정** 클릭
(이미 설정되어 있으면 "API 사용 중지" 버튼이 보임)

**② 새 API 키 만들기**

다시 **사용자 인증 정보** → 상단 **+ 사용자 인증 정보 만들기** → **API 키** → 생성

→ 키가 팝업으로 뜨면 복사해서 여기에 붙여넣어 주세요.
