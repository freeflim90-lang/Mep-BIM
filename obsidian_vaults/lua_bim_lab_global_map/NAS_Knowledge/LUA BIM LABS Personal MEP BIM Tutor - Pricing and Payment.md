# LUA BIM LABS Personal MEP BIM Tutor - Pricing and Payment

## Pricing Principle

This service should be priced as personalized BIM mentoring, not as a content newsletter.

The paid value is:

- level diagnosis
- personalized daily BIM lessons
- Telegram Q&A
- weak-point tracking
- practical project guidance
- Obsidian-based learning record

## Launch Decision

First launch only the Starter plan.

Reason:

- validate global demand through PayPal
- keep delivery scope simple
- avoid unlimited 1:1 support too early
- build a customer base before opening higher-touch mentoring

## Public Launch Package

| Package | Monthly Price | Client |
| --- | ---: | --- |
| Starter | USD 39 | beginner, student, light learner |

## Starter Scope

Included:

- daily MEP BIM lesson through Telegram
- beginner-friendly explanation
- checklist or action item
- monthly learning path

Not included:

- unlimited 1:1 Q&A
- project file review
- Revit model QA
- clash report review
- urgent support

Future packages stay Coming Soon:

- Personal Tutor: USD 119/month
- Coordinator Mentor: USD 229/month
- Project Mentor: USD 490+/month

## Payment Strategy

### Default

Use PayPal as the primary payment method for both domestic Korean users and global users.

### Fast Launch

Create only the Starter PayPal Payment Link for launch. Use that link in Blogger CTA, Telegram onboarding, email, and social posts. Keep Personal Tutor, Coordinator Mentor, and Project Mentor as Coming Soon services.

### Subscription Operation

After demand is validated, convert Starter to PayPal Subscriptions.

### Fallbacks

- No bank transfer in the standard offer.
- Toss Payments later only if PayPal creates a major conversion problem.
- Stripe later only if PayPal is not enough for some regions.

## Operational Rule

Payment should trigger onboarding:

1. confirm payment
2. collect Telegram ID
3. collect level diagnosis
4. create client profile
5. create Obsidian client note
6. start daily personalized lessons

## Refund Draft

- Full refund before diagnosis or first personalized lesson.
- No full refund after personalized work begins.
- Monthly cancellation applies to the next cycle.
- Project Mentor refunds follow milestone agreement.
