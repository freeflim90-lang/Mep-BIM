# QA - 이력서를 기반으로 수행했던 프로젝트 넣으면 어떨까?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

이력서를 기반으로 수행했던 프로젝트 넣으면 어떨까?

## 답변

좋습니다. 이력서의 주요 프로젝트들이 LUA BIM LABS의 실력을 직접 보여주는 레퍼런스가 됩니다. 웹사이트에 프로젝트 섹션을 추가하겠습니다.

프로젝트를 카테고리별로 나누면 어떨까요:

| 카테고리 | 프로젝트 |
|---|---|
| **상업시설** | 청라 스타필드 |
| **공공·인프라** | 제2철도 관제센터, 코엑스 무역센터, 인천공항 2터미널 |
| **반도체·플랜트** | 삼성디스플레이, 삼성전자, 삼성SDI, 포스코 등 |

추가로 두 가지 확인이 필요합니다:

1. **회사명 표기** — "신세계건설 청라 스타필드" 같은 실제 발주처/현장명을 그대로 노출해도 괜찮으신가요?
2. **이력서 사진** — 웹사이트에 대표자 프로필로 넣을까요, 아니면 프로젝트만 넣을까요?
