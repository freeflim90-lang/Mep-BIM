# QA - 운영체제 docker mac 어떤걸로해야되?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

운영체제 docker mac 어떤걸로해야되?

## 답변

**macOS** 선택하세요.

현재 백엔드 서버가 Mac에서 네이티브로 실행 중이고, 이미 LaunchAgent로 cloudflared를 등록해놨기 때문입니다.

Docker 옵션은 cloudflared 자체를 컨테이너 안에서 돌릴 때 쓰는 건데, 지금 구조에선 불필요합니다.

macOS 선택하면 화면에 토큰 방식 명령어가 나옵니다:
```bash
cloudflared tunnel run --token eyJhI...
```

그 토큰 값 복사해서 저한테 주시면 LaunchAgent에 바로 적용해드릴게요.

Docker 옵션은 cloudflared 자체를 컨테이너 안에서 돌릴 때 쓰는 건데, 지금 구조에선 불필요합니다.

macOS 선택하면 화면에 토큰 방식 명령어가 나옵니다:
```bash
cloudflared tunnel run --token eyJhI...
```
