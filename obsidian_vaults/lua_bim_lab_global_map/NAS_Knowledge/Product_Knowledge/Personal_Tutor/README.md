# Personal Tutor Product Knowledge

## Purpose

Prepare personalized MEP BIM tutoring by collecting anonymized learning diagnosis patterns and reusable curriculum structures.

## Collect

- Level diagnosis patterns
- Revit MEP skill gaps
- Discipline-specific blockers
- Weak-point tracking methods
- Personalized lesson templates

## Reuse

- Diagnosis rubric
- Personalized learning path
- Paid curriculum modules
- Anonymized Q&A examples

## Boundary

Client learning profiles are private. Reuse only generalized and anonymized patterns.
