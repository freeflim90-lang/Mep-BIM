# Project Mentor Product Knowledge

## Purpose

Prepare high-scope project mentoring with strict boundaries around confidentiality, scope, responsibility, and reuse.

## Collect

- Scope definition patterns
- Risk and decision logs
- Custom workflow templates
- Project mentoring playbooks
- Deliverable boundary examples

## Reuse

- Fully anonymized lessons
- Internal process improvements
- Non-client-specific templates

## Boundary

Project-derived material requires explicit review before reuse. Avoid code compliance, engineering verification, construction approval, and professional certification claims unless a future contract explicitly defines the scope.
