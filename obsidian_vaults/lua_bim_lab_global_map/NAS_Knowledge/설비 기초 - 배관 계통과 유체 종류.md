# 설비 기초 — 배관 계통과 유체 종류

> **핵심**: 배관은 액체·기체·증기를 운반한다. 어떤 유체인지에 따라 재료·압력·온도·이격 기준이 모두 달라진다.

---

## 1. 배관 계통 전체 분류

```
배관 계통
├── 공조/냉난방 배관 (Mechanical)
│   ├── CHW — 냉수 (Chilled Water)
│   ├── HW  — 온수 (Hot Water / Heating Water)
│   ├── CW  — 냉각수 (Condenser Water)
│   ├── Refrigerant — 냉매
│   └── Steam — 증기
├── 위생/급배수 배관 (Plumbing)
│   ├── CWS — 급수 (Cold Water Supply)
│   ├── HWS — 급탕 (Hot Water Supply)
│   ├── HWR — 급탕환수 (Hot Water Return)
│   ├── Sanitary — 오배수
│   ├── Vent — 통기
│   └── Storm — 우수
├── 소방 배관 (Fire Protection)
│   ├── Sprinkler — 스프링클러 (습식/건식/준비작동식)
│   ├── Standpipe — 연결송수관
│   └── Gas System — 가스계 소화 (CO₂, HFC 등)
└── 특수 배관
    ├── 가스 (Natural Gas / LPG)
    ├── 압축공기 (Compressed Air)
    └── 진공 (Vacuum)
```

---

## 2. 공조/냉난방 배관 상세

### 2-1. CHW — 냉수 배관 (Chilled Water)

| 항목 | 내용 |
|---|---|
| 유체 | 냉각된 물 (물 + 부동액 혼합 가능) |
| 역할 | 냉동기(칠러)에서 냉각된 물을 AHU·FCU로 공급 → 냉방 |
| 온도 | 공급 7℃ / 환수 13℃ (온도차 6℃) |
| 압력 | 최고 사용압력 1.0 MPa 이하 |
| 단열 | 결로 방지 필수 — 배관 외면에 단열재 + 방습층 |
| 관재 | 탄소강관 (KS D 3507) 또는 스테인리스 |
| Revit 계통 | Hydronic Supply / Hydronic Return (Chilled) |
| 색상 표시 | 파란색 (공급) / 남색 (환수) |

### 2-2. HW — 온수 배관 (Heating Hot Water)

| 항목 | 내용 |
|---|---|
| 유체 | 가열된 물 |
| 역할 | 보일러·열교환기에서 가열된 물을 AHU·FCU·방열기로 공급 → 난방 |
| 온도 | 공급 60℃ / 환수 50℃ (온도차 10℃) |
| 단열 | 열손실 방지 — 배관 외면 단열재 |
| 관재 | 탄소강관, 동관, CPVC |
| Revit 계통 | Hydronic Supply / Hydronic Return (Heating) |
| 색상 표시 | 빨간색 (공급) / 주황색 (환수) |

### 2-3. CW — 냉각수 배관 (Condenser Water)

| 항목 | 내용 |
|---|---|
| 유체 | 냉각탑과 냉동기 응축기 사이 순환하는 물 |
| 역할 | 냉동기에서 발생한 열을 냉각탑으로 운반 → 대기 방출 |
| 온도 | 공급 32℃ / 환수 37℃ (온도차 5℃) |
| 특징 | 외기에 노출된 냉각탑과 연결 → 수질 관리 필수 |
| 단열 | 불필요 (일반적으로 미단열) |
| 관재 | 탄소강관 (부식 방지 처리) |
| Revit 계통 | Condenser Hydronic Supply / Return |

### 2-4. Refrigerant — 냉매 배관

| 항목 | 내용 |
|---|---|
| 유체 | 냉매 (R-410A, R-32, R-22 등) |
| 역할 | 실외기(압축기·응축기)와 실내기(증발기) 사이 냉매 순환 |
| 계통 | 액관 (Liquid Line): 실외기→실내기 / 흡입관 (Suction Line): 실내기→실외기 |
| 단열 | 흡입관 20~25mm 발포폼, 액관 10mm |
| 전기 이격 | 단열 포함 외경 기준 최소 300mm |
| 가스관 이격 | 60mm 이상 |
| 주의 | 냉매 누설 → 질식·환경오염 → 밀폐 공간 설치 시 환기 검토 |

### 2-5. Steam — 증기 배관

| 항목 | 내용 |
|---|---|
| 유체 | 수증기 (포화증기 또는 과열증기) |
| 역할 | 병원·산업시설에서 고온 열원 공급, 소독, 가습 |
| 압력 | 저압: 0.1~0.35 MPa / 고압: 0.35 MPa 이상 |
| 온도 | 저압 증기 120~140℃, 고압 증기 150℃ 이상 |
| 응축수 | 증기가 식으면 응축수 발생 → 증기트랩(Steam Trap)으로 제거 |
| 안전 | 고압·고온으로 누설 시 화상·폭발 위험 → 법적 검사 대상 |

## 3. 위생/급배수 배관 상세

### 3-1. CWS — 급수 배관 (Cold Water Supply)

| 항목 | 내용 |
|---|---|
| 유체 | 음용·위생용 냉수 |
| 역할 | 수도 본관 → 저수조 → 가압 펌프 → 각 위생기구 |
| 압력 | 말단 기구 최소 0.05 MPa (세면기 기준) |
| 관재 | PVC, 스테인리스, 동관 (음용 가능 재질) |
| 단열 | 결로 방지 — 건물 외부 노출부 보온 |

### 3-2. HWS/HWR — 급탕·급탕환수 배관

| 항목 | 내용 |
|---|---|
| 유체 | 위생용 온수 (음용 가능 온도) |
| 공급 온도 | 60℃ (레지오넬라균 사멸 기준) |
| 공급/분기 온도 | 분기점 최소 55℃ 이상 유지 |
| 순환 | HWS 공급 → 위생기구 → HWR 환수 → 가열기 (순환 배관으로 온도 유지) |
| 구분 | 공조용 HW(난방)와 전혀 다른 계통 — 반드시 분리 |

### 3-3. Sanitary — 오배수 배관

| 항목 | 내용 |
|---|---|
| 유체 | 생활하수 (오수 + 잡배수) |
| 역할 | 위생기구 → 집수정 or 공공 하수관 |
| 구배 필수 | 자연유하 — 구배 없으면 막힘 |
| 유속 | 0.6m/s 이상 (퇴적 방지) |
| 가스 | 황화수소(H₂S) 등 유독 가스 발생 → 트랩 필수 |

### 3-4. Storm — 우수 배관

| 항목 | 내용 |
|---|---|
| 유체 | 빗물 (오염도 낮음) |
| 역할 | 옥상·주차장 우수 → 집수정 → 공공 우수관 |
| 오수와 분리 | 오수와 혼합 금지 (법적 분리 의무) |
| 활용 | 중수도로 재활용 가능 (화장실 세정수 등) |

## 4. 소방 배관 상세

### 4-1. Sprinkler — 스프링클러 배관

| 유형 | 내용 |
|---|---|
| 습식 (Wet) | 배관 내 항상 물 충만 — 헤드 개방 즉시 방수. 동결 우려 없는 구역 |
| 건식 (Dry) | 배관 내 가압 공기 → 화재 시 공기 배출 후 물 충전. 동결 우려 구역 |
| 준비작동식 (Pre-action) | 배관 내 공기 + 감지기 신호 2중 확인 후 방수. IT실·전산실 |
| 일제살수식 (Deluge) | 개방형 헤드 — 감지기 신호로 전 헤드 동시 방수. 위험물 창고 |

| 항목 | 기준 |
|---|---|
| 배관 재료 | 배관용 탄소강관 (KSD 3507), 동관, 스테인리스 |
| 최소 압력 | 0.1 MPa |
| 최대 압력 | 1.2 MPa |
| 배관 색상 | 빨간색 (Red) |

### 4-2. Standpipe — 연결송수관

| 항목 | 내용 |
|---|---|
| 유체 | 소방차에서 가압 공급하는 물 |
| 역할 | 소방대원이 건물 내부에서 직접 호스 연결 → 방수 |
| 설치 | 11층 이상 건물 또는 연면적 5,000㎡ 이상 |
| 배관 위치 | 피난 계단 부근 (1층에 송수구, 각 층에 방수구) |

### 4-3. 가스계 소화 배관

| 시스템 | 소화약제 | 주요 설치 장소 |
|---|---|---|
| CO₂ | 이산화탄소 | 전기실, 발전기실 (질식 소화) |
| HFC-125 (청정소화약제) | 할로카본 계열 | 전산실, 통신실 |
| IG-541 (INERGEN) | 질소·아르곤·CO₂ 혼합 | 미술관, 문서보관실 |

## 5. 특수 배관

### 5-1. 가스 배관 (Natural Gas / LPG)

| 항목 | 내용 |
|---|---|
| 유체 | 도시가스 (LNG: 메탄 주성분) 또는 LPG (프로판·부탄) |
| 압력 | 저압 (0.1 MPa 미만) → 건물 내부 일반 |
| 이격 | 전기 배선과 150mm 이상, 전기계량기·개폐기 300mm 이상 |
| 관재 | 가스용 강관 (KS D 3631), 폴리에틸렌 (PE) 매설 |
| LNG vs LPG | LNG는 공기보다 가벼워 위로 확산, LPG는 무거워 바닥 축적 — 환기 방향이 반대 |

### 5-2. 압축공기 배관 (Compressed Air)

| 항목 | 내용 |
|---|---|
| 유체 | 압축된 건조 공기 |
| 압력 | 0.5~0.8 MPa (일반 산업용) |
| 용도 | 공압 기기 구동, 의료용 (클린에어) |
| 특징 | 수분 제거 필수 (에어드라이어 설치) |
