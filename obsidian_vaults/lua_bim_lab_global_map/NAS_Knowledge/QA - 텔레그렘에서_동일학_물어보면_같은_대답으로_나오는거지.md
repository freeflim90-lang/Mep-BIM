# QA - 텔레그렘에서 동일학 물어보면 같은 대답으로 나오는거지?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

텔레그렘에서 동일학 물어보면 같은 대답으로 나오는거지?

## 답변

반은 그렇고, 반은 다릅니다.

**현재 상황:**
- 방금 제가 드린 답변은 Claude 모델의 학습 데이터 기반입니다 — 지식 베이스 파일에 저장된 내용이 아닙니다.
- Telegram 봇도 동일한 Claude 모델을 사용하므로, 인천공항 물량 같은 일반 지식은 비슷하게 답변할 수 있습니다.
- 하지만 **지식 베이스에 없는 내용**이라 봇이 검색 결과를 못 찾으면, 답변이 달라지거나 더 짧아질 수 있습니다.

**정확히 동일한 답변을 원한다면:**

방금 정리한 인천공항 물량 내용을 QA 파일에 저장해야 합니다. Telegram에서 `/fix` 명령어로 저장하거나, 지금 바로 파일에 기록할 수 있습니다.

지금 `qa/건축_QA.md` 등 관련 파일에 저장해드릴까요?
