## 1. BIM_프로젝트_견적산정.md — 2026 표준시장단가 BIM 단가·AI 견적도구 보강

### 지식 공백 해소
- 기존 자동보강 섹션(2026-06-01~06-06): 일부 중국어 포함, 근거 없는 단가 수치, 저품질
- 교체: 정확한 공식 출처 기반 구조화 지식으로 교체

### 핵심 내용

**2026 하반기 표준시장단가 BIM 단가 상세설명서 (국토교통부 공고 제2026-652호, 2026-05-08 시행):**
- BIM 수행 비용이 건설공사 표준단가 체계에 공식 편입 → 분리 계상 가능
- 공동 작성: 국토교통부 + 한국건설기술연구원(KICT)
- 총 1,457페이지 공고에 BIM 단가 상세설명서 부록 수록
- 적용: 국가·지방자치단체·공공기관 등 모든 공공발주처 (2026-05-08 즉시 적용)

**BIM 단가 상세설명서 주요 구성:**
- 설계 단계별 MM 기준, 공종별 BIM 작업 단가
- LOD 수준별 보정계수 (LOD 200/300/400/500)
- BIM 라이브러리 제작 단가, 납품 검수 비용, 4D/5D BIM 단가

**AI 기반 물량 산출·견적 도구 (2026):**
| 도구 | 정확도 | 비고 |
|------|--------|------|
| Togal.AI | 98% (도면 기준) | 2D 도면 CV 자동 산출, $299/월 |
| RIB CostX | 90~95% | 3D BIM + 견적 워크북 |
| Kreo | 85~92% | IFC/Revit 네이티브 |
| Autodesk Takeoff | 88~93% | ACC 포함 |

**산업 도입 현황:**
- AI 견적 도입률: 2024년 19% → 2026년 38% (2년 만에 2배)
- 조기 도입 기업 68%: 연 $50,000+ 절감
- BIM+AI 통합 시 물량 산출 시간 50%+ 단축

## 2. BIM_인력파견_기준.md — BIM 자격제도·ISO 19650-5 보안 요건 보강

### 지식 공백 해소
- 기존 자동보강 섹션: "최소 2년 Revit 경력" 반복 수준의 저품질 정보
- 교체: 자격제도 현황 + 국제 보안 기준 정밀 보강

### 핵심 내용

**한국 BIM 전문가 자격제도 (2026):**
- BIM 운용전문가 1급·2급 (민간자격, 한국BIM학회+한솔아카데미)
- 국가공인 전환 추진 중 (2022년~, 전환 시 공공입찰 요건 지정 가능)
- 2026 인력 수요: BIM 500억+ 의무화로 발주 건수 ~200건 (2024: 75건에서 2.7배)
- 희소 인력: MEP BIM 전문가 / BIM 코디네이터 / Add-in 개발자

**ISO 19650-5 보안 인식형 BIM 관리 (2025년 재확인, 현행 유효):**
- 적용: 국방·교정시설·발전소·공항 등 보안 민감 공공 프로젝트
- 5대 보안 요소: BASS / BASMgr / Security Management Plan / Breach Plan / SIR
- LUA BIM LABS 계약서: 데이터 반출 금지, 접근 권한 최소화, 파견 종료 후 삭제 확인 등 6개 조항 추가 권장
