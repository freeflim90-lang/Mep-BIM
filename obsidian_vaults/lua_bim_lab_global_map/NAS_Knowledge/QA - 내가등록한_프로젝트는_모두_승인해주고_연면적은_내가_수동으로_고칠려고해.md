# QA - 내가등록한 프로젝트는 모두 승인해주고 연면적은 내가 수동으로 고칠려고해

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-25
> **출처:** Claude Code 세션

내가등록한 프로젝트는 모두 승인해주고 연면적은 내가 수동으로 고칠려고해

## 답변

17개 전부 승인됐습니다. 이제 지도에서 내부 GFA 원이 실제 면적 기준으로 표시됩니다.

연면적 수정은 관리자 패널 "프로젝트 등록" 탭의 프로젝트 목록에서 직접 수정할 수 있어야 하는데, 현재는 수정 버튼이 없습니다. 프로젝트 항목에 연면적 인라인 수정 기능을 추가할까요? 아니면 다른 방식으로 수정하실 건가요?
