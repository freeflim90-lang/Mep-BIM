# QA - 실제 바닥면적으로는 표기가 안되?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-25
> **출처:** Claude Code 세션

실제 바닥면적으로는 표기가 안되?

## 답변

`L.circleMarker`(픽셀 고정)가 아니라 `L.circle`(미터 단위, 줌에 따라 스케일)을 쓰면 됩니다. `GFA = π × r²` 역산으로 실제 반지름(m)을 계산해 그립니다.
