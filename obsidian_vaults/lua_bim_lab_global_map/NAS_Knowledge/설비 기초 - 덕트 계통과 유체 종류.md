# 설비 기초 — 덕트 계통과 유체 종류

> **핵심**: 덕트는 공기(기체)를 운반하는 통로다. 무엇을 위한 공기인지에 따라 계통이 달라진다.

---

## 1. 덕트 계통 전체 분류

```
덕트 계통
├── 공조 (HVAC) 덕트
│   ├── OA — 외기 (Outside Air)
│   ├── SA — 급기 (Supply Air)
│   ├── RA — 환기/환수 (Return Air)
│   ├── EA — 배기 (Exhaust Air)
│   └── MA — 혼합기 (Mixed Air)
├── 제연 덕트
│   ├── PS — 가압 (Pressurization)
│   └── SE — 배연 (Smoke Exhaust)
├── 환기 전용 덕트
│   ├── 주방 배기 (Kitchen Exhaust)
│   ├── 욕실 배기 (Toilet Exhaust)
│   └── 주차장 환기
└── 특수 덕트
    ├── 클린룸 (Clean Room)
    └── 산업용 집진
```

---

## 2. 공조(HVAC) 덕트 계통 상세

### 2-1. OA — 외기 덕트 (Outside Air)

| 항목 | 내용 |
|---|---|
| 유체 | 외부 신선 공기 |
| 역할 | 실내 CO₂ 희석, 법정 환기량 확보 |
| 연결 | 외벽 루버 → 에어핸들링유닛(AHU) |
| 특징 | 여름 고온다습, 겨울 저온건조 → AHU에서 온습도 조정 |
| 법정 기준 | 사람 거주 공간 1인당 최소 25m³/h (ASHRAE 62.1) |

### 2-2. SA — 급기 덕트 (Supply Air)

| 항목 | 내용 |
|---|---|
| 유체 | AHU에서 처리(냉각·가열·가습·제습)된 공기 |
| 역할 | 실내 온도·습도 유지, 환기 |
| 연결 | AHU → 분기 덕트 → 취출구(디퓨저) |
| 온도 | 냉방 시 12~16℃, 난방 시 30~40℃ |
| Revit 계통 | Supply Air System |

### 2-3. RA — 환기 덕트 (Return Air)

| 항목 | 내용 |
|---|---|
| 유체 | 실내 사용된 공기 (온도 조건이 살아있음) |
| 역할 | AHU로 회수 → 재처리 후 재공급 (에너지 회수) |
| 연결 | 흡입구(그릴) → RA 덕트 → AHU |
| 특징 | OA와 혼합하여 혼합기(MA) 형성 → 에너지 절감 |
| Revit 계통 | Return Air System |

### 2-4. EA — 배기 덕트 (Exhaust Air)

| 항목 | 내용 |
|---|---|
| 유체 | 외부로 버려지는 오염·습기 포함 공기 |
| 역할 | 오염 공기 제거, 음압 유지 |
| 연결 | 배기구 → EA 덕트 → 외벽/옥상 배출구 |
| 주요 설치 위치 | 화장실, 주방, 탕비실, 주차장, 기계실 |
| Revit 계통 | Exhaust Air System |

### 2-5. MA — 혼합기 (Mixed Air)

| 항목 | 내용 |
|---|---|
| 유체 | OA + RA의 혼합 공기 |
| 역할 | AHU 내부 믹싱 챔버에서 형성 |
| 특징 | 별도 덕트 구간 없음 — AHU 내부 구간 |

## 3. 제연 덕트 계통

> 소방법 우선 보호 대상. 일반 HVAC 덕트와 **반드시** 별도 계통으로 분리.

### 3-1. PS — 가압 덕트 (Pressurization)

| 항목 | 내용 |
|---|---|
| 유체 | 신선 외기 (연기 없는 공기) |
| 역할 | 피난 계단실·전실에 양압 형성 → 연기 침입 차단 |
| 가압 기준 | 부속실(전실) 40~60 Pa, 계단실 50 Pa |
| 연결 | 가압팬 → PS 덕트 → 계단실·전실 |
| 재료 | 내화 덕트 또는 1시간 내화 피복 필수 |

### 3-2. SE — 배연 덕트 (Smoke Exhaust)

| 항목 | 내용 |
|---|---|
| 유체 | 화재 발생 구역의 고온 연기 |
| 역할 | 거실·복도 등 재실 공간의 연기 배출 → 피난 시간 확보 |
| 배연량 | 화재실 바닥면적 × 1m³/min 이상 |
| 연결 | 배연 댐퍼 → SE 덕트 → 배연팬 → 외부 |
| 이격 기준 | 일반 덕트·배관 외면에서 100mm 이상 |
| 재료 | 아연도강판 + 내화 피복 또는 스테인리스 |

---

## 4. 환기 전용 덕트

### 4-1. 주방 배기 (Kitchen Exhaust)

| 항목 | 내용 |
|---|---|
| 유체 | 고온(60~80℃), 유증기(油蒸氣), 냄새 포함 공기 |
| 역할 | 조리 시 발생 열·유증기·냄새 제거 |
| 특이사항 | 유증기로 인해 덕트 내 기름 축적 → 화재 위험 |
| 요구사항 | 스테인리스 덕트, 유분 필터, 정기 청소 |
| 배출 | 외벽 또는 옥상 직접 배출 (재순환 금지) |

### 4-2. 욕실/화장실 배기 (Toilet Exhaust)

| 항목 | 내용 |
|---|---|
| 유체 | 습기·냄새 포함 공기 |
| 역할 | 음압 유지, 냄새 확산 차단 |
| 풍량 | 환기횟수 10회/h 이상 |
| 덕트 경로 | 각 유닛 → 수직 샤프트 → 옥상 배출 |

### 4-3. 주차장 환기

| 항목 | 내용 |
|---|---|
| 유체 | CO·NOx·매연 포함 공기 |
| 역할 | CO 농도 25ppm 이하 유지 (법정 기준) |
| 시스템 | 급기 팬 + 배기 팬 (양방향) 또는 제트팬 |
| 덕트 규모 | 대구경 덕트 또는 제트팬으로 덕트리스 구성 가능 |

## 5. Revit 덕트 계통 (System Type) 매핑

| 계통명 (영문) | 한국어 | 역할 |
|---|---|---|
| Supply Air | 급기 | SA |
| Return Air | 환기(환수) | RA |
| Exhaust Air | 배기 | EA |
| Outside Air | 외기 | OA |
| Other Air | 기타 | 특수 용도 |

> Revit에서 계통이 잘못 지정되면 Navisworks 간섭 검토 시 공종 레이어 분류가 틀림 → 반드시 계통 지정 확인.

## 6. 덕트 재료 종류

| 재료 | 용도 | 특징 |
|---|---|---|
| 아연도 강판 (galvanized steel) | 일반 공조 SA/RA | 가장 일반적, 저비용 |
| 스테인리스 (STS) | 주방 배기, 제연 | 내식성, 고온 적합 |
| 글라스울 덕트보드 | 기계실 접속부 | 단열 일체형 |
| FRP | 부식 환경 (수처리실 등) | 내화학성 |
| 알루미늄 | 클린룸, 냉장창고 | 경량, 내습 |
| 플렉시블 (flexible) | 기기 접속부 | 진동 차단, 300~600mm |

## 7. 덕트 계통 간 이격 기준 (BIM 간섭 검토용)

| 계통 A | 계통 B | 최소 이격 | 비고 |
|---|---|---|---|
| 제연 덕트 | 일반 덕트 | 100mm | 방화구획 관통 시 300mm |
| 제연 덕트 | 소방 배관 | 50mm | 동일 우선순위 |
| 주방 배기 | 일반 급기 | 500mm | 유증기 오염 방지 |
| 급기 (SA) | 환기 (RA) | 없음 | 동일 천장 내 공존 가능 |
