# QA - 신규로 추가하면서 시스템명을 모르는 경우가 있는데 이부분은 비워도 되?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-27
> **출처:** Claude Code 세션

신규로 추가하면서 시스템명을 모르는 경우가 있는데 이부분은 비워도 되?

## 답변

네, 비워도 됩니다. 현재 코드에서 시스템명이 비어 있거나 Revit에 없는 이름이면 해당 행은 자동으로 건너뜁니다.

다만 지금은 결과 메시지가 "몇 개 적용됨"만 나오고 건너뛴 행이 몇 개인지 안 보입니다. 건너뛴 행 수도 같이 보여주면 더 유용할 것 같은데, 추가할까요?

```
붙여넣기 완료: 8개 시스템에 적용 / 3개 행 건너뜀 (시스템명 없음 또는 불일치)
```
