# QA - 캘린더에 등록된 사항들은 완료되면 반영되어야해

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-06-10
> **출처:** Claude Code 세션

캘린더에 등록된 사항들은 완료되면 반영되어야해

## 답변

자동으로 완료 반영이 되려면 `daily_routine_checklist.py`에서 Google Calendar API를 직접 호출해야 합니다. 먼저 프로젝트에 Google API 인증 정보가 있는지 확인합니다.
