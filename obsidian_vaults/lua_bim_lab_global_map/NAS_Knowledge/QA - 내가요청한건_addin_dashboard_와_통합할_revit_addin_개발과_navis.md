# QA - 내가요청한건 addin dashboard 와 통합할 revit addin 개발과 navis

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-31
> **출처:** Claude Code 세션

내가요청한건 addin dashboard 와 통합할 revit addin 개발과 navisworks addin을 말한거야

## 답변

이해했습니다. 정리하면:

**개별 아이템** → Qwen Coder로 C# 코드 생성 → 이메일 발송 → 개발 PC에서 빌드

통합이 필요한 묶음 아이템(Model Health Pro, Data Sync Hub, Clash Center 등)은 제외하고, **개별 독립 기능**만 먼저 개발합니다.
