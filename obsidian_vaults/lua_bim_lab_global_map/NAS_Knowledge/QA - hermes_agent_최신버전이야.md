# QA - hermes agent 최신버전이야?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

hermes agent 최신버전이야?

## 답변

이 프로젝트에서 "hermes agent"는 별도로 구현되어 있지 않습니다. 검색 결과 `hermes-parser` (React Native/Flow용 JS 파서 라이브러리)만 아카이브된 패키지에서 발견됐습니다.

혹시 아래 중 어느 것을 말씀하시는 건가요?

- **ollama의 Hermes 모델** (예: `nous-hermes`, `hermes3` 등 로컬 LLM)
- **프로젝트 내 특정 에이전트 스크립트** (이름이 hermes인)
- **다른 서비스/툴의 Hermes 에이전트**

어떤 hermes agent를 말씀하시는지 알려주시면 정확히 확인해 드리겠습니다.
