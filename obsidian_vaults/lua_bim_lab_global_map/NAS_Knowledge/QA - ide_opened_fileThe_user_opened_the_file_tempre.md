## 커리큘럼 구성안 (365일)

| 단계 | 기간 | 내용 |
|------|------|------|
| 1단계 | 1~30일 | BIM 개념/ISO 19650/BEP/LOD 기초 |
| 2단계 | 31~90일 | Revit 실무 (모델링, 패밀리, 시트) |
| 3단계 | 91~150일 | MEP 분야별 심화 (기계/전기/소방) |
| 4단계 | 151~180일 | 구조 BIM / 충돌검토 / Navisworks |
| 5단계 | 181~220일 | Dynamo 자동화 / API 기초 |
| 6단계 | 221~260일 | 물량산출 / 공정시뮬레이션 / BIM 납품 |
| 7단계 | 261~300일 | 설계/시공/유지관리 단계별 BIM 실무 |
| 8단계 | 301~365일 | Digital Twin / GIS / AI+BIM 트렌드 |

## 구현에 필요한 것

텔레그램 자동 발송 시스템을 만들려면:

1. **텔레그램 봇 토큰** — BotFather로 봇 생성 여부
2. **각 직원 Chat ID** — 봇에 먼저 메시지를 보내야 확인 가능
3. **레벨 구분 여부** — 허진석(1년차) vs 오수빈·김선정(2년차) 동일 내용으로 갈건지, 레벨별 분리할건지

몇 가지 확인이 필요해요:

- 텔레그램 봇이 이미 있나요, 새로 만들어야 하나요?
- 세 명 모두 동일한 내용을 받나요, 아니면 경력에 맞게 다르게 가나요?
- 메시지 형식 선호도가 있나요? (텍스트만 / 이미지 포함 / 예시 코드 포함 등)
