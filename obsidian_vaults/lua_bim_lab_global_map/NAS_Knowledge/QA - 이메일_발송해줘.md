# QA - 이메일 발송해줘

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-31
> **출처:** Claude Code 세션

이메일 발송해줘

## 답변

죄송합니다. 현재 제가 사용 가능한 Gmail MCP 도구 목록에는 **이메일 발송(send) 기능이 없습니다.** `create_draft`(초안 생성)만 지원됩니다.

방금 만든 드래프트를 Gmail에서 직접 열어서 발송해 주세요:

1. [Gmail 열기](https://mail.google.com) → 임시보관함(Drafts)
2. `[BCC 개발완료] NWC/IFC 내보내기 신버전 — 소스 경로 안내` 선택
3. 필요 시 소스 파일 첨부 후 발송

또는, 이전 19:59 이메일들처럼 **자동 발송 스크립트가 있다면** 그 스크립트를 실행하면 됩니다. 해당 스크립트 경로를 알려주시면 실행해드릴 수 있습니다.
