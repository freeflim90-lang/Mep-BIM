# QA - 컴퓨터 이동해야해서 컴퓨터 끄려고해

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-06-04
> **출처:** Claude Code 세션

컴퓨터 이동해야해서 컴퓨터 끄려고해

## 답변

끄시면 번역 프로세스가 중단됩니다. 나중에 다시 이어서 실행하면 됩니다 — 이미 번역된 파일은 건너뜁니다.

**재개 방법 (다음에 컴퓨터 켰을 때):**
```bash
cd ~/LUA\ BIM\ LABS
python3 scripts/translate_starter_messages.py --lang ar
```

ar만 미완성이라 이것만 실행하면 됩니다. 자동으로 남은 파일(fire + electrical)만 번역합니다.

편하게 끄세요!
