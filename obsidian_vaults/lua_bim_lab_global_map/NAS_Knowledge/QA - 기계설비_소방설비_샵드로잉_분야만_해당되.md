# QA - 기계설비 소방설비 샵드로잉 분야만 해당되

> **카테고리:** Claude대화 | **날짜:** 2026-05-31
> **출처:** Claude Code 세션

기계설비 소방설비 샵드로잉 분야만 해당되

## 답변

배포 완료됐습니다.

수행 범위를 **기계설비**와 **소방설비** 두 공종으로 정리했습니다:

- **기계설비**: 공조·배관·덕트 시공 상세도 / 위생·냉난방 배관 샵드로잉 / 장비 기초·앵커 상세
- **소방설비**: 스프링클러·소화배관 상세도 / 알람밸브·가압송수 배치도 / 현장 실측 기반 도면 작성
