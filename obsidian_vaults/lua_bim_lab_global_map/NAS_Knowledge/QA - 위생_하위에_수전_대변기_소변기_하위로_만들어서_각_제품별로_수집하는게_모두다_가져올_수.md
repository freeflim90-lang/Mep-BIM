# QA - 위생 하위에 수전 대변기 소변기 하위로 만들어서 각 제품별로 수집하는게 모두다 가져올 수

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

위생 하위에 수전 대변기 소변기 하위로 만들어서 각 제품별로 수집하는게 모두다 가져올 수 있는 방법 같아

## 답변

맞아요. 리프(leaf) 소분류 단위로 각각 수집하면 캡을 최대한 피할 수 있습니다. Obsidian 구조도 계층형으로 만들게요.

```
BIMobject/
  위생/
    BIMobject - 수도꼭지.md        (15,950개)
    BIMobject - 욕실소품.md        (6,083개)
    BIMobject - 샤워실.md          (5,609개)
    BIMobject - 세면기.md          (5,357개)
    BIMobject - 대변기.md          (2,580개)
    BIMobject - 소변기.md          ...
    ...
  공조(HVAC)/
    BIMobject - 공기조화.md
    BIMobject - 덕트설비.md
    ...
```

크롤러 전체를 이 구조로 재설계할게요.
