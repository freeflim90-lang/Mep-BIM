# QA - 제대로 작동안하는데?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

제대로 작동안하는데?

## 답변

두 가지 문제 확인됩니다:
1. 저장된 검색 결과가 "📚 내부 지식:" 형태로 그대로 노출됨
2. 내부 지식 사용 시 헤더/메타데이터 정리가 안 됨

`line_noise` 필터 강화 + 내부 지식을 메인 답변으로 직접 사용하도록 수정합니다.
