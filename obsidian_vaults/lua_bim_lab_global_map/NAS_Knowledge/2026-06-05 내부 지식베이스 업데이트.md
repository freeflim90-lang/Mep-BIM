## 핵심 변경 4: 계약 범위 자동 분석 API (server_total.py)

```
_scope_limit_assessment() — 예산·면적·LOD·기간 기반 범위 판정
_parse_krw_budget()       — 억/만원/원 자동 파싱
_contract_scope_items()   — 납품 범위 항목 자동 생성
```

예산 5단계 티어 자동 분류:
- < 500만: 사전검토/범위산정
- 500만~3000만: 소규모 MEP BIM
- 3000만~1억: 중규모 BIM 용역
- 1억~5억: 대규모/복합 BIM
- 5억+: 특수/대형 프로젝트

**관련**: [[BIM 프로젝트 견적산정 로직 내부 기준]]
