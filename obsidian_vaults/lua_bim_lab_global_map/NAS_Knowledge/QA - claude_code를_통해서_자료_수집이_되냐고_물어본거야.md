# QA - claude code를 통해서 자료 수집이 되냐고 물어본거야

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-28
> **출처:** Claude Code 세션

claude code를 통해서 자료 수집이 되냐고 물어본거야

## 답변

가능하다. Claude Code(지금 이 대화)가 직접 지식 수집을 하는 방식은 이렇게 된다:

**Claude Code가 하는 것:**
- WebSearch로 최신 정보 검색
- 검색 결과 분석·합성
- KB 파일에 직접 쓰기

**Ollama(현재 자동화)와 비교:**

| | Ollama 자동화 | Claude Code |
|---|---|---|
| 실행 | 매일 자정 무인 | 대화 중 수동 트리거 |
| 품질 | 중간 (7B 로컬 모델) | 높음 (Sonnet 4.6) |
| 비용 | 무료 | Claude Code 구독 내 포함 |
| 검색 | 4개 엔진 API | WebSearch |

**실용적인 조합:**
- **매일 자정**: Ollama가 자동으로 75개 파일 업데이트 (현행 유지)
- **필요할 때**: Claude Code에게 "소방기계 KB 업데이트해줘" → Sonnet 품질로 깊이 있는 내용 생성

지금 바로 해볼 수도 있다. 어떤 KB 파일을 Claude Code로 업데이트해볼까?
