# QA - cli로 하는데도 토큰이 필요해?

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-30
> **출처:** Claude Code 세션

cli로 하는데도 토큰이 필요해?

## 답변

네, 필요합니다. CLI도 Cloudflare 서버에 파일을 올려야 하니까 인증이 필요합니다.

다만 토큰 말고 **브라우저 로그인** 방식도 있습니다. 터미널에서 직접 실행하시면 브라우저가 열리면서 로그인할 수 있어요:

```bash
npx wrangler login
```

터미널에서 위 명령어 실행해보세요. 브라우저가 자동으로 열립니다.
