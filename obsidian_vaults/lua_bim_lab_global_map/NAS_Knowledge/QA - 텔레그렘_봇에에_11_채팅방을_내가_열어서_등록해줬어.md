# QA - 텔레그렘 봇에에 1:1 채팅방을 내가 열어서 등록해줬어

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-24
> **출처:** Claude Code 세션

텔레그렘 봇에에 1:1 채팅방을 내가 열어서 등록해줬어

## 답변

그럼 봇 토큰과 각 직원 Chat ID만 있으면 바로 구축할 수 있습니다.

구현에 필요한 정보를 알려주세요:

1. **텔레그램 봇 토큰** — BotFather에서 발급받은 `123456:ABC-DEF...` 형태
2. **Chat ID 3개** — 오수빈, 김선정, 허진석 각각
   - 확인 방법: 봇에 메시지 보낸 후 `https://api.telegram.org/bot{토큰}/getUpdates` 접속하면 `chat.id` 확인 가능
3. **메시지 형태** — 텍스트만인가요, 이미지/다이어그램도 포함할까요?

Chat ID를 아직 확인 못 하셨으면 방법 안내해드릴게요.
