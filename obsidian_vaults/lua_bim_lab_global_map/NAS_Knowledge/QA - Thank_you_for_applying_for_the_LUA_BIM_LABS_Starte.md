# QA - Thank you for applying for the LUA BIM LABS Starte

## 질문
> **카테고리:** Claude대화 | **날짜:** 2026-05-29
> **출처:** Claude Code 세션

Thank you for applying for the LUA BIM LABS Starter Plan.

This is a Telegram-based MEP BIM education service for beginners and early-stage BIM learners.

Price: USD 39 / month
Payment: PayPal

Included:
- One practical MEP BIM lesson per day
- Beginner-friendly explanation
- Basic checklist or action item
- Monthly learning direction

Not included:
- Project file review
- Revit model QA
- Clash report review
- Engineering design verification
- Code compliance confirmation
- Construction approval
- Unlimited 1:1 consulting

This service provides general educational content only.

Please do not submit confidential project files, private drawings, contracts, personal data, or materials you do not have permission to share.

Thank you for applying for the LUA BIM LABS Starter Plan.

This is a Telegram-based MEP BIM education service for beginners and early-stage BIM learners.

**Price: USD 39 / month**
**Payment: PayPal**
**Program: 90-Day MEP BIM Starter Program**

**Included:**
- One practical MEP BIM lesson per day through Telegram
- Beginner-friendly explanation with checklist or action item
- Weekly BIM Check Friday quiz (every Friday)
- Quick Reference Card at the end of each learning track (8 cards total)
- Day 60 Foundation Completion Certificate
- Day 90 Discipline Specialist Completion Certificate
- Days 1–60: MEP BIM Foundation (orientation, Revit basics, clash coordination, data management, site-readiness)
- Days 61–90: Your chosen discipline deep-dive (HVAC / Piping / Plumbing / Fire Protection / Electrical)

**Not included:**
- Project file review
- Revit model QA
- Clash report review
- Engineering design verification
- Code compliance confirmation
- Construction approval
- Unlimited 1:1 consulting
- Live video or voice sessions

This service provides general educational content only.

Please do not submit confidential project files, private drawings, contracts, personal data, or materials you do not have permission to share.

---

**변경된 핵심 포인트 3가지:**

1. **"90-Day MEP BIM Starter Program"** 명시 — 기존 60일에서 업그레이드된 것을 명확히 전달
2. **Included 항목 확장** — BIM Check Friday, Reference Cards, 수료 인증서 2개, Day 1~60 / 61~90 커리큘럼 구조 설명 추가
3. **공종 선택 안내** — Days 61~90이 선택한 공종으로 진행된다는 것을 명시
