```
Thank you for applying for the LUA BIM LABS Starter Plan.

This is a Telegram-based MEP BIM education service for beginners 
and early-stage BIM learners.

Price: USD 39 / month
Payment: PayPal
Program: 90-Day MEP BIM Starter Program

Included:
- One practical MEP BIM lesson per day through Telegram
- Beginner-friendly explanation with checklist or action item
- Weekly BIM Check Friday quiz (every Friday)
- Quick Reference Card at the end of each track (8 cards total)
- Day 60 Foundation Completion Certificate
- Day 90 Discipline Specialist Completion Certificate
- Days 1–60: MEP BIM Foundation
- Days 61–90: Your chosen discipline deep-dive

Not included:
- Project file review
- Revit model QA
- Clash report review
- Engineering design verification
- Code compliance confirmation
- Construction approval
- Unlimited 1:1 consulting
- Live video or voice sessions

This service provides general educational content only.

Please do not submit confidential project files, private drawings, 
contracts, personal data, or materials you do not have permission 
to share.
```
