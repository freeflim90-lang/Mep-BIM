# 2026-06-04 Knowledge Update Queue

이 문서는 2026-06-04 지식 수집·업데이트·Obsidian 반영 결과를 실제 실행 큐로 전환한 운영 노트다. 자동 분류 리포트가 검토 대상을 보여준다면, 이 큐는 오늘 먼저 처리할 항목과 보류할 항목을 정한다.

## 큐 요약

| 우선순위 | 목적 | 오늘 판단 |
|---|---|---|
| `PRI01 즉시 차단` | 보안·개인정보·고객 신뢰 위험 차단 | 원문 링크 제거와 보안검토 기록을 먼저 확인 |
| `PRI02 납품 영향` | 납품, 검수, 고객 응답 영향 해소 | 납품 검수·QA·고객지원 문서 연결 우선 |
| `PRI03 기준 병목` | 반복을 줄이는 공통 기준 고정 | 뉴스 트렌드·지식 큐레이션 반복 기준 분리 |
| `PRI04 재검증 예약` | 공식 출처 확인 필요 항목 관리 | Autodesk, 시장 수치, 정책 후보는 Watch 유지 |
| `PRI05 링크 회복` | Obsidian 고립 지식 연결 | 관련 위키링크 2개 이상과 MOC 연결 확인 |
| `PRI06 중복 압축` | 반복 QA와 중복 기준 정리 | 대표 FAQ 지정 후 반복 문서 링크 전환 |
| `PRI07 승격 후보` | 교육·SOP·상품·자동화 후보 관리 | 파일럿 조건 또는 needs-pilot 상태 기록 |
| `PRI08 백로그 유지` | 낮은 영향도 참고 기록 보류 | 보류 사유와 다음 처리 조건 기록 |

## 오늘 실행 큐

| 티켓 | 개선 코드 | 우선순위 | 대상 | 오늘 실행 | 완료 증빙 |
|---|---|---|---|---|---|
| 민감정보 포함 자료 차단 기준 확인 | `FIX05 보안 정리` | `PRI01 즉시 차단` | `docs/generated_contracts/`, 보안검토 후보 문서 | 원문 링크 반영 금지, 보안검토 기록만 유지 | 고객명·계약·비용·내부 경로 미노출 확인 |
| 납품 검수 지식 링크 회복 | `FIX01 링크 보강` | `PRI02 납품 영향` | `BIM_납품검수`, `QA_테스터`, `고객지원운영` | 관련 링크와 검수 체크리스트 연결 | 관련 위키링크 2개 이상, 다음 검토일 |
| 뉴스 트렌드 공통 기준 분리 | `FIX07 기준 분리` | `PRI03 기준 병목` | `지식업데이트`, `산업동향_데일리브리핑`, `건설시장_트렌드` | 공통 기준 노트 후보 지정 | 날짜별 로그가 기준 노트로 링크됨 |
| Autodesk 정책 후보 재검증 예약 | `FIX03 출처 재검증` | `PRI04 재검증 예약` | `스토어심사`, `Revit_Addin` | 공식 문서 확인 전 Watch 유지 | 공식 출처 또는 미확인 사유, 다음 확인일 |

| 반복 Telegram QA 전체 병합 | `PRI06 중복 압축` | 대표 FAQ 기준 선정 필요 | 공식 출처가 확인된 대표 문서 1개 지정 |

## Obsidian 반영 기준

- 확정 지식은 관련 위키링크 2개 이상과 다음 검토일을 포함한다.
- 보안검토 항목은 원문 링크를 남기지 않고 보안검토 기록만 남긴다.
- Watch 항목은 확정 문서가 아니라 후보 상태와 다음 확인일을 기록한다.
- 고립 지식은 `PRI05 링크 회복` 대상으로 다시 큐에 올린다.
