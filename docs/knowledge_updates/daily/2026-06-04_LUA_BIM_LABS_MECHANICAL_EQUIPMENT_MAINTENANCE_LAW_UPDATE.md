# 2026-06-04 LUA BIM LABS Mechanical Equipment Maintenance Law Update

| 프로젝트 적용 | `KST03 적용주의` | 대상 건축물, 유지관리 주체, 발주처 요구, 성능점검 범위를 확인해야 한다. |

국가법령정보센터의 기계설비법, 기계설비 유지관리기준, 시행령 제정·개정이유를 기준으로 LUA BIM LABS의 공조·위생·설비장비·FM BIM 검토에 필요한 기계설비 유지관리/성능점검 운영 기준을 정리한다. 이 문서는 법률 자문이 아니라 `needs-review` 지식 업데이트이며, 실제 프로젝트 적용 전 최신 법령·고시 전문, 발주처 요구, 유지관리 주체, 기계설비 전문가 확인을 거친다.

## LUA BIM LABS 반영 판단

### 1. 기계설비 BIM은 유지관리·성능점검 데이터까지 본다

공조배관, 공조덕트, 위생, 자동제어, 설비장비 BIM은 납품 시 형상과 clash만 확인하면 부족하다. 기계설비법 체계에서는 유지관리지침서, 점검대상 기계설비 현황, 유지관리 결과, 성능점검 검토사항과 연결되는 데이터가 필요하다.

운영 기준:
- 장비번호, 장비명, 계통, 용량, 설치 위치, 제조사, 모델명, 점검 접근공간을 필수 후보로 둔다.
- 밸브, 스트레이너, 필터, 코일, 펌프, 팬, 냉동기, 보일러, 급수펌프, 오수펌프, 자동제어 포인트는 유지관리 접근성을 별도 검토한다.
- COBie/FM 이관이 있는 프로젝트는 기계설비 유지관리 기준 대응 필드를 별도 정의한다.

### 2. 성능점검 대상성과 BIM 납품 범위를 분리한다

모든 모델링 항목이 성능점검 대상은 아니다. 하지만 성능점검 대상 장비·계통은 점검 가능성, 접근성, 측정 위치, 기준값, 기록 가능성을 BIM 검수에서 분리해야 한다.

게이트:
- 기계설비법 적용 대상 여부 확인
- 기계설비 유지관리기준 적용 여부 확인
- 점검대상 기계설비 현황표 작성 필요 여부 확인
- 성능점검 대상 장비/계통 식별
- 측정 접근성(유량, 압력, 온도, 전류, 차압 등) 확인
- 점검기록·성능점검 보고서와 BIM 파라미터 연결 여부 확인

### 3. 자동 수집 유지관리 수치와 LOD 문장은 확정 기준으로 승격하지 않는다

기존 설비장비·공조·위생 지식베이스에는 자동 수집된 LOD, 유지관리 주기, 장비 파라미터 수치가 섞여 있다. 법령·고시·발주처 요구·제조사 O&M 문서 확인 전에는 납품 판정 기준이나 견적 책임으로 확정하지 않는다.

운영 기준:
- 유지관리 주기와 성능점검 주기는 법령·고시·발주처 O&M 요구로 확인한다.
- 장비 점검 공간은 제조사 도면, 유지관리 지침서, 현장 반입·보수 계획을 함께 확인한다.
- 자동 수집 LOD 문장은 발주처 EIR/BEP/LOD 매트릭스 확인 전 확정하지 않는다.

### 4. 견적에는 유지관리·성능점검 대응 공수를 별도 반영한다

기계설비 유지관리/성능점검 대응 BIM은 일반 설계 모델보다 데이터 정리와 검수표 작성 공수가 커진다. FM 이관, 장비대장, COBie, 유지관리 지침서 대응이 있으면 별도 견적 항목으로 둔다.

견적 반영 항목:
- 기계설비법/유지관리기준 적용성 확인
- 점검대상 장비·계통 식별
- 장비대장·COBie·FM 파라미터 매핑
- 점검 접근성 검토
- 유지관리지침서/O&M 문서 대조
- 성능점검 보고서 대응 데이터 정리
