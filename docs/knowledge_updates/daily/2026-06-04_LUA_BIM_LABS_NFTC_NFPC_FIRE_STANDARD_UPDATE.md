국가법령정보센터의 화재안전기술기준(NFTC) 전자법령집, NFPC/NFTC 행정규칙, 소방시설 설치 및 관리에 관한 법률 체계를 기준으로 LUA BIM LABS의 소방기계·소방전기 BIM 검토, 견적, 납품검수에 필요한 소방 기준 운영 원칙을 정리한다. 이 문서는 법률·소방설계 자문이 아니라 `needs-review` 지식 업데이트이며, 실제 프로젝트 적용 전 최신 행정규칙 전문, 관할 소방서 해석, 소방 설계자 확인을 거친다.

## LUA BIM LABS 반영 판단

### 1. NFPC와 NFTC를 역할별로 분리한다

NFPC는 화재안전성능기준, NFTC는 화재안전기술기준이다. BIM 검토에서 두 기준을 한 줄로 "소방법 기준"이라고 처리하면 성능 요구와 기술 세부 요구가 섞여 견적·검수 책임이 흐려진다.

운영 기준:
- NFPC: 성능 요구 확인축
- NFTC: 설치·관리·기술 세부 기준 확인축
- KEC/KDS 32: 전기적 안전과 배선·전원 기준 확인축
- KDS/KCS 31: 설비 시공·기계 기준 보조축
- 관할 소방서·소방 설계자 해석: 프로젝트 최종 적용 확인축

### 2. 구 NFSC 명칭과 자동 수집 수치는 확정 지식으로 승격하지 않는다

기존 지식베이스에는 NFSC 구명칭, 자동 수집된 NFTC/KCS 조항번호, 검증되지 않은 소방 수치가 섞여 있다. 최신 행정규칙 전문과 관할 소방서 해석 전에는 납품 판정 기준으로 쓰지 않는다.

게이트:
- 최신 NFTC/NFPC 행정규칙명 확인
- 공고 번호와 시행일 확인
- 구 NFSC 인용 여부 확인
- NFPC 성능기준과 NFTC 기술기준의 대응 관계 확인
- KEC/KDS/KCS/건축 방화구획 기준과 충돌 여부 확인
- 수치 기준을 Revit 파라미터나 검토표에 넣기 전 근거 URL 또는 문서번호 기록

### 3. 소방 BIM 검수는 일반 MEP clash와 분리한다

소방기계·소방전기 이슈는 단순 물리 간섭보다 인허가·성능·방재 연동에 직접 영향을 준다. 스프링클러 헤드 장애, 감지기 사각, 내화배선 경로, 제연 구역, 방화셔터 연동, 비상전원은 일반 clash 시트와 별도 소방 검수 시트로 관리한다.

검수 필드:
- 적용 NFPC 코드
- 적용 NFTC 코드
- 공고 번호 및 시행일
- 소방기계/소방전기/제연/피난 기준 구분
- 관할 소방서 또는 소방 설계자 확인 여부
- Revit 파라미터와 기준 항목 매핑 여부
- 자동 검토 가능/수동 검토 필요 구분

### 4. 견적에는 소방 기준 검토·소방 협의 공수를 별도 반영한다

소방 BIM 용역은 모델링 공수 외에 기준 확인, 소방 설계자 질의응답, 관할 소방서 해석 반영, 재검토 버퍼가 필요하다. 특히 창고, 지하주차장, 고층건축물, 데이터센터, 전기저장시설, 병원은 별도 NFTC 기준과 특수 위험 기준을 확인해야 한다.

견적 반영 항목:
- NFPC/NFTC 최신 전문 확인
- 설비별 기준 코드 매핑
- 소방기계·소방전기·제연·피난 기준 충돌 검토
- 관할 소방서 또는 소방 설계자 질의응답
- 소방 전용 검수표 작성
- 기준 변경 및 인허가 해석 변경 버퍼
