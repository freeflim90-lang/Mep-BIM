## 2. 핵심 업데이트

BF 및 편의시설 기준은 건축 BIM에서 다음 항목과 직접 연결된다.

- 주출입구 접근로
- 장애인전용주차구역
- 주출입구 높이차이 제거
- 출입구, 문, 복도
- 계단 또는 승강기
- 장애인등이 이용 가능한 화장실, 욕실, 샤워실, 탈의실
- 점자블록, 유도 및 안내설비
- 경보 및 피난설비
- 객실, 관람석, 접수대, 작업대, 매표소, 판매기, 음료대, 임산부 등을 위한 휴게시설

법령과 별표에는 대상시설, 의무/권장 구분, 구조·재질 세부기준이 함께 걸려 있다. 따라서 BIM 검수는 단순 치수 체크가 아니라 `대상시설 판정`, `의무/권장 구분`, `설계도서·특기시방서 확인`, `예비인증/본인증 단계 확인`을 함께 기록해야 한다.

## 3. BIM 적용 게이트

건축 BIM 모델에서 BF/편의시설을 검토할 때 다음 필드를 추가한다.

| 구분 | 확인 필드 |
|---|---|
| 대상성 | 대상시설 여부, 시설 용도, 규모, 공공/민간 여부 |
| 기준 계층 | 장애인등편의법, 시행령 별표, 시행규칙 별표, BF 인증 기준, 지자체 조례/사전검사 |
| 인증 단계 | 예비인증, 본인증, 인증연장, 미대상 |
| 모델 항목 | 접근로, 주차, 출입구, 복도, 승강기, 화장실, 안내설비, 피난설비, 객실/관람석/접수대 |
| 판정 구분 | 자동 검토 가능, 수동 검토 필요, 전문가 확인 필요 |
| 증빙 | 도면 번호, 모델 뷰, 체크리스트, 인증 신청/심사 문서, RFI 번호 |
