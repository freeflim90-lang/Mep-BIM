한국전기설비규정(KEC), 전기설비기술기준, 국가법령정보센터, KDS/KCS 32 공식 신호를 기준으로 LUA BIM LABS의 전기 BIM 검토, 견적, 납품검수, 법규 모니터링에 필요한 운영 기준을 정리한다. 이 문서는 법률·전기설계 자문이 아니라 `needs-review` 지식 업데이트이며, 실제 프로젝트 적용 전 최신 KEC 전문, 전기설비기술기준, KDS/KCS 32, 발주처 특기시방서, 전기 설계자 확인을 거친다.

## LUA BIM LABS 반영 판단

### 1. 전기 BIM 검토는 KEC 단독이 아니다

전기 BIM 검토는 KEC, 전기설비기술기준, KDS/KCS 32, KEC 관련 상세 지침, KS, 발주처 특기시방서, 소방 기준, 통신 기준이 함께 영향을 준다. 특히 전기 트레이, 분전반, 접지, 피뢰, 비상전원, EV 충전, ESS, UPS는 프로젝트 용도와 설비 종류에 따라 기준 조합이 달라진다.

운영 기준:
- KEC: 전기설비 안전·기술 기준의 핵심 확인축
- KDS/KCS 32: 건설기준 체계의 설비(전기) 설계·시공 기준
- KS/IEC: 전선, 케이블, 외함, 트레이, 시험 기준 확인축
- NFTC/NFPC: 소방전기, 비상전원, 내화·내열 배선 관련 우선 확인축
- 발주처 특기시방서: 프로젝트 계약 기준

### 2. KEC 조항번호와 수치 문장은 공식 전문 확인 전 확정 지식으로 승격하지 않는다

기존 전기 지식베이스에는 자동 수집된 KEC 조항번호와 수치 문장이 다수 포함되어 있다. 조항번호가 실제 KEC 체계와 맞지 않거나, 구 판단기준·내선규정·프로젝트 특기시방서와 섞였을 수 있으므로 납품 판정 기준으로 바로 사용하지 않는다.

게이트:
- KEC 최신 공고 번호와 시행일 확인
- 실제 KEC 전문 조항번호 확인
- KDS/KCS 32 또는 발주처 특기시방서와의 우선순위 확인
- 전기 설계자 또는 감리 확인 필요 여부 표시
- 수치 기준을 Revit 파라미터나 검토표에 넣기 전 근거 URL/문서번호 기록

### 3. 전기 BIM 납품검수는 형상, 접근성, 기준 근거를 분리한다

전기 BIM 검수는 케이블 트레이 간섭, 분전반 접근 공간, 강전·약전 이격 같은 형상 검토와 KEC/KDS/KCS 기준 근거 검토를 분리해야 한다. 모델에서 특정 이격이 확보되어도 해당 프로젝트 기준이 무엇인지, 격벽 대체가 가능한지, 통신·소방 기준과 충돌하지 않는지는 별도 확인이 필요하다.

검수 필드:
- KEC 공고 번호 및 기준일
- 적용 KEC 장/절/조항
- 적용 KDS/KCS 32 코드
- 발주처 특기시방서 조항
- 전기 설계자 확인 여부
- 모델 파라미터와 기준 항목 매핑 여부
- 자동 검토 가능/수동 검토 필요 구분

### 4. 견적에는 전기 기준 해석·검증 공수를 별도 반영한다

전기 BIM 용역은 트레이/분전반 모델링 공수만으로 끝나지 않는다. KEC 조항 확인, 기준 충돌 검토, 전기 설계자 질의응답, 검토표 작성, 납품 증빙 정리 공수를 별도로 산정해야 한다.

견적 반영 항목:
- KEC 최신 공고 및 기준 전문 확인
- KDS/KCS 32와 KEC 대조
- 강전·약전·소방전기·통신 기준 충돌 검토
- Revit 파라미터와 검토 항목 매핑
- 전기 설계자/감리 질의응답
- 기준 변경 및 발주처 해석 변경 버퍼
