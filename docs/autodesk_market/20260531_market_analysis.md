- 예상 개발 기간: 2~3개월 (Navisworks API + 엑셀 내보내기)
- 수익 모델: 구독 $15~$30/월 또는 연간 $150. 한국 현장 대형 프로젝트 1건당 ROI 명확
- 근거: 이미 개발 큐 존재, 경쟁 앱 없음, LUA BIM LABS 핵심 역량 일치

**2위: 한국 BIM 납품 자동 검증기 (IFC + Revit)**
- 앱 컨셉: 발주처 BIM 지침 룰셋 로드 → 모델 자동 검증 → 미달 항목 리포트
- 예상 개발 기간: 3~4개월
- 수익 모델: B2B 연간계약 $500~$2,000/프로젝트. 공공 발주 BIM 의무화로 수요 급증 중
- 근거: 한국 고유 규격 = 글로벌 경쟁자 진입 불가 구조

**3위: Revit 한글 파라미터 엑셀 동기화 Pro**

- 예상 개발 기간: 1~2개월
- 수익 모델: 일회성 $30~$50 또는 구독 $5/월. 빠른 수익화, 낮은 진입장벽
