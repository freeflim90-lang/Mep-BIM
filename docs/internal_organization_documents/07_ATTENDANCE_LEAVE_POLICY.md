## 1. 목적

본 문서는 근무시간, 휴가, 재택/외근, 지각/결근 보고 기준을 내부 운영 차원에서 정의한다. 실제 근로계약과 법정 기준이 있는 경우 해당 기준을 우선한다.

## 2. 근태 기준

| 항목 | 기준 |
|---|---|
| 근무시간 | 계약 또는 내부 합의 기준 |
| 출근/퇴근 기록 | 지정 방식으로 기록 |
| 외근/현장 방문 | 사전 공유 및 일정 등록 |
| 재택근무 | 업무 내용과 연락 가능 시간 명시 |
| 지각/결근 | 가능한 한 사전 보고 |

## 3. 휴가 요청

| 단계 | 내용 |
|---|---|
| 요청 | 휴가일, 사유 유형, 업무 인수인계 작성 |
| 검토 | 일정/납품/회의 영향 확인 |
| 승인 | 담당 리드 또는 경영지원 승인 |
| 기록 | 휴가대장 반영 |

## 4. 인수인계

휴가 또는 부재 전 진행 중인 모델, 보고서, 고객 응답, 회의 일정, 차단 이슈를 문서로 남긴다.
