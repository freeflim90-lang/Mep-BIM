## 1. 목적

본 문서는 신규 구성원의 30/60/90일 온보딩, 수습 평가, 멘토링, 업무 투입 기준을 정의한다.

## 2. 30/60/90일 운영

| 기간 | 목표 | 확인 항목 |
|---|---|---|
| 30일 | 조직/보안/CDE 적응 | 계정, 문서, 보안, 기본 실습 |
| 60일 | 기본 업무 수행 | 모델링, 체크리스트, 회의 참여 |
| 90일 | 부분 독립 수행 | 산출물 제출, 리뷰 반영, 업무 태도 |

## 3. 수습 평가 항목

| 평가 항목 | 기준 |
|---|---|
| 기술 습득 | Revit/CDE/문서 기준 이해 |
| 업무 태도 | 일정 준수, 질문 품질, 피드백 반영 |
| 보안 준수 | 고객 자료 취급 기준 이해 |
| 협업 | 회의, 보고, 이슈 공유 |
| 성장 가능성 | 학습 속도와 책임감 |

## 4. 수습 종료 판단

| 결과 | 조치 |
|---|---|
| 전환 | 정규 역할/교육 로드맵 지정 |
