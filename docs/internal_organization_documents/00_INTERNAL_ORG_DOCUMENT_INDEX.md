## 4. 내부 문서 작성 원칙

1. 내부 문서는 역할, 책임, 승인권자를 명확히 한다.
2. 외부 제출 문서와 달리 미확정 내부 판단, 개선 과제, 리스크를 숨기지 않는다.
3. 비용, 인사, 보안, 계약 관련 문서는 접근 권한을 제한한다.
4. 프로젝트 종료 후 발견된 반복 문제는 내부 운영 문서 개정 후보로 등록한다.
