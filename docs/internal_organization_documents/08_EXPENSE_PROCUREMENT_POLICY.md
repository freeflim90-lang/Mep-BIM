## 1. 목적

본 문서는 프로젝트 수행과 조직 운영에 필요한 비용 정산, 장비/소프트웨어 구매, 외주 비용 요청 기준을 정의한다.

## 2. 비용 유형

| 유형 | 예시 | 승인 |
|---|---|---|
| 프로젝트 비용 | 출장, 현장 방문, 출력, 장비 임대 | PM/대표 |
| 소프트웨어 | Revit, Navisworks, ACC, 생산성 도구 | 대표/경영지원 |

| 외주 | 모델링, 스캔, 문서 지원 | PM/대표 |

| 필드 | 설명 |
|---|---|
| 요청자 | 이름/소속 |

## 4. 정산 기준

영수증, 세금계산서, 거래명세서 등 증빙을 제출한다. 고객 청구 가능 비용과 내부 비용은 구분한다.

## 5. Telegram 영수증 자동 정리 담당 AI

| 항목 | 기준 |
|---|---|
| 담당 AI | 경비정산_AI |
| 소유 부서 | 경영지원 |
| 처리 채널 | 팀원 Telegram 영수증/증빙 제출 |
| 기본 배치 | 로컬 전용, DeepSeek API 사용 금지 |
| 협업 담당 | 경영지원, 엑셀자동화, Qwen_Coder_8B, 라이선스_보안관, CFO |

`경비정산_AI`는 Telegram으로 제출된 영수증, 세금계산서, 거래명세서, 카드전표를 수집하고 날짜, 금액, 사용처, 결제자, 비용 유형, 프로젝트/내부 비용 구분을 정리한다.

누락, 중복, 금액 불일치, 사용처 불명확 항목은 확정 처리하지 않고 질문자에게 추가 확인 요청으로 돌린다. 개인정보, 카드번호, 사업자번호, 고객명, 프로젝트명은 외부 AI API로 전송하지 않는다.

월별 정산표 또는 CSV/XLSX가 필요한 경우 `엑셀자동화`와 `Qwen_Coder_8B`가 로컬에서 초안을 만들고, 최종 회계/세무 제출 여부는 `경영지원`이 확정한다.
