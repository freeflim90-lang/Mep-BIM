## 1. 목적

본 문서는 구성원의 성과, 역량, 산출물, 협업 태도를 기준으로 평가와 승급을 운영하기 위한 기준이다.

## 2. 평가 축

| 평가 축 | 비중 | 설명 |
|---|---:|---|
| BIM 기술 | 30% | 모델링, 품질, CDE, 납품 |
| 설계/시공 이해 | 20% | MEP 공종 이해, 현장성 판단 |
| 문서/보고 | 15% | 회의록, 보고서, 제출대장 |
| 협업/책임 | 20% | 일정, 소통, 리뷰 반영 |
| 개선/자동화 | 15% | 반복 업무 개선, AI/도구 활용 |

## 3. 승급 기준

| 단계 | 승급 조건 |
|---|---|
| Junior → Modeler | 기준 모델링과 CDE 제출 독립 수행 |
| Modeler → Practitioner | 공종 품질/간섭/변경 보고 가능 |
| Practitioner → Coordinator | 다공종 조율과 회의 운영 가능 |
| Coordinator → Lead | BEP/납품/협력사 검수 가능 |
| Lead → PM | 제안/SOW/고객 의사결정 지원 가능 |

## 4. 평가 산출물

| 산출물 | 설명 |
|---|---|
| 대표 모델 | 본인이 수행한 모델 또는 검토 결과 |
| 보고서 | 간섭/품질/물량/변경 보고 |
| 피드백 기록 | 리뷰 반영 이력 |
| 개선 사례 | 자동화, 표준 개선, 교육 기여 |
