# LUA BIM LABS Personal MEP BIM Tutor

## Positioning

LUA BIM LABS Personal MEP BIM Tutor is a Telegram-based 1:1 MEP BIM mentoring service tailored to each learner's current skill level, discipline, weak points, and project goals.

It is not a generic BIM newsletter. It is a personalized daily training system for practical BIM growth.

## Core Offer

- Daily personalized MEP BIM lesson through Telegram
- Level-based explanation for beginner, modeler, coordinator, lead, or automation learner
- Discipline-specific guidance for HVAC, piping, plumbing, fire protection, electrical, and coordination
- Practical checklist and one action item per lesson
- Telegram Q&A for weak-point tracking
- Obsidian knowledge capture for reusable training assets

## Client Levels

| Level | Learner Type | Training Focus |

| Level 5 | BIM Automation Learner | Dynamo, Python, Revit API concepts, repeatable checks |
