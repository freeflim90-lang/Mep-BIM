## 1. 목적

본 문서는 프로젝트 수행 중 고객으로부터 수령하거나 LUA BIM LAB이 작성하는 자료의 보안 및 비밀유지 원칙을 설명한다.

## 2. 보호 대상 자료

| 자료 | 예시 |
|---|---|
| 설계 자료 | 도면, BIM 모델, 계산서, 시방서 |
| 현장 자료 | 사진, 스캔 데이터, 출입/보안 정보 |
| 프로젝트 문서 | 회의록, RFI, 변경대장, 보고서 |
| 계약/비용 자료 | 견적, 단가, 계약 조건 |
| 개인정보 | 담당자 연락처, 계정 정보 |

## 3. 자료 관리 원칙

- 프로젝트 자료는 지정된 CDE 또는 고객 승인 저장소에서 관리한다.
- 고객 승인 없이 공개 저장소나 개인 클라우드에 업로드하지 않는다.
- 외부 제출 자료는 필요한 범위로만 구성한다.
- 보안 구역, 중요 설비, 개인정보가 포함된 자료는 접근 권한을 제한한다.

## 4. 접근 권한

| 역할 | 접근 기준 |
|---|---|
| 고객/발주처 | 공식 배포 자료 |
| LUA BIM LAB PM | 프로젝트 전체 자료 |
| 모델러/검토자 | 담당 공종/구역 자료 |
| 협력사 | 승인된 업무 범위 내 최소 자료 |

## 5. 외부 공유 및 협력사 관리

협력사에게 자료를 제공할 경우 업무 범위, 보안 의무, 재공유 금지, 종료 후 삭제/반납 기준을 명확히 한다.

## 6. 보관 및 종료

| 단계 | 기준 |
|---|---|
| 수행 중 | CDE 기준으로 버전 관리 |
| 납품 후 | 최종 승인본과 제출대장 보관 |
| 종료 후 | 계약/고객 요구사항에 따라 보관 또는 삭제 |

## 7. 사고 대응

자료 오발송, 권한 오류, 외부 노출이 확인될 경우 즉시 접근을 차단하고, 영향 범위와 조치 결과를 고객과 내부 승인권자에게 보고한다.
