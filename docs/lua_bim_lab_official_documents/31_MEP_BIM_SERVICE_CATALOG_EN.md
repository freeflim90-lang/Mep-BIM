## 1. Service Overview

LUA BIM LABS provides MEP (Mechanical, Electrical, Plumbing) BIM services for construction projects across design, procurement, and construction phases. Our services are available individually or as integrated packages tailored to project requirements.

We combine deep domain expertise in MEP engineering with proprietary Revit add-in tools to deliver models that are accurate, coordinated, and ready for downstream FM use.

---

## 2. Service Portfolio

| Service | Key Content | Applicable Phase |
|---|---|---|
| BIM Execution Plan (BEP) | BIM goals, organization, CDE, deliverable plan | Project Initiation |
| MEP BIM Modeling | Mechanical / Electrical / Fire Protection / Telecom models | Design / Construction |
| Scan-to-BIM Support | As-built capture for existing facilities | Renovation / Retrofit |
| Clash Detection | Cross-discipline clash review and issue management | Pre-construction |
| Constructability Review | Access space, equipment clearance, installation sequence | Construction Prep |
| Quantity Takeoff & Change Management | Model-based quantities and change impact analysis | Contract / Billing |
| Quality Assurance | Model compliance, deliverable acceptance inspection | Milestones / Closeout |
| As-Built BIM Delivery | IFC, COBie, FM data integration | Closeout |

## 6. Revit Add-in Solutions

LUA BIM LABS develops proprietary Revit add-ins that augment our BIM service delivery:

| Add-in | Function | Benefit |
|---|---|---|
| BIM Command Center | Centralized control panel for MEP BIM workflows | Reduces multi-click tasks to single operations |
| Auto Clash Tagger | Automates clash annotation and issue numbering | Eliminates manual clash log creation |
| MEP Sheet Generator | Auto-generates MEP layout sheets from model views | Hours of drawing setup reduced to minutes |
| Quantity Exporter | Exports model quantities to structured XLSX | No manual parameter mapping required |

> Add-ins are available on Autodesk App Store. Enterprise customization available on request.

## 7. Items for Discussion

Before project commencement, the following should be confirmed:

- [ ] Project phase and applicable BIM LOD (LOD 200 / 300 / 350 / 400)
- [ ] Supported Revit version(s)
- [ ] CDE platform (Autodesk Construction Cloud, BIM 360, SharePoint, etc.)
- [ ] Clash detection frequency and participating disciplines
- [ ] Deliverable format and submission schedule
- [ ] FM handover requirements (IFC schema, COBie requirements)
- [ ] NDA / data security requirements
