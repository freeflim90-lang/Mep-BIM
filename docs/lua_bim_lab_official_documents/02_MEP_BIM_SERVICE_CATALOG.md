## 1. 서비스 개요

본 문서는 LUA BIM LAB이 제공하는 MEP BIM 서비스의 범위와 산출물을 설명한다. 서비스는 프로젝트 특성에 따라 단독 또는 패키지 형태로 적용할 수 있다.

## 2. 서비스 구성

| 서비스 | 주요 내용 | 적용 대상 |
|---|---|---|
| BIM 수행계획/BEP | BIM 목표, 조직, CDE, 산출물 계획 | 착수 단계 |
| MEP BIM 모델링 | 기계/전기/소방/통신 모델 구축 | 설계/시공 단계 |
| Scan to BIM 지원 | 기존 시설 현황 반영 | 리모델링/개보수 |
| 간섭 검토 | 공종 간 Clash 및 이슈 관리 | 설계/시공 전 |
| 시공성 검토 | 점검공간, 장비 반입, 설치 순서 | 시공 준비 |
| 물량/변경 관리 | 모델 기반 수량 및 변경 영향 | 공무/정산 |
| 품질 검증 | 모델 품질, 납품 기준 검수 | 단계별/준공 |
| 준공 BIM 납품 | IFC, COBie, FM 데이터 연계 | 준공 단계 |

## 3. 단계별 수행 내용

| 단계 | 업무 | 고객 제공 자료 | 산출물 |
|---|---|---|---|
| 착수 | 요구사항 분석, CDE 구성 | 도면, 모델, 공정표, EIR | 착수자료 목록, BEP 초안 |
| 모델 구축 | 공종별 BIM 모델 작성 | DWG/PDF/RVT/IFC | 공종별 모델 |
| 통합 검토 | 간섭/품질 검토 | 통합 모델 | 간섭 보고서, 이슈대장 |
| 현장 운영 | 변경/RFI/회의 관리 | 변경 지시, 현장 이슈 | 회의록, 변경대장 |
| 납품 | 최종 패키지 구성 | 검수 기준 | 납품 패키지, 인수확인 |
