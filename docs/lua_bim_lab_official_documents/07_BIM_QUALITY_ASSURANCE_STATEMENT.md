## 1. 품질보증 목적

LUA BIM LAB은 BIM 모델이 시공 검토, 간섭 검토, 물량 산출, 납품 검수에 활용될 수 있도록 단계별 품질 검증 절차를 운영한다.

## 2. 품질 검토 범위

| 구분 | 검토 항목 |
|---|---|
| 파일 품질 | 파일 열림, 링크, 버전, 명명 규칙 |
| 모델 정합 | 좌표, 레벨, 그리드, 링크 중첩 |
| 정보 품질 | 필수 파라미터, 시스템명, 공종/구역 정보 |
| 간섭 품질 | Clash 분류, 책임자, 조치기한 |
| 납품 품질 | 제출대장, 형식, Revision, 승인본 일치 |

## 3. 검토 기준

| 항목 | 기준 |
|---|---|
| 좌표/레벨 | 프로젝트 기준과 일치 |
| LOD/LOI | BEP 또는 고객 요구사항 기준 |
| 필수 정보 | 합의된 파라미터 누락 없음 |
| 간섭 상태 | 미해결 이슈는 책임자와 조치기한 보유 |
| 납품 파일 | 지정 소프트웨어에서 열람 가능 |

## 4. 보고 체계

| 보고서 | 주기 | 대상 |
|---|---|---|
| 간섭 검토 보고서 | 주간/격주 | 시공사, 설계사, 감리/CM |
| 모델 품질 보고서 | 단계별 | 고객 PM, BIM PM |
| 변경 영향 보고서 | 변경 발생 시 | 고객 PM, 공종 담당 |
| 납품 검수 보고서 | 납품 전 | 승인권자 |

## 5. 고객 확인 사항

품질 기준은 프로젝트 착수 시 고객 요구사항, EIR/BEP, 계약 범위, 납품 형식을 기준으로 확정한다.
