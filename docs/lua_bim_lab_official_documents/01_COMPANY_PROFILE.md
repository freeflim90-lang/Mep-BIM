## 1. 회사 개요

LUA BIM LAB은 MEP BIM 수행, BIM 데이터 관리, 간섭 검토, 물량 및 변경 관리, 품질 검증, 납품 문서화를 통합 지원하는 BIM 전문 조직이다.

LUA BIM LAB은 AX(AI Transformation)를 BIM 실무에 적용하여 설계, 시공, 품질검토, 납품, 교육, 자동화 전반을 지능화하고 BIM 업계에서 선도적인 기술 기업으로 성장하는 것을 목표로 한다.

| 항목 | 내용 |
|---|---|
| 조직명 | LUA BIM LAB |
| 핵심 분야 | MEP BIM, CDE, 간섭 검토, 품질 검증, 납품 관리, BIM AX 자동화 |
| 주요 고객 | 발주처, 시공사, 설계사, 감리/CM, 협력사 |
| 수행 방식 | 기준 수립, 모델 운영, 보고, 검증, 납품, 자동화를 하나의 흐름으로 관리 |

## 2. 핵심 업무

| 업무 | 설명 |
|---|---|
| BIM 수행계획 | BEP, CDE, 모델링 기준, 산출물 계획 수립 |
| MEP BIM 모델링 | 기계, 배관, 위생, 소방, 전기, 통신 모델 구축 |
| 간섭 검토 | Navisworks 기반 Clash Detection 및 이슈 관리 |

| 물량/변경 관리 | 모델 기반 물량 산출과 변경 영향 분석 |
| 납품 관리 | RVT/IFC/NWD/보고서/COBie/FM 데이터 패키지 구성 |

## 3. MEP BIM 전문성

LUA BIM LAB은 MEP 시스템을 단순 객체 배치가 아니라 설계 의도, 시공 가능성, 유지관리, 납품 데이터까지 연결되는 정보 체계로 관리한다.

| 공종 | 주요 검토 항목 |
|---|---|
| 공조덕트 | 풍량, 정압, 점검구, 천장고, 제연 |
| 공조배관 | 관경, 단열, 밸브 조작 공간, 장비 접속 |
| 위생 | 급수/배수, 통기, 구배, 슬리브 |
| 소방기계 | 스프링클러, 소화배관, 알람밸브, 법규 민감 구간 |
| 전기 | 케이블트레이, 분전반, 이격, 점검 공간 |
| 통신/보안 | 네트워크, CCTV, 출입통제, 배선 경로 |

## 4. 프로젝트 수행 방식

| 단계 | 수행 내용 |
|---|---|
| 착수 | 자료 수령, 요구사항 확인, BEP/CDE 구성 |
| 구축 | 공종별 모델 작성, 좌표/레벨/링크 기준 정리 |
| 검토 | 간섭, 품질, 변경, 물량 검토 |
| 운영 | 회의, RFI, 변경대장, 이슈대장 관리 |
| 납품 | 최종 모델, 보고서, 제출대장, 인수확인 관리 |

## 5. 차별화 요소

- MEP 설계와 시공성을 함께 고려한 BIM 수행
- CDE, 문서, 보고, 납품 기준까지 포함한 프로젝트 운영
- 모델을 물량, 변경, 품질, 준공 데이터의 근거로 활용
- 고객 의사결정에 필요한 보고서와 이슈 관리 체계 제공
- AI를 반복 문서화와 검토 보조에 활용하되 최종 판단은 전문가가 검증
- 반복 BIM 업무를 지식화, 데이터화, 자동화하여 AX 기반 업무 체계로 전환
- Revit/Navisworks/IFC/IDS 기반 모델 품질 검증과 Add-in 제품화 역량 보유

## 7. 협업 원칙

LUA BIM LAB은 고객의 프로젝트 요구사항과 현장 운영 조건을 기준으로 수행 범위, 책임, 일정, 산출물을 명확히 정의한다. 모든 공식 의사결정은 회의록, RFI, 변경대장, 제출대장을 통해 추적 가능하게 관리한다.
