## 1. 교육 목적

본 교육은 고객이 BIM 모델, CDE, 간섭 보고서, 물량/변경 자료, 납품 패키지를 실제 업무에서 활용할 수 있도록 지원하기 위한 과정이다.

## 2. 교육 대상

| 대상 | 교육 초점 |
|---|---|
| 발주처/CM | 납품 검수, 품질 보고, FM 데이터 이해 |
| 시공사 | 현장 BIM 활용, 간섭/변경/물량 관리 |
| 설계사 | 모델 기준, RFI, 변경 반영 |
| 협력사 | CDE 사용, 이슈 확인, 모델 기준 |

## 3. 교육 구성

| 세션 | 시간 | 내용 |
|---|---:|---|
| 1 | 1h | BIM 수행 범위와 산출물 이해 |
| 2 | 1h | CDE 접속, 폴더, Revision, 제출 흐름 |
| 3 | 1h | 간섭 보고서와 이슈대장 활용 |
| 4 | 1h | 물량/변경 보고서 이해 |
| 5 | 1h | 납품 패키지 검수 방법 |

## 4. 실습 자료

| 자료 | 제공 여부 | 비고 |
|---|---|---|
| 샘플 모델 | 협의 | 고객 자료 사용 시 보안 승인 필요 |
| 샘플 보고서 | 제공 | 민감 정보 제거 |
| CDE 실습 계정 | 협의 | 고객 플랫폼 정책 기준 |
| 체크리스트 | 제공 | PDF/XLSX |

## 5. 제공 산출물

| 산출물 | 형식 |
|---|---|

| Q&A 정리 | PDF |
| 참석자 명단 | XLSX/PDF |

## 6. 협의 필요 항목

교육 범위, 시간, 대상 인원, 장소, 자료 보안, 녹화 여부, 추가 질의 대응 범위는 계약 또는 별도 SOW 기준에 따라 확정한다.
