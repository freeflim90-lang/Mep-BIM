## 1. 패키지 목적

본 패키지는 LUA BIM LAB 명칭으로 외부 고객, 발주처, 시공사, 감리/CM, 협력사에 제출하거나 내부 프로젝트 운영에 사용할 수 있는 비개발 공식 문서 묶음이다.

본 패키지는 Add-in 개발, 소프트웨어 판매, Store 제출 문서를 제외하고 MEP BIM 수행, 품질, 보안, 회의, 변경, 납품, 교육, 고객 커뮤니케이션에 필요한 문서를 우선 구성한다.

## 4. 명칭 및 테마 기준

외부 제출 문서의 공식 명칭은 `LUA BIM LAB`으로 통일한다. 내부 저장소명 또는 기존 문서의 `LUA BIM LABS` 표기는 시스템/저장소 맥락으로 유지할 수 있으나, 본 패키지의 표지와 본문 주어는 `LUA BIM LAB`을 사용한다.

문서 테마는 다음 기준을 따른다.

| 항목 | 기준 |
|---|---|
| 표지 | 문서명, 구분선, 작성주체, 제출처, 작성일 |
| 구조 | 목차, 번호형 장/절, 표 중심 구성 |
| 문체 | 공식적, 실무 중심, 과장 없는 표현 |
| 색감/인상 | 첨부 제안서와 같은 정돈된 제안/수행 문서 분위기 |
| 수치 | 목표/기준/협의 필요를 구분 |
