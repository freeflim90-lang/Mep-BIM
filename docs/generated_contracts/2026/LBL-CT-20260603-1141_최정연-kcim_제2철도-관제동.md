## 제3조 (예산·기간·LOD 기준 수행 가능 범위)

본 조항은 문의자가 입력한 희망 예산, 일정, LOD, 연면적, 요청 공종을 기준으로 자동 산정한 **초기 범위 제한 기준**이다. 최종 범위는 원본 도면·모델·BEP/EIR·납품 기준 검토 후 확정한다.

|---|---|
| 수행 등급 | 단일 공종 중심 수행 |

### 3.1 현재 조건에서 가능한 업무

- 단일 공종 또는 제한된 복수 공종의 LOD 300 모델링
- 우선 구역 중심 간섭 검토 1~2회
- RVT/IFC/NWD 등 합의된 형식의 중간·최종 납품

### 3.2 현재 조건에서 제외하거나 별도 견적이 필요한 업무

- 전체 연면적·전 공종 LOD 350 이상 정밀 코디네이션
- 샵드로잉 수준의 상세 시공 도면 전체 납품
- 무제한 수정 및 범위 외 설계 변경 대응

### 3.3 주요 리스크 및 범위 조정 조건

- 연면적 30,000m² 이상 프로젝트에서 3천만원 미만 예산은 전체 범위보다 우선 구역 중심 수행이 적정

### 3.4 금액·기간 증가 시 확장 가능한 업무

- 예산 3,000만원 이상 범위에서는 단일 공종 또는 제한된 복수 공종의 LOD 300 수행 가능
- 예산 7,000만원 이상 확보 시 전 공종 또는 주요 공종 LOD 350 이상 코디네이션 검토 가능
- 샵드로잉, 현장 대응, 반복 설계 변경은 별도 금액과 일정이 확정될 때 추가 가능
- 수행 기간 90일 이상 확보 시 복수 공종·대형 구역의 단계별 납품 계획 수립 가능

### 3.5 전제 조건

- 발주자가 최신 도면, 기준층/주요 구역 정보, 기존 모델, 납품 기준을 착수 전 제공하는 것을 전제로 함
- 희망 예산은 상한 또는 참고 금액으로 보며, 최종 계약 금액은 자료 검토 후 별도 견적으로 확정
