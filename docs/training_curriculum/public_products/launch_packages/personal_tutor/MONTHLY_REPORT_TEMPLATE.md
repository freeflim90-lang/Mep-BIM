# Personal Tutor — Monthly Progress Report Template

**Document Type:** Client-facing PDF (2 pages)  
**Format:** Google Docs → Export as PDF  
**File naming:** `PT_[Client_Handle]_Report_[YYYY-MM].pdf`  
**Version:** 1.0  
**Document Owner:** COO

---

## Instructions for Report Preparation

1. Copy this template into a new Google Doc for each client each month
2. Replace every [FIELD_NAME] placeholder with real data from the client's logs
3. Do not leave any placeholder unfilled — if data is unavailable, write "N/A" with a brief note
4. Keep the report to exactly 2 pages when exported as PDF (use font: Noto Sans KR or Malgun Gothic, 11pt body, 1-inch margins)
5. Add LUA BIM LABS logo to the top-right of Page 1 header
6. Export as PDF and send to the client via Telegram
