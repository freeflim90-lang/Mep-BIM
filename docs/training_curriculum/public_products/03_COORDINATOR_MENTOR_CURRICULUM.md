## 상품 정의

Coordinator Mentor는 BIM 코디네이터 역할을 수행 중이거나 진입을 준비 중인 학습자를 위한 실무 중심 멘토링 서비스다.

단순 개념 설명이 아니라, 학습자의 현재 프로젝트 상황(일반화된 문제)을 기반으로 간섭 조율, 회의 운영, 이슈 추적, 보고서 작성 역량을 개발한다.

**$229/월이 제공하는 것**: 전문가 접촉 + 실무 도구 + 실사례 학습 + 과정 완료 인증.

| BIM 리드 준비 | Level 4로 이행 전 코디네이션 역량 완성 필요 |

## 주간 실무 이슈 리뷰

Personal Tutor와의 주요 차이:
- 학습자가 현재 프로젝트에서 직면한 일반화된 이슈를 주 1회 공유
- LUA BIM LABS가 접근 방법, 판단 기준, 대안을 Telegram으로 피드백
- 기밀 프로젝트 파일, 도면, 모델 공유 불가 — 텍스트/캡처 기반 논의

### 리뷰 형식

```
학습자 제출:
"이번 주 조율 이슈: L3층 AHU-03 공급 덕트가 3축 방향 구조 보와
간섭 발생. 덕트 사이즈는 1200×600. 3가지 루트 대안 중 어떤
기준으로 선택해야 할지 판단이 어렵습니다."

LUA BIM LABS 피드백:
- 덕트 루트 변경 vs. 사이즈 다운 vs. 보 위로 우회 판단 기준 제시
- 각 대안의 시공성, 유지보수, 간섭 해결 효과 비교 설명
- 결정권자(ME, 구조 엔지니어) 보고 방법 안내
```

[300~500단어 분석 및 권장 접근 방법 제시]

## Coordination Scenario Challenge (v2 신규)

월 1회 학습자가 스스로 판단하고 LUA BIM LABS가 피드백하는 시나리오 연습.

### 진행 방식

1. LUA BIM LABS가 조율 시나리오 상황 설명 발송 (Telegram)
2. 학습자가 3~5일 내 자신의 접근 방법을 텍스트로 작성하여 제출
3. LUA BIM LABS가 학습자 접근 방법에 대한 서면 피드백 발송

### 평가 기준

| 기준 | 설명 |
|---|---|
| 간섭 우선순위 판단 | 어떤 간섭을 먼저 처리했는가, 이유는? |
| 책임 귀속 판단 | 어떤 공종이 이슈를 소유하는가? |
| 회의 전 준비 | 어떤 정보를 사전에 준비하는가? |
| 의사소통 방법 | 고객/CM에게 어떻게 보고하는가? |
