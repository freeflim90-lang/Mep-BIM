## 상품 정의

Project Mentor는 실제 프로젝트를 수행 중인 BIM 리드, 시니어 코디네이터, 또는 BIM PM을 위한 고강도 멘토링 서비스다.

단순 교육이 아니라, 현재 프로젝트에서 발생하는 실제 문제(납품 리스크, 팀 관리, BEP 작성, 고객 보고)에 대한 실시간 전략적 가이던스를 제공한다.

**$490/월이 제공하는 것**: Zoom 전략 세션 + 24시간 응답 + 즉시 사용 가능한 문서 도구 + 분기 커리어 성장 측정.

## 대상 학습자

| 프로필 | 설명 |
|---|---|
| MEP BIM 리드 | BEP 작성, 품질 관리, 팀 관리 역할 수행 중 |
| BIM PM 전환 준비 | 프로젝트 관리 역할로 이행하려는 시니어 |
| BIM 자동화 개발자 | 실무 자동화 도구 개발 중인 학습자 |

## 서비스 구조

Project Mentor는 시간표 기반 커리큘럼이 아니라 **마일스톤 기반 멘토링**으로 운영된다.

학습자의 현재 프로젝트 단계에 따라 멘토링 포커스가 달라진다.

---

## 프로젝트 단계별 멘토링 포커스

### Phase 1: Project Kickoff (1~4주)

| 주제 | 멘토링 내용 |
|---|---|
| BEP 작성 지원 | EIR 분석, BEP 섹션별 작성 기준, 고객 승인 전략 |
| 팀 구성 계획 | 역할 정의, 업무 배분, 교육 계획 수립 |
| CDE 세팅 | 폴더 구조, 명명 규칙, 버전 관리 설계 |
| 킥오프 회의 준비 | 발표 자료 구성, 질의응답 대비 |

### Phase 2: Active Modeling and Coordination (월 단위)

| 주제 | 멘토링 내용 |
|---|---|
| 모델 품질 관리 | QA/QC 루틴 설계, 검수 기준 수립, 반려 기준 |
| 간섭 조율 전략 | 복잡한 조율 이슈 판단, 대안 비교, 의사결정 지원 |
| 팀 관리 | 모델러 업무 추적, 오류 피드백, 생산성 향상 방안 |
| 고객 보고 준비 | 진행 상황 보고서 구성, 리스크 커뮤니케이션 |

### Phase 3: Delivery and Closeout (납품 전 4~8주)

| 주제 | 멘토링 내용 |
|---|---|
| 납품 패키지 준비 | 파일 구성, IFC 검토, 제출 대장 완성 |
| 최종 QA 점검 | 납품 전 전체 품질 점검 루틴 |
| 준공 BIM 대응 | 준공 기록 기준, As-Built 처리 방법 |
| 프로젝트 회고 | 오류 사례 정리, 다음 프로젝트 개선 포인트 |

## 월간 1시간 Zoom 전략 세션 (v2 신규)

**$490 Standard 패키지의 핵심 차별화 요소.** Telegram 멘토링 + Zoom 전략 세션의 조합이 "파트너십 경험"을 만든다.

### 세션 구조 (60분)

| 구간 | 내용 | 시간 |
|---|---|---|
| 현황 공유 | 이번 달 프로젝트 진행 상황, 주요 이슈 리뷰 | 10분 |
| 핵심 이슈 분석 | 가장 복잡한 이슈 1개 집중 분석 + 대안 검토 | 20분 |
| 전략 수립 | 팀 관리, 납품 계획, 고객 보고 전략 논의 | 20분 |
| 다음 달 계획 | 집중 과제, 마일스톤, 학습 우선순위 확정 | 8분 |
| 마무리 | 다음 달 세션 예약 | 2분 |

### 세션 후 산출물

- LUA BIM LABS가 세션 요약 메모 (5~7줄) Telegram 발송
- 합의한 다음 달 집중 과제 목록 공유
- 필요 시 관련 레슨 또는 템플릿 추가 발송

## 프로젝트 문서 템플릿 팩 (v2 신규)

서비스 시작 후 3일 이내 발송. $490 고객에게 즉시 실무 가치를 제공.

### 팩 구성

| 파일명 | 형식 | 내용 |
|---|---|---|
| `bep_template_full.docx` | Word | BEP 전체 구조 (15개 섹션, 작성 가이드 포함) |
| `qa_qc_checklist_hvac.xlsx` | Excel | HVAC 공종 모델 품질 체크리스트 |
| `qa_qc_checklist_piping.xlsx` | Excel | 배관 공종 모델 품질 체크리스트 |
| `qa_qc_checklist_electrical.xlsx` | Excel | 전기 공종 모델 품질 체크리스트 |
| `coordination_report_template.docx` | Word | 월간 조율 현황 보고서 양식 |
| `model_submission_checklist.xlsx` | Excel | 납품 전 최종 점검 체크리스트 |
| `team_task_tracker.xlsx` | Excel | 팀 업무 배분 및 진도 추적 표 |

> BEP 템플릿은 ISO 19650 기반 구조. 학습자 프로젝트에 맞게 수정하여 사용 가능.

## 분기 커리어 성장 Assessment (v2 신규)

3개월마다 1회, 30분 구조화된 커리어 성장 평가 실시.

### Assessment 구성

**역량 자가 평가 (학습자가 사전 작성):**

```
Section A: 현재 역할 역량
─ BEP 작성 독립 수행 가능? (1~5)
─ 고객 보고 독립 수행 가능? (1~5)
─ 팀 업무 배분 및 추적 가능? (1~5)
─ 납품 패키지 구성 독립 수행? (1~5)
─ 간섭 조율 의사결정 독립 가능? (1~5)

Section B: 다음 역할 준비도
─ 목표 역할: [BIM PM / 컨설턴트 / 독립 사업]
─ 현재 갭: [학습자 서술]
─ 6개월 내 목표: [학습자 서술]
```

## 납품 시뮬레이션 (v2 신규)

월 1회, 학습자가 가상 납품 패키지 구성을 준비하고 LUA BIM LABS가 서면 피드백 제공.

### 진행 방식

1. LUA BIM LABS가 가상 프로젝트 납품 요구사항 발송
2. 학습자가 5~7일 내 납품 패키지 구성 계획 (파일 목록, 폴더 구조, 점검 기준) 텍스트 작성
3. LUA BIM LABS가 아래 기준으로 서면 피드백 발송:

| 평가 기준 | 설명 |
|---|---|
| 완성도 | 요구된 산출물이 모두 포함되었는가? |
| 품질 기준 | LOD, 명명, 파라미터 기준이 명시되었는가? |
| 리스크 식별 | 납품 위험 요소를 사전에 파악했는가? |
| 실행 가능성 | 제시된 일정과 인력으로 현실적인가? |
