# LUA BIM LABS 상품별 가격-가치 갭 분석

문서번호: LBL-TRN-PUB-006
문서상태: 상품 디밸롭먼트 전략
작성일: 2026-05-29

---

## 분석 기준

소비자가 특정 가격을 지불할 때 기대하는 것은 **콘텐츠 양**이 아니라
**가격 수준에 맞는 경험 품질, 개인화 깊이, 결과물의 가시성**이다.

각 티어는 아래 4가지 축으로 평가한다:

| 축 | 정의 |
|---|---|
| 콘텐츠 밀도 | 레슨 수, 트랙 깊이, 난이도 커버리지 |
| 개인화 수준 | 공통 → 수준별 → 상황별 → 프로젝트별 |
| 가시적 결과물 | 수료증, 보고서, 템플릿, 포트폴리오 자산 |
| 접근 깊이 | 텍스트 → 비동기 Q&A → 동기 1:1 세션 |

## Tier별 가격-기대치-갭 분석

---

### Tier 1: Starter — USD 39/월

#### 소비자가 $39에 기대하는 것

$39는 스트리밍 서비스(Netflix $15~22)와 전문 뉴스레터($10~50) 사이의 가격이다.
이 가격대 소비자 심리는 **"부담 없이 매일 가치를 받는다"** 는 습관형 구독이다.

| 소비자 기대 | 기대 근거 |
|---|---|
| 매일 짧고 실용적인 레슨 | $39 = 일 $1.30 — 커피 한 잔값 수준 |
| 진행감(progress) 느낌 | "내가 성장하고 있다"는 시각적 신호 |
| 60일 이후 지속 이유 | 구독을 유지할 이유가 있어야 재결제 |
| 무언가 남는 것 | 텍스트 외 다운로드 가능한 자료 |

#### 현재 상품이 제공하는 것

- 60일 공통 커리큘럼 레슨 (1일 1개)
- Telegram 텍스트 전달
- 월 1회 짧은 확인 Q&A 허용

#### 갭 (Gap)

| 갭 | 위험 |
|---|---|
| 60일 이후 재구독 이유 없음 | 이탈률 급등 → LTV 붕괴 |
| 진행감 표시 없음 | 레슨을 받아도 "나 뭐 배웠지?" 느낌 |
| 다운로드 자료 없음 | Telegram 메시지는 스크롤하면 사라짐 |
| 공종 구분 없음 | 전기 학습자에게 배관 레슨은 관련성 약함 |
| 마일스톤 없음 | 30일, 60일에 아무 이벤트 없음 |

### Tier 2: Personal Tutor — USD 119/월

#### 소비자가 $119에 기대하는 것

$119는 온라인 피트니스 코칭($80~150)과 어학 개인 튜터($100~200) 사이다.
이 가격대 소비자 심리는 **"나만을 위한 선생님이 있다"** 는 개인화 투자다.

| 소비자 기대 | 기대 근거 |
|---|---|
| 내 수준에 맞는 레슨 | "모든 사람이 같은 레슨"이면 Starter와 차별 없음 |
| 내가 성장했다는 증거 | 수강 후 "어디까지 왔는지" 가시적 기록 |
| 질문에 대한 실질적 답변 | $119 → Q&A 횟수 제한이 아닌 질 높은 응답 |
| 내 약점을 알고 있는 선생님 | "저번에 이 실수했었죠" 기억하는 선생님 |

#### 현재 상품이 제공하는 것

- 레벨 진단 후 5가지 경로 중 선택
- 개인화 일일 레슨
- 주 3회 Q&A
- 약점 패턴 기록 (Obsidian)

#### 갭 (Gap)

| 갭 | 위험 |
|---|---|
| 개인화가 "경로 선택"에 그침 | $119인데 콘텐츠 자체가 달라지지 않음 |
| 진행 기록이 내부 Obsidian에만 존재 | 학습자가 자기 성장을 볼 수 없음 |
| 공종 특화 레슨이 미약 | HVAC 학습자와 전기 학습자가 같은 레슨 받는 경우 발생 |
| Starter 대비 3배 가격의 체감 근거 부족 | "그냥 Q&A 좀 더 되는 Starter"로 느낄 위험 |

---

### Tier 3: Coordinator Mentor — USD 229/월

#### 소비자가 $229에 기대하는 것

$229는 전문 자격증 교육 과정($150~400/월)과 전문가 그룹 코칭($200~400)의 영역이다.
이 가격대 소비자 심리는 **"실무 문제를 같이 해결해주는 전문가"** 다.

| 소비자 기대 | 기대 근거 |
|---|---|
| 전문 도구와 자료 | 교육만이 아니라 "쓸 수 있는 것" |
| 전문가와 직접 접촉 | 텍스트만으로는 $229 정당화 어려움 |
| 내 프로젝트 문제 해결 지원 | "이 이슈 어떻게 하면 되나요?" 즉각 답변 |
| 수료 후 남는 자산 | 45일 후 내 손에 무언가 있어야 함 |

#### 현재 상품이 제공하는 것

- 45일 4개 모듈 커리큘럼
- 주 1회 텍스트 이슈 리뷰
- 서비스 범위 내 Q&A

#### 갭 (Gap)

| 갭 | 위험 |
|---|---|
| 동기적(synchronous) 접점 없음 | $229 = 프리랜서 BIM 코디네이터 2시간 — 음성/영상 없으면 체감 낮음 |
| 실무 도구/템플릿 없음 | 교육 후 바로 쓸 수 있는 것이 없음 |
| 45일 후 수료의 가시화 없음 | 과정 끝나도 "뭔가 완성됐다"는 느낌 없음 |
| 케이스 스터디 부재 | 실제 사례 기반 학습 없음 |

---

### Tier 4: Project Mentor — USD 490+/월

#### 소비자가 $490에 기대하는 것

$490+는 임원 코칭($400~800/월), 전문 컨설팅 리테이너의 하단 영역이다.
이 가격대 소비자 심리는 **"이 사람이 내 커리어와 프로젝트에 투자하고 있다"** 는 파트너십이다.

| 소비자 기대 | 기대 근거 |
|---|---|
| 정기적 1:1 세션 | $490에 Telegram 텍스트만이면 가치 불일치 명확 |
| 내 프로젝트 산출물 검토 | "내 BEP 봐줘", "내 보고서 어떤가요" |
| 커리어 계획 공동 수립 | 단순 교육이 아닌 성장 파트너십 |
| 측정 가능한 아웃컴 | "이 돈 쓰고 나서 뭐가 달라졌나" |

#### 현재 상품이 제공하는 것

- 마일스톤 기반 멘토링
- Telegram Q&A
- 포트폴리오/커리어 가이던스
- 프로젝트 이슈 검토 (텍스트 기반)

#### 갭 (Gap)

| 갭 | 위험 |
|---|---|
| 월간 1:1 세션 없음 | $490 = Coordinator Mentor의 두 배인데 구조적 차이 불명확 |
| 납품 가능한 템플릿 부재 | BEP 템플릿, QA 체크리스트 등 "쓸 수 있는 것" 없음 |
| 아웃컴 측정 없음 | 분기별 성장 측정이 없으면 지속 구독 이유 약함 |
| 우선 응답 보장 없음 | $490 고객이 48시간 기다리는 것은 가치 불일치 |

## 갭 해결을 위한 상품별 개발 방향

### Starter: "보이는 성장" 추가

| 추가 요소 | 목적 | 비용/공수 |
|---|---|---|
| 30일/60일 수료 인증서 (PDF 자동 발급) | 마일스톤 가시화 | 템플릿 1회 제작 |
| Track별 퀵 레퍼런스 카드 (PDF 8장) | 다운로드 가능한 잔여 자산 | 1회 제작 |
| 공종별 Day 61~90 확장 트랙 | 재구독 이유 + 공종 특화 | 레슨 30개 추가 |
| 주간 "BIM Check Friday" 퀴즈 메시지 | 참여도 + 진행감 | 운영 루틴 추가 |

### Personal Tutor: "개인화의 증거" 추가

| 추가 요소 | 목적 | 비용/공수 |
|---|---|---|
| 월간 학습 진도 보고서 (PDF, 2페이지) | 개인화를 학습자가 직접 볼 수 있게 | Obsidian→PDF 자동화 |
| 공종별 시나리오 레슨 (10일마다 1개) | 공종 특화 경험 강화 | 레슨 세트 공종별 제작 |
| 월간 레벨 체크 (5문항 진단) | 성장 측정 + 다음 달 커리큘럼 조정 근거 | 월 1회 운영 |
| 약점 보정 레슨 라이브러리 (30개) | 동일 실수 반복 방지 | 사전 제작 후 재사용 |

### Coordinator Mentor: "전문가와의 접촉" 추가

| 추가 요소 | 목적 | 비용/공수 |
|---|---|---|
| 월 1회 30분 Telegram 음성/Zoom 세션 | 동기적 접촉 → 가격 정당화 | 월 0.5시간 투입 |
| 조율 템플릿 팩 (입문 시 제공) | 즉시 실무 적용 가능한 자산 | 1회 제작 |
| 월 1회 익명 케이스 스터디 | 실제 사례 기반 학습 | 월 1개 제작 |
| 45일 수료 Coordinator 인증서 | 과정 완료 가시화 | 템플릿 1회 제작 |

|---|---|---|
| 월 1회 1시간 화상 전략 세션 (Zoom) | $490 정당화 핵심 요소 | 월 1시간 투입 |
