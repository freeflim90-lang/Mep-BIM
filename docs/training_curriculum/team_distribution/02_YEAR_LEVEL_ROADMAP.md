# 연차별 MEP BIM 교육 로드맵

문서상태: 팀 배포용 초안  
작성일: 2026-05-20

---

## 1. 단계 구분

| 단계 | 연차 | 역할 | 핵심 목표 |
|---|---:|---|---|
| Foundation | 1~2년차 | 입문 실무자 | 정확한 모델링과 기준 준수 |
| Practitioner | 3~5년차 | 공종 실무자 | 공종별 품질, 간섭, 변경, 물량 이해 |
| Coordinator | 6~8년차 | 조율 담당자 | 다공종 조율, CDE, 회의/이슈 관리 |
| Lead | 9~12년차 | 공종/BIM 리드 | BEP, 납품, 협력사 검수, 기준 수립 |
| PM/Consultant | 13~16년차 | PM/컨설팅 리드 | 제안, 범위, 비용, 리스크, 고객 의사결정 |
| Principal | 17~20년차 | 기술/교육 리더 | 조직 표준화, 교육 체계, 방법론 정립 |

## 2. 1~2년차: Foundation

| 영역 | 교육 내용 | 평가 산출물 |
|---|---|---|
| Revit MEP | 카테고리, 패밀리, 시스템, 뷰, 링크 | 기본 MEP 모델 |
| 도면 이해 | 장비표, 계통도, 평면도, 상세도 | 도면-모델 매칭표 |
| CDE/문서 | 파일명, Revision, 업로드, 제출대장 | 파일 제출대장 |
| 품질 기초 | 좌표, 링크, 명명, 필수 파라미터 | 기본 품질 체크리스트 |
| AI 활용 | 용어 정리, 체크리스트 초안 | AI 활용 기록 |

## 3. 3~5년차: Practitioner

| 영역 | 교육 내용 | 평가 산출물 |
|---|---|---|
| 공종 모델 품질 | 단열, 구배, 점검공간, 시스템명 | 공종별 품질 검토표 |
| 간섭 검토 | Clash 이슈 분류, 우선순위 | 간섭 보고서 초안 |

| 물량 기초 | Revit Schedule, Excel 정리 | 물량 산출서 |
| AI/자동화 | 보고서 초안, Excel 자동화 기초 | 자동화 결과와 검토 기록 |

## 4. 6~8년차: Coordinator

| 영역 | 교육 내용 | 평가 산출물 |
|---|---|---|
| 다공종 조율 | 건축/구조/MEP 이슈 구조화 | 조율 이슈 대장 |

## 5. 9~12년차: Lead

### 9년차: BEP 작성·표준 수립 집중

| 영역 | 교육 내용 | 평가 산출물 |
|---|---|---|
| BEP 작성 | EIR 분석, BEP 섹션 독립 작성 | BEP 초안 (2개 이상) |
| 모델링 표준 개정 | 기존 표준 오류·누락 발굴, 수정안 제안 | 표준 개정안 |
| 후배 지도 시작 | 3~5년차 대상 작업 기준 설명, 교안 작성 | 교육 교안 |
| 자동화 요구사항 | 반복 업무 1개 기능 명세서 작성 | 기능 명세서 |
| 납품 검수 참여 | 선배와 함께 납품 패키지 검수 경험 | 검수 의견서 |

### 10년차: 공종 리드 본격 수행

| 영역 | 교육 내용 | 평가 산출물 |
|---|---|---|
| BEP 주관 | BEP 전 섹션 작성 및 발주처 협의 | BEP 완성본 |
| 납품 주관 | 납품 패키지 최종 승인 및 제출 대응 | 납품 검수표 |
| 협력사 온보딩 | 표준 교육, 반려·보완 절차 운영 | 검수 의견서 |
| 팀 관리 | 2~3명 업무 배분, 주간 업무 점검 | 주간 업무 현황표 |
| Add-in QA | QA 시나리오 설계·실행 | QA 테스트표 |

### 11~12년차: 복수 프로젝트 리딩·표준화 주도

| 영역 | 교육 내용 | 평가 산출물 |
|---|---|---|
| 복수 프로젝트 | 2개 이상 프로젝트 BEP·납품 병행 운영 | 프로젝트별 BEP |
| 교육 체계 기여 | 팀 교육자료 제작, 오류 사례집 정리 | 오류 사례집 |
| 기술 사양 검토 | 발주처 BIM 요건 분석, 수행 가능성 판단 | 검토 의견서 |
| 제품화 후보 발굴 | 반복 Add-in 수요 PM/Principal에 제안 | 제품화 후보 목록 |
