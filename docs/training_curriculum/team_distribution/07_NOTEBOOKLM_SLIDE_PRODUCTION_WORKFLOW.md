# NotebookLM 연계 슬라이드 제작 협업 흐름

문서상태: 팀 배포용 초안  
작성일: 2026-05-20

---

## 1. 목적

본 문서는 MEP BIM 교육 커리큘럼을 NotebookLM 또는 유사 지식 기반 도구와 연계하여 슬라이드, 강의안, 퀴즈, 실습 안내서로 제작하는 협업 흐름을 정의한다.

---

## 2. 역할 구분

| 역할 | 책임 |
|---|---|
| 교육컨설팅 | 교육 목표, 대상 연차, 커리큘럼, 평가 기준 확정 |
| 러닝콘텐츠디자이너 | NotebookLM 자료 구성, 슬라이드 목차, 강의안, 퀴즈, FAQ 제작 |
| BIM 리드 | 기술 내용, 실습 모델, 공종별 판단 검수 |
| QA_테스터 | 실습 과제와 퀴즈가 평가 가능한지 검증 |
| 테크니컬_라이터 | 문장, 용어, 배포 형식 최종 정리 |

---

## 3. 제작 프로세스

1. 교육컨설팅이 교육 목표와 대상자를 확정한다.
2. 러닝콘텐츠디자이너가 원본 문서 목록을 구성한다.
3. 고객명, 프로젝트명, 개인정보, 모델 원본 정보 등 민감정보를 제거한다.
4. NotebookLM에 투입할 자료를 정리한다.
5. 요약본, FAQ, 핵심 질문, 강의 흐름 초안을 생성한다.
6. 슬라이드 목차와 강사용 스크립트를 만든다.
7. BIM 리드가 기술 정확성을 검수한다.
8. QA_테스터가 퀴즈와 실습 과제를 검증한다.
9. 테크니컬_라이터가 최종 배포본을 정리한다.
10. 교육 후 피드백을 다음 교육자료 개정에 반영한다.

---

## 4. 제작 산출물

| 산출물 | 설명 |
|---|---|
| 슬라이드 | 교육 진행용 PPT/PDF |
| 강사용 스크립트 | 설명 흐름, 질문, 예시 답변 |
| 학습자 요약본 | 핵심 개념과 체크리스트 |
| FAQ | 교육 중 예상 질문과 답변 |
| 퀴즈 | 이해도 점검 문항 |
| 실습 안내서 | 과제 목적, 입력 자료, 제출 형식 |

---

| 항목 | 기준 |

| 프로젝트명 | 필요 시 익명화 |
| 도면/계약 조건 | 민감정보 제거 후 사용 |
| AI 생성 결과 | 기술 검수 전 배포 금지 |
| 설계/시공 판단 | 담당 리드 검토 후 표현 |
