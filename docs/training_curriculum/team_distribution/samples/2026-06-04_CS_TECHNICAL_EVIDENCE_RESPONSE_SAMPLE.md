## 1. 샘플 상황

고객이 다음과 같이 문의했다.

> 위생 배관 경사와 물탱크 청소 기준을 LUA BIM LABS 모델 검수 기준으로 확정해서 봐도 되나요? 납품 전에 바로 수정해야 하는지 알고 싶습니다.

## 2. 근거기반 CS 답변 샘플

```text
안녕하세요, [고객명]님.

문의하신 위생 배관 경사와 물탱크 청소 기준의 BIM 납품검수 적용 여부에 대한 현재 검토 결과입니다.

1. 결론
현재 단계에서는 해당 항목을 LUA BIM LABS의 단독 확정 검수 기준으로 바로 적용하지 않습니다.
수도법, 하수도법, 관련 시행규칙, 프로젝트 설계도서, 발주처 특기시방서, 지자체 상하수도 조건을 함께 확인한 뒤 납품검수 기준으로 확정해야 합니다.

2. 근거
- `2026-06-04_LUA_BIM_LABS_PLUMBING_WATER_TANK_DRAINAGE_STANDARD_UPDATE.md`
- `위생.md`의 급수·저수조·오배수 기준 적용 게이트
- `BIM_납품검수.md`의 공식 기준, 특기시방서, 프로젝트 적용 조건 확인 필드
- `27_CS_RESPONSE_SCRIPT.md`의 근거기반 기술 답변 템플릿

3. 적용 범위
이 답변은 위생 BIM 모델 납품검수, 급수·저수조·오배수 모델 확인, 고객 기술 문의 1차 응대에 적용됩니다.
실제 설계 변경, 시공 승인, 인허가 판단은 발주처와 설계자, 감리자, 관할 기관 확인 범위입니다.

4. 주의
공식 법령과 고시 출처가 확인된 항목은 `KST01 공식확인` 신호로 봅니다.
다만 프로젝트 적용 여부는 설계도서와 지자체 조건에 따라 달라질 수 있으므로 `KST03 적용주의`로 안내해야 합니다.
기존 지식베이스에 자동 수집된 배관 경사, 청소 주기, 수질 검사 수치 문장은 `KST04 자동수집` 상태로 두며, 고객에게 확정 기준처럼 안내하지 않습니다.

5. 다음 액션
고객사에서 발주처 특기시방서, 위생 설계도서, 지자체 상하수도 협의 조건을 공유해 주시면 LUA BIM LABS에서 모델 검수 항목과 기준 적용 여부를 대조하겠습니다.
기준 충돌이 있거나 설계 판단이 필요한 경우 RFI 또는 설계자 확인 절차로 전환하겠습니다.

LUA BIM LABS | [이메일]
```

## 3. 교육 포인트

좋은 CS 답변의 조건:
- 고객에게 바로 필요한 결론을 먼저 말한다.
- 공식 기준과 프로젝트 적용 조건을 분리한다.
- 자동 수집 수치나 일반론을 확정 납품 기준으로 말하지 않는다.
- 고객이 다음에 어떤 자료를 주면 되는지 분명히 안내한다.
- 설계자, 감리자, 관할 기관 확인이 필요한 영역을 CS 답변 안에서 넘겨짚지 않는다.

피해야 할 답변:
- “물탱크 기준은 법에 있으니 그대로 모델 검수하면 됩니다.”
- “배관 경사는 일반적으로 이 수치이므로 바로 수정하세요.”
- “LUA BIM LABS 기준상 무조건 불합격입니다.”
- “지자체 조건은 큰 차이가 없으니 내부 기준만 보면 됩니다.”
