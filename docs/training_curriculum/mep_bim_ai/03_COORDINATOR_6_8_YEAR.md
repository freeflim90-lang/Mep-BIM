## 1. 과정 목표

6~8년차는 공종별 모델을 보는 수준을 넘어 통합 모델에서 문제를 발견하고, 책임자를 배정하며, 회의와 보고를 통해 이슈를 종결시키는 역할을 수행한다.

## 2. 핵심 책임

| 책임 | 설명 |
|---|---|
| 통합 모델 관리 | 공종별 모델 수집, 링크, 좌표, 버전 확인 |
| 간섭 조율 | Clash 그룹핑, 우선순위, 책임자, 기한 관리 |
| CDE 운영 | WIP/Shared/Published 상태와 배포 이력 관리 |
| 회의 운영 | 주간 BIM 회의, 간섭 회의, 액션아이템 추적 |
| 품질 보고 | 월간 품질 지표, 미해결 이슈, 리스크 보고 |
| 자동화 개선 | 반복 보고/검토 업무 자동화 후보 발굴 |

## 3. 교육 모듈

| 모듈 | 내용 | 산출물 |
|---|---|---|
| M01 통합 모델 운영 | 모델 수집, 좌표, 링크, NWD/NWC | 통합 모델 검토표 |
| M02 간섭 전략 | Matrix, tolerance, 그룹핑, 이슈 상태 | 간섭 운영 계획 |
| M03 CDE 배포 | Shared/Published 승인, 제출대장 | 배포 이력표 |
| M04 회의 운영 | 안건, 의사결정, Action Item | 회의록 |
| M05 변경/리스크 | 변경대장, 리스크대장, 조치기한 | 변경/리스크 보고 |
| M06 자동 QA | 파라미터/파일명/링크 점검 | QA 스크립트 초안 |
| M07 Revit API 개념 | Transaction, Collector, ElementId | API 개념 노트 |
| M08 AI 회의 보조 | 회의록 요약, 결정사항 추출 | 회의 요약본 |

## 4. 협업 운영 훈련

| 상황 | 훈련 내용 |
|---|---|
| 간섭 과다 | Critical/High 중심으로 우선순위 재분류 |
| 설계 변경 | 변경 접수, 영향 분석, 모델 반영 일정 수립 |
| 공종 충돌 | 건축/구조/MEP 의견을 회의 안건으로 정리 |
| 납품 지연 | 원인, 책임, 회복 일정, 고객 보고 초안 작성 |
| CDE 혼선 | 최신본/검토본/승인본 구분 재정리 |

## 5. AI/개발 교육

| 항목 | 교육 내용 | 주의 |
|---|---|---|
| AI 회의록 | 녹취/메모 기반 안건과 결정사항 정리 | 민감 정보 제거 |
| AI 이슈 요약 | Clash/RFI/변경 이슈 요약 | 책임자/기한 수동 확인 |
| Python QA | 파일명, 파라미터 CSV, 링크 목록 점검 | 원본 모델 직접 수정 금지 |
| Revit API 개념 | 모델 변경과 읽기 작업 차이 | 실행 전 백업/검증 |
| 자동 보고 | XLSX/PDF 보고서 초안 생성 | 제출 전 수동 검수 |

## 6. 실습 과제

| 과제 | 요구사항 | 제출물 |
|---|---|---|
| 통합 간섭 회의 운영 | 간섭 100건을 회의 안건화 | 회의록, 이슈대장 |
| CDE 배포 관리 | Shared→Published 승인 흐름 구성 | 배포 이력표 |
| 월간 BIM 보고 | 지표, 리스크, 다음 조치 작성 | 월간 보고서 |
| 자동 QA 초안 | 파일명/파라미터 누락 점검 | Python 스크립트 |

## 7. 평가 기준

| 평가 항목 | 합격 기준 |
|---|---|
| 조율 품질 | 이슈 책임자/기한/상태 명확 |
| 회의 운영 | 결정사항과 후속 조치가 추적 가능 |
| CDE 정확도 | 승인본과 검토본 혼선 없음 |
| 자동화 판단 | 자동화 대상과 검증 방법을 구분 |
| 리스크 대응 | 차단 이슈를 사전에 보고 |

## 8. 수료 산출물

- 간섭 운영 계획
- 주간/월간 BIM 보고서
- CDE 배포 이력표
- 변경/리스크 대장
- 자동 QA 스크립트 초안
