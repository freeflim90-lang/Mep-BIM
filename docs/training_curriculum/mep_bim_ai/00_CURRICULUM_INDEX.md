## 3. 공통 운영 원칙

1. AI는 초안, 요약, 비교, 자동화 보조 도구로 사용한다.
2. 설계 기준, 법규, 시공 가능성, 납품 적합성은 담당자가 검증한다.
3. 교육은 “강의 → 실습 → 산출물 → 리뷰 → 표준 반영” 순서로 운영한다.
4. 모든 실습 결과는 포트폴리오와 교육 평가 자료로 보관한다.
5. 반복 오류는 교육컨설팅 지식베이스와 표준문서 개정 후보로 등록한다.

| 포트폴리오 | 연차별 승급 평가용 대표 산출물 |

| 평가 축 | 비중 | 설명 |
