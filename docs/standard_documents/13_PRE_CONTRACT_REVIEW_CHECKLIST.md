# 계약 전 검토 체크리스트

━━━━━━━━━━━━━━━━━━━━

문서번호: LBL-STD-013  
문서상태: 내부 기준 초안  
작성일: 2026-05-20  
적용범위: 제안서 제출 전, 견적 제출 전, 계약/발주 수락 전

## 목 차

1. 목적
2. 사업성 검토
3. 업무범위 검토
4. 자료 및 일정 검토
5. 계약 조건 검토
6. 보안 및 지식재산 검토
7. 최종 승인 체크

## 1. 목적

본 체크리스트는 계약 전 무리한 범위, 불명확한 책임, 과도한 보안/납품 조건, 비용 회수 불가능 리스크를 사전에 확인하기 위한 내부 검토 문서다. 법률 검토가 필요한 항목은 전문가 검토 대상으로 분류한다.

## 2. 사업성 검토

| 체크 항목 | 확인 |
|---|---|
| 예상 매출과 투입 공수가 합리적인가 | [ ] |
| 일정 지연 또는 자료 부족에 따른 추가 비용 조건이 있는가 | [ ] |
| 고객사의 의사결정권자와 승인 절차가 확인되었는가 | [ ] |
| 프로젝트 실적/포트폴리오 가치가 있는가 | [ ] |
| 수행 리스크 대비 계약 금액이 적정한가 | [ ] |

| 검토 회차와 회의 참석 범위가 정의되어 있는가 | [ ] |
