# 보안 및 NDA 운영 기준

━━━━━━━━━━━━━━━━━━━━

문서번호: LBL-STD-014  
문서상태: 기준 초안  
작성일: 2026-05-20  
적용범위: 고객 자료, BIM 모델, 도면, 현장 사진, 계약/견적 자료, 협력사 공유 자료

## 목 차

1. 목적
2. 자료 등급
3. 접근권한 기준
4. 외부 공유 기준
5. 협력사 보안 기준
6. 보관 및 파기
7. 보안 사고 대응

## 1. 목적

본 기준은 고객으로부터 수령한 도면, BIM 모델, 현장 정보, 계약 자료를 안전하게 관리하고, NDA 체결 여부와 관계없이 프로젝트 자료의 무단 공유를 방지하기 위한 운영 기준이다.

## 2. 자료 등급

| 등급 | 설명 | 예시 |
|---|---|---|
| Public | 공개 가능 자료 | 공개 포트폴리오, 승인된 홍보 이미지 |
| Internal | 내부 업무 자료 | 내부 검토표, 작업 메모 |
| Client Confidential | 고객 비밀 자료 | 도면, 모델, 회의록, 견적 |
| Restricted | 제한 자료 | 보안 구역, 중요 설비, 개인정보 포함 자료 |

## 3. 접근권한 기준

| 역할 | 접근 기준 |
|---|---|
| PM | 프로젝트 전체 자료 접근 |
| 모델러 | 담당 공종/구역 자료 접근 |
| 협력사 | 계약 범위 내 최소 자료 접근 |
| 고객/감리/CM | 공식 배포 자료 접근 |
| 외부 제3자 | 사전 승인 없이는 접근 불가 |

## 4. 외부 공유 기준

- 고객 자료는 고객 승인 없이 공개 저장소, 개인 클라우드, 포트폴리오에 업로드하지 않는다.
- 외부 이메일 발송 시 수신자, 첨부 파일, 문서등급을 확인한다.
- 편집 가능한 원본 파일 제공은 계약 또는 승인 조건이 있을 때만 수행한다.
- 보안 자료는 다운로드 제한, 링크 만료, 비밀번호 등 추가 보호 조치를 적용한다.

## 5. 협력사 보안 기준

| 항목 | 기준 |
|---|---|
| NDA | 필요 시 업무 투입 전 체결 |
| 자료 범위 | 수행 범위에 필요한 최소 자료만 제공 |
| 재위탁 | 고객 또는 PM 승인 없이 금지 |
| 반출 | 승인된 저장소와 장비만 사용 |
| 종료 | 업무 종료 후 자료 반납/삭제 확인 |

## 6. 보관 및 파기

| 자료 | 보관 위치 | 보관 기간 |
|---|---|---|
| 계약/견적 | 내부 관리 폴더 | 계약/세무 기준에 따름 |
| 작업 모델 | CDE/WIP | 프로젝트 기간 |
| 공식 제출본 | Published/Archive | 계약 기준 |
| 임시 파일 | 개인 작업 폴더 | 업무 완료 후 정리 |

## 7. 보안 사고 대응

| 단계 | 조치 |
|---|---|
| 발견 | 유출/오발송/권한 오류 확인 |
| 차단 | 링크 만료, 접근 권한 회수, 파일 삭제 요청 |
| 보고 | PM 및 고객 담당자에게 사실관계 보고 |
| 영향 분석 | 자료 종류, 수신자, 노출 시간 확인 |
| 재발 방지 | 권한/배포 절차 수정 |
