## 11. 정보 보안 계획

- 보안 구역, 중요 설비, 개인정보가 포함된 자료는 접근 권한을 제한한다.
- 외부 반출 자료는 발주처 승인 후 배포한다.
- CDE 권한은 최소 권한 원칙을 따른다.
- 보안 요구가 높은 프로젝트는 ISO 19650-5 기준을 참고한다.

## 12. 승인 및 변경 관리

변경 요청은 접수, 영향 분석, 승인, 모델 반영, 배포의 순서로 관리한다. 변경 반영 전에는 변경 사유, 영향 범위, 물량 영향, 일정 영향을 기록해야 한다.
