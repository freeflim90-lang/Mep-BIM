## 1. 모델링 기준 목적

본 기준은 프로젝트별 BIM 모델이 동일한 좌표, 명명, 파라미터, LOD, 납품 형식을 따르도록 정의한다. 모델은 시각화뿐 아니라 간섭 검토, 물량 산출, 시공 검토, 준공 데이터로 활용 가능해야 한다.

## 2. LOD 기준

| 단계 | 목표 LOD | 주요 내용 |
|---|---|---|
| 기본 설계 | LOD 200 | 주요 공간, 시스템, 개략 크기 |
| 실시 설계 | LOD 300 | 실제 위치, 크기, 주요 사양 |
| 시공 검토 | LOD 350~400 | 접속부, 지지대, 시공 상세 |
| 준공 | LOD 500 | 실제 설치 정보, 자산/FＭ 데이터 |

## 3. 좌표 및 단위

| 항목 | 기준 |
|---|---|
| 좌표 | 발주처/측량 기준 또는 프로젝트 공유좌표 |
| 레벨 | 건축 기준 레벨과 공종별 레벨 명칭 일치 |
| 길이 | mm |
| 면적 | m2 |
| 체적 | m3 |
| 수량 | EA |

## 4. 공종별 모델 구성

| 공종 | 모델 파일 기준 | 포함 요소 |
|---|---|---|
| 건축 | ARC | 벽, 문, 창, 천장, 마감, 공간 |
| 구조 | STR | 기둥, 보, 슬래브, 기초, 구조 보강 |
| 기계 | MEC | 덕트, 배관, 장비, 밸브, 단열 |
| 전기 | ELE | 케이블트레이, 전선관, 조명, 분전반 |
| 소방 | FPS | 소화배관, 스프링클러, 감지기 |
| 통신 | COM | 통신배관, 랙, 단말, 보안 설비 |

## 5. 명명 규칙

| 대상 | 형식 | 예시 |
|---|---|---|
| 모델 파일 | `[Project]-[Discipline]-[Zone]-R[Rev]` | `ICN-T1-MEP-L01-R02.rvt` |
| 패밀리 | `[공종]_[기능]_[규격]` | `MEP_배관_DN100` |
| 타입 | `[규격]_[재질]_[비고]` | `DN100_STS_Insulated` |
| 뷰 | `[용도]_[공종]_[층/구역]` | `Clash_MEP_L01` |
| 시트 | `[번호]_[제목]_[Rev]` | `M-101_MEP Plan_R02` |

## 6. 파라미터 기준

필수 공통 파라미터:

| 파라미터 | 설명 | 필수 |
|---|---|---|
| LBL_ProjectCode | 프로젝트 코드 | 예 |
| LBL_Discipline | 공종 | 예 |
| LBL_Zone | 구역/공구 | 예 |
| LBL_Level | 층/레벨 | 예 |
| LBL_SystemName | 시스템명 | MEP 필수 |
| LBL_ItemCode | 내역/자산 코드 | 물량/FM 적용 시 |
| LBL_InstallStatus | 설치 상태 | 시공/준공 적용 시 |
| LBL_Revision | 모델 Revision | 예 |

## 7. 뷰/시트/필터 기준

| 유형 | 기준 |
|---|---|
| 협의 뷰 | 공종별 주요 요소와 간섭 검토 대상만 표시 |
| 간섭 뷰 | Clash 마커 및 관련 요소 중심 |
| 현장 배포 뷰 | 층/구역/공종별로 단순화 |
| 색상 필터 | 공조=파랑, 소방=빨강, 전기=노랑, 통신=초록을 기본값으로 사용 |
| 시트 | 발주처/시공사 도곽 기준에 맞춰 별도 확정 |

## 8. 납품 패키지 기준

```text
Deliverables/
  01_Models/
  02_IFC/
  03_Navisworks/
  04_Drawings/
  05_Reports/
  06_COBie_FM/
  07_QA_Evidence/
  08_Transmittal/
```

| 산출물 | 형식 | 필수 여부 |
|---|---|---|
| 공종별 Revit 모델 | RVT | 프로젝트 기준 |
| 통합 모델 | NWD/NWC/IFC | 예 |
| IFC 모델 | IFC2x3/IFC4 | 발주처 기준 |
| 간섭 보고서 | PDF/XLSX/HTML | 예 |
| 품질 검증 증빙 | PDF/XLSX | 예 |
| COBie/FM 데이터 | XLSX/CSV/DB | 발주처 기준 |
| 제출대장 | XLSX/PDF | 예 |

## 9. 검수 기준

| 검수 항목 | 합격 기준 |
|---|---|
| 파일 열람 | 지정 소프트웨어에서 오류 없이 열림 |
| 좌표 정합 | 통합 모델 중첩 시 기준점 일치 |
| 필수 파라미터 | 필수 항목 누락 없음 |
| 명명 규칙 | 파일/뷰/패밀리 기준 준수 |
| 간섭 상태 | 미해결 이슈가 승인된 상태로 관리 |
| 납품 목록 | 제출대장과 실제 파일 일치 |
