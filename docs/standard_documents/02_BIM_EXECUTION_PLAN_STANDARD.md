## 1. 수행계획서 목적

BIM 수행계획서는 제안 단계의 약속을 실제 업무 절차로 전환하는 문서다. 본 문서는 프로젝트 참여자가 동일한 수행 범위, 일정, 책임, 산출물 기준을 공유하도록 작성한다.

## 4. 수행 범위

| 범위 | 포함 | 제외/협의 필요 |
|---|---|---|
| 모델링 | 공종별 BIM 모델 작성/수정 | 설계 책임 변경은 별도 계약 |
| 간섭 검토 | 주요 공종 간 Clash Detection | 현장 시공 상세 결정은 시공사 승인 필요 |
| 물량 산출 | 모델 기반 수량 추출 | 단가, 계약 금액 확정은 공무 기준 |
| CDE | 파일/이슈/승인 운영 | 플랫폼 라이선스 비용은 협의 |
| 준공 데이터 | IFC, COBie/FM 데이터 | 발주처 시스템 직접 연동은 별도 협의 |

## 10. 완료 기준

프로젝트 단계별 BIM 업무는 다음 조건을 충족할 때 완료로 본다.

- 합의된 산출물이 지정 형식으로 CDE에 제출되어 있다.
- 모델 품질 체크리스트의 필수 항목이 통과되어 있다.
- 미해결 간섭 및 변경 이슈가 책임자와 조치 예정일을 가진 상태로 관리되고 있다.
- 발주처/시공사/CM의 단계 승인 또는 조건부 승인을 확보했다.
