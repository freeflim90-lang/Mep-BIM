## 1. 검증 목적

간섭 및 품질 검증은 시공 전 오류를 줄이고, 시공 중 변경 사항을 추적하며, 준공 단계에서 발주처 검수에 필요한 객관적 근거를 확보하기 위한 절차다.

## 2. 간섭 검토 기준

| 검토 조합 | 우선순위 | 주요 검토 내용 |
|---|---|---|
| 구조 vs MEP | High | 보/슬래브 관통, 장비실 간섭 |
| 건축 vs MEP | High | 천장고, 점검구, 방화구획 |
| 기계 vs 전기 | High | 덕트/배관과 케이블트레이 간섭 |
| 소방 vs 기타 MEP | High | 스프링클러, 소화배관 설치 공간 |
| 통신/보안 vs 건축/MEP | Medium | 단말 위치, 전원/통신 경로 |
| 가설 vs 기존 시설 | Medium | 임시 설비, 작업 동선, 양중 |

## 3. 간섭 이슈 등급

| 등급 | 기준 | 조치기한 |
|---|---|---|
| Critical | 시공 불가, 안전/법규/운영 중단 영향 | 즉시 또는 2영업일 이내 |
| High | 공정 지연 또는 재시공 가능성 높음 | 5영업일 이내 |
| Medium | 시공 상세 조정 필요 | 다음 검토 회의 전 |
| Low | 모델 표현/문서 정리 수준 | 단계 종료 전 |

## 4. 검토 회의 운영

| 회의 | 주기 | 참석자 | 결과물 |
|---|---|---|---|
| 주간 간섭 검토 | 주 1회 | BIM팀, 시공사, 공종 담당자 | 간섭 보고서, 이슈 대장 |
| 긴급 검토 | 필요 시 | 관련 공종 책임자 | 긴급 조치 기록 |
| 월간 품질 회의 | 월 1회 | BIM PM, 시공사, 감리/CM | 월간 품질 보고 |

## 5. 모델 품질 검증

| 검증 항목 | 방법 | 합격 기준 |
|---|---|---|
| 파일 무결성 | 모델 열람, 링크 확인 | 오류 없이 열림 |
| 좌표/레벨 | 통합 모델 중첩 | 기준점 일치 |
| 카테고리 | 요소 카테고리 검토 | 잘못된 카테고리 없음 |
| 파라미터 | 필수값 검사 | 필수 필드 입력 |
| 중복 요소 | 중복/겹침 검사 | 중복 제거 또는 승인 |
| 뷰/시트 | 명명 및 표시 기준 확인 | 기준 준수 |

## 6. Scan vs BIM 검증

| 검증 항목 | 검증 방법 | 권장 허용 기준 |
|---|---|---|
| 배관/덕트 위치 | BIM 좌표와 점군 비교 | 프로젝트별 협의 |
| 장비 설치 위치 | 기준점/방향 비교 | 프로젝트별 협의 |
| 천장 내 여유 공간 | 점군과 모델 중첩 | 유지관리 공간 확보 |
| 시공 완료율 | 설치 객체와 모델 상태 비교 | 구간별 완료율 산정 |
| 미시공/오시공 | 점군 미검출/위치 차이 | 이슈 등록 |

허용 기준은 발주처 요구사항, 시공 허용오차, 감리 기준에 따라 프로젝트 착수 시 확정한다.

## 7. 보고서 구성

간섭 및 품질 보고서는 다음 항목을 포함한다.

| 항목 | 내용 |
|---|---|
| 보고 개요 | 회차, 기준일, 대상 모델, 검토 범위 |
| 요약 지표 | 신규/미해결/종결 이슈 수, Critical/High 건수 |
| 주요 이슈 | 위치, 스크린샷, 책임자, 조치기한 |
| 추이 분석 | 회차별 이슈 증감 |
| 조치 결과 | 해결 방법, 반영 모델 버전 |
| 다음 조치 | 다음 회의 전 결정 필요 사항 |

## 8. 완료 기준

- Critical 이슈는 모두 종결 또는 승인된 예외 상태여야 한다.
- High 이슈는 조치기한과 책임자가 명확해야 한다.
- 납품 모델의 품질 체크 필수 항목이 통과되어야 한다.
- 보고서와 CDE 이슈 상태가 일치해야 한다.
