## 1. 물량 관리 목적

BIM 기반 물량 관리는 모델 객체와 내역 항목의 연결성을 확보하여 견적, 발주, 변경, 기성, 준공 정산의 근거를 명확히 하는 데 목적이 있다.

## 2. 물량 산출 범위

| 단계 | 산출 내용 | 활용 목적 |
|---|---|---|
| 초기 | 기존/철거/신설 대상 개략 물량 | 예산 및 범위 검토 |
| 설계 | 설계 모델 기반 상세 물량 | 발주/내역 검토 |
| 시공 | 변경 전/후 물량 | 추가/감소 비용 근거 |
| 기성 | 완료 구간별 물량 | 월별 기성 검토 |
| 준공 | 최종 설치 물량 | 정산 및 자산 대장 |

## 3. 모델-내역 매핑 기준

| 모델 정보 | 내역 정보 | 비고 |
|---|---|---|
| 카테고리 | 공종 | 기계/전기/소방/통신 등 |
| 타입/규격 | 품명/규격 | 관경, 덕트 크기, 장비 사양 |
| 시스템명 | 공종 세부 분류 | HVAC, 급수, 배수 등 |
| LBL_ItemCode | 내역 코드 | 고객사 코드 체계 연계 |
| 수량 파라미터 | 단위 | m, m2, m3, EA |
| 구역/층 | 공구/기성 구간 | 월별 정산 연계 |

## 4. 변경 물량 관리

변경 물량은 변경 전 모델과 변경 후 모델의 차이를 기준으로 산정한다.

| 항목 | 기준 |
|---|---|
| 변경 ID | 변경 요청/지시와 동일한 번호 사용 |
| 기준 모델 | 변경 전 승인 모델 Revision |
| 비교 모델 | 변경 반영 후 모델 Revision |
| 산출 방식 | 객체 추가/삭제/수정분 분리 |
| 보고 내용 | 증가/감소 물량, 영향 공종, 비용 영향, 일정 영향 |

## 5. 기성 및 정산 지원

| 업무 | BIM 활용 방식 | 산출물 |
|---|---|---|
| 월별 기성 | 설치 완료 구간 모델 필터링 | 기성 물량표 |
| 하도급 정산 | 협력사별 공종/구역 물량 분리 | 협력사별 내역 |
| 변경 정산 | 변경 ID별 증가/감소 물량 추적 | 변경 물량 보고서 |
| 준공 정산 | 준공 모델 기준 최종 물량 | 최종 물량 산출서 |

## 6. ROI 산정 기준

| 효과 항목 | 산정 방식 | 근거 자료 |
|---|---|---|
| 재시공 감소 | 예방된 간섭 건수 x 평균 재작업 비용 | 간섭 이슈 대장 |
| 공기 단축 | 지연 예방일 x 일일 간접비 | 공정표, 회의록 |
| 물량 정확도 | 기존 산출 대비 오차 감소율 | 내역/모델 비교표 |
| 보고 공수 절감 | 자동화 전후 작성 시간 비교 | 업무 시간 기록 |
| 분쟁 감소 | 물량 이슈 종결률, 미해결 건수 | 변경/정산 대장 |

정량 효과가 확정되지 않은 경우 “예상”, “목표”, “프로젝트 검증 필요”를 명시한다.

## 7. 보고서 구성

물량 및 ROI 보고서는 다음 구조를 따른다.

| 섹션 | 내용 |
|---|---|
| 요약 | 기준일, 대상 모델, 총 물량, 변경 건수 |
| 공종별 물량 | 공종/시스템/구역별 수량 |
| 변경 물량 | 변경 ID별 증가/감소 |
| 비용 영향 | 단가 적용 가능 시 금액 영향 |
| 기성 연계 | 완료 구간 및 청구 가능 물량 |
| 리스크 | 물량 신뢰도, 미확정 항목 |
| 다음 조치 | 고객 확인 필요 사항 |
