## 1. 요청 목적

본 자료요청서는 BIM 업무 착수에 필요한 기본 자료를 사전에 수령하여 착수 지연, 모델 기준 불일치, 산출물 누락을 방지하기 위한 문서다.

| 요청 자료 | 형식 | 비고 |
|---|---|---|
| 프로젝트 개요서 | PDF/DOCX | 위치, 규모, 용도, 공사 범위 |
| 계약/업무범위 관련 자료 | PDF | BIM 수행 범위 확인용 |
| 참여 조직도 | PDF/XLSX | 발주처, 시공사, 설계사, 감리/CM |
| 담당자 연락처 | XLSX | 이름, 소속, 역할, 이메일, 전화 |

## 7. 자료 제출 형식

자료는 가능하면 CDE 또는 고객 지정 공식 저장소로 제공한다. 이메일 첨부만으로 전달할 경우 용량, 버전, 누락 이슈가 발생할 수 있으므로 자료 목록과 함께 제출한다.

| 항목 | 기준 |
|---|---|
| 파일명 | 프로젝트명-공종-자료명-Revision-날짜 |
| 압축 | 공종/구역별 ZIP 분리 |
| 제출목록 | 파일명, 설명, Revision, 작성일 포함 |
| 우선순위 | 도면, 모델, 좌표, 공정표, EIR 순으로 우선 제공 |
