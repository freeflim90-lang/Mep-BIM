## 1. 회사 개요

LUA BIM LABS는 BIM 데이터, Revit/Navisworks 자동화, 모델 품질 검증, CDE 운영, 납품 문서화를 통합 수행하는 BIM 기술 조직이다.

LUA BIM LABS는 AX(AI Transformation)를 BIM 실무에 적용하여 설계, 시공, 품질검토, 납품, 교육, 자동화 전반을 지능화하고 BIM 업계에서 선도적인 기술 기업으로 성장하는 것을 목표로 한다.

| 항목 | 내용 |
|---|---|
| 조직명 | LUA BIM LABS |
| 핵심 분야 | BIM 수행, BIM AX 자동화, Add-in 개발, 품질/납품 문서화 |
| 주요 도구 | Revit, Navisworks, Autodesk Construction Cloud, Excel/CSV, Revit API, Obsidian/RAG |
| 수행 방식 | 기준 문서 수립, 모델링, 검토, 보고, 납품, 자동화를 통합 운영 |

## 2. 핵심 역량

| 역량 | 설명 | 고객 가치 |
|---|---|---|
| BIM 기준 수립 | LOD, 명명, 파라미터, 납품 기준 정의 | 프로젝트 초반 혼선 방지 |
| 모델 품질 관리 | 좌표, 파라미터, 중복, 간섭 검토 | 납품 반려 및 재작업 감소 |
| 현장 활용 BIM | 구역/공종별 검토 모델, 이슈 위치화 | 현장 의사결정 속도 향상 |
| 물량/변경 추적 | 모델 기반 수량 및 변경분 관리 | 공무/정산 근거 확보 |
| 자동화 도구 개발 | 반복 업무를 Add-in/스크립트로 전환 | 장기 운영 효율 향상 |
| AX 지식 운영 | BIM 지식, QA, 기준, 프로젝트 경험을 검색/재사용 가능한 지식망으로 축적 | 답변 품질, 교육 속도, 자동화 전환성 향상 |

## 3. BIM 수행 서비스

- BIM 수행계획서 및 BEP 작성
- 공종별 BIM 모델링 및 통합 모델 운영
- Navisworks 기반 간섭 검토 및 보고서 작성
- CDE 폴더/권한/승인 체계 운영
- 모델 기반 물량 산출 및 변경 물량 추적
- Scan vs BIM 및 준공 품질 검증 지원
- IFC, COBie, FM 데이터 납품 패키지 구성

## 4. 자동화 및 Add-in 역량

LUA BIM LABS는 단순 모델링 용역을 넘어 반복 업무를 소프트웨어 기능으로 전환하는 역량을 보유한다.

| 영역 | 예시 |
|---|---|
| Revit 자동화 | 요소 필터링, 파라미터 점검, 도면/일람표 자동화 |
| Navisworks 검토 | Clash 결과 정리, 이슈 보고 자동화 |
| 데이터 처리 | Excel/CSV 기반 물량, 변경, 검토 결과 정리 |
| 상용 Add-in | BIM Command Center for Revit 제품화 기준 보유 |
| AI/RAG 운영 | 반복 Q&A, 기준 문서, 프로젝트 경험을 지식 그래프와 에이전트 답변으로 연결 |

## 5. 프로젝트 적용 가치

| 구분 | 적용 가치 |
|---|---|
| 발주처 | 준공 BIM과 FM 데이터의 검수 가능성 향상 |
| 시공사 | 간섭, 변경, 물량, 감리 대응 업무 부담 감소 |
| 설계사 | 설계 변경 및 조치 요청의 추적성 확보 |
| 협력사 | 구역별 시공 정보와 이슈 위치를 명확히 확인 |
| 감리/CM | 품질 검증 근거와 승인 이력 확보 |

## 6. 제안서 삽입용 문구

LUA BIM LABS는 BIM 모델을 단순한 3D 표현물이 아니라 프로젝트 정보관리의 기준 데이터로 운영한다. 모델링, 간섭 검토, 물량 산출, 변경 관리, 품질 검증, 준공 납품을 하나의 흐름으로 연결하여 시공 현장의 의사결정과 발주처 검수 대응을 동시에 지원한다.

특히 Revit/Navisworks 기반 자동화와 문서화 기준을 함께 보유하고 있어, 프로젝트별 반복 업무를 안정적인 운영 절차와 재사용 가능한 도구로 전환할 수 있다. 이를 통해 고객은 BIM 업무를 일회성 산출물 제작이 아니라 공정, 품질, 비용 리스크를 줄이는 실무 체계로 활용할 수 있다.

LUA BIM LABS는 BIM을 지식화하고, 지식을 자동화하고, 자동화를 AX 제품으로 전환하는 방향으로 성장한다. 이를 통해 고객의 프로젝트 데이터가 단순 납품물이 아니라 의사결정, 품질관리, 교육, 운영 개선에 재사용되는 자산이 되도록 지원한다.
