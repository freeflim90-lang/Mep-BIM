## 1. 목적

본 문서는 LUA BIM LABS 신규 직원이 회사의 BIM 수행 기준, CDE 운영, 모델 품질, 납품 문서, 보안 기준을 빠르게 습득하고 연차별 책임 수준에 맞게 성장하도록 교육 체계를 정의한다.

## 2. 운영 책임

| 역할 | 책임 |
|---|---|
| 교육컨설팅 | 교육 로드맵, 커리큘럼, 실습 과제, 평가 기준 설계 |
| 경영지원 | 입사 행정, 계정, 수습 일정, 필수 규정 교육 |
| BIM_템플릿기획관 | LOD, 파라미터, 모델링 기준 교육 검토 |
| 테크니컬_라이터 | 교육자료 문장, 매뉴얼, 배포 형식 정리 |
| QA_테스터 | 실습 평가, 체크리스트, 수료 기준 검증 |
| 조율차장 | 부서 간 handoff와 교육 개선 과제 병합 |

| 기간 | 목표 | 주요 교육 | 산출물 |
|---|---|---|---|
| 1~30일 | 회사와 문서 체계 적응 | 조직 구조, 보안/NDA, CDE, 파일명, 기본 Revit | CDE 업로드 실습, 보안 체크 |

## 8. 교육자료 관리

교육자료는 문서관리 기준에 따라 Revision을 관리한다. 프로젝트 수행 중 발견된 반복 오류는 교육컨설팅이 교육자료 개정 후보로 등록하고, 조율차장이 표준문서 개정 여부를 판단한다.

주간 공식 기준 업데이트는 역할별 지식학습 스프린트로 전환한다. 각 조직원은 자기 역할에 맞는 산출물을 제출하고, 교육 기록에는 학습 지식, `KST` 상태 코드, 적용 산출물, 다음 검토일을 남긴다.
