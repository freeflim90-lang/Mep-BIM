# 서비스 지원 및 SLA 기준

문서번호: LBL-STD-019  
문서상태: 기준 초안  
작성일: 2026-05-20  
적용범위: BIM 프로젝트 운영지원, 납품 후 보완, Add-in 고객지원

## 목 차

1. 목적
2. 지원 범위
3. 지원 채널
4. 장애/요청 등급
5. 응답 및 처리 목표
6. 제외 범위
7. 지원 이력 관리

## 1. 목적

본 기준은 고객지원 요청의 우선순위, 응답시간, 처리 범위를 정하여 운영 중 혼선을 줄이고, 납품 후 보완과 신규 요청을 구분하기 위한 기준이다.

## 2. 지원 범위

| 구분 | 지원 내용 |
|---|---|
| 프로젝트 지원 | 납품 파일 열람, 보고서 설명, 검수 의견 보완 |
| 모델 지원 | 파일 오류, 링크 누락, 형식 변환 이슈 |
| 문서 지원 | 제출대장, 보고서, 회의록 보완 |
| Add-in 지원 | 설치, 실행, 라이선스, 오류 로그 확인 |
| 교육 지원 | 계약 범위 내 사용자 안내 |

## 3. 지원 채널

| 채널 | 용도 |
|---|---|
| 이메일 | 공식 요청 및 답변 |
| CDE 이슈 | 모델/도면 위치 기반 이슈 |
| 회의 | 복합 이슈 협의 |
| 원격 지원 | 설치/실행 문제 확인 |

## 4. 장애/요청 등급

| 등급 | 기준 | 예시 |
|---|---|---|
| P1 Critical | 업무 중단 또는 납품 불가 | 파일 전체 열람 불가, 제출본 누락 |
| P2 High | 주요 업무 지연 | 주요 링크 누락, 보고서 오류 |
| P3 Medium | 일반 보완 | 일부 표기 오류, 추가 설명 요청 |
| P4 Low | 개선/문의 | 기능 개선 요청, 향후 반영 제안 |

## 5. 응답 및 처리 목표

| 등급 | 1차 응답 목표 | 처리 목표 |
|---|---|---|
| P1 | 1영업일 이내 | 원인 확인 후 일정 즉시 공유 |
| P2 | 2영업일 이내 | 합의된 기한 내 보완 |
| P3 | 3영업일 이내 | 다음 정기 배포 또는 회차 반영 |
| P4 | 5영업일 이내 | 검토 후 반영 여부 회신 |

위 기준은 기본 목표이며, 계약서 또는 별도 SLA가 있는 경우 해당 조건을 우선한다.

## 6. 제외 범위

| 제외 항목 | 비고 |
|---|---|
| 계약 범위 외 신규 모델링 | 별도 견적 |
| 설계 책임 검토 | 설계사/기술사 책임 범위 |
| 고객 PC/네트워크 문제 | 원인 확인 지원 가능 |
| 타사 소프트웨어 장애 | 제조사 지원 필요 |
| 무제한 교육/상주 지원 | 별도 계약 |

## 7. 지원 이력 관리

| 필드 | 설명 |
|---|---|
| Ticket ID | 요청 번호 |
| 접수일 | YYYY-MM-DD |

| 등급 | P1~P4 |

| 담당자 | 내부 담당 |
| 상태 | 접수/진행/대기/완료 |
| 완료일 | YYYY-MM-DD |
| 후속 조치 | 재발 방지 또는 개선 항목 |
