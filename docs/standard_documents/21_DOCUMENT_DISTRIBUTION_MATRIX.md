# 내부/외부 문서 배포 매트릭스

━━━━━━━━━━━━━━━━━━━━

문서번호: LBL-STD-021  
문서상태: 기준 초안  
작성일: 2026-05-20  
적용범위: 표준 문서의 내부 사용, 외부 제출, 협력사 공유 판단

## 목 차

1. 목적
2. 배포 등급
3. 문서별 배포 기준
4. 외부 제출 전 승인 기준
5. 추가 필요 문서 판단 기준

## 1. 목적

본 문서는 표준 문서 세트 중 어떤 문서를 내부 운영용으로만 사용할지, 어떤 문서를 고객/발주처/협력사에게 제출하거나 공유할 수 있는지 구분하기 위한 기준이다.

## 2. 배포 등급

| 등급 | 의미 | 승인 기준 |
|---|---|---|
| Internal Only | 내부 운영용 | 외부 배포 금지 |
| External Ready | 프로젝트 정보 반영 후 외부 제출 가능 | PM 검토 후 제출 |
| External With Approval | 조건/법무/보안 검토 후 외부 제출 | PM + 대표/법무/보안 승인 |
| Partner Share | 협력사 업무 범위 내 공유 가능 | PM 승인 |

## 4. 외부 제출 전 승인 기준

| 문서 유형 | 필수 검토 |
|---|---|
| 제안서/회사소개 | PM, 대표 또는 영업 책임 |
| 견적/SOW | PM, 재무/대표 |
| 계약/보안/NDA | PM, 법무/보안 담당 |
| BEP/수행계획 | PM, BIM Lead |
| 납품/검수 | PM, Document Controller |
| SLA/지원 조건 | PM, 운영/대표 |

## 5. 추가 필요 문서 판단 기준

다음 조건이 발생하면 별도 문서를 추가한다.

| 조건 | 추가 문서 |
|---|---|
| 특정 발주처 양식이 있음 | 발주처별 제출 양식 템플릿 |
| 특정 시설 유형 반복 | 공항/병원/데이터센터별 제안 문구 세트 |
| 법무 검토가 반복됨 | 표준 계약 특약 검토표 |
| 제품 판매가 확대됨 | 제품별 EULA/Privacy/Support 문서 |
| 협력사 풀이 커짐 | 협력사 평가표 및 성과관리 기준 |
| 내부 인력이 늘어남 | 역할별 업무 매뉴얼과 교육 체크리스트 |
