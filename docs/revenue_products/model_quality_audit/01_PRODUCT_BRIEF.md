## 1. 상품 개요

`MEP BIM 모델 품질진단`은 고객이 보유한 MEP BIM 모델의 품질, 간섭, 데이터 완성도, 납품 적합성을 진단하고 개선 우선순위를 제시하는 단기 컨설팅형 서비스다.

고객은 모델을 보유하고 있어도 실제 납품 가능한 상태인지, 시공 전 검토에 충분한 수준인지, 공종 간 간섭과 데이터 누락이 어느 정도인지 빠르게 판단하기 어렵다. 본 상품은 그 불확실성을 정리된 보고서와 실행 가능한 조치 목록으로 전환한다.

---

## 2. 고객 문제

| 고객 상황 | 발생 문제 |
|---|---|
| 모델은 있으나 검수 기준이 없음 | 납품 반려, 재작업, 책임 소재 불명확 |
| 공종별 모델이 따로 작성됨 | 통합 시 간섭과 좌표 문제 발생 |
| 설계 변경이 반복됨 | 모델, 도면, 물량 간 불일치 |
| 외주 모델 품질 확인이 어려움 | 내부 검수 공수 증가 |
| 준공 BIM 요구가 있음 | 납품 형식과 데이터 누락 리스크 |

---

## 3. 제공 가치

| 가치 | 설명 |
|---|---|
| 납품 리스크 조기 확인 | 제출 전 품질 문제와 형식 문제를 사전에 확인 |
| 수정 우선순위 제공 | 모든 문제를 나열하지 않고 영향도 기준으로 정리 |
| 고객 회의 근거 확보 | CM, 감리, 시공사, 외주사와 협의 가능한 보고서 제공 |
| 재작업 비용 절감 | 반복 오류와 범위 외 요청을 조기에 식별 |
| 후속 업무 연결 | 교육, 재검토, BEP/CDE 세팅, 물량/변경관리로 확장 가능 |

---

## 4. 주요 산출물

| 산출물 | 설명 |
|---|---|
| 모델 품질진단 보고서 | 모델 상태, 오류 유형, 리스크, 개선 방향 |
| 간섭/이슈 대장 | 주요 간섭, 책임 공종, 우선순위, 조치 의견 |
| 납품 적합성 체크 | 좌표, 파일 구조, 명명, 파라미터, 형식 기준 검토 |
| 요약 발표 자료 | 의사결정자를 위한 핵심 이슈 요약 |
| 재검토 제안서 | 필요 시 보완 후 재검토 또는 교육 범위 제안 |

---

## 5. 제외 범위

아래 항목은 기본 품질진단 범위에 포함하지 않으며, 별도 견적 또는 별도 계약으로 수행한다.

| 제외 항목 | 비고 |
|---|---|
| 모델 직접 수정 | 고객 승인 후 별도 모델링 용역으로 전환 |
| 설계 검토 책임 | 설계 적합성 최종 판단은 설계 책임자/감리 기준 |
| 법규 검토 확정 | 법규 자문이 필요한 경우 별도 전문가 확인 필요 |
| 상세 물량 산출 | 기본 데이터 상태 확인은 가능하나 정식 물량 산출은 별도 |
| 무제한 재검토 | 패키지별 포함 횟수 외 별도 비용 |

## 6. 후속 상품 연결

| 진단 결과 | 후속 제안 |
|---|---|
| 모델 품질 기준 부재 | BEP/CDE/납품 기준 세팅 컨설팅 |
| 반복 모델링 오류 | 고객사 MEP BIM 실무 교육 |
| 외주 품질 불안정 | 협력사 품질 검수 체계 구축 |
| 물량/변경 이슈 | 모델 기반 물량/변경관리 패키지 |
| 반복 검토 공수 과다 | 자동화/Add-in 진단 도구 기획 |
