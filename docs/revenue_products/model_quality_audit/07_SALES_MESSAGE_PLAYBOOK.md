## 3. 제안 메일 초안

제목: MEP BIM 모델 품질진단 서비스 제안

안녕하세요. LUA BIM LAB입니다.

현재 보유 중인 MEP BIM 모델이 납품, 간섭 협의, 준공 제출 기준에 적합한지 사전에 확인할 수 있는 `MEP BIM 모델 품질진단` 서비스를 제안드립니다.

본 서비스는 모델을 직접 수정하기보다, 현재 모델의 품질 상태와 주요 리스크를 진단하고 수정 우선순위를 보고서로 정리하는 단기 검토 서비스입니다.

주요 검토 항목은 다음과 같습니다.

- 파일 구조, 버전, 링크 상태
- 좌표 및 레벨 정합성
- MEP 공종별 모델 품질
- 건축/구조/MEP 주요 간섭
- 파라미터 및 납품 데이터 상태
- 제출 전 보완 필요 항목

검토 후에는 품질진단 보고서와 주요 이슈 대장을 제공드리며, 필요 시 결과 설명 회의도 함께 진행할 수 있습니다.

검토 가능 여부 확인을 위해 모델 형식, 공종 범위, 희망 일정을 알려주시면 적합한 패키지와 견적 범위를 안내드리겠습니다.

감사합니다.  
LUA BIM LAB

## 4. 첫 미팅 질문

| 질문 | 목적 |
|---|---|
| 현재 모델을 어떤 목적으로 사용하려고 하시나요? | 납품/시공/검수/외주관리 구분 |
| 가장 걱정되는 부분은 간섭, 납품, 물량, 품질 중 무엇인가요? | 구매 동기 확인 |
| 모델은 어떤 형식과 버전으로 보유 중인가요? | 수행 가능성 확인 |
| 건축/구조 링크 모델도 함께 제공 가능한가요? | 간섭 검토 범위 확인 |
| 납품 또는 회의 마감일이 정해져 있나요? | 긴급도 확인 |
| 내부 BIM 기준이나 BEP가 있나요? | 기준 존재 여부 확인 |

## 5. 반대 의견 대응

| 고객 반응 | 대응 방향 |
|---|---|
| 내부에서 검토할 수 있다 | 내부 검토 전 제3자 기준으로 위험 항목을 빠르게 추리는 용도라고 설명 |
| 모델을 수정해주는 게 아닌가 | 기본 상품은 진단이며, 수정은 별도 범위로 전환 가능하다고 설명 |
| 비용이 부담된다 | Lite 진단으로 시작해 핵심 리스크만 확인하는 방안 제안 |
| 자료 보안이 걱정된다 | NDA, 자료 보관/삭제 기준, 접근 제한을 먼저 합의 |
| 시간이 없다 | 납품 전 Critical 이슈 중심의 긴급 진단 제안 |

## 6. 후속 제안 문구

품질진단 결과 반복 오류가 확인된 경우:

> 이번 진단에서 발견된 이슈는 단순 수정만으로 끝내기보다, 모델링 기준과 검수 기준을 함께 정리하면 이후 프로젝트에서 같은 오류를 크게 줄일 수 있습니다.

교육 연계가 필요한 경우:

> 보고서 결과를 바탕으로 실무자 대상 1일 또는 2일 MEP BIM 품질 교육을 구성할 수 있습니다. 실제 고객사 모델에서 발견된 이슈를 익명화해 교육 사례로 전환하는 방식이 효과적입니다.
