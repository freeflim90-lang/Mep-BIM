## 1. Product Name

Model Quality Auditor

---

## 2. Short Description

A model quality checking add-in for BIM teams who need fast model health review, issue prioritization and report-ready audit results inside Autodesk Revit® workflows.

---

## 3. Long Description

Model Quality Auditor helps BIM managers, coordinators and MEP teams review model quality before coordination meetings, external submissions or project handovers.

The add-in checks practical model health items such as file structure, model warnings, linked model status, workset conditions, view/template consistency, key parameter completeness and reportable model issues. Results are organized into clear categories so teams can understand what should be fixed first.

The first release focuses on local model review and report generation. Project model data is processed on the user's computer and is not sent to an external AI or cloud service by the core audit commands.

Model Quality Auditor is designed for teams who need a repeatable quality review workflow rather than a one-time manual checklist.

## 7. What This App Does Not Do

- It does not replace engineering judgment.
- It does not certify code compliance.
- It does not automatically fix project models in v1.0.
- It does not send project model data to an external AI service in the core audit workflow.
- It does not include unlimited consulting or manual model review.
