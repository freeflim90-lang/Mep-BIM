# Link Health & Reload

Priority: P2

## Commercial Value

Coordination teams need to see broken, outdated, or wrongly pathed links quickly before publishing deliverables.

## Minimum Scope

- List Revit, CAD, and point cloud link references.
- Show loaded/unloaded/missing status.
- Export link health report.
- Reload selected links after confirmation.

## First Safe Step

Implement a read-only link health report before adding reload behavior.
