# Schedule Excel Sync

Priority: P2

## Commercial Value

Schedule-driven parameter editing is a high-value productivity feature, especially for repetitive MEP and documentation data.

## Minimum Scope

- Export schedule data to CSV/XLSX.
- Preserve element identifiers for import.
- Skip read-only parameters.
- Import changed values with preview.

## First Safe Step

Start with CSV export/import mapping and strict read-only protection before direct Excel dependency.
