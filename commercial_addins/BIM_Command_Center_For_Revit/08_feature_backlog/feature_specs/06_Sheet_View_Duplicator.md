# Sheet/View Duplicator

Priority: P2

## Commercial Value

Drawing teams often need repeatable sheet/view creation while keeping viewport placement consistent.

## Minimum Scope

- Duplicate selected sheets.
- Duplicate dependent views or views with detailing based on user choice.
- Preserve viewport positions where Revit allows it.
- Rename sheets and views by naming rule.

## First Safe Step

Review existing drawing automation modules before adding a new duplicated workflow.
