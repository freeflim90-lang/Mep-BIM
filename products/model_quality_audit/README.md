# products/model_quality_audit

MEP BIM 모델 품질 감리(B2B 서비스)의 **운영 상태**를 둔다 — 수임 건(engagements.json) 등 런타임 데이터.

- 제품 정의·서비스 메뉴·가격·딜리버리 SOP·세일즈 플레이북 등 **문서 원본**은
  [docs/revenue_products/model_quality_audit/](../../docs/revenue_products/model_quality_audit/) 에 있다
  (Obsidian 글로벌 맵·지식 큐레이션이 이 경로를 분류 참조하므로 이동하지 않는다).
- 제품 단일 인덱스는 [config/products.json](../../config/products.json) 의 `model_quality_audit` 항목.

이로써 products/ 하위에 starter_plan·bim_land·bim_command_center·autodesk_market 와 동일하게
서비스 제품도 운영 상태 폴더를 갖는다(택소노미 대칭화).
