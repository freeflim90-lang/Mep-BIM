اليوم 87 - قائمة التفتيش على المعلمات الكهربائية — اسم لوحة / مجمع التوزيع، رقم الدارة، الحمولة، الجهد، النظام

1. لماذا هذا يهم
المعلمات الكهربائية هي طبقة البيانات التي تجعل نموذج BIM مفيدًا خارج نطاق اكتشاف الاصطدام. نموذج مع هندسة صحيحة لكنه يفتقر للمعلومات أو المعلومات غير المتسقة لا يمكن أن ينتج جداول لوحة كهربائية موثوقة أو ملخصات الحمولة أو بيانات التسليم. تلتقط قائمة التفتيش النظامية للعوامل فجوات البيانات قبل تقديم النموذج — مما يمنع إصلاحات مكلفة بعد قبول العميل.

2. المفهوم الأساسي
قائمة التفتيش على المعلمات الكهربائية — عنصرًا بعنصر:

لوحة / مجمع التوزيع:
- اسم لوحة — يتطابق تمامًا مع ID لوحة SLD
- الجهد — مثل 400V، 230V
- عدد الأبواب الفازية — 1 أو 3
- تصنيف التوصيل (أ) — قدرة المفتاح الرئيسي
- تصنيف العطل (كأ) — تصنيف خط المرور أثناء اختبار القصر الكهربائي
- عدد الأبواب / الدارات — 总容量
- اسم الغرفة — الغرفة التي تقع فيها لوحة التوزيع
- الطابق المخدوم / المنطقة — الطوابق أو المناطق الممده منها هذه اللوحة
- طوارئ / عادية — تصنيف نوع الطاقة

دارة (للدارات في جدول الدارات):
- رقم الدارة — يتطابق مع رقم الدارة في جدول لوحة الهندسة
- وصف الدارة — يتطابق مع وصف لوحة الدارات في التصميم الهندسي
- فئة الحمولة — إنارة / قوى التشغيل / HVAC / تحذيرات الحرائق / طوارئ
- عدد الأبواب — 1فاز (أحد أبواب) أو 3فاز (ثلاثة أبواب)
- حجم الإطار / تصنيف التوصيل (أ) — تصنيف المفتاح
- الحمولة المرئية (VA) أو الكيلوواط (W) — قيمة الحمولة المرتبطة
- ID العنصر المرتبط — يرتبط بعائلة الأجهزة ذات الحمولة

سلك الشبكة / الأنابيب:
- نوع النظام — قوى-HV / قوى-LV / بيانات / تحذيرات الحرائق / أمن
- المادة — حديد / صلب / PVC / فولاذ مغلف
- العرض (مم) — عرض السلك
- العمق (مم) — عمق السلك
- الحجم التجاري (للأنابيب فقط) — بالمتر أو بوصة

الأجهزة (الحمولات المرتبطة بالدارات):
- اسم الجهاز / العلامة التجارية — يتطابق مع جدول الأجهزة الميكانيكية أو الكهربائية
- اسم لوحة التوزيع — التي تغذي هذه الأجهزة لوحة التوزيع
- رقم الدارة — الذي يغذي هذا الجهاز مفتاح دارة معين
- الجهد — 230V أو 400V
- الحمولة (كيلوواط) أو VA — قيمة الحمولة المرتبطة

أداة الفحص: مشاهد جداول Revit — إنشاء مشهد جدول متعدد الأقسام يشمل جميع العناصر الكهربائية مع المعلمات فوق. تصفية لعرض الخلايا الفارغة — أي خلية فارغة هي فجوة بيانات.

3. نظرة على المشروع الواقع
يقوم م co-ordinator BIM بإنشاء مشهد جدول متعدد الأقسام في Revit يغطي جميع لوائف التوزيع، الدارات والأجهزة المرتبطة بمشروع الفندق. تصفية للفجوات الفارغة "اسم لوحة" على الأجهزة، وجدت 31 عائلة أجهزة بدون اسم لوحة — كلها مثبتة لكنها لم يتم تعيين الدارة إليها. تصفية للفجوات الفارغة "فئة الحمولة" على الدارات، وجدت 18 دارة بدون تصنيف. تم إصلاح كلا الفجوة قبل تصدير بيانات التسليم، مما يتجنب رفض جودة البيانات من فريق إدارة المرافق.

4. تحقق BIM
- تحقق: هل مشهد Revit متعدد الأقسام يظهر صفوفًا فارغة لاسم لوحة، رقم الدارة وفئة الحمولة لكل العناصر الكهربائية؟
- تحقق: هل تم تعيين معلم الجهد على كل لوائف التوزيع والعائلات المرتبطة بالأجهزة — وليس تركها في الافتراض الافتراضي لـ Revit؟
- تحقق: هل تم تعيين معلم نظام النوع على كل سلك الشبكة والأنابيب — مما يسمح بتصفية النظام وتعيين قواعد الاشتباك؟

5. الخطأ الشائع
تعبئة المعلمات فقط في لوائف التوزيع ونسيان تعبئة الأجهزة المرتبطة بالحمولة. تبدو جداول لوائف التوزيع مكتملة، لكن عائلات الأجهزة المرتبطة بالحمولة لا تحتوي على اسم لوحة أو رقم الدارة — مما يجعل من المستحيل تتبعها من الأجهزة إلى لوحة التوزيع دون البحث اليدوي في سجل النظام.

6. العمل اليومي
في Revit، إنشاء جدول بسيط للأجهزة الكهربائية (أو الإضاءة) مع ثلاث أعمدة: اسم العائلة، اسم لوحة التوزيع، رقم الدارة. تصديره. عد عدد الصفوف التي تحتوي على خلايا فارغة لاسم لوحة أو رقم الدارة. هذه العدد هو قاعدة الفجوة البيانات للمodel الكهربائي الخاص بك.
