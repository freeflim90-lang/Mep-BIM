Day 64 - 배관 사이징 매개변수

1. 중요성
배관 사이징은 엔지니어링 작업이지만, 모델러는 Revit 모델의 사이즈가 엔지니어의 설계와 일치하는지 반드시 확인해야 합니다. 모델 내 배관 사이즈가 잘못되면 자재 산출량, 간섭 체크, shop drawing 모두 오류가 발생합니다.

2. 핵심 개념
- 배관 사이즈는 인치 계열에서는 NPS(Nominal Pipe Size), 미터법에서는 DN(Diameter Nominal)으로 정의됩니다
  - DN50 = NPS 2인치 (약)
  - DN100 = NPS 4인치
  - DN150 = NPS 6인치
  - DN200 = NPS 8인치
- 배관 사이징은 다음 요소에 의해 결정됩니다:
  - 유량 (L/s 또는 GPM)
  - 속도 제한 (냉수 기준 일반적으로 1.5–2.5 m/s)
  - 압력 손실 제한 (일반적으로 100–300 Pa/m)
- 유량이 분기될수록 각 분기점에서 배관 사이즈는 작아집니다 — 라이저(riser)가 가장 크고, FCU 연결부가 가장 작습니다
- 벽 두께도 중요합니다: 배관 스케줄(Schedule 40, Schedule 80)은 외경에 영향을 주며, 간섭 체크 시 고려해야 합니다
- Revit에서: 배관 유형(재질 + 스케줄)과 사이즈는 유형 속성 및 인스턴스 속성에서 설정합니다

냉수 배관 사이즈 빠른 참조표

| 유량 (L/s) | 권장 DN |
|---|---|
| 0.1–0.5 | DN20–DN25 |
| 0.5–2.0 | DN32–DN50 |
| 2.0–6.0 | DN65–DN100 |
| 6.0–15.0 | DN125–DN150 |
| 15.0–35.0 | DN200–DN250 |

3. 실제 프로젝트 사례
엔지니어의 배관 스케줄에는 7층 메인 분기관이 DN100 CHWS 및 CHWR로 명시되어 있습니다. 주니어 모델러가 도면을 잘못 읽어 DN80으로 모델링했습니다. Revit에서 내보낸 배관 스케줄이 엔지니어의 Excel 스케줄과 일치하지 않아 문제가 발견되었고, 값비싼 현장 변경을 막을 수 있었습니다.

4. BIM 체크리스트
- 체크: Revit에서 배관 스케줄 내보내기(마크, 사이즈, 시스템, 유량). 엔지니어 스케줄과 행별로 비교
- 체크: 배관 유형(재질)이 시방서와 일치하는지 확인 — 탄소강, CPVC, 동관, 스테인리스강 등 시스템 유형별 확인
- 체크: 모든 분기점에서 배관 사이즈가 줄어드는지 확인 — 하류가 상류보다 더 큰 구간이 없어야 함

5. 흔한 실수
배관 유형을 업데이트하지 않고 모델에서 배관 사이즈만 변경하는 경우 — Revit이 사이즈 입력을 받아들이더라도 해당 배관 패밀리의 사이즈 테이블에 일치하는 값이 없으면 "pipe size not in type" 경고가 발생하며, 이를 무시하는 경우가 많습니다.

6. 오늘의 실천 과제
엔지니어의 배관 스케줄(또는 샘플)을 엽니다. 배관 하나를 선택합니다. 다음 공식을 사용하여 속도가 1.5–2.5 m/s 범위 내에 있는지 계산합니다: 속도 = 유량 (L/s) ÷ 배관 단면적 (m²). 선택한 DN이 적절한지 확인합니다.
