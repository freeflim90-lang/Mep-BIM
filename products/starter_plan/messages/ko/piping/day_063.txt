Day 63 - Revit에서의 차단밸브와 부속

1. 왜 중요한가
차단밸브와 배관 부속은 장식이 아닙니다 — 어느 구간을 유지보수를 위해 차단할 수 있는지를 결정합니다. 모델에서 차단밸브가 누락되면 현장에서도 밸브가 누락되고, 결국 FCU 한 대를 정비하려고 계통 전체를 정지시켜야 합니다.

2. 핵심 개념
- 주요 배관 부속과 기능:
  - 게이트밸브 / 볼밸브: 완전 차단(열림 또는 닫힘). 장비 접속부와 분기 인출부에 사용
  - 글로브밸브: 유량 조절(스로틀링). 정밀 유량 조정이 필요한 곳에 사용
  - 체크밸브: 한 방향으로만 흐름 허용. 펌프 토출, 바이패스 라인에 사용
  - 스트레이너(Y형): 유체의 이물질 제거. 펌프 흡입측과 제어밸브 전단에 항상 설치
  - 밸런싱밸브: 수동/자동 유량 평형. 각 FCU·AHU 코일 접속부에 사용
  - 감압밸브(PRV): 계통 구역에서 고압을 설정값으로 감압
  - 플렉시블 커넥터: 펌프·장비의 진동 절연
- Revit에서: 단순 배관 피팅이 아니라 배관 부속(pipe accessory) 패밀리를 사용합니다. 각 부속은 커넥터와 Mark 파라미터를 가진 고유 패밀리입니다

밸브 배치 기본 원칙

| 위치 | 밸브 유형 |
|---|---|
| 장비 입구/출구 | 차단(볼/게이트)밸브 |
| 입상관 분기 인출부 | 차단밸브 |
| 펌프 흡입 | 스트레이너 + 차단밸브 |
| 펌프 토출 | 차단밸브 + 체크밸브 |
| FCU 코일 접속 | 2방/3방 제어밸브 + 밸런싱밸브 |
| 구역 차단 | 각 층 분기 양측 볼밸브 |

3. 실무 관점
4층의 FCU 한 대는 다음을 요구합니다: CHWS: 볼밸브 → 2방 제어밸브 → 코일 → 볼밸브 → CHWR. 모델러는 이 부속들을 순서대로 배치하고, 각 밸브의 Mark를 설정하며(예: V-CHW-L4-FCU07-IN), 관경이 FCU 접속 사이즈(팬코일은 대표 예시로 DN20~DN25)와 일치하는지 확인합니다.

4. BIM 점검
- 점검: 모든 장비 접속점의 공급·환수 양측에 차단밸브가 모델링되어 있는가
- 점검: 모든 펌프의 흡입측에 스트레이너가 있는가
- 점검: 제어밸브(2방·3방)가 FCU 코일의 올바른 쪽에 배치되어 있는가(2방은 공급측)

5. 흔한 실수
너무 큰 밸브 패밀리를 배치하는 것 — DN25 분기에 DN50 볼밸브를 놓으면 Revit 커넥터 불일치 오류가 발생하고 도면에도 잘못 출력됩니다.

🇰🇷 국내 적용 포인트
- 밸브·스트레이너 점검공간(손잡이 조작·청소 인출)을 확보해야 하며, 이는 국내 기계설비 표준시방서/설계기준에서도 강조됩니다.
- 밸브 Mark·계통 약어(CHW 등)는 프로젝트 BIM 지침/도면 표기 규칙에 맞춰 통일하세요.

6. 오늘의 실습
Revit(또는 샘플 파일)에서 배관 분기에 볼밸브 1개를 배치하세요. 관경과 사이즈가 일치하는지 확인하고, Mark 파라미터를 설정한 뒤, 배관 부속 일람표에 나타나는지 확인하세요.
