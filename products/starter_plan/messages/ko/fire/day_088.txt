Day 88 - 소방 설비 조정 도면 패키지

1. 왜 중요한가
소방 설비 BIM 모델은 단순한 3D 파일이 아니라, 현장에서 시공사, 소방 당국, 현장 감독관이 사용할 수 있는 조정 도면 세트로 전달되어야 합니다. 적절한 도면 패키지가 없는 모델은 불완전한 인도물입니다. 모델 데이터는 존재하지만, 이를 필요로 하는 사람들이 실제로 활용할 수 없기 때문입니다.

2. 핵심 개념
소방 설비 조정 도면 패키지에는 일반적으로 다음과 같은 시트가 포함됩니다.

1. 소방 설비 평면도 (층별)
- 축척: 층의 복잡성에 따라 1:50 또는 1:100
- 표시: 모든 헤드의 Mark 번호, 모든 배관의 직경 라벨, 구역 밸브 위치, 호스릴 위치
- 주석: 외벽 경계에서 헤드 간격 치수 (벽 오프셋 준수 여부 확인)
- 평면도 위에 위험 구역 경계선 중첩 표시

2. 입상관 계통도 (Schematic)
- 소방펌프/시 상수도 연결 지점에서 최상층까지의 전체 배관 계층 구조 표시
- 라벨: 배관 직경, 구역 밸브 위치, 층별 연결부, 시험 및 배수 지점
- 축척 불필요 — 다이어그램 형식이 표준

3. 관통부 / 슬리브 일람표
- 모든 내화 관통부를 나열한 테이블 형식: 위치 (Grid 기준, 층), 배관 직경, 슬리브 ID, 관통 부재의 내화 등급, 방화구획 충전재 사양 참조

4. 스프링클러 헤드 일람표
- Revit에서 생성: Mark, 층, 구역, 헤드 유형, 커버리지 면적, 작동 온도, 시스템 명칭
- 구역별 물량 산출 — 자재 조달에 사용

5. 조정 간섭 검토 보고서
- 해결된 모든 간섭 요약 — 분류 (Day 76 프레임워크 기준 Type 1 / Type 2)
- 표시: 간섭 설명, 해결 조치, 승인 상태 (엔지니어/시공사 서명 완료)

6. 커버리지 검증 계획도
- 평면도 위에 커버리지 원 (Day 81 방법론 기준) 중첩 표시
- 확인: 사각지대 없음. 초기 배치 후 추가된 보조 헤드는 강조 표시

도면 명명 규칙:
[프로젝트 코드]-FP-[유형]-[층]-[개정]
예시: LUA-FP-FPL-L03-C02 = 소방 설비 평면도, Level 3, 개정 C, 발행 02

3. 실제 프로젝트 사례
소방 설비 하도급업체가 관통부 일람표 없이 모델을 인도받았습니다. 현장에서 설치업체는 1차 타설 전에 어떤 벽에 어떤 슬리브가 필요한지 알 수 없었습니다. 작업이 지연되었습니다. 이 사건 이후, 프로젝트 팀은 규칙을 수립했습니다: 완전한 관통부 일람표와 입상관 계통도가 포함되지 않은 모델은 시공용으로 접수하지 않는다.

4. BIM 체크리스트
- 체크: 평면도에 헤드 Mark 번호와 배관 직경 라벨이 표시되어 있는가 — 단순한 형상만 있는 것은 아님
- 체크: 입상관 계통도가 포함되어 있으며 현재 모델 상태를 반영하여 업데이트되었는가
- 체크: 관통부 일람표에 모든 내화 관통부의 슬리브 ID, 배관 직경, 방화구획 충전재 사양 참조가 나열되어 있는가

5. 흔한 실수
소방 설비 모델을 2D 도면 패키지 없이 Revit NWD/IFC 파일로만 발행하는 경우. 정보가 필요한 모든 사람이 모델 파일을 열 수 있는 것은 아닙니다. 현장 감독관, 소방 당국 검사관, FM(Facility Management) 팀은 도면이 필요합니다. 모델 파일만으로는 인도 요구 사항을 충족할 수 없습니다.

6. 오늘의 실천 과제
현재 진행 중인 프로젝트 또는 샘플 프로젝트의 소방 설비 도면 패키지에 포함되어야 할 시트들을 목록으로 작성하세요. 어떤 시트가 이미 존재하고 어떤 시트가 누락되었는지 확인하세요. 관통부 일람표가 누락된 경우, 필요한 열이 포함된 템플릿 테이블을 초안으로 작성하세요.
