Day 64 - 스탠드파이프 및 호스릴 위치 선정

1. Why This Matters
스탠드파이프 시스템과 호스릴은 수동 소화 능력을 제공합니다. 이들의 위치는 도달 거리에 의해 결정됩니다. 호스릴이 건물 내 어느 지점에서든 너무 멀리 떨어져 있으면, 설치된 장비로 수동 소화가 불가능합니다. BIM 모델은 법규를 준수하는 배치를 반영해야 합니다.

2. Core Concept
- 스탠드파이프: 소방대 호스용 연결구가 있는 고정 배관 시스템. 건물 내에서 수직으로 상승. 일반적으로 DN100~DN150
  - Class I: 소방대 전용 (계단실 내 2.5" 연결구)
  - Class II: 건물 재실자용 (1.5" 호스릴)
  - Class III: 두 용도 모두 포함
- 호스릴: 초기 화재 시 재실자 사용을 위한 가요성 호스(직경 25mm 또는 19mm, 길이 30m)가 포함된 캐비닛
- 커버리지 규칙: 건물 내 모든 지점은 호스릴로부터 30m 호스 도달 거리 이내에 있어야 함 (직선 거리가 아닌 벽과 복도를 고려한 경로 거리)
- 배치: 일반적으로 계단실 및 비상구 근처 복도에 위치, 최대 30m 간격
- Revit에서: 호스릴 캐비닛을 장비 패밀리(벽걸이형 또는 매립형)로 모델링. 격리 밸브를 사용하여 가정용/소화용 급수관에 연결
- 스탠드파이프 연결구 높이: 완성된 바닥 마감면 위 750mm~1000mm

호스릴 배치 체크리스트

| 요구사항 | 값 |
|---|---|
| 최대 호스 길이 | 30m (가요성 호스) |
| 커버리지 반경 | 캐비닛으로부터 30m (경로 거리, 직선 거리 아님) |
| 최대 간격 | 호스릴 간 30~35m |
| 연결구 높이 | AFF 750~1000mm |
| 공급 배관 크기 | DN25 (호스릴) / DN100 (스탠드파이프) |

3. Real Project Lens
양쪽 끝에 비상계단이 있는 60m 길이의 복도에는 각 끝에서 20m 지점에 두 개의 호스릴 캐비닛이 설치되어 있습니다. BIM 모델에서 경로 커버리지 확인을 수행한 결과, 복도 내 모든 사무실 출입구가 최소 하나의 호스릴로부터 30m 경로 거리 이내에 있음이 확인되었습니다.

4. BIM Check
- 확인: 호스릴을 단순 심볼이 아닌 배치된 패밀리로 모델링할 것. 벽 매립 깊이가 벽체 두께 이내인지 확인
- 확인: 스탠드파이프 연결구 높이를 AFF 750~1000mm로 모델링할 것 — 바닥 레벨이나 천장 레벨에 배치하지 않음
- 확인: 각 호스릴에 대해, 해당 호스릴이 담당해야 하는 최원점까지 30m 경로(측정 도구 사용)를 추적하여 도달 가능 여부 확인

5. Common Mistake
호스릴 커버리지를 직선 거리로 측정하는 것 — 직선 거리로 30m를 커버하는 호스릴이 L자형 복도에서는 호스가 모퉁이를 돌아야 하므로 실제로는 18m만 커버할 수 있습니다.

6. Today's Action
평면도(스케치 또는 실제 도면)에서 복도에 호스릴 하나를 배치하십시오. 실제 복도 경로(모퉁이 및 출입문 통과)를 따라 30m 경로 커버리지를 추적하십시오. 커버되지 않은 영역을 식별하십시오. 필요한 경우 두 번째 호스릴을 추가하십시오.
