Day 76 - 헤드 vs. 덕트 및 케이블 트레이 간섭

1. Why This Matters
스프링클러 헤드와 덕트 및 케이블 트레이 간의 간섭은 천장 코디네이션에서 가장 흔히 발생하는 소방 설비 간섭 유형입니다. 덕트가 경로를 막아 설계 위치에 헤드를 설치할 수 없는 경우, 현장에서 헤드 위치가 변경됩니다. 이는 종종 부적절한 방식으로 이루어집니다. 시공 전에 BIM에서 이러한 간섭을 해결하는 것이 소방 설비 BIM 코디네이션의 핵심 가치입니다.

2. Core Concept
헤드와 덕트/트레이 간 간섭은 두 가지 유형으로 발생합니다.

Type 1 — 천장 평면 위 덕트에 의해 헤드가 차단된 경우
- 급기 또는 환기 덕트가 의도된 헤드 위치 상부를 지나감
- 헤드가 천장 면에 도달할 수 없음 — 드롭(스프리그)이 차단됨
- 해결 방법: 덕트 분기관 경로를 변경하거나, 헤드를 측면의 빈 공간으로 재배치
- 덕트를 피하기 위해 드롭 길이를 절대 줄이지 말 것 — 헤드는 올바른 천장 이격 거리를 유지해야 함

Type 2 — 헤드 아래에 덕트가 있는 경우 (스프레이 음영)
- 덕트가 헤드 아래에 있음 — 이는 장애물 음영 구역을 생성함 (Day 69 규칙 참조)
- 덕트가 스프레이 콘(살수 원뿔)이 그 뒤쪽 바닥 면적에 도달하는 것을 차단함
- 해결 방법: 헤드를 이동하거나 덕트 아래에 보조 헤드를 추가

케이블 트레이 간섭
- 케이블 트레이는 일반적으로 덕트보다 얇지만(깊이 50–150 mm), 폭은 넓음(300–600 mm)
- 헤드 위에 있는 넓은 케이블 트레이는 장애물로 작용함 — 3배 음영 규칙 확인 필요
- 드롭 구역을 통과하는 트레이는 스프리그 설치를 차단함
- 해결 방법: 드롭 구역 주변으로 트레이 경로를 변경하거나, 드롭 위치에 컷아웃/브릿지를 제공

BIM에서의 코디네이션 접근법
- Navisworks 또는 Revit 간섭 검토 실행: 분야 필터 = 소방 설비 vs. 기계 + 전기
- 간섭 허용 오차를 0mm가 아닌 50mm로 설정하여 현장 문제를 유발할 근접 간섭을 포착
- 각 간섭에 대해: Type 1(드롭 차단) 또는 Type 2(스프레이 음영)로 분류
- Type 1은 항상 중요함. Type 2는 보조 헤드 필요 여부에 대한 엔지니어링 판단이 필요함.

3. Real Project Lens
한 병원 프로젝트의 천장 코디네이션 회의에서 4층에 소방 설비 드롭과 기계 덕트 간 47건의 간섭이 발견되었습니다. 소방 설비 모델러는 31건을 Type 1(드롭 차단), 16건을 Type 2(스프레이 음영)로 분류했습니다. Type 1 간섭은 헤드 재배치를 통해 재코디네이션되었습니다. 16건의 Type 2 간섭은 소방 엔지니어에게 판단이 요청되었으며, 그중 9건은 보조 헤드가 필요했고, 7건은 덕트 크기를 고려할 때 허용 가능한 것으로 판정되었습니다.

4. BIM Check
- Check: 소방 설비 드롭과 모든 기계 덕트 간 간섭 검토가 실행되었으며, 미해결된 Type 1(드롭 차단) 간섭이 없음
- Check: 스프링클러 헤드 아래에 있는 넓은 덕트(> 300 mm)에 대해 스프레이 음영이 평가되었으며, 결과가 문서화됨
- Check: 헤드 커버리지 구역 내 폭 300mm 초과 케이블 트레이에 대해 장애물 영향이 확인됨

5. Common Mistake
헤드와 덕트 간 간섭을 해결하기 위해 단순히 드롭 파이프 길이를 줄여 간섭을 피하는 것. 이는 헤드를 허용 가능한 천장 이격 거리 범위 이상으로 올리게 되어 — 규정을 준수하지 않는 설치를 초래합니다. 드롭 길이는 코디네이션의 편의가 아닌, 헤드-천장 간 규칙에 의해 결정됩니다.

6. Today's Action
Navisworks 또는 Revit에서 한 개 층을 대상으로 소방 설비와 기계/전기 간 간섭 검토 테스트를 실행하십시오. 총 간섭 건수를 집계하고, 그중 최소 3건을 Type 1(드롭 차단) 또는 Type 2(스프레이 음영)로 분류하십시오.
