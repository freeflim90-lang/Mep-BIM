Day 75 - 3주차 복습 — 배관 라우팅 요약

1. 왜 중요한가
소화 배관 라우팅은 모델러에게 가장 직접적인 책임이 주어지면서도 가장 흔한 오류가 발생하는 분야입니다. 모델 내에서 계층 구조, 사이즈, 배치 구성 및 수리(유체역학) 규칙을 올바르게 적용하는 것은 시스템이 설계 압력으로 물을 제대로 공급하는지, 아니면 실패하는지를 가르는 핵심 요소입니다. 3주차는 간섭 검토 패턴으로 넘어가기 전에 이러한 규칙들을 통합 정리합니다.

2. 핵심 개념
3주차 소화 — 배관 라우팅 요약

| Day | 주제 | 핵심 규칙 |
|---|---|---|
| 71 | 크로스 메인 및 브랜치 사이징 | 계층 구조가 내려갈수록 관경이 축소됩니다. 항상 엔지니어의 사이징 스케줄을 따라야 합니다 |
| 72 | 스탠드파이프 및 호스릴 | 30m 호스 커버리지 = 이동 경로 거리입니다. 직선 거리가 아닌 복도를 따라 측정하세요 |
| 73 | 수리 계산 기초 | 길이나 피팅 개수가 변경되는 배관 재라우팅은 반드시 소방 엔지니어에게 보고되어야 합니다 |
| 74 | 트리 / 루프 / 그리드 시스템 | 올바른 구성을 확인하세요. 루프 시스템을 절대 개방 상태로 두지 마세요. 그리드 메인은 먼저 구조 부재와 조정하세요 |

핵심 규칙 — 배관 라우팅
- 엔지니어의 승인 없이 관경을 절대 변경하지 마세요
- 루프 시스템을 절대 개방 상태로 두지 마세요 — 양쪽 끝이 모두 연결되어야 합니다
- 호스릴은 이동 경로 거리, 직선 거리가 아닙니다 — 항상 이동 경로를 측정하세요
- 배관이나 피팅이 추가되는 모든 재라우팅은 소방 엔지니어에게 반드시 통보되어야 합니다
- 크로스 메인은 최우선 라우팅입니다 — HVAC 및 전기 메인보다 먼저 조정하세요
- 펜던트 헤드 드롭: 항상 사이징 스케줄에 따라 정확한 관경으로 모델링하세요

통합 실행 체크리스트 (Day 61–75)
1. 구역별 올바른 시스템 유형 (습식 / 건식 / 준비작동식)
2. 구역별 위험 등급
3. 헤드당 최대 커버리지 면적 확인
4. 헤드 간 최소/최대 간격 확인
5. 벽 이격 거리 ≤ 헤드 최대 간격의 절반
6. 3D 장애물 확인 (보, 덕트, 트레이)
7. 헤드-천장 거리가 허용 범위 내에 있는지 확인
8. 배관 사이즈가 엔지니어의 사이징 스케줄과 일치하는지 확인
9. 호스릴 커버리지가 이동 경로 거리로 확인되었는지 확인
10. 배관 재라우팅 사항이 소방 엔지니어에게 보고되었는지 확인
11. 시스템 구성 (트리/루프/그리드)이 올바르게 모델링되었는지 확인

3. 실제 프로젝트 사례
한 선임 BIM 코디네이터는 다중 임대 오피스 빌딩의 Level 3 소화 모델을 발행하기 전에 위의 11개 항목 체크리스트를 구조화된 모델 QA 패스로 사용합니다. 두 가지 문제가 발견됩니다: 엔지니어가 DN40으로 지정한 브랜치 라인이 DN50으로 모델링된 것과, 호스릴이 최원점까지의 이동 경로 거리로 33m가 측정된 것입니다. 두 문제 모두 발행일 이전에 수정 및 보고되었습니다.

4. BIM 체크
- 체크: 현재 층 또는 구역에 대해 실행 체크리스트의 11개 항목이 모두 검토되었는지 확인
- 체크: 배관 사이징 스케줄이 모델 발행 패키지에 첨부되어 검토자가 스케줄과 사이즈를 대조 확인할 수 있는지 확인
- 체크: 시스템 구성(트리, 루프 또는 그리드)이 올바르게 반영되었는지 확인 — 데드 루프나 누락된 연결이 없어야 함

5. 흔한 실수
문서화된 QA 체크 없이 소화 모델을 간섭 검토용으로 제출하는 경우. 검토자들은 15분짜리 자체 QA 패스에서 발견되었을 관경 오류와 개방 루프 연결을 발견하게 되어, 피할 수 있었던 RFI로 모두의 시간을 낭비하게 됩니다.

6. 오늘의 실천 과제
현재 작업 중이거나 샘플 소화 모델의 한 개 층에 대해 11개 항목 실행 체크리스트를 적용하세요. 통과한 항목, 수정이 필요한 항목, 그리고 소방 엔지니어에게 질문이 필요한 항목을 기록하세요.
