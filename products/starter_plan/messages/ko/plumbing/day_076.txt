Day 76 - 변기 및 WC 배관 연결

1. Why This Matters
변기 및 WC 배관 연결은 건물에서 가장 많은 양의 오수(soil discharge)를 발생시킵니다. BIM 모델에서 배관 사이즈, 구배, 연결 경로를 올바르게 설정하는 것은 배수 시스템이 막힘 없이 작동하는지에 직접적인 영향을 미칩니다. 또한 오수 수직관(soil stack)의 위치를 결정하는 핵심 요소이므로, WC 연결 배치 계획은 화장실 코어를 설계할 때 가장 먼저 모델링되는 작업 중 하나입니다.

2. Core Concept
WC 변기는 오수 배관(soil pipe)으로 배출되며, 이 배관은 오수 수직관 또는 오수 지관(branch soil pipe)에 연결됩니다.

일반적인 WC 연결 요구 사항:
- 최소 배관 사이즈: DN100 (100 mm 공칭 내경)
- WC 배출구에서 오수 수직관까지의 최대 수평 거리: 6m (일부 기준에서는 특정 조건 하에 최대 9m 허용)
- 수평 지관 구배: 1:40 ~ 1:80 (고형물 함량으로 인해 일반 오수 배관보다 더 급한 구배 적용)
- 단일 무환기(unvented) 100mm 지관에는 최대 2개의 WC 변기만 연결 가능
- 수직관 연결 시 90도 티(Tee)보다 45도 분기 연결이 난류 감소에 유리

BIM 모델링 단계:
1. 건축 평면도에 표시된 기구 위치에 WC 변기 패밀리 배치
2. 각 WC 배출구에서 수평 지관까지 DN100 오수 배관 연결
3. 수평 지관 구배를 1:40 또는 프로젝트에서 지정된 기울기로 설정
4. 45도 와이(Wye) 또는 조합 피팅(combination fitting)을 사용하여 지관을 오수 수직관으로 연결
5. 수직관 연결 지점의 인버트 레벨(invert level)이 상류 측 모든 지관 인버트보다 낮은지 확인
6. 배관 시스템을 위생 — 오수(Sanitary — Soil)로 지정

추가 고려 사항:
- P-트랩(P-trap) 배출구가 있는 WC 변기는 별도의 P-트랩 피팅 없이 수평 오수 배관에 직접 연결
- 두 개의 WC 변기가 백투백(back-to-back)으로 배치된 경우 공통 지관 사용 가능, 단 지관 직경은 반드시 확인 필요

3. Real Project Lens
한 상업용 오피스 인테리어 공사에서 화장실 코어에 4개의 WC 변기가 모두 단일 DN100 수평 지관에 연결되었습니다. 무환기 DN100 지관의 최대 허용 기구 수는 2개였습니다. 시스템 검토관이 하나의 지관에 4개의 기구가 연결된 것을 확인하면서 BIM 모델에서 이 문제가 드러났습니다. 설계는 수정되어 WC를 두 개의 개별 지관으로 분할하고, 각 지관을 서로 다른 높이에서 오수 수직관에 연결했습니다.

4. BIM Check
- Check: 모든 WC 배관 연결이 DN100 이상으로 모델링되었는지 확인
- Check: 모든 WC 지관의 수평 오수 지관 구배가 최소 1:40 이상으로 설정되었는지 확인
- Check: 무환기 지관당 WC 변기 개수가 프로젝트 또는 기준 한도를 초과하지 않는지 확인

5. Common Mistake
흔한 실수는 WC 지관을 반대편에 있는 다른 지관과 정확히 동일한 인버트 레벨에서 오수 수직관에 연결하는 것입니다. 이로 인해 교차 흐름(cross-flow) 현상이 발생하여 한 지관의 배출수가 수직관을 따라 내려가지 않고 반대편 지관으로 유입될 수 있습니다. 지관은 서로 다른 높이(최소 200mm 수직 이격)에서 연결되어야 합니다.

6. Today's Action
Revit에서 작은 화장실 공간에 두 개의 WC 변기를 모델링하고, DN100 수평 지관으로 오수 수직관에 연결하세요. 구배를 1:40으로 설정합니다. 각 WC 배출구에서 수직관까지의 수평 거리를 측정합니다. 두 연결부가 모두 최대 6m 이내에 있는지 확인합니다. 수직관 접합부의 인버트 레벨을 문서화합니다.
