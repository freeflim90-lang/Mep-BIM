LUA BIM LABS Starter
Day 79 - 위생기구 일람표 (Sanitary Fixture Schedule)

1. Why This Matters
기구 일람표는 단순한 행정 문서가 아닙니다. 이는 품질 관리 도구이자, 자재 조달 참고 자료이며, 인계 기록으로서의 역할을 동시에 수행합니다. MEP BIM에서 기구 일람표는 모델로부터 생성되며, 그 정확성은 전적으로 Revit 패밀리와 매개변수가 얼마나 잘 설정되었는지에 달려 있습니다. 올바른 기구 일람표를 작성하려면 형상(geometry)뿐만 아니라 모델 데이터를 점검해야 합니다.

2. Core Concept
위생기구 일람표는 프로젝트 내 모든 배관 설비 기구를 주요 매개변수와 함께 나열한 표입니다. Revit에서 기구 일람표는 모델 데이터베이스를 조회하는 뷰(view)로, 의도된 사항이 아닌 모델에 실제로 존재하는 내용을 반영합니다.

위생기구 일람표에 포함해야 할 매개변수:

| 매개변수 | 기록 내용 | 출처 |
|-----------|----------------|--------|
| 표시(Mark) / ID | 기구 고유 태그 | Revit 패밀리 인스턴스 |
| 기구 유형(Fixture Type) | 변기, 세면기, 소변기, 싱크대, 바닥 배수구 | 패밀리 이름 |
| 레벨(Level) | 층 위치 | 레벨 매개변수 |
| 실 이름(Room Name) | 기구가 위치한 실 | Revit 룸 태그 |
| 시스템 이름(System Name) | 오수, 폐수, 통기 | 배관 시스템 지정 |
| 배관 크기(출구) | 토출관 DN | 배관 매개변수 |
| 기구 단위 수(Fixture Unit Count) | 배수 설계용 배수 부하 단위 값 | 공유 매개변수 또는 패밀리 매개변수 |
| 트랩 유형(Trap Type) | P-트랩, 딥씰, AAV | 패밀리 공유 매개변수 |
| 비고(Remarks) | 접근성, 특수 기구 여부 | 수동 입력 |

일람표 작성 방법:
1. 모든 기구 패밀리에 필요한 공유 매개변수가 로드되어 있는지 확인
2. 각 패밀리 배치 시 기구별 매개변수 입력
3. 카테고리: 배관 설비(Plumbing Fixtures)로 필터링된 Revit 일람표 뷰 생성
4. 필요한 필드를 올바른 순서로 추가
5. 레벨(Level)별, 이어서 실 이름(Room Name)별로 정렬 적용
6. 일람표를 검토하여 누락되거나 빈 매개변수 값(모델 내 데이터 결손) 식별

3. Real Project Lens
한 병원 프로젝트에서 FM 인계 시 모든 340개 기구에 대해 실 이름, 기구 단위 수, 트랩 유형이 포함된 기구 일람표가 요구되었습니다. BIM 코디네이터는 프로젝트에 사용된 Revit 패밀리에 기구 단위 값이 전혀 추가되지 않았다는 사실을 발견했습니다. 해당 값은 설계 시방서에는 있었지만 BIM 모델에는 없었던 것입니다. 이를 사후에 추가하려면 각 패밀리를 열고 공유 매개변수를 추가한 후 패밀리를 다시 배치해야 했습니다. 이 작업에 3일이 소요되었으며, 모델링 시작 시 일람표 요구사항을 확인했다면 충분히 방지할 수 있었던 문제였습니다.

4. BIM Check
- 확인: 모든 배관 설비 패밀리에 기구 단위 수(Fixture Unit Count), 트랩 유형(Trap Type), 시스템 이름(System Name) 매개변수가 로드되어 있고 값이 입력되었는가
- 확인: Revit 일람표에 레벨(Level), 실 이름(Room Name), 배관 크기(출구) 항목에 빈 셀이 없는가
- 확인: 일람표의 기구 개수가 건축 기구 배치 도면의 개수와 일치하는가

5. Common Mistake
흔한 실수는 프로젝트 마감 단계에서 모델에 존재하는 데이터만으로 기구 일람표를 만드는 것입니다. 그러면 빈 매개변수 필드가 임시 텍스트로 채워지거나 그대로 비워둡니다. 이렇게 작성된 일람표는 완성된 것처럼 보이지만 부정확한 데이터를 포함하게 되며, 이는 FM 인계나 시운전(commissioning) 시 문제가 됩니다.

6. Today's Action
Revit에서 레벨(Level), 기구 유형(Fixture Type), 시스템 이름(System Name), 배관 크기(Pipe Size), 기구 단위 수(Fixture Unit Count)를 포함하여 최소 6개 이상의 매개변수 열로 구성된 배관 설비 일람표(Plumbing Fixture Schedule)를 작성하세요. 결과물에서 빈 셀이 있는지 검토하세요. 빈 셀이 발견되면 해당 패밀리나 배관 시스템으로 돌아가 누락된 값을 추가하세요. 일람표를 다시 실행하여 모든 값이 완전히 입력되었는지 확인하세요.
