LUA BIM LABS Starter
Day 67 - 1주차 배관 리뷰

1. 중요성
배관 BIM은 시스템 분류, 구배 형상, 슬리브 조정, 설비기구 연결성, 공간 간섭 관리에 정밀함을 요구합니다. 1주차 리뷰를 통해 이러한 기본기를 확실히 다져야 더 복잡한 배관 작업으로 넘어갈 수 있습니다.

2. 핵심 개념
1주차 배관 요약

| 일차 | 주제 | 핵심 포인트 |
|---|---|---|
| 61 | 시스템 분류 | 오수, 통기, 냉수, 온수 — 4개의 개별 시스템, 절대 혼용 금지 |
| 62 | 중력 배수 및 구배 | DN100 = 최소 구배 1:80. Revit에서 구배 설정. 총 낙차가 슬래브 두께 내에 들어가는지 확인 |
| 63 | 라이저 및 슬리브 계획 | 슬리브 = 배관 외경 + 여유. 슬래브 타설 전 슬리브 일정 제출 |
| 64 | 설비기구 연결 | 각 설비기구마다 지정된 오수 배관 크기와 급수 배관 크기가 있음. 미연결 커넥터 금지 |
| 65 | 급수 분배 | 각 분기점에서 배관 크기 축소. 3리터 초과 데드레그(Dead Leg) 금지 |
| 66 | 오수 간섭 | 긴 관로는 슬래브 아래로 낙차 발생. 스택 코어에는 단열재 외경 포함 필수 |

배관 BIM 핵심 규칙
- 오수와 통기는 별도 시스템 — Revit에서 병합 금지
- 모든 수평 오수 배관에는 구배가 있어야 함 — 수평(Flat) 절대 금지
- 슬리브 요청은 정확한 좌표와 함께 슬래브 타설 전 제출 필수
- 모든 설비기구에 P-트랩 모델링 — 트랩 누락 시 배관 검사 불합격
- 온수 데드레그(Dead Leg)는 법적 한계 이내 유지 — 레지오넬라균 위험

3. 실제 프로젝트 적용
한 배관 BIM 모델러가 호텔 프로젝트에서 새 층 작업 첫날에 투입되었습니다. 1주차 기술을 활용하여: 모든 배관에 시스템 유형 지정, 모든 분기관에 구배 설정, 슬리브 일정 제출, 모든 설비기구 연결, 데드레그 길이 확인, 오수 간섭 테스트 수행 — 이후 모델을 조정팀에 인계합니다. 이러한 체계적인 접근 방식은 가장 흔한 배관 재작업 상황을 방지합니다.

4. BIM 체크리스트
- 확인: 시스템 브라우저 — "미지정(Unassigned)" 상태의 배관이 없는지 확인
- 확인: 모든 오수 분기관에 구배 매개변수가 설정되어 있고 구배 방향이 올바른지 확인
- 확인: 슬리브 일정이 작성되어 구조 제출 준비가 완료되었는지 확인

5. 흔한 실수
배관 BIM이 HVAC나 배관(파이핑)보다 단순하다고 생각하여 배관 크기가 작다고 판단 — 그 결과 오수 시스템에 특화된 구배 및 여유 공간 확인을 누락하여 조정 리뷰에서 탈락하는 경우.

6. 오늘의 실천 과제
배관 모델을 열고(또는 간단한 테스트 모델을 생성), 다음을 확인하세요: (1) 시스템 지정, (2) 오수 분기관 구배, (3) 설비기구 연결. 모든 배관 모델 리뷰 시작 시 사용할 개인 체크리스트를 간단히 작성해 보세요.
