Day 70 - Week 2 Review — 경사 모델링 요약

1. Why This Matters
배관 심화 학습의 첫 10일 동안, 모든 MEP BIM 코디네이터에게 필요한 기술적 기반을 다졌습니다: 위생 설비 시스템이 어떻게 분류되는지, 어떤 경사 규칙이 적용되는지, 그리고 Revit에서 경사를 올바르게 모델링하는 방법입니다. 이번 복습은 실제 프로젝트에서 필요할 때 바로 활용할 수 있도록 해당 내용을 체계적으로 정리합니다.

2. Core Concept
Day 61~70 요약:

| Day | 주제 | 핵심 규칙 |
|-----|------|-----------|
| 61 | 위생 시스템 개요 | 오수(Soil) / 폐수(Waste) / 통기(Vent) / 우수(Stormwater)는 별도 시스템 |
| 62 | 오수 시스템 | 변기 배출 이송; 최소 DN100; 오수 수직관(soil stack) 연결 |
| 63 | 폐수 시스템 | 세면대/싱크대 배출 이송; 모든 기구에 P-트랩 필수 |
| 64 | 통기 시스템 | 기압 유지; 트랩 봉수 파손 방지 |
| 65 | 합류식 vs 분류식 | 프로젝트가 단일 배수 수직관 또는 이중 배수 수직관 사용 여부 확인 |

| 70 | 주간 복습 | 본 강의 |

이번 주에 확립된 핵심 모델링 습관:
- 모든 수평 배수관에는 경사값이 설정되어야 함
- 모든 배관에는 시스템 분류(오수, 폐수, 통기)가 적용되어야 함
- 모든 분기 접합부에서 관저고를 반드시 확인해야 함

3. Real Project Lens
주거용 타워의 간섭 검토(coordination review) 중, BIM 매니저가 Revit 필터를 실행하여 경사(Slope) = 0인 모든 배수관을 강조 표시했습니다. 40개가 넘는 배관이 표시되었으며, 대부분 변기와 오수 수직관을 연결하는 짧은 분기관이었습니다. 간섭 검토 도면을 발행하기 전에 이 모든 배관을 수정해야 했고, 수정 작업에 반나절이 소요되었습니다. 주간 자체 점검 습관이 있었다면 모델링한 당일에 해당 오류를 발견할 수 있었을 것입니다.

4. BIM Check
- Check: 모델 내 모든 수평 배수관의 경사값이 0보다 큰지 확인
- Check: 모든 배관 세그먼트에 시스템 파라미터(오수, 폐수, 통기)가 할당되었는지 확인 — 미분류 배관 없음
- Check: 분기 접합부의 관저고가 문서화되어 있으며, 일관된 하향 흐름을 보이는지 확인

5. Common Mistake
흔한 실수는 모델링 단계에서 시스템 분류를 선택 사항으로 간주하고 나중에 추가하려는 것입니다. 실제로는 간섭 검토 이전에 "나중"이 오는 경우가 거의 없습니다. 분류되지 않은 배관은 일정표(schedule) 수치 오류, 간섭 검토 필터 누락, 그리고 팀 전체의 작업 속도를 늦추는 간섭 문제를 초래합니다.

6. Today's Action
배관 모델에서 Revit 일정표 또는 필터를 실행하여 다음 세 가지를 확인하세요: (1) 모든 수평 배수관의 경사 > 0, (2) 모든 배관에 시스템 분류가 할당됨, (3) 인접한 두 배관 세그먼트의 관저고가 동일하지 않음. 발견된 문제점을 문서화하세요.
