Day 87 - 배관 파라미터 체크리스트

1. 왜 중요한가
배관 BIM 모델의 가치는 그 모델이 포함하는 데이터에 달려 있습니다. 형상(Geometry)만 있고 데이터가 없으면 단순한 시각화에 불과합니다. 그러나 완전하고 정확한 데이터를 갖춘 형상은 시공 및 설비 관리 자산이 됩니다. 파라미터 체크리스트는 '보기에는 맞는 모델'과 '실제로도 맞는 모델' 사이의 간극을 메우는 도구이며, 검토자는 작업물을 발행하기 전에 감사(Audit)할 때 이 체크리스트를 사용합니다.

2. 핵심 개념
배관 파라미터 체크리스트는 배관 BIM 모델의 모든 모델 요소에 대해 요구되는 모든 데이터 필드가 입력되었는지 확인합니다.

배관(배수관)의 핵심 파라미터:

| 파라미터 | 요구 값 | 확인 주체 |
|-----------|---------------|------------|
| 시스템 이름 | 프로젝트별 시스템 이름 (예: "오수 스택 - A구역") | BIM 코디네이터 |
| 시스템 유형 | 오수, 급탕, 급수 | Revit 시스템 |
| 호칭 지름 | 설계 의도에 부합하는 DN 값 | BIM 코디네이터 |
| 구배 | 모든 수평 배수관에 대해 0보다 큰 값 | 일정표 필터 |
| 재질 | PVC-U, 주철, CPVC, 동관 등 | 공유 파라미터 |
| 단열 두께 | 시방서 기준 (예: 냉수 25mm) | 공유 파라미터 |
| 압력 등급 | 해당 시 PN 값 | 공유 파라미터 |
| 내화 등급 | 각 슬래브/벽 관통부에 대해 내화 등급 있음/없음 | 슬리브 패밀리 |

위생기구의 핵심 파라미터:

| 파라미터 | 요구 값 | 확인 주체 |
|-----------|---------------|------------|
| 표시(Mark) | 고유 기구 태그 | BIM 코디네이터 |
| 기구 단위 수 | 설계 시방서 기준 DFU 값 | 공유 파라미터 |
| 트랩 유형 | P-트랩, 딥씰, AAV, 병 트랩 | 패밀리 파라미터 |
| 시스템 이름 | 연결된 시스템 이름 | Revit 시스템 |
| 실 이름 | 건축 실 | 실 태그 |

밸브의 핵심 파라미터:

| 파라미터 | 요구 값 | 확인 주체 |
|-----------|---------------|------------|
| 밸브 유형 | 게이트, 볼, 체크, PRV 등 | 패밀리 이름 |
| 용도 | 온수, 냉수, 오수 등 | 공유 파라미터 |
| 접근 필요 여부 | 예 / 아니오 | 공유 파라미터 |
| 태그 | P&ID와 일치하는 밸브 태그 | 표시(Mark) 파라미터 |

체크리스트 실행 방법:
1. 배관, 기구, 밸브에 대한 Revit 일정표 생성 — 요소 유형별로 하나씩
2. 주요 필드(구배, 시스템 이름, 재질, 기구 단위 수)에서 빈 셀 필터링
3. Excel로 내보낸 후 구역별 빈 셀 개수로 정렬
4. 구역별로 모델러에게 수정 작업 배정
5. 수정 완료 후 일정표를 다시 실행하여 완료 확인

3. 실제 프로젝트 사례
대규모 상업용 개발 프로젝트에서 BIM 매니저는 매주 파라미터 체크리스트를 내보내는 방식을 도입했습니다. 매주 월요일 아침 일정표를 내보내고, 두 개 이상의 파라미터가 비어 있는 요소는 수요일까지 수정하도록 지정했습니다. 6주 후, 지적된 요소 수는 480개에서 12개로 감소했습니다. 해당 분야 BIM 리드는 이 루틴을 프로젝트에서 가장 효과적인 품질 도구라고 평가했습니다.

4. BIM 점검
- 점검: 구배 = 0으로 필터링한 배관 일정표 결과가 0건 (모든 배수관에 구배가 설정됨)
- 점검: 기구 단위 수 = 공백으로 필터링한 기구 일정표 결과가 0건
- 점검: 모든 밸브에 태그/표시 값이 표시된 밸브 일정표 — 모델 내 태그 없는 밸브 없음

5. 흔한 실수
흔한 실수는 프로젝트 막바지, 조정도면(Coordination Drawing) 발행 직전에만 체크리스트를 실행하는 것입니다. 그 시점에는 수백 개의 요소에 빈 파라미터가 있을 수 있으며, 마감 압박 속에서 이를 수정하다 보면 오류가 발생합니다. 상세 모델링 시작 시점부터 매주 체크리스트를 실행하십시오.

6. 오늘의 실천 과제
Revit에서 네 개의 열(시스템 이름, 호칭 지름, 구배, 재질)로 구성된 배관 일정표를 만드십시오. 구배가 공백이거나 0인 배관만 표시하도록 필터링하십시오. 결과 개수를 세어 보십시오. 이 숫자가 현재 배수관 파라미터 품질 격차입니다. 다음 주까지 0으로 만드는 것을 목표로 설정하십시오.
