LUA BIM LABS Starter
Day 83 - 회로-부하 추적 — 모델 내 차단기에서 설비까지

1. 왜 중요한가
회로-부하 추적은 패널 스케줄의 차단기부터 BIM 모델에서 해당 회로가 공급하는 실제 설비까지 전기 회로를 따라가는 프로세스입니다. 이 검증을 통해 모델이 실제 회로 토폴로지를 정확히 반영하는지 확인할 수 있습니다. 즉, 설비가 올바른 공간에 배치되었을 뿐만 아니라 올바른 패널의 올바른 회로에 정확히 연결되었는지를 검증합니다.

2. 핵심 개념
회로-부하 추적 — 단계별 가이드:

Step 1: 패널에서 시작
- Revit에서 대상 분전반(DB)의 패널 스케줄 뷰를 엽니다
- 회로 번호, 설명, 부하 유형, 차단기 정격을 확인합니다

Step 2: 회로를 따라 부하까지 이동
- Revit에서 패널 스케줄의 회로를 선택하고 "표시" 또는 "모델에서 강조" 기능을 사용하여 연결된 요소를 찾습니다
- 또는 시스템 브라우저(System Browser)를 사용합니다: 패널 확장 → 회로 찾기 → 회로 확장하여 연결된 모든 부하 확인

Step 3: 부하 요소 확인
- 연결된 요소(조명 기구, 설비, 콘센트)가 회로 설명에 명시된 올바른 요소인지 확인합니다
- 확인 사항: 요소가 올바른 공간 또는 존(Zone)에 있는가?
- 확인 사항: 올바른 요소 유형인가(예: "AHU-5"가 아닌 "AHU-4")?

Step 4: 매개변수 확인
- 연결된 요소의 피상 부하(VA) 또는 와트(W) — 패널 스케줄 항목과 일치해야 함
- 전압(230V / 400V) — 회로 상(Phase) 구성과 일치해야 함
- 부하 분류(조명, 동력, HVAC 등) — 회로 설명 카테고리와 일치해야 함

일반적인 회로 추적 문제:
- 부하가 잘못된 패널에 연결됨 (예: 4층 설비가 DB-L5에 연결)
- 전용 회로여야 하는데 여러 부하가 하나의 회로를 공유 (예: AHU가 일반 동력 회로에 연결)
- 회로에 연결된 부하가 없음 — 고아 회로(Orphan Circuit, 차단기는 할당되었으나 요소 미연결)
- 모델에 요소는 있으나 회로가 없음 — 고아 부하(Orphan Load, 요소는 배치되었으나 차단기 미할당)

추적 도구: Revit의 시스템 브라우저(System Browser)는 회로-부하 추적에 가장 효율적인 도구입니다. 패널별로 필터링하고, 회로를 확장하며, 부하를 검증하세요.

3. 실제 프로젝트 사례
한 모델러가 DB-L2의 회로 09를 추적합니다. 해당 회로는 "배기팬 EF-2"로 설명되어 있으며 16A 차단기가 지정되어 있습니다. 그녀는 시스템 브라우저를 사용하여 회로 09를 따라가다가 "팬 유닛"이라는 레이블이 붙은 일반 기계 설비 패밀리에 연결된 것을 발견합니다. 요소의 Type Mark 매개변수를 확인하니 "EF-2"로 표시됩니다. 회로 자체는 올바르지만, 요소의 패밀리 이름이 기계 모델과 조정되지 않은 상태입니다. 그녀는 이를 명명법(Nomenclature) 조정 이슈로 기록하고 이슈 등록부에 추가합니다.

4. BIM 체크리스트
- 확인: 패널 스케줄의 모든 회로에 대해, 시스템 브라우저를 사용하여 연결된 부하 요소를 모델에서 찾고 식별할 수 있는가? — 고아 회로가 0개인가?
- 확인: 부하 요소의 공간 위치가 패널 스케줄의 회로 설명과 일치하는가? (예: 3층 회로는 3층 요소에만 연결되어 있는가?)
- 확인: 모든 전용 회로(AHU, 펌프, UPS, 서버실)가 단일 부하 회로로 확인되었는가? — 일반 동력 부하와 공유되지 않는가?

5. 흔한 실수
패널 스케줄에 회로가 존재하는 것만 확인하고 물리적 요소까지 추적하지 않는 경우 — 즉, 패널 스케줄을 최종 확인 단계로 간주하는 실수입니다. 스케줄상으로는 회로가 올바르게 보일 수 있지만, 모델에서 연결된 요소가 잘못된 설비이거나, 잘못된 위치에 있거나, 아예 존재하지 않을 수 있습니다.

6. 오늘의 실천 과제
Revit 또는 교육용 모델에서 분전반(DB) 하나를 선택하고 시스템 브라우저를 사용하여 세 개의 회로를 추적하세요. 각 회로에 대해 다음을 기록하세요: 회로 번호, 설명, 연결된 요소 이름, 요소의 공간, 부하 값이 패널 스케줄과 일치하는지 여부. 불일치 사항이 있으면 메모하세요.
