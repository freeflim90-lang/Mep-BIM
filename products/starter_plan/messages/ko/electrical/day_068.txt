Day 68 - 부하 일정 — 패널 일정과 BIM 모델 매칭하기

1. 왜 중요한가
패널 일정(Panel Schedule)은 배전반의 핵심 엔지니어링 문서입니다. 일정에 기재된 부하가 Revit 모델과 일치하지 않으면, 해당 모델은 간섭 검토, 충돌 감지, 또는 인수인계에 신뢰할 수 없습니다. 패널 일정을 BIM 모델과 매칭하는 것은 주니어 MEP BIM 실무자에게 가장 빈번하게 주어지는 업무 중 하나입니다.

2. 핵심 개념
패널 일정은 배전반(DB)의 모든 회로를 다음과 같이 나열합니다:

| 열 | 의미 |
|---|---|
| 회로 번호 | 차단기 위치 — 홀수는 좌측, 짝수는 우측 |
| 설명 | 연결된 부하 (예: "조명 — 3층 사무실") |
| 상(Phase) | A, B, C (단상) 또는 3상 |
| 차단기 정격 | MCB의 전류 용량 (예: 16A, 20A, 32A) |
| 연결 부하 (kW) | 해당 회로의 설계 부하 |
| 비고 | 예비 회로, 특수 장비, 향후 회로 |

패널 일정을 BIM 모델과 매칭하는 방법:
- Revit에서 해당 DB 패밀리를 찾습니다 — 패널 이름이 일치하는지 확인
- Revit에서 DB의 전기 패널 일정 뷰를 엽니다
- 종이 일정의 모든 회로가 모델에 할당된 회로와 일치하는지 확인
- 회로 설명이 모델 내 연결된 장비와 일치하는지 확인
- 일정의 차단기 정격이 Revit 패밀리의 MCB 파라미터와 일치하는지 확인
- "예비(Spare)"로 표시된 회로는 누락 데이터가 아닌 미사용 차단기 슬롯으로 모델링합니다

도구: Revit 전기 패널 일정 뷰, 파라미터 매핑, 프로젝트 브라우저 패널 패밀리

3. 실제 프로젝트 관점
한 코디네이터가 M&E 컨설턴트의 DB-L4(4층 배전반) 패널 일정을 PDF로 받습니다. 그녀는 Revit 모델을 열고 DB-L4 패밀리를 찾은 후 패널 일정 뷰를 엽니다. 24개 회로를 한 줄씩 검토하며 각 부하가 올바른 Revit 요소에 연결되었는지, 차단기 정격이 일치하는지 확인합니다. 회로 07번에서 "AHU-4"가 기재되어 있지만 어떤 요소에도 연결되지 않은 것을 발견하고, 이를 간섭 이슈로 플래그 처리합니다.

4. BIM 점검
- 점검: Revit 패밀리의 패널 이름이 엔지니어링 패널 일정의 패널 ID와 정확히 일치합니까? (예: "DB-L4", "DB4" 또는 "Panel-L4"가 아님)
- 점검: Revit 패널 일정에 표시된 총 연결 부하가 엔지니어링 문서의 총 부하와 5% 이내로 일치합니까?
- 점검: 모든 예비 회로가 누락 항목이 아닌 미사용 슬롯으로 모델링되었습니까?

5. 흔한 실수
패널 패밀리가 모델에 배치되었으므로 회로가 자동으로 올바르다고 가정하는 것입니다. 패널 패밀리 배치는 단순한 형상일 뿐입니다 — 회로 연결과 부하 파라미터는 별도로 할당되어야 하며, 이 단계는 마감 압박 속에서 자주 생략됩니다.

6. 오늘의 실천 과제
샘플 패널 일정을 찾으세요 (프로젝트 파일, 교재, 또는 온라인 참고 자료). Revit 패널 일정 뷰와 대조 검증할 다섯 가지 정보 항목을 나열하세요. 각 일정 데이터 항목에 대응하는 Revit 파라미터를 기록하세요.
