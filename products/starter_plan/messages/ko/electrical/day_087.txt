LUA BIM LABS Starter
Day 87 - 전기 파라미터 체크리스트 — 패널명, 회로 번호, 부하, 전압, 시스템

1. 왜 중요한가
전기 파라미터는 BIM 모델이 간섭 검토 이상의 가치를 발휘하도록 만드는 데이터 계층입니다. 올바른 형상을 갖추었더라도 파라미터가 누락되었거나 일관성이 없는 모델은 신뢰할 수 있는 패널 스케줄, 부하 요약, 또는 인계 데이터를 생성할 수 없습니다. 체계적인 파라미터 체크리스트는 모델이 제출되기 전에 데이터 결함을 포착하여, 클라이언트 승인 이후 발생하는 고비용 수정을 방지합니다.

2. 핵심 개념
전기 파라미터 체크리스트 — 요소별 점검:

패널 / 분전반 (Distribution Board):
- 패널명 (Panel Name) — SLD 패널 ID와 정확히 일치
- 전압 (Voltage) — 예: 400V, 230V
- 상 수 (Number of Phases) — 1 또는 3
- 인입 차단기 정격 (Incomer Rating, A) — 주 차단기 전류 용량
- 단락 용량 (Fault Rating, kA) — 부스바 단락 정격
- 극 수 / 회로 수 (Number of Poles / Circuits) — 전체 회로 용량
- 실명 (Room Name) — 패널이 위치한 실
- 담당 층 / 구역 (Served Floor / Zone) — 해당 패널이 공급하는 층 또는 영역
- 비상 / 일반 (Emergency / Normal) — 전원 유형 분류

회로 (패널 스케줄 내 개별 회로):
- 회로 번호 (Circuit Number) — SLD 회로 번호와 일치
- 회로 설명 (Circuit Description) — 엔지니어링 패널 스케줄 설명과 일치
- 부하 분류 (Load Classification) — 조명 / 동력 / HVAC / 화재 경보 / 비상
- 극 수 (Number of Poles) — 1Ph(1극) 또는 3Ph(3극)
- 프레임 크기 / 정격 (Frame Size / Rating, A) — 차단기 정격
- 피상 부하 (Apparent Load, VA) 또는 전력 (Wattage, W) — 연결 부하 값
- 연결 요소 ID (Connected Element ID) — 부하 장비 패밀리와 연결

케이블 트레이 / 도관 (Cable Tray / Conduit):
- 시스템 유형 (System Type) — Power-HV / Power-LV / Data / 화재 경보 / 보안
- 재질 (Material) — 강재 / GI / PVC / 알루미늄
- 폭 (Width, mm) — 트레이 폭
- 깊이 (Depth, mm) — 트레이 깊이
- 호칭 사이즈 (Trade Size, 도관 전용) — mm 또는 인치

장비 (회로에 연결된 부하):
- 장비명 / 마크 (Equipment Name / Mark) — 기계 또는 전기 스케줄과 일치
- 패널명 (Panel Name) — 해당 장비에 전원을 공급하는 분전반
- 회로 번호 (Circuit Number) — 해당 장비에 전원을 공급하는 차단기
- 전압 (Voltage) — 230V 또는 400V
- 부하 (Load, kW) 또는 VA — 연결 부하 값

점검 도구: Revit 스케줄 뷰 — 위 파라미터를 포함한 모든 전기 요소의 멀티 카테고리 스케줄을 생성합니다. 빈 필드를 표시하도록 필터링하면, 빈 셀이 데이터 결함입니다.

3. 실제 프로젝트 관점
한 BIM 코디네이터가 호텔 프로젝트에서 모든 분전반, 회로 및 연결 장비를 포함하는 Revit 멀티 카테고리 파라미터 스케줄을 생성합니다. 장비의 "패널명" 필드가 비어 있는 항목을 필터링한 결과, 31개의 장비 패밀리에서 패널명이 누락된 것을 발견했습니다 — 모두 배치되었지만 회로가 할당되지 않은 상태였습니다. 회로의 "부하 분류"가 비어 있는 항목을 필터링한 결과, 18개의 회로에서 분류가 누락된 것을 발견했습니다. 두 결함 모두 인계 데이터 내보내기 전에 수정되어, 시설 관리팀의 데이터 품질 거부를 방지할 수 있었습니다.

4. BIM 점검
- 점검: Revit 멀티 카테고리 스케줄에서 모든 전기 요소의 패널명, 회로 번호, 부하 분류에 대해 빈 셀이 0개인가?
- 점검: 모든 분전반과 모든 연결 장비 패밀리에 전압 파라미터가 Revit 기본값이 아닌 실제 값으로 설정되어 있는가?
- 점검: 모든 케이블 트레이 및 도관 런에 시스템 유형 파라미터가 설정되어 있어, 시스템 기반 필터링과 간섭 규칙 할당이 가능한가?

5. 흔한 실수
패널에만 파라미터를 입력하고 연결 부하 장비에는 입력하지 않는 경우입니다. 패널 스케줄은 완전해 보이지만, 부하 장비 패밀리에 패널명이나 회로 번호가 없어 System Browser를 수동으로 검색하지 않고는 장비에서 패널로 역추적이 불가능합니다.

6. 오늘의 실천 과제
Revit에서 전기 장비(또는 조명 기구)에 대한 간단한 스케줄을 생성하고, 세 개의 열(패밀리명, 패널명, 회로 번호)을 추가하세요. 이를 내보내고, 패널명 또는 회로 번호 셀이 비어 있는 행의 개수를 세어보세요. 이 숫자가 전기 모델의 데이터 결함 기준선입니다.
