Day 78 - HVAC 기자재 승인도서와 BIM 모델 업데이트

1. 왜 중요한가
BIM 모델은 설계 엔지니어가 가정한 크기가 아닌, 실제 현장에 설치될 기자재를 반영해야 합니다. 승인도서 검토는 BIM 모델을 실제 현장 반입 자재와 일치시키는 프로세스입니다.

2. 핵심 개념
- 기자재 승인도서는 시공사가 각 HVAC 품목에 대해 특정 제조사와 모델 번호를 제시하는 문서입니다
- 설계 가정과 승인도서 간 일반적인 차이점: 기자재가 더 클 수 있고, 접속부가 다른 면에 위치하거나, 다른 이격거리(Clearance)가 필요할 수 있습니다
- 승인도서 검토 시 BIM 모델러의 역할: 제출된 치수와 접속부 위치를 모델 패밀리와 비교 — 합의된 공차(Tolerance)를 초과하는 치수 차이가 있을 경우 업데이트
- 확인해야 할 주요 파라미터: 폭, 깊이, 높이, 중량(구조적 영향), 접속부 방향, 접속부 크기, 점검구(Access Panel) 요구사항, 전기 접속점
- 제출된 기자재가 모델링된 공간에 맞지 않는 경우, 시공사는 대안을 찾거나 실 배치를 수정해야 합니다 — 이는 콘크리트 타설 전 BIM에서 해결됩니다

3. 실제 프로젝트 사례
상업용 프로젝트에서 HVAC 시공사가 설계 가정보다 200mm 더 긴 냉동기(Chiller)를 제출했습니다. BIM 코디네이터는 추가 길이가 비구동단(Non-drive End)의 배관 유지보수 공간을 차단할 것을 발견했습니다. 시공사는 동일한 용량이지만 케이싱이 더 짧은 대체 모델을 조달했습니다 — 설계 단계에서는 비용이 들지 않는 변경이었으나, 설치 후에는 고비용이 발생했을 사안입니다.

4. BIM 체크리스트
- 체크: 승인도서가 승인되면, Revit 패밀리 치수를 승인된 기자재 치수에 맞게 업데이트합니다
- 체크: 승인된 승인도서의 접속부 방향과 접속부 크기가 BIM에 이미 모델링된 배관과 일치하는지 확인합니다
- 체크: 승인된 기자재의 중량이 구조 설계에서 가정한 구조 하중(Structural Loading) 범위 내에 있는지 확인합니다

5. 흔한 실수
초보자는 실제 기자재가 현장에 도착하는 시공이 시작된 후에야 모델을 업데이트합니다. 그때는 크기 차이로 인해 이미 간섭 문제가 발생한 후입니다.

6. 오늘의 실천 과제
온라인에서 기자재 데이터시트 1개(AHU, FCU 또는 냉동기 — 제조사 무관)를 찾아보세요. 해당 치수를 기록하고, 동일한 기자재 유형의 일반적인 Revit 패밀리와 비교해 보세요.
범위 안내:
본 내용은 일반 교육용 콘텐츠이며, 프로젝트 검토, 엔지니어링 검증, 코드 준수 확인 또는 시공 승인을 대체하지 않습니다.
