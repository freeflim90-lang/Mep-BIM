Day 67 - 1주차 HVAC 복습

1. 왜 중요한가
배운 내용을 복습하지 않으면 정보는 희미해집니다. 이번 복습 세션은 61~66일차의 내용을 실무에서 바로 활용 가능한 참고 자료로 정리하고, 더 심화된 HVAC 주제로 넘어가기 전에 놓친 부분이 있는지 확인하는 데 목적이 있습니다.

2. 핵심 개념
1주차 HVAC 요약

| 일차 | 주제 | 핵심 포인트 |
|---|---|---|
| 61 | 시스템 흐름 | AHU → 급기 덕트 → VAV → 디퓨저 → 환기 → AHU |
| 62 | 덕트 사이징 | 풍량 ÷ 풍속 = 덕트 단면적. 엔지니어 스케줄과 일치시킬 것 |
| 63 | 시스템 지정 | 급기 / 환기 / 배기 — Revit 시스템 브라우저에서 절대 혼용 금지 |
| 64 | VAV 연결 | 입구는 급기 분기관에, 출구는 플렉스 덕트에 연결, Mark 및 풍량 설정 |
| 65 | AHU 실 배치 | 900mm 필터 접근 공간, 1000mm 유지보수 공간, 실제 장비 치수 적용 |
| 66 | HVAC-구조 간섭 | 덕트와 보의 충돌이 가장 빈번. 우회하거나 승인된 관통부 적용 |

꼭 기억해야 할 규칙
- 덕트 사이즈는 엔지니어 스케줄에서 가져올 것 — 절대 임의 추정 금지
- Revit에서 시스템 지정은 덕트 개체가 아닌 시스템 레벨에서 설정
- VAV 마크는 엔지니어링 스케줄과 정확히 일치해야 함
- AHU 여유 공간은 반드시 모델링해야 함 — 가정 금지
- 모든 P1 간섭은 조정 모델 제출 전에 반드시 해결

3. 실제 프로젝트 관점
주니어 HVAC 모델러가 프로젝트 첫날 Level 7 HVAC 모델을 인계받았습니다. 61~66일차에서 배운 기술을 활용하여 (1) 시스템 브라우저에서 미지정 덕트 확인, (2) 덕트 사이즈와 엔지니어 스케줄 교차 검토, (3) VAV 마크 검증, (4) AHU 여유 공간 모델링, (5) 구조 모델과 간섭 테스트 실행, (6) 조정 회의 전 간섭 보고서 발행까지 하루 만에 작업을 완료했습니다.

4. BIM 점검
- ✓ 점검: 모델에서 AHU에서 디퓨저까지, 그리고 다시 AHU로 돌아오는 공기 흐름을 연결 끊김 없이 추적할 수 있습니까?
- ✓ 점검: 모든 VAV 박스가 올바르게 마킹되고 연결되었습니까?
- ✓ 점검: AHU 여유 공간이 모델링되었고 장애물이 없습니까?

5. 흔한 실수
1주차 복습을 선택 사항으로 간주하는 것. HVAC 작업 2개월 차에서 발생하는 오류의 가장 빈번한 원인은 61~66일차의 기본기를 건너뛰고 기초 결함을 해결하지 않은 채 진도만 나가는 데 있습니다.

6. 오늘의 실천 과제
오늘 새로운 HVAC Revit 모델을 인계받았다면 가장 먼저 확인할 5가지를 개인 체크리스트로 작성하세요. 저장해 두고 — 90일차까지 이 리스트를 확장해 나가게 됩니다.
