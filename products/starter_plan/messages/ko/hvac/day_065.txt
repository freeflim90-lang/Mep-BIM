LUA BIM LABS Starter
Day 65 - AHU 실 배치 및 여유 공간

1. Why This Matters (중요성)
도면상으로는 실 안에 들어맞는 AHU라도 유지보수 접근 공간이 확보되지 않으면 설치 후 서비스가 불가능합니다. MEP 설비가 너무 좁게 배치되면 시공 중 장애가 발생하고, 운영 중에는 장기간의 유지보수 중단으로 이어집니다.

2. Core Concept (핵심 개념)
- AHU 여유 공간 요구사항:
  - 필터 접근 측: 최소 900mm~1200mm (필터 슬라이드 아웃용)
  - 코일 접근 측: 최소 900mm (코일 인출 또는 세척용)
  - 팬 모터 측: 최소 600mm
  - 일반 통로: 유닛 간 1000mm
- 바닥 하중: AHU는 중량이 크므로 구조 엔지니어와 슬래브 하중 용량을 반드시 확인
- 진동 방진: AHU는 스프링 방진기 또는 패드 위에 설치해야 하며, 이로 인해 전체 높이가 100~150mm 증가
- 서비스 경로: AHU 연결 덕트가 접근 패널을 막지 않도록 확인
- AHU 실 필요 사항: 바닥 배수구, 출입문 (설치 시 유닛 반입 가능한 크기), 조명, 환기
- AHU 실 배치 체크리스트:
  - 필터 접근 여유 공간: 최소 900~1200mm
  - 코일 인출 여유 공간: 최소 900mm
  - 모터/팬 측 여유 공간: 최소 600mm
  - 일반 통로 경로: 최소 1000mm 확보
  - 설치용 출입문 폭: 장비 폭 + 최소 200mm
  - 바닥 배수구: AHU 드레인 팬으로부터 2m 이내
  - 방진 여유 높이: 100~150mm 추가

3. Real Project Lens (실무 사례)
호텔 14층의 AHU 실에 3대의 옥상형 유닛이 배치되었습니다. 모델 1차 검토 결과, 유닛 1과 벽 사이 간격이 500mm에 불과해 필터 접근이 불가능했습니다. BIM 코디네이터는 이를 간섭 체크 보고서에서 소프트 클래시(여유 공간 위반)로 플래그 처리했습니다. 이후 유닛 1을 500mm 이동하는 방식으로 레이아웃이 수정되었고, 이에 따라 급기 덕트 경로도 변경되었습니다.

4. BIM Check (BIM 점검 사항)
- Check: 실제 제조사 치수가 반영된 AHU 모델 사용 (임시 박스 아님)
- Check: Revit 내부 패밀리 또는 참조 평면을 활용한 여유 공간 영역 생성 — 접근 구역이 겹치지 않는지 시각적으로 확인
- Check: AHU 상단 레벨 + 방진 패드 높이가 천장 또는 보 하부를 초과하지 않는지 확인

5. Common Mistake (흔한 실수)
실제 지정된 유닛이 2400 x 1200 x 1600mm인데, 기본 치수(1000 x 500 x 800mm)의 일반 AHU 패밀리를 사용하여 모델상 실 배치가 문제없어 보였으나, 현장 설치 시 전혀 맞지 않는 상황이 발생합니다.

6. Today's Action (오늘의 실천 과제)
샘플 또는 실제 프로젝트 모델에서 AHU 하나를 찾으세요. 해당 AHU의 속성을 열어 실제 치수를 확인하세요. 그런 다음 모델에서 AHU와 가장 가까운 벽 사이의 여유 공간을 측정하세요. 최소 요구사항인 900mm와 비교하여 통과 또는 실패 여부를 기록하세요.

Scope note:
