Day 80 - HVAC 모델 품질 — 10가지 자가 점검 항목

1. 중요성
제출 전 모델 품질 자가 점검은 재작업을 방지하고, 전문가로서의 평판을 보호하며, 선임 코디네이터가 인정하고 보상하는 규율을 입증합니다.

2. 핵심 개념
- HVAC 모델 제출 전, 다음 10가지 항목을 점검하세요:
- 1. 시스템 할당: 모든 덕트와 피팅이 올바른 Revit MEP 시스템에 할당되어 있는가
- 2. 연결되지 않은 커넥터: System Inspector로 확인 시, 연결되지 않은 덕트 끝단이 없는가
- 3. 요소 명명: 모든 장비에 Mark 값이 있으며, 덕트는 올바른 유형 이름을 사용하는가
- 4. LOD 상태: 모든 요소가 해당 제출 단계에서 합의된 LOD를 충족하는가
- 5. 단열: 프로젝트 시방서에 따라 단열이 필요한 모든 덕트에 적용되었는가
- 6. 방화 댐퍼: 모든 내화 벽체 및 슬래브 관통부에 방화 댐퍼가 배치되었는가
- 7. 여유 공간: 모든 장비 주변에 접근 여유 공간이 확보되었는가
- 8. 간섭 검토 구역: 합의된 범위 경계 밖에 배치된 요소가 없는가
- 9. Revit 경고: 경고 개수가 50개 미만이며, 중요 경고가 없는가
- 10. NWC 내보내기: 제출 전, 새로운 NWC 파일을 내보내 Navisworks에서 테스트했는가

3. 실제 프로젝트 사례
한 BIM 코디네이터가 이 10가지 점검 항목을 팀 루틴으로 도입한 결과, HVAC 제출물당 평균 간섭(clash) 개수가 2개월 만에 340개에서 95개로 감소했습니다. 이러한 감소는 제출 전에 연결되지 않은 커넥터와 시스템 할당 오류를 수정했기 때문이며, 이 오류들은 이전에 Navisworks에서 가상 간섭(phantom clash)을 발생시키고 있었습니다.

4. BIM 점검
- 점검: Revit System Inspector를 실행하여 연결되지 않은 HVAC 요소가 0개인지 확인
- 점검: 모든 HVAC 장비의 요소 특성(properties)에 Mark 값이 표시되는지 확인
- 점검: 테스트용 NWC 내보내기 파일이 Navisworks에서 올바르게 열리고, 선택 트리(selection tree)에 모든 요소가 표시되는지 확인

5. 흔한 실수
초보자들은 자가 점검이 추가 작업처럼 느껴져 건너뛰곤 합니다. 하지만 실제로는 점검되지 않은 모델이 수정을 위해 반환되면, 점검에 소요된 시간보다 더 많은 시간이 낭비됩니다.

6. 오늘의 실천 과제
접근 가능한 HVAC 모델에서 10가지 점검 항목 중 3가지를 실행하세요. 각 항목에 대해 발견한 내용을 기록하세요.

범위 안내:
본 내용은 일반 교육 콘텐츠이며, 프로젝트 검토, 엔지니어링 검증, 코드 준수 확인 또는 시공 승인을 대체하지 않습니다.
