Day 66 - 管道与结构及 MEP 的碰撞

1. 为什么这很重要
管道碰撞——尤其是穿越结构梁或与其他 MEP 系统交叉的大口径冷冻水主管——是现场最昂贵的协调失误之一。每次协调失败都可能导致整个楼层的工作停滞，等待碰撞解决。

2. 核心概念
- 最常见的管道碰撞：
  - 管道穿越结构梁（最关键——绝不切割翼缘）
  - 管道穿越剪力墙（需要结构套管和防火封堵）
  - CHW 供水和回水距离太近（违反保温净空——保温管道表面之间通常需要 50mm 间距）
  - 管道与桥架交叉（电气分离和净空冲突）
  - 立管竖井边界处的管道（管道太大，无法通过竖井开口）
- 碰撞解决策略：
  - 重新规划管道走向以避开结构构件
  - 使用结构穿洞（仅限梁腹板——需结构工程师批准）
  - 与其他 MEP 专业协商垂直叠放（管道在管道下方，桥架在上方）
- 在 Navisworks 中：分别运行管道与结构和管道与所有 MEP 的碰撞测试——这样可以得到更清晰的碰撞分组

管道碰撞优先级

| 碰撞 | 优先级 | 解决方案 |
|---|---|---|
| 管道穿越梁翼缘 | P1 | 重新规划——绝不切割 |
| 管道穿越剪力墙无套管 | P1 | 建模套管，协调 |
| CHW 管道保温相接触 | P2 | 增加间距 |
| 管道穿越桥架间距 < 100mm | P2 | 调整一个系统的高程 |
| 竖井内管道太宽 | P1 | 修改竖井尺寸或管道走向 |

3. 真实项目视角
200mm CHWS 立管穿越 3 层楼板。建模的核心为 100mm 直径——但管道保温为 50mm 厚，使得保温后外径为 300mm。核心太小。BIM 团队在协调模型中发现这一问题，并申请 350mm 核心以容纳保温和净空。

4. BIM 检查
- 检查：所有穿越楼板和墙体的管道包含了建模套管（或至少提交了穿洞申请）
- 检查：保温管道尺寸——在检查净空前将保温厚度加到外径上（例如 DN100 管道 + 50mm 保温 = 214mm 保温后外径）
- 检查：所有管道重新规划走向后重新运行碰撞测试——确认该段管道走向无剩余 P1 碰撞

5. 常见错误
只用裸管几何体运行碰撞测试——保温在每侧增加 50–100mm，这会将一个"接近碰撞"变成一个硬碰撞，一旦现场安装了保温即会显现。

6. 今日行动
对模型（或示例）中的任意一根管道，使用以下公式计算保温后外径：保温后外径 = 管道外径 + (2 × 保温厚度)。与管道和最近结构构件之间的可用间隙进行比较。能通过吗？
