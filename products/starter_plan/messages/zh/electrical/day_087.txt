LUA BIM LABS 始动计划
第87天 - 电气参数检查表 — 配电板名称、回路编号、负载、电压、系统

1. 这为什么重要
电气参数是使BIM模型超越碰撞检测而有用的数据层。一个几何正确但缺少或不一致的参数的模型无法生成可靠的配电板时间表、负载汇总或移交数据。在提交模型之前，系统的参数检查列表可以捕捉到数据缺口 — 防止客户验收后昂贵的修正。

2. 核心概念
电气参数检查表 — 逐项检查：

配电板 / 分配器：
- 配电板名称 — 完全匹配SLD分配器ID
- 电压 — 如400V, 230V
- 相数 — 1相或3相
- 进线额定值 (A) — 主断路器安培数
- 故障额定值 (kA) — 母排短路额定值
- 回路数量 / 路数 — 总回路容量
- 房间名称 — 分配板所在的房间
- 服务楼层 / 区域 — 此分配器供电的楼层或区域
- 应急 / 正常 — 电源类型分类

回路（每个面板时间表中的回路）：
- 回路编号 — 完全匹配SLD回路编号
- 回路描述 — 匹配工程面板时间表描述
- 负载分类 — 照明 / 动力 / HVAC / 火灾报警 / 应急
- 相数 — 1相（1个接线端子）或3相（3个接线端子）
- 框架尺寸 / 额定值 (A) — 断路器额定值
- 视在负载 (VA) 或瓦特数 (W) — 连接负载值
- 所连接元件ID — 与负载设备族链接

电缆桥架 / 导管：
- 系统类型 — 动力-HV / 动力-LV / 数据 / 火灾报警 / 安全
- 材料 — 钢材 / 镀锌铁 / PVC / 铝合金
- 宽度 (mm) — 桥架宽度
- 深度 (mm) — 桥架深度
- 贸易尺寸（仅导管） — mm或in

设备（连接到回路的负载）：
- 设备名称 / 标记 — 与机械或电气时间表匹配
- 配电板名称 — 哪个分配器为该设备供电
- 回路编号 — 哪个断路器为该设备供电
- 电压 — 230V 或 400V
- 负载 (kW) 或 VA — 连接负载值

检查工具：Revit时间表视图 — 创建包含上述所有电气元素的多类别时间表。筛选显示空字段 — 任何空白单元格都是数据缺口。

3. 实际项目视角
BIM协调员在酒店项目中创建了一个涵盖所有配电板、回路和连接设备的多类别参数时间表。过滤出具有无配电板名称的设备空字段，她发现有31个设备族没有分配器名称 — 所有放置但从未电路分配。过滤出回路中无负载分类的空字段，她发现有18个回路没有分类。在数据导出移交之前，这两个缺口都被修复了，避免了设施管理团队的数据质量拒绝。

4. BIM检查
- 检查：Revit多类别时间表是否显示所有电气元素的配电板名称、回路编号和负载分类字段均无空值？
- 检查：每个分配器和每个多连接设备族上的电压参数设置了吗 — 不留用Revit默认值？
- 检查：每个电缆桥架和导管运行上的系统类型参数设置了吗 — 以便基于系统的筛选和碰撞规则分配？

5. 常见错误
只在配电板上填充参数，而忘记在连接负载设备上填充。配电板时间表看起来完整，但连接的设备族没有配电板名称或回路编号 — 使无法从设备追溯到配电板，除非手动搜索系统浏览器。

6. 今天的行动
在Revit中为电气设备（或灯具）创建一个简单的具有三列的时间表：族名、配电板名称、回路编号。导出它。计算有多少行的配电板名称或回路编号单元格为空。这个计数是电气模型的数据缺口基线。
