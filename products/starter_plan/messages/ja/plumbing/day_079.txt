Day 79 - 衛生器具スケジュール

1. なぜ重要か
器具スケジュールは単なる管理書類ではなく、品質管理ツール・調達リファレンス・引き渡し記録のすべてが一体となったものです。MEP BIMでは、器具スケジュールはモデルから生成され、その精度はRevitのファミリとパラメータがどれだけ適切に設定されているかにかかっています。正しい器具スケジュールの作成はジオメトリだけでなくモデルデータを確認することを強制します。

2. 核心概念
衛生器具スケジュールはプロジェクト内のすべての給排水器具を主要パラメータとともにリストします。Revitでは、器具スケジュールはモデルデータベースを照会するビューです — モデルに実際に入力されているものを反映し、意図したものではありません。

衛生器具スケジュールに含めるパラメータ：

| パラメータ | 記録内容 | ソース |
|-----------|----------------|--------|
| 記号 / ID | 固有の器具タグ | Revitファミリインスタンス |
| 器具種別 | 便器・洗面台・小便器・シンク・床排水 | ファミリ名 |
| 階 | フロア / 階数位置 | 階パラメータ |
| 室名 | 器具が設置される部屋 | Revit室タグ |
| システム名 | 汚水・雑排水・通気 | 管システム割り当て |
| 管径（出口） | 排水管のDN呼び径 | 管パラメータ |
| 器具単位数 | サイズ決定のための排水単位値 | 共有パラメータまたはファミリパラメータ |
| トラップ種別 | Pトラップ・深水封・AAV | ファミリ共有パラメータ |
| 備考 | アクセシビリティ・特殊器具 | 手動入力 |

スケジュールの作成方法：
1. すべての器具ファミリに必要な共有パラメータがロードされていることを確認する
2. 各ファミリを配置する際に器具固有のパラメータを入力する
3. カテゴリ「給排水器具」でRevitスケジュールビューを作成する
4. 必要なフィールドを正しい順序で追加する
5. 階でソートし、次に室名でソートを適用する
6. スケジュールを確認して欠落または空白のパラメータ値を特定する — これらがモデルのデータギャップ

3. 実プロジェクトの視点
病院プロジェクトで、FM引き渡しに340台すべての器具の室名・器具単位数・トラップ種類を含む器具スケジュールが必要だった。BIMコーディネーターは器具単位値がプロジェクトで使用されているRevitファミリに一度も追加されていないことを発見した — 設計仕様書にはあったがBIMモデルにはなかった。遡及的に追加するには各ファミリを開き、共有パラメータを追加し、ファミリを再配置する必要があった。このプロセスに3日かかり、モデリング開始時にスケジュール要件を確認していれば回避できた。

4. BIMチェック
- 確認：すべての給排水器具ファミリに器具単位数・トラップ種別・システム名のパラメータがロードされ入力されている
- 確認：Revitスケジュールに階・室名・管径（出口）の空白セルがない
- 確認：スケジュール内の器具数が建築の器具配置図の数と一致している

5. よくある間違い
プロジェクト終了時にモデルに存在するデータから器具スケジュールを作成する。空白のパラメータフィールドがプレースホルダーテキストで埋められるか空のままになる。これにより、見た目は完成しているが不正確なデータを含むスケジュールが生成され、FM引き渡しまたは試運転時に問題となる。

6. 本日のアクション
Revitで階・器具種別・システム名・管径・器具単位数を含む少なくとも6列のパラメータを持つ給排水器具スケジュールを作成する。空白セルの結果を確認する。空白セルがある場合は、ファミリまたは管システムに戻り、欠落値を追加する。スケジュールを再実行して完成していることを確認する。
