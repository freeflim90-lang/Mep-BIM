Day 71 - Cross Main and Branch Line Sizing

1. これの重要性
火災防止用の配管サイズはモデリング上の決定ではありません—それはエンジニアリング計算です。しかし、BIM モデラとして、階層の各レベルで正しい配管直径をモデル化する責任があります。階層が間違っていると、水力計算は無効になり、設計圧力での必要な流量が出力されません。

2. 核心概念
火災防止用の配管は、供給配管から下に厳格なサイズ階層を追っていきます：

| 配管レベル | 一般的な名称 | 機能 |
|---|---|---|
| 1 | Riser / Main Feed | 火災ポンプや都市水道から建物内への接続 |
| 2 | Cross Main | フロア沿いに水平に走る、供給配管から給水される |
| 3 | Branch Line | ターミナルヘッドをサービスする行方向の配管から直角に分岐 |
| 4 | Drop / Sprig | 分岐から個別のペンデントヘッドへの短い垂直配管 |

サイズ減少ルール
- 階層の下に向かって配管の直径は減少します—増えることはありません（エンジニアが特定の理由がある場合を除く）
- 分岐線は、接続するクロスメインの直径を超えてはなりません
- ドロップは、接続する分岐線の直径を超えてはなりません
- 一般的な危険度フロアのクロスメインサイズ：普通の危険度フロアでは DN100 から DN150
- 分岐線のサイズ：サービスするヘッド数によって異なるが、通常 DN25 から DN50
- ペンデントヘッドへのドロップ：軽量または普通の危険度では通常 DN25（1インチ）

Revit での処理: 正しいノーマル直径を持つパイプタイプファミリーを使用します。配管サイズはエンジニアリングによって決定されるので、未承認で配管を再サイズしないでください。不明な場合は、エンジニアの配管サイズスケジュールを参照文書として使用します。

3. 実際のプロジェクト視点
一般的なオフィスフロアの場合、火災工事担当者のサイズスケジュールは次の通りです：クロスメイン DN100、6～8 つのヘッドをサービスする分岐線は DN40、3～5 のヘッドをサービスする分岐線は DN32、そして各ヘッドへのペンデントヘッド用のドロップは通常 DN25。モデラーはこのスケジュールに従います。分岐線のルーティングが必要な距離が計画より長い場合、モデラーは新しい長さでの DN40 のサイズがまだ機能するかどうかではなく、エンジニアに報告します。

4. BIM チェック
- 検証: 階層の各レベルで火災工事担当者のサイズスケジュールと一致していることを確認—推定値やデフォルトサイズを使用しないこと
- 検証: クロスメインから分岐線、ドロップへ配管の直径が減少する（または一定）—予期せぬ直径の増加がないこと
- 検証: ペンデントヘッドへのドロップは適切な直径でモデル化されていることを確認（誤ってクロスメインサイズに残されていない）

5. 一般的なミス
フロア全体の分岐線に対して単一の配管直径を使用し、モデリングが速いからです。異なる数のヘッドをサービスする分岐線には異なるサイズが必要であり—8 のヘッドをサービスする DN40 ラインは DN25 に置き換えることができません（水力計算を再計算する必要があります）。

6. 今日のアクション
現在のプロジェクトまたはサンプルプロジェクトで火災工事担当者の配管サイズスケジュールを見つけます。次の3つの直径をリストします：(1) クロスメイン、(2) 最長の分岐線、(3) ドロップ。これらの3つのサイズがRevitモデルに正しく反映されていることを確認します。
