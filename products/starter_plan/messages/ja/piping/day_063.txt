Day 63 - Revitでの仕切弁とアクセサリ

1. なぜ重要か
仕切弁とパイプアクセサリは装飾ではありません——メンテナンスのためにどのパイプセクションをシャットダウンできるかを制御します。モデルに仕切弁が欠落しているということは、現場でバルブが欠落していることを意味し、つまり1台のFCUをサービスするためにシステム全体をシャットダウンする必要があります。

2. 基本概念
- 主要なパイプアクセサリとその機能：
  - ゲートバルブ / ボールバルブ：完全遮断（開または閉）。機器接続と分岐取り出しに使用
  - グローブバルブ：流量制御（絞り）。微細な流量調整が必要な場合に使用
  - チェックバルブ：1方向のみの流れを許可。ポンプ吐き出し、バイパスラインに使用
  - ストレーナ（Y型）：流体から異物を除去。ポンプの吸い込み側とコントロールバルブの前に常に設置
  - バランシングバルブ：手動または自動の流量バランス調整。各FCUまたはAHUコイル接続に使用
  - 減圧弁（PRV）：システムゾーンで高圧を設定値に下げる
  - フレキシブルコネクタ：ポンプと機器での振動絶縁
- Revitで：パイプアクセサリファミリを使用——単なるパイプ継手ではなく。各アクセサリにはユニークなファミリとコネクタとマークパラメータがある

バルブ配置の目安

| 場所 | バルブタイプ |
|---|---|
| 機器入口/出口 | 仕切弁（ボールまたはゲート） |
| ライザーからの分岐取り出し | 仕切弁 |
| ポンプ吸い込み | ストレーナ + 仕切弁 |
| ポンプ吐き出し | 仕切弁 + チェックバルブ |
| FCUコイル接続 | 2方向または3方向コントロールバルブ + バランシングバルブ |
| ゾーン遮断 | 各フロア分岐の両側にボールバルブ |

3. 実プロジェクトの視点
4階のFCUに必要なもの：CHWS：ボールバルブ → 2方向コントロールバルブ → コイル → ボールバルブ → CHWR。モデラーはこれらのアクセサリを順番に配置し、各バルブのマークを設定し（例：V-CHW-L4-FCU07-IN）、パイプサイズがFCU接続サイズに一致することを確認します（通常DN20〜DN25）。

4. BIM チェック
- チェック：すべての機器接続ポイントに供給と還水側の両方に仕切弁がモデル化されている
- チェック：すべてのポンプに吸い込み側にストレーナがある
- チェック：コントロールバルブ（2方向、3方向）がFCUコイルの正しい側（2方向の場合は供給側）に配置されている

5. よくある間違い
DN25分岐にDN50ボールバルブのような大きすぎるバルブファミリを配置すること——Revitのコネクタ不一致エラーを作成し、図面に誤って印刷されます。

6. 本日のアクション
Revit（またはサンプルファイル）でパイプ分岐に1つのボールバルブを配置します。そのサイズがパイプに一致することを確認します。マークパラメータを設定します。パイプアクセサリスケジュールに表示されることを確認します。
