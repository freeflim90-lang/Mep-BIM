## 図面タイプリファレンス

| 図面タイプ | 定義 | 作成者 | 主な用途 |
|---|---|---|---|
| 設計図 | エンジニアの設計意図（システムレイアウト、サイズ、仕様）を示す | 設計エンジニア | BIMモデリングおよび調達の基礎 |
| 調整図 | 施工用に確認された、干渉を解消したMEPルートを示す（レベル、グリッド参照を含む） | BIMコーディネーター | 現場施工および墨出し |
| 製作図 | 製作レベルの詳細（切断長、フランジ、ボルトパターンを含む）を示す | 施工業者 / 製作会社 | 工場製作（スプール、パネル） |
| 竣工図 | 実際の設置状況（設計からの差異を記録）を示す | 施工業者（BIM支援） | ファシリティ管理および引渡し |

## MEP平面図読み取りのヒント

**HVAC図面 — 確認すべき項目：**
- AHUからディフューザーへの給気ルート（各ダクトのシステム割り当てを確認）
- グリルからAHUへの還気ルート（給気と混同しないこと）
- VAVボックスの位置と接続分岐ダクト
- 機器タグ（AHU、FCU、EF）が機器リストと一致していること
- 主要なルート変更箇所のダクトサイズ注釈

**配管図面 — 確認すべき項目：**
- 往き管と還り管の流れ方向矢印
- 配管サイズ注釈 — 設計仕様と一致していることを確認
- バルブ記号 — 仕切弁、逆止弁、減圧弁
- 機器接続ポイント（ポンプ、コイル、熱交換器）
- 衛生設備系統の排水管および通気管ルート（勾配表記を確認）

**電気図面 — 確認すべき項目：**
- 主配電盤から分電盤へのケーブルラックルート
- 配電盤の位置が単線結線図と一致していること
- 配電盤から主要機器への回路ホームラン線
- 電線管ルートおよびジャンクションボックス位置
- 配電機器の負荷ラベル

## シンボル記号リファレンス

| 系統 | 一般的な記号 |
|---|---|
| HVAC 給気 | 平面図では実線の長方形ダクト、流れ方向を示す矢印 |
| HVAC 還気 | 平面図では破線の長方形ダクト、AHU方向への逆矢印 |
| 冷水 | CWS（往き）およびCWR（還り）、流れ矢印付き |
| 温水 | HWSおよびHWRとラベル表示、仕切弁記号付き |
| 排水（重力式） | 破線、勾配表記付き（例：1:50） |
| スプリンクラー | 実線、ヘッドを示す「X」または点記号 |
| ケーブルラック | 平面図では二重線の長方形、サイズ表記付き（例：600×100） |
| 電線管 | 単線、サイズ呼び出し付き、配電盤へのホームラン矢印 |

## 調整図と設計図の違い

| | 設計図 | 調整図 |
|---|---|---|
| **目的** | 設計意図を示す | 施工用に干渉を解消したルートを示す |
| **LOD** | LOD 200–300 | LOD 350 |
| **レベルデータ** | 概略または未表示 | 確定した配管インバートおよびダクト天井伏せレベル |
| **構造参照** | 梁が含まれない場合あり | 構造グリッドおよび主要梁を表示 |
| **干渉ステータス** | 該当なし | 発行前に主要な干渉をすべて解消 |
| **発行者** | 設計エンジニア | BIMコーディネーター |
| **使用者** | モデリングチーム、QA | 現場施工チーム |
