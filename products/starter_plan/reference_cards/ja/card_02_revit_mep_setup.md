## プロジェクトセットアップチェックリスト

**レベルとグリッド**
- [ ] レベルは、建築モデルまたは構造モデルからコピーまたは監視されます（単独で作成しないでください）
- [ ] 階高が建築モデルと照合されていることを確認します
- [ ] レベル名がプロジェクトの命名規則（例：L01、B1、RF）に準拠していることを確認します
- [ ] MEP平面図で、空間参照用に構造グリッドが表示されていることを確認します

**共有座標**
- [ ] 建築モデルがMEPモデルにリンクされています
- [ ] 建築モデルから共有座標を取得します
- [ ] 測量点とプロジェクト基点を確認します（MEPチームは移動しないでください）
- [ ] すべての専門モデル（HVAC、配管、電気）が同じ共有座標基準に位置合わせされています

**ワークセット**
- [ ] 専門分野またはシステムごとにワークセットを作成します（例：HVAC、配管、電気、リンクモデル）
- [ ] 各ワークセットを担当チームメンバーまたは役割に割り当てます
- [ ] 中央ファイルをプロジェクトCDEに保存します（ローカルデスクトップフォルダには保存しないでください）
- [ ] すべてのモデラーは、中央ファイルから直接ではなく、ローカルコピーで作業します

**リンクモデル**
- [ ] 建築モデルがリンクされ、ピン留めされています
- [ ] 構造モデルがリンクされ、ピン留めされています
- [ ] 他のMEP専門モデルが調整参照用にリンクされています
- [ ] モデリング開始前に、すべてのリンクパスが確認されています（破損していないこと）

**専門分野別設定**
- [ ] 正しいRevit MEPテンプレートが選択されています（機械、電気、または配管）
- [ ] ビューの専門分野フィルターが、ビューごとに正しいMEPシステムに設定されています
- [ ] ビューの範囲が天井内MEPゾーンを捉えるように設定されています（カット高さと下側オフセットを確認）
- [ ] リンクモデルの表示がハーフトーンに設定され、MEP要素が視覚的に区別できるようにします

## システム命名規則

| 要素 | 形式例 | 備考 |
|---|---|---|
| ファイル名 | `[プロジェクト]-[専門分野]-[レベル]-[作成者]-[日付].rvt` | 例：`PRJ-MECH-ALL-LUA-20260529.rvt` |
| システム名 | `HVAC-SA-L03` | HVAC、給気、レベル3 |
| 機器タグ | `AHU-01-L03` | 空調機01、レベル3 |
| 平面図 | `MEP-MECH-L03-PLAN` | MEP、機械、レベル3、平面図 |
| 調整ビュー | `3D-COORD-ZONE-A` | ゾーンA調整用3Dビュー |

## テンプレート確認項目

モデリングを開始する前に、以下を確認します：
- [ ] 正しいMEPテンプレートがアクティブであること（ファイル > プロジェクト情報で確認）
- [ ] 必要なすべてのシステム（給気、還気、冷水供給など）のシステムタイプが存在すること
- [ ] ファミリライブラリのパスにアクセス可能であり、プロジェクト固有のファミリがロードされていること
- [ ] 図面テンプレートに改訂テーブルとタイトルブロックが設定されていること
- [ ] 各専門分野のビューテンプレートが作成され、適用されていること
