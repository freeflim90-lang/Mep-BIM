## 現場チームがBIMに求めるもの

| 出力 | 形式 | 目的 |
|---|---|---|
| 調整図 | A1/A3印刷、PDF | 各ゾーンの確定したレベルとグリッド参照を伴うMEPルートを示す |
| 墨出しデータ | 寸法表、グリッド参照図 | スラブまたは壁における孔位置、ハンガー間隔、機器ベース位置をマークする |
| モデルアクセス（ビューア） | NWDファイル、Autodesk Viewerリンク、BIM 360 Docs | 現場監督が3Dモデルを操作し、空間関係を理解する |
| 干渉クリアランス承認書 | 書面による覚書またはトラッカーエクスポート | 作業開始前に、当該ゾーンのCriticalおよびMajorな干渉がすべて解決されたことを確認する |
| スプール図 | ISO図（アイソメ図）、スプール納入スケジュール | 固有のスプールIDタグにより、プレハブ組立品を現場に適合させる |
| RFI回答 | 書面によるRFI回答（24時間以内） | 現場状況がモデルや図面と異なる場合の正式な解決策 |

## RFIプロセスフロー

```
現場状況がモデルまたは図面と異なる
        ↓
現場監督が正式なRFIを発行する
（RFI番号、説明、写真を付与）
        ↓
RFIがBIMコーディネーター/エンジニアに提出される
        ↓
BIMコーディネーター/エンジニアがモデルと現場情報を確認する
        ↓
合意された期間内（通常24～48時間）に回答が発行される
        ↓
モデル変更が必要な場合：
  → モデルが更新され、図面が新しいリビジョンで再発行される
        ↓
RFIは、課題トラッカーにおける解決策への参照とともにクローズされる
        ↓
現場が修正モデルと異なる場合、その差異はアズビルド記録に反映される
```

## アズビルド更新チェックリスト

設置がBIMモデルと異なる場合：
- [ ] 差異が特定され、説明される（要素、位置、変更の性質）
- [ ] RFIが発行され、正式に回答されている
- [ ] BIMモデラーが実際の設置位置を反映するようモデルを更新する
- [ ] 更新されたモデルバージョンがファイル名に「AS-BUILT」と付けて保存される
- [ ] 修正された調整図が新しいリビジョンで発行される
- [ ] 現場チームに更新図面が通知され、旧リビジョンが無効化される
- [ ] 差異がプロジェクト変更台帳に記録される

## スプール図の基礎

| 用語 | 意味 |
|---|---|
| スプール | 配管またはダクトのプレハブセクション（継手、フランジ、サポート付き）— オフサイトで製造 |
| ISO図 | スプールのアイソメ図。3Dルート、全接続部、寸法を示す |
| 切断長 | 継手代を考慮した、各直管セグメントの正確な長さ |
| スプールID | 各スプールに付与される固有のタグ。スプール図と一致し、現場はタグと図面を照合して設置する |
| 端部処理 | 各スプール端部の継手タイプ（フランジ、グルーブ、突合せ溶接） |
| BIMから製作へ | スプール図を作成する前に、モデルはLOD 350であり、干渉が解決されている必要がある |

## 一般的な現場からBIMへのフィードバックタイプ

| フィードバックタイプ | 典型的な原因 | BIMの対応 |
|---|---|---|
| スラブのずれ | 床の打設レベルがモデルと異なる | 実際のレベルを測量し、モデルのレベルを調整し、調整図を再発行する |
| 構造体の位置ずれ | 柱や梁が設計からずれて設置された | 構造エンジニアにRFIを発行し、承認されればモデルを更新し、干渉を再確認する |
| モデル未反映の設備 | 設計変更が現場に発行されたがBIMに反映されていない | モデル更新が必要。影響を受けるすべての調整図を再発行する |
| 現場でのクリアランス不足 | 仕上げ厚の影響でボイドがモデルより狭い | 仕上げ構成を確認し、確定データでモデルを調整し、再調整する |
| 継手の調達不可 | 指定された継手タイプが入手不可または代替された | エンジニアが代替を承認し、モデルを代替ファミリで更新する |
