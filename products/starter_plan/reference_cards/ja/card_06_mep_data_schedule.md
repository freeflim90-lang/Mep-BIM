## 分野別のRevitスケジュールタイプ

| 分野 | スケジュールタイプ | 主な目的 |
|---|---|---|
| HVAC | 機械設備スケジュール | AHU、FCU、VAVボックスを記号、レベル、系統、流量で一覧表示 |
| HVAC | ダクトスケジュール | サイズと系統別のダクト総延長 — 数量拾い出し |
| 配管 | 配管スケジュール | サイズと材質別の配管総延長 — 調達 |
| 配管 | 機械設備スケジュール | ポンプ、熱交換器を能力、流量、圧力で表示 |
| 電気 | 電気設備スケジュール | 配電盤、変圧器を電圧、相、負荷で表示 |
| 電気 | ケーブルトレイスケジュール | 幅とタイプ別のトレイ延長 |
| 衛生 | 衛生器具スケジュール | 器具を記号、タイプ、水使用量で表示 |
| 衛生 | 配管スケジュール | 上水・排水管をサイズと材質別に表示 |
| 消火 | 機械設備スケジュール | 消火ポンプ、スプリンクラーヘッド — 数量と位置 |
| 消火 | 配管スケジュール | スプリンクラー主管・枝管をサイズ別に表示 |
| 全分野 | ダクト/配管継手スケジュール | タイプ別の継手数量 — 調達と数量確認 |

## MEP分野別の主要パラメータ

**HVAC**
- 記号（機器固有ID）、ファミリ/タイプ、レベル、系統区分、系統名、設計流量、風量（CFM/L/s）、フィルタータイプ

**配管（機械）**
- 記号、ファミリ/タイプ、レベル、系統名、配管サイズ（DN/NPS）、材質、設計流量、使用圧力、保温仕様

**電気**
- 記号、ファミリ/タイプ、レベル、電圧、相、負荷区分、回路番号、配電盤名、負荷（kW/VA）

**衛生**
- 記号、ファミリ/タイプ、レベル、系統タイプ（上水、給湯、排水）、配管サイズ、勾配（排水用）、材質

**消火**
- 記号、ファミリ/タイプ、レベル、系統名（湿式/乾式）、配管サイズ、スプリンクラーヘッドタイプ、Kファクター、防護面積

## COBie必須フィールド

COBie（Construction Operations Building Information Exchange）は、BIMからFMへの引継ぎにおける標準データ形式です。

| COBieシート | 主要必須フィールド |
|---|---|
| Facility | 名称、説明、プロジェクト名、サイト名、長さ単位、面積単位 |
| Floor | 名称、説明、標高、高さ |
| Space | 名称、説明、階名、面積 |
| Component | 名称、説明、作成者、タイプ名、スペース名 |
| Type | 名称、説明、カテゴリ、メーカー、型番、保証期間 |
| Attribute | 名称、カテゴリ、値、単位、作成者 |

## データ書き出しチェックリスト — FM引継ぎ

- [ ] すべての機器に一意の記号が設定されている — 空白や重複なし
- [ ] COBie対象機器すべてにメーカー名と型番が入力されている
- [ ] 系統名がプロジェクトBEPの命名規則に従っている — 「既定」系統なし
- [ ] 階およびスペースデータがスケジュール内の機器とリンクしている
- [ ] IFC書き出しが完了し、検証済み — SolibriまたはBIM Collabで確認
- [ ] COBieスプレッドシートが書き出され、必須フィールドの空白が確認されている
- [ ] 保証および保守データフィールドがFM要件に従って入力されている
- [ ] スケジュールCSVが書き出され、設計仕様の数量と照合されている
- [ ] ファイルバージョンと発行日が書き出しログに記録されている
