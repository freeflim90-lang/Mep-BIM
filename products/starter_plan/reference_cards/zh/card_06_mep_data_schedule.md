## 按专业划分的 Revit 明细表类型

| 专业 | 明细表类型 | 主要用途 |
|---|---|---|
| 暖通空调 | 机械设备明细表 | 列出 AHU、FCU、VAV 箱体，包含标记、标高、系统、流量 |
| 暖通空调 | 风管明细表 | 按尺寸和系统统计风管总长度 — 工程量提取 |
| 管道 | 管道明细表 | 按尺寸和材质统计管道总长度 — 采购用 |
| 管道 | 机械设备明细表 | 水泵、换热器，包含容量、流量、压力 |
| 电气 | 电气设备明细表 | 配电箱、变压器，包含电压、相数、负荷 |
| 电气 | 电缆桥架明细表 | 按宽度和类型统计桥架长度 |
| 给排水 | 卫生器具明细表 | 卫生器具，包含标记、类型、用水量 |
| 给排水 | 管道明细表 | 生活给水和排水管道，按尺寸和材质统计 |
| 消防 | 机械设备明细表 | 消防泵、喷淋头 — 数量与位置 |
| 消防 | 管道明细表 | 喷淋主管和支管，按尺寸统计 |
| 全部 | 风管/管道管件明细表 | 按类型统计管件数量 — 采购与数量核对 |

## 各 MEP 专业的关键参数

**暖通空调**
- 标记（唯一设备编号）、族/类型、标高、系统分类、系统名称、设计流量、风量（CFM/升/秒）、过滤器类型

**管道（机械）**
- 标记、族/类型、标高、系统名称、管道尺寸（DN/NPS）、材质、设计流量、工作压力、保温规格

**电气**
- 标记、族/类型、标高、电压、相数、负荷分类、回路编号、配电箱名称、负荷（kW/VA）

**给排水**
- 标记、族/类型、标高、系统类型（生活冷水、生活热水、污水）、管道尺寸、坡度（排水用）、材质

**消防**
- 标记、族/类型、标高、系统名称（湿式/干式）、管道尺寸、喷淋头类型、K 系数、覆盖面积

## COBie 必填字段

COBie（施工运营建筑信息交换）是从 BIM 移交至设施管理的标准数据格式。

| COBie 工作表 | 关键必填字段 |
|---|---|
| Facility（设施） | 名称、描述、项目名称、场地名称、线性单位、面积单位 |
| Floor（楼层） | 名称、描述、标高、高度 |
| Space（空间） | 名称、描述、楼层名称、面积 |
| Component（构件） | 名称、描述、创建者、类型名称、空间名称 |
| Type（类型） | 名称、描述、类别、制造商、型号、保修期限 |
| Attribute（属性） | 名称、类别、数值、单位、创建者 |

## 数据导出检查清单 — 设施管理移交

- [ ] 所有设备具有唯一标记值 — 无空白或重复
- [ ] 所有 COBie 要求设备的制造商和型号字段已填写
- [ ] 系统名称遵循项目 BEP 约定 — 无“默认”系统
- [ ] 楼层和空间数据已在明细表中关联至设备
- [ ] IFC 导出已完成并验证 — 在 Solibri 或 BIM Collab 中检查
- [ ] COBie 电子表格已导出并检查必填字段是否空白
- [ ] 保修和维护数据字段已按设施管理要求填写
- [ ] 明细表 CSV 已导出并与设计规格数量交叉核对
- [ ] 文件版本和发布日期已记录在导出日志中
