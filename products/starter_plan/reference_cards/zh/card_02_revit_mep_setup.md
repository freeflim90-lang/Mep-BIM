## 项目设置清单

**标高与网格**
- [ ] 标高从建筑/结构模型复制或监测而来 — 非独立创建
- [ ] 楼层高度已与建筑模型核对
- [ ] 标高名称符合项目命名规则（例如：L01、B1、RF）
- [ ] 结构网格在MEP平面视图中可见，用于空间定位

**共享坐标**
- [ ] 建筑模型已链接至MEP模型
- [ ] 已从建筑模型获取共享坐标
- [ ] 测量点与项目基点已确认 — MEP团队不得移动
- [ ] 所有专业模型（暖通、管道、电气）对齐至同一共享坐标基准

**工作集**
- [ ] 按专业或系统创建工作集（例如：暖通、给排水、电气、链接模型）
- [ ] 每个工作集分配给负责的团队成员或角色
- [ ] 中心文件保存至项目CDE（通用数据环境） — 非本地桌面文件夹
- [ ] 所有建模人员使用本地副本工作，而非直接操作中心文件

**链接模型**
- [ ] 建筑模型已链接并固定
- [ ] 结构模型已链接并固定
- [ ] 其他MEP专业模型已链接，用于协调参考
- [ ] 所有链接路径在建模开始前已验证（无断开）

**专业特定设置**
- [ ] 已选择正确的Revit MEP样板（机械、电气或给排水）
- [ ] 每个视图的视图专业过滤器已设置为正确的MEP系统
- [ ] 视图范围已设置以捕捉吊顶MEP区域（剖切高度与底部偏移已确认）
- [ ] 链接模型可见性设置为半色调，使MEP构件视觉上清晰可辨

## 系统命名约定

| 元素 | 示例格式 | 备注 |
|---|---|---|
| 文件名 | `[项目]-[专业]-[标高]-[作者]-[日期].rvt` | 例如：`PRJ-MECH-ALL-LUA-20260529.rvt` |
| 系统名称 | `HVAC-SA-L03` | 暖通、送风、三层 |
| 设备标记 | `AHU-01-L03` | 空气处理机组01、三层 |
| 平面视图 | `MEP-MECH-L03-PLAN` | MEP、机械、三层、平面 |
| 协调视图 | `3D-COORD-ZONE-A` | A区协调三维视图 |

## 样板验证项

在开始建模前，请验证：
- [ ] 正确的MEP样板已激活（检查文件 > 项目信息）
- [ ] 所有必需系统的系统类型已存在（送风、回风、冷冻水供水等）
- [ ] 族库路径可访问，且项目专用族已载入
- [ ] 图纸样板上的修订表与标题栏已设置完成
- [ ] 各专业的视图样板已创建并应用
