## 图纸类型参考

| 图纸类型 | 定义 | 编制方 | 主要用途 |
|---|---|---|---|
| 设计图 | 表达工程师的设计意图——系统布局、尺寸及规格 | 设计工程师 | BIM 建模与采购依据 |
| 协调图 | 展示经碰撞检查确认可用于施工的 MEP 路由——包含标高与网格参照 | BIM 协调员 | 现场安装与放样 |
| 加工图 | 提供用于制造的加工级细部——包含切割长度、法兰、螺栓布置 | 承包商/加工厂 | 工厂预制（分段、模块） |
| 竣工图 | 记录实际安装状况——设计变更已纳入 | 承包商（BIM 辅助） | 设施管理与移交 |

## MEP 平面图识读要点

**暖通平面图——检查项：**
- 从 AHU（空调处理机组）至散流器的送风路径（确认每段风管的系统归属）
- 从回风格栅返回 AHU 的回风路径（切勿与送风混淆）
- VAV（变风量箱）位置及连接的支管
- 设备编号（AHU、FCU、EF）与设备明细表一致
- 关键路由变更处的风管尺寸标注

**管道平面图——检查项：**
- 供回水管路上的流向箭头
- 管道尺寸标注——确认与设计规格一致
- 阀门符号——截止阀、止回阀、减压阀
- 设备连接点（水泵、盘管、换热器）
- 排水系统的排污管与透气管路由（注意坡度标注）

**电气平面图——检查项：**
- 从主配电柜至分配电盘的电缆桥架路由
- 配电盘位置与单线图一致
- 从配电盘至主要设备的电路回路线
- 线管路由及接线盒位置
- 配电设备上的负荷标签

## 图例符号参考

| 系统 | 常见符号 |
|---|---|
| 暖通送风 | 平面图中实线矩形风管；箭头指示流向 |
| 暖通回风 | 平面图中虚线矩形风管；箭头指向 AHU |
| 冷冻水 | CWS（供水）与 CWR（回水），带流向箭头 |
| 热水 | 标注 HWS 与 HWR，带截止阀符号 |
| 排水（重力流） | 虚线，带坡度标注（如 1:50） |
| 喷淋 | 实线管道，喷头用“X”或圆点符号表示 |
| 电缆桥架 | 平面图中双线矩形；标注尺寸（如 600×100） |
| 线管 | 单线，带尺寸标注；至配电盘的回路线箭头 |

## 协调图与设计图差异

| | 设计图 | 协调图 |
|---|---|---|
| **目的** | 表达设计意图 | 展示经碰撞检查可用于安装的路由 |
| **LOD** | LOD 200–300 | LOD 350 |
| **标高数据** | 近似值或未显示 | 已确认的管道反底标高与风管梁底标高 |
| **结构参照** | 可能不包含梁 | 显示结构网格及主要梁 |
| **碰撞状态** | 不适用 | 出图前所有重大碰撞已解决 |
| **出图方** | 设计工程师 | BIM 协调员 |
| **使用方** | 建模团队、QA | 现场安装团队 |
