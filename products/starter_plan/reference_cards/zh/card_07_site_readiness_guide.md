## 现场团队需要BIM提供什么

| 交付物 | 格式 | 用途 |
|---|---|---|
| 协调图纸 | A1/A3打印件、PDF | 显示各区域MEP（机电）路由，含确认标高及网格参照 |
| 放样数据 | 尺寸表、网格参照图纸 | 在楼板或墙体上标记孔洞位置、吊架间距及设备基座位置 |
| 模型访问（查看器） | NWD文件、Autodesk Viewer链接、BIM 360文档 | 现场主管浏览3D模型以理解空间关系 |
| 碰撞清除签认 | 书面备忘录或追踪表导出文件 | 确认工作开始前该区域内所有关键及重大碰撞已解决 |
| 分段图 | ISO图（轴测图）、分段交付计划 | 预制组件通过唯一分段ID标签与现场匹配 |
| RFI（信息请求）回复 | 书面RFI回复（24小时内） | 当现场条件与模型或图纸不符时，提供正式解决方案 |

---

## RFI流程

```
现场条件与模型或图纸不符
        ↓
现场主管提出正式RFI
（分配RFI编号、描述、照片）
        ↓
RFI提交至BIM协调员/工程师
        ↓
BIM协调员/工程师审查模型及现场信息
        ↓
在约定时限内（通常24–48小时）发出回复
        ↓
如需修改模型：
  → 更新模型，重新发布图纸并标注新版本
        ↓
RFI在问题追踪表中记录解决方案后关闭
        ↓
若现场与修改后模型仍有差异，则在竣工记录中记录偏差
```

---

## 竣工更新检查清单

当安装与BIM模型存在偏差时：
- [ ] 识别并描述偏差（构件、位置、变更性质）
- [ ] 提出RFI并得到正式回复
- [ ] BIM建模员更新模型，反映实际安装位置
- [ ] 更新后的模型版本以文件名包含"AS-BUILT"（竣工）保存
- [ ] 重新发布协调图纸并标注新版本
- [ ] 通知现场团队图纸已更新，旧版本作废
- [ ] 在项目变更登记表中记录偏差

## 分段图基础

| 术语 | 含义 |
|---|---|
| 分段 | 管道或风管的预制段（含管件、法兰、支架）——在工厂预制 |
| ISO图 | 分段轴测图，显示3D路由、所有连接及尺寸 |
| 切割长度 | 每段直管的精确长度，已考虑管件余量 |
| 分段ID | 每个分段上的唯一标签，与分段图对应——现场通过匹配标签与图纸进行安装 |
| 端部处理 | 每个分段端部的接头类型（法兰连接、沟槽连接、对焊连接） |
| BIM到制造 | 模型必须达到LOD 350（详细度等级350）且碰撞已解决，方可生成分段图 |

## 常见现场到BIM反馈类型

| 反馈类型 | 典型原因 | BIM应对措施 |
|---|---|---|
| 楼板偏差 | 浇筑的楼板不平——与模型不符 | 实测实际标高，调整模型标高，重新发布协调图纸 |
| 结构位置偏移 | 柱子或梁安装位置偏离设计 | 向结构工程师发出RFI，若接受则更新模型，重新检查碰撞 |
| 模型中遗漏管线 | 设计变更已发至现场但未录入BIM | 需更新模型，重新发布所有受影响的协调图纸 |
| 现场净空不足 | 因饰面厚度导致实际空间比模型更窄 | 检查饰面构造层，根据确认数据调整模型，重新协调 |
| 管件采购不到 | 指定管件类型缺货或被替代 | 工程师批准替代方案，更新模型并替换族文件 |
