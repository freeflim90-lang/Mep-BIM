## 四域质量框架

模型质量需在四个领域进行验证。提交任何MEP模型前，须完成全部20项检查。

---

### 领域1：几何

- [ ] 所有构件均使用数值偏移放置于正确标高，而非目测估算
- [ ] 无构件悬空、埋入楼板或偏离网格放置
- [ ] 所有风管和管道均具有有效且经确认的尺寸（非默认或近似值）
- [ ] 设备相对于结构网格放置于正确位置
- [ ] 同一位置无重复构件（通过“管理 > 检查警告”进行核查）

### 领域2：数据/参数

- [ ] 所有构件均具有唯一的标记值——无空白，无重复
- [ ] 所有构件均已分配至命名系统（无“默认”或未命名系统）
- [ ] 所需参数已填写：尺寸、标高、系统名称、材质/规格
- [ ] 设备标记与设计图纸或设备明细表一致
- [ ] 文件名遵循项目BEP命名规则

### 领域3：系统连接

- [ ] 所有风管和管道末端均已连接——无未连接接口
- [ ] 系统浏览器中无未分配或“未命名”系统
- [ ] 设备（AHU、水泵、配电盘）已与其关联的风管或管道连接
- [ ] 重力排水系统已按所需坡度建模（非平坡）——排水通常为1:50
- [ ] 系统检查器显示每个活动系统均无错误

### 领域4：协调就绪

- [ ] 所有链接模型均已加载且为最新版本（无断链）
- [ ] 共享坐标已确认——模型与建筑底图对齐
- [ ] 模型已清除未使用的族和类型（文件 > 清除未使用项）
- [ ] Revit警告已审查——关键警告已解决或已记录
- [ ] 移交前文件已同步至中心模型

## LOD自检表

| LOD | 几何确认 | 参数已填写 | 包含吊架/支架 | 已碰撞检查 |
|---|---|---|---|---|
| LOD 200 | 近似 | 最少 | 否 | 否 |
| LOD 300 | 精确 | 核心字段已填写 | 否 | 可选 |
| LOD 350 | 精确 | 全套字段已填写 | 是 | 是 — 必需 |
| LOD 400 | 加工详图 | 所有字段含制造商 | 是（详细） | 是 |
| LOD 500 | 竣工状态 | 所有字段 + 现场核实 | 按竣工状态 | 不适用 — 记录 |

## 快速参考：常见错误检查

| 错误 | 检查位置 |
|---|---|
| 未连接接口 | 系统浏览器 — 未托管构件 |
| 系统类型错误 | 属性面板 → 系统分类 |
| 标记空白 | 明细表视图，按标记字段排序 |
| 警告数量 | 管理选项卡 → 检查警告 |
| 文件过大 | 文件 > 清除未使用项；检查RVT文件大小 |
