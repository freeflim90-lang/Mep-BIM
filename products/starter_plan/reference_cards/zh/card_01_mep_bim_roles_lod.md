## BIM 角色参考

| 角色 | 主要职责 |
|---|---|
| BIM 经理 | 制定 BIM 标准，管理 CDE（通用数据环境），审查整个项目的模型质量 |
| BIM 协调员 | 执行碰撞检测与协调，管理问题追踪清单，主持协调会议 |
| MEP BIM 建模员 | 创建并维护专业 BIM 模型（暖通空调、管道、电气等） |
| BIM 主管 | 负责 BEP（BIM 执行计划），监控交付里程碑，指导建模团队 |
| MEP 工程师 | 提供设计意图与工程决策，审批协调变更 |
| 业主 / 开发商 | 通过 EIR（雇主信息要求）设定 BIM 需求；定义交付物与验收标准 |

## LOD 定义

| LOD | 名称 | 描述 | MEP 示例 |
|---|---|---|---|
| LOD 100 | 概念级 | 仅表示整体尺寸与位置 — 不可测量 | 平面图上标注的风管示意区域 |
| LOD 200 | 近似级 | 近似几何形状、尺寸与位置 | 带有近似尺寸与标高的风管走向 |
| LOD 300 | 精确级 | 精确几何形状、确认尺寸与连接 | 具有正确尺寸、标高及系统分配的风管 |
| LOD 350 | 协调级 | 包含协调所需的所有元素，包括支吊架 | 带有吊架、经多专业碰撞检查的风管 |
| LOD 400 | 加工级 | 完整的加工详图级别细节，用于制造 | 带有法兰、支吊架及安装公差的风管分段 |
| LOD 500 | 竣工级 | 经核实的现场条件 — 记录模型 | 实际安装的风管，记录与设计图纸的偏差 |

## 关键缩略语

| 缩略语 | 全称 | 实际含义 |
|---|---|---|
| BEP | BIM 执行计划 | 由团队编写的项目文件，定义谁、何时、以何种方式交付何种模型内容 |
| EIR | 雇主信息要求 | 由业主定义的文件，规定项目的 BIM 交付物与标准 |
| CDE | 通用数据环境 | 用于存储和管理所有项目模型与文档的共享数字平台 |
| IFC | 工业基础类 | 用于在不同软件平台之间共享 BIM 数据的开放、非专有文件格式 |
| NWC | Navisworks 缓存文件 | 从 Revit 导出的轻量化模型，用于 Navisworks 碰撞检测与审查 |
| NWD | Navisworks 文档 | 将多个 NWC 模型打包的项目文件，用于共享与归档 |
| RFI | 信息请求 | 当现场条件与模型或图纸不符时提出的正式询问 |
