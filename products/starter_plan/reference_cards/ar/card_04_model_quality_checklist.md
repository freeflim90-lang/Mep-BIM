# قائمة التحقق الذاتي لجودة النموذج
**دليل البدء السريع من LUA BIM LABS — المسار 4**
*بطاقة إتمام المسار — اليوم 28*

---

## إطار الجودة رباعي المجالات

يتم التحقق من جودة النموذج عبر أربعة مجالات. أكمل جميع الفحوصات العشرين قبل تسليم أي نموذج MEP.

---

### المجال 1: الهندسة

- [ ] جميع العناصر موضوعة على الارتفاعات الصحيحة باستخدام الإزاحات الرقمية — وليس التقدير البصري
- [ ] لا توجد عناصر عائمة في الفراغ، أو مدفونة في بلاطة، أو موضوعة خارج الشبكة
- [ ] جميع مجاري الهواء والأنابيب ذات أحجام صالحة ومؤكدة (وليست افتراضية أو تقريبية)
- [ ] المعدات موضوعة في الموضع الصحيح بالنسبة لشبكات العناصر الإنشائية
- [ ] لا توجد عناصر مكررة في نفس الموقع (تحقق عبر إدارة > مراجعة التحذيرات)

### المجال 2: البيانات / المعاملات

- [ ] جميع العناصر لها قيمة Mark فريدة — لا فراغات ولا تكرار
- [ ] جميع العناصر مسندة إلى نظام مسمى (لا توجد أنظمة "افتراضية" أو غير مسماة)
- [ ] المعاملات المطلوبة مملوءة: الحجم، المستوى، اسم النظام، المادة/المواصفات
- [ ] تسميات المعدات تطابق الرسم التصميمي أو جدول المعدات
- [ ] اسم الملف يتبع اصطلاح التسمية الخاص بـ BEP للمشروع

### المجال 3: توصيلات الأنظمة

- [ ] جميع نهايات مجاري الهواء والأنابيب متصلة — لا توجد موصلات مفتوحة
- [ ] متصفح الأنظمة لا يظهر أي أنظمة غير مسندة أو "غير مسماة"
- [ ] المعدات (AHU، مضخة، لوحة) متصلة بمجاري الهواء أو الأنابيب المرتبطة بها
- [ ] التصريف بالجاذبية مصمم بالميل المطلوب (وليس مسطحًا) — عادة 1:50 للصرف
- [ ] مدقق الأنظمة يظهر صفر أخطاء لكل نظام نشط

### المجال 4: جاهزية التنسيق

- [ ] جميع النماذج المرتبطة محملة ومحدثة (وليست روابط مقطوعة)
- [ ] تم تأكيد الإحداثيات المشتركة — النموذج متوافق مع القاعدة المعمارية
- [ ] تم تنظيف النموذج من العائلات والأنواع غير المستخدمة (ملف > تنظيف غير المستخدم)
- [ ] تم مراجعة تحذيرات Revit — تم حل التحذيرات الحرجة أو توثيقها
- [ ] تمت مزامنة الملف مع النموذج المركزي قبل التسليم

## جدول التحقق الذاتي لمستوى تطور النموذج (LOD)

| LOD | الهندسة مؤكدة | المعاملات مملوءة | الشماعات/الدعامات مضمنة | تم فحص التصادم |
|---|---|---|---|---|
| LOD 200 | تقريبية | دنيا | لا | لا |
| LOD 300 | دقيقة | الحقول الأساسية مملوءة | لا | اختياري |
| LOD 350 | دقيقة | المجموعة الكاملة مطلوبة | نعم | نعم — مطلوب |
| LOD 400 | تفاصيل تصنيعية | جميع الحقول بما فيها الشركة المصنعة | نعم (تفصيلية) | نعم |
| LOD 500 | كما تم التركيب | جميع الحقول + التحقق الميداني | كما تم التركيب | غير قابل للتطبيق — سجل |

## مرجع سريع: الأخطاء الشائعة التي يجب فحصها

| الخطأ | أين يتم الفحص |
|---|---|
| موصلات مفتوحة | متصفح الأنظمة — العناصر غير المستضافة |
| نوع نظام خاطئ | لوحة الخصائص → تصنيف النظام |
| Mark فارغ | عرض الجدول، الفرز حسب حقل Mark |
| عدد التحذيرات | علامة التبويب إدارة → مراجعة التحذيرات |
| حجم ملف كبير | ملف > تنظيف غير المستخدم؛ التحقق من حجم ملف RVT |
