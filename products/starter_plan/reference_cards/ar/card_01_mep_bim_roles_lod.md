**LUA BIM LABS — مرجع سريع للمسار الأول**
*بطاقة إتمام المسار — اليوم 07*

## مراجع أدوار BIM

| الدور | المسؤولية الأساسية |
|---|---|
| مدير BIM | يضع معايير BIM، ويدير بيئة البيانات المشتركة (CDE)، ويراجع جودة النماذج عبر المشروع |
| منسق BIM | يُجري فحص التصادمات والتنسيق، ويدير سجل المشكلات، ويقود اجتماعات التنسيق |
| مصمم نماذج BIM لأنظمة MEP | يُنشئ ويُحدّث نماذج BIM الخاصة بالتخصصات (HVAC، أنابيب، كهرباء، إلخ) |
| قائد BIM | يمتلك خطة تنفيذ BIM (BEP)، ويتابع معالم التسليم، ويُوجّه فريق النمذجة |
| مهندس MEP | يُقدّم نية التصميم والقرارات الهندسية، ويُوافق على تغييرات التنسيق |
| العميل / المطور | يحدد متطلبات BIM عبر متطلبات معلومات صاحب العمل (EIR)؛ ويُعرِّف مخرجات التسليم ومعايير القبول |

## تعريفات مستويات التطور (LOD)

| LOD | الاسم | الوصف | مثال على أنظمة MEP |
|---|---|---|---|
| LOD 100 | مفهومي | يُشار إلى الحجم والموقع الإجمالي — غير قابل للقياس | منطقة مجرى هواء تخطيطي مُحدّدة على مخطط الأرضية |
| LOD 200 | تقريبي | هندسة تقريبية، حجم، شكل، وموقع | مسار مجرى هواء بحجم وارتفاع تقريبيين |
| LOD 300 | دقيق | هندسة دقيقة، أبعاد مؤكدة، توصيلات | مجرى هواء بالحجم والارتفاع الصحيحين، مع تعيين النظام |
| LOD 350 | تنسيقي | جميع العناصر اللازمة للتنسيق، بما في ذلك الدعامات | مجرى هواء مع معلّقات، تم فحص تصادماته مع جميع التخصصات |
| LOD 400 | تصنيعي | تفاصيل كاملة بمستوى رسومات الورشة للتصنيع | أجزاء مجرى هواء مع حواف، دعامات، تفاوتات تركيب |
| LOD 500 | كما تم التنفيذ | ظروف ميدانية مُتحقق منها — نموذج سجلي | مجرى هواء كما تم تركيبه فعلياً، مع توثيق الانحرافات عن التصميم |

## الاختصارات الرئيسية

| الاختصار | الاسم الكامل | المعنى العملي |
|---|---|---|
| BEP | خطة تنفيذ BIM | وثيقة مشروع يُعدّها الفريق تُحدّد من يُسلّم أي محتوى نموذجي، ومتى، وكيف |
| EIR | متطلبات معلومات صاحب العمل | وثيقة يُحدّدها العميل تُحدّد مخرجات BIM ومعاييرها للمشروع |
| CDE | بيئة البيانات المشتركة | منصة رقمية مشتركة تُخزّن وتُدار فيها جميع نماذج المشروع ووثائقه |
| IFC | فئات أساسيات الصناعة | تنسيق ملف مفتوح غير مملوك لمشاركة بيانات BIM بين منصات برمجية مختلفة |
| NWC | ذاكرة تخزين مؤقت لـ Navisworks | تصدير نموذج خفيف من Revit يُستخدم لفحص التصادمات والمراجعة في Navisworks |
| NWD | وثيقة Navisworks | ملف مشروع مُجمّع يضم نماذج NWC متعددة للمشاركة والأرشفة |
| RFI | طلب معلومات | استفسار رسمي يُرفع عندما تختلف الظروف الميدانية عن النموذج أو الرسومات |
