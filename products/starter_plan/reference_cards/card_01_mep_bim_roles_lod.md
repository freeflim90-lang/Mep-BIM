## BIM Roles Reference

| Role | Primary Responsibility |
|---|---|
| BIM Manager | Sets BIM standards, manages the CDE, reviews model quality across the project |
| BIM Coordinator | Runs clash detection and coordination, manages issue tracker, leads coordination meetings |
| MEP BIM Modeler | Creates and maintains discipline-specific BIM models (HVAC, piping, electrical, etc.) |
| BIM Lead | Owns the BEP, monitors delivery milestones, mentors the modeling team |
| MEP Engineer | Provides design intent and engineering decisions, approves coordination changes |
| Client / Developer | Sets BIM requirements via the EIR; defines deliverables and acceptance criteria |

## LOD Definitions

| LOD | Name | Description | MEP Example |
|---|---|---|---|
| LOD 100 | Concept | Overall size and location indicated — not measurable | Schematic duct zone marked on floor plan |
| LOD 200 | Approximate | Approximate geometry, size, shape, and location | Duct routing with approximate size and elevation |
| LOD 300 | Precise | Accurate geometry, confirmed dimensions, connections | Duct with correct size, elevation, system assignment |
| LOD 350 | Coordination | All elements needed for coordination, including supports | Duct with hangers, clash-checked against all disciplines |
| LOD 400 | Fabrication | Full shop-drawing level detail for manufacturing | Duct sections with flanges, supports, installation tolerances |
| LOD 500 | As-Built | Verified field conditions — record model | Duct as actually installed, deviations from design captured |

## Key Acronyms

| Acronym | Full Form | What It Means in Practice |
|---|---|---|
| BEP | BIM Execution Plan | Project document written by the team defining who delivers what model content, when, and how |
| EIR | Employer's Information Requirements | Client-defined document specifying BIM deliverables and standards for the project |
| CDE | Common Data Environment | Shared digital platform where all project models and documents are stored and managed |
| IFC | Industry Foundation Classes | Open, non-proprietary file format for sharing BIM data between different software platforms |
| NWC | Navisworks Cache | Lightweight model export from Revit used for Navisworks clash detection and review |
| NWD | Navisworks Document | Packaged project file bundling multiple NWC models for sharing and archive |
| RFI | Request for Information | Formal query raised when site conditions differ from the model or drawings |
