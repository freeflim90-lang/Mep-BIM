## 프로젝트 설정 체크리스트

**레벨 및 그리드**
- [ ] 레벨은 건축/구조 모델에서 복사 또는 모니터링되어야 함 — 독립적으로 생성하지 않음
- [ ] 층고가 건축 모델과 일치하는지 확인
- [ ] 레벨 이름이 프로젝트 명명 규칙과 일치 (예: L01, B1, RF)
- [ ] MEP 평면 뷰에서 공간 참조를 위해 구조 그리드가 표시되어야 함

**공유 좌표**
- [ ] 건축 모델이 MEP 모델에 링크됨
- [ ] 건축 모델로부터 공유 좌표를 획득
- [ ] 측량점 및 프로젝트 기준점 확인 — MEP 팀이 이동하지 않음
- [ ] 모든 분야 모델(HVAC, 배관, 전기)이 동일한 공유 좌표 기준에 정렬됨

**워크셋**
- [ ] 워크셋은 분야 또는 시스템별로 생성 (예: HVAC, 배관, 전기, 링크 모델)
- [ ] 각 워크셋은 담당 팀원 또는 역할에 할당
- [ ] 중앙 파일은 프로젝트 CDE에 저장 — 로컬 데스크톱 폴더 아님
- [ ] 모든 모델러는 중앙 파일이 아닌 로컬 복사본에서 작업

**링크 모델**
- [ ] 건축 모델이 링크 및 고정됨
- [ ] 구조 모델이 링크 및 고정됨
- [ ] 기타 MEP 분야 모델이 조정 참조용으로 링크됨
- [ ] 모델링 시작 전 모든 링크 경로 확인 (끊어짐 없음)

**분야별 설정**
- [ ] 올바른 Revit MEP 템플릿 선택 (기계, 전기 또는 배관)
- [ ] 뷰 분야 필터가 뷰별 올바른 MEP 시스템으로 설정됨
- [ ] 뷰 범위가 천장 MEP 영역을 포함하도록 설정 (절단 높이 및 하단 오프셋 확인)
- [ ] 링크 모델 가시성이 하프톤으로 설정되어 MEP 요소가 시각적으로 구분됨

## 시스템 명명 규칙

| 요소 | 예시 형식 | 비고 |
|---|---|---|
| 파일 이름 | `[프로젝트]-[분야]-[레벨]-[작성자]-[날짜].rvt` | 예: `PRJ-MECH-ALL-LUA-20260529.rvt` |
| 시스템 이름 | `HVAC-SA-L03` | HVAC, 급기, 3층 |
| 장비 태그 | `AHU-01-L03` | 공조기 01, 3층 |
| 평면 뷰 | `MEP-MECH-L03-PLAN` | MEP, 기계, 3층, 평면 |
| 조정 뷰 | `3D-COORD-ZONE-A` | A구역 조정 3D 뷰 |

## 템플릿 확인 항목

모델링 시작 전 다음을 확인:
- [ ] 올바른 MEP 템플릿이 활성화되어 있는지 확인 (파일 > 프로젝트 정보)
- [ ] 필요한 모든 시스템 유형이 존재하는지 확인 (급기, 환기, 냉수 공급 등)
- [ ] 패밀리 라이브러리 경로에 접근 가능하고 프로젝트별 패밀리가 로드되었는지 확인
- [ ] 시트 템플릿에 개정표 및 표제란 설정 완료
- [ ] 각 분야별 뷰 템플릿이 생성 및 적용되었는지 확인
