## 도면 유형 참조

| 도면 유형 | 정의 | 작성 주체 | 주요 용도 |
|---|---|---|---|
| 설계 도면 | 엔지니어의 설계 의도(시스템 배치, 크기, 사양)를 나타냄 | 설계 엔지니어 | BIM 모델링 및 자재 조달의 기초 |
| 조정 도면 | 간섭이 해소된 MEP 경로를 확정하여 시공에 반영 — 레벨 및 그리드 기준 포함 | BIM 코디네이터 | 현장 설치 및 측설(Setting-out) |
| 제작 도면 | 제조를 위한 fabrication 수준의 상세 정보 제공 — 절단 길이, 플랜지, 볼트 패턴 포함 | 시공사 / 제작사 | 오프사이트 제조(스풀, 패널) |
| 준공 도면 | 실제 설치 상태 기록 — 설계 대비 변경 사항 반영 | 시공사(BIM 지원) | 시설 관리 및 인계 |

## MEP 평면도 판독 요령

**HVAC 평면도 — 확인 사항:**
- AHU에서 디퓨저까지의 급기 경로(각 덕트의 시스템 할당 확인)
- 그릴에서 AHU로의 환기 경로(급기와 혼동하지 말 것)
- VAV 박스 위치 및 연결된 분기 덕트
- 장비 일람표와 일치하는 장비 태그(AHU, FCU, EF)
- 주요 경로 변경 지점의 덕트 치수 표기

**배관 평면도 — 확인 사항:**
- 공급관 및 환수관의 유량 방향 화살표
- 배관 치수 표기 — 설계 사양과 일치하는지 확인
- 밸브 기호 — 게이트 밸브, 체크 밸브, 압력 조절 밸브
- 장비 연결 지점(펌프, 코일, 열교환기)
- 위생 설비용 배수 및 통기관 경로(구배 표기 확인)

**전기 평면도 — 확인 사항:**
- 주배전반에서 분전반까지의 케이블 트레이 경로
- 단선 결선도와 일치하는 분전반 위치
- 분전반에서 주요 장비까지의 회로 홈런(Homerun) 선
- 케이블 트레이 경로 및 정션 박스 위치
- 분전 장비의 부하 레이블

## 배선도 기호 참조

| 시스템 | 일반 기호 |
|---|---|
| HVAC 급기 | 평면도에서 실선 사각형 덕트; 유량 방향 화살표 |
| HVAC 환기 | 평면도에서 점선 사각형 덕트; AHU 방향 역화살표 |
| 냉수 | CWS(공급) 및 CWR(환수) 유량 화살표 포함 |
| 온수 | HWS 및 HWR 레이블, 차단 밸브 기호 포함 |
| 배수(중력식) | 구배 표기(예: 1:50)가 포함된 점선 |
| 스프링클러 | 헤드용 "X" 또는 점 기호가 있는 실선 파이프 |
| 케이블 트레이 | 평면도에서 이중선 사각형; 크기 표기(예: 600×100) |
| 케이블 트레이 | 크기 표기가 있는 단일선; 패널 방향 홈런 화살표 |

## 조정 도면과 설계 도면의 차이점

| | 설계 도면 | 조정 도면 |
|---|---|---|
| **목적** | 설계 의도 표시 | 시공을 위한 간섭 해소 경로 표시 |
| **LOD** | LOD 200–300 | LOD 350 |
| **레벨 데이터** | 근사치 또는 미표시 | 확정된 배관 하단 및 덕트 천장 높이 |
| **구조 참조** | 보(Beam) 미포함 가능 | 구조 그리드 및 주요 보 표시 |
| **간섭 상태** | 해당 없음 | 발행 전 모든 주요 간섭 해소 완료 |
| **발행 주체** | 설계 엔지니어 | BIM 코디네이터 |
| **사용 주체** | 모델링 팀, QA | 현장 설치 팀 |
