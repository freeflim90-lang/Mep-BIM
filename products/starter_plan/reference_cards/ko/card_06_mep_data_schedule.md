## 분야별 Revit 일정 유형

| 분야 | 일정 유형 | 주요 목적 |
|---|---|---|
| HVAC | 기계 장비 일정 | AHU, FCU, VAV 박스 목록 — 마크, 레벨, 시스템, 유량 포함 |
| HVAC | 덕트 일정 | 크기 및 시스템별 총 덕트 길이 — 물량 산출 |
| 배관 | 배관 일정 | 크기 및 재질별 총 배관 길이 — 조달 |
| 배관 | 기계 장비 일정 | 펌프, 열교환기 — 용량, 유량, 압력 포함 |
| 전기 | 전기 장비 일정 | 패널, 변압기 — 전압, 상, 부하 포함 |
| 전기 | 케이블 트레이 일정 | 폭 및 유형별 트레이 길이 |
| 위생 | 위생 기구 일정 | 기구 목록 — 마크, 유형, 용수 수요 포함 |
| 위생 | 배관 일정 | 급수 및 배수관 — 크기 및 재질별 |
| 소방 | 기계 장비 일정 | 소방 펌프, 스프링클러 헤드 — 개수 및 위치 |
| 소방 | 배관 일정 | 스프링클러 주관 및 지관 — 크기별 |
| 전체 | 덕트/배관 피팅 일정 | 유형별 피팅 개수 — 조달 및 물량 확인 |

## MEP 분야별 주요 매개변수

**HVAC**
- 마크(고유 장비 ID), 패밀리/유형, 레벨, 시스템 분류, 시스템 이름, 설계 유량, 공기량(CFM/L/s), 필터 유형

**배관(기계)**
- 마크, 패밀리/유형, 레벨, 시스템 이름, 배관 크기(DN/NPS), 재질, 설계 유량, 작동 압력, 단열 사양

**전기**
- 마크, 패밀리/유형, 레벨, 전압, 상, 부하 분류, 회로 번호, 패널 이름, 부하(kW/VA)

**위생**
- 마크, 패밀리/유형, 레벨, 시스템 유형(급수, 급탕, 배수), 배관 크기, 구배(배수용), 재질

**소방**
- 마크, 패밀리/유형, 레벨, 시스템 이름(습식/건식), 배관 크기, 스프링클러 헤드 유형, K-계수, 커버리지 면적

## COBie 필수 필드

COBie(Construction Operations Building Information Exchange)는 BIM에서 FM으로의 인계를 위한 표준 데이터 형식입니다.

| COBie 시트 | 주요 필수 필드 |
|---|---|
| Facility | 이름, 설명, 프로젝트명, 부지명, 선형 단위, 면적 단위 |
| Floor | 이름, 설명, 표고, 높이 |
| Space | 이름, 설명, 층 이름, 면적 |
| Component | 이름, 설명, 작성자, 유형 이름, 공간 이름 |
| Type | 이름, 설명, 카테고리, 제조사, 모델 번호, 보증 기간 |
| Attribute | 이름, 카테고리, 값, 단위, 작성자 |

## 데이터 내보내기 체크리스트 — FM 인계

- [ ] 모든 장비에 고유한 마크 값이 있음 — 공백 또는 중복 없음
- [ ] 모든 COBie 요구 장비에 제조사 및 모델 번호 필드가 입력됨
- [ ] 시스템 이름이 프로젝트 BEP 규칙을 따름 — "기본값" 시스템 없음
- [ ] 층 및 공간 데이터가 일정 내 장비와 연결됨
- [ ] IFC 내보내기 완료 및 검증 — Solibri 또는 BIM Collab에서 확인
- [ ] COBie 스프레드시트 내보내기 완료 및 필수 필드 공백 검토
- [ ] 보증 및 유지보수 데이터 필드가 FM 요구사항에 따라 입력됨
- [ ] 일정 CSV 내보내기 완료 및 설계 사양 물량과 상호 참조
- [ ] 파일 버전 및 발행일자가 내보내기 로그에 기록됨
