## 현장팀이 BIM으로부터 필요로 하는 사항

| 산출물 | 형식 | 목적 |
|---|---|---|
| 조정 도면 | A1/A3 출력물, PDF | 각 구역별로 확정된 레벨과 그리드 기준이 표시된 MEP 경로 표시 |
| 측설 데이터 | 치수표, 그리드 기준 도면 | 슬래브 또는 벽체에 구멍 위치, 행거 간격, 장비 베이스 위치 표기 |
| 모델 접근 (뷰어) | NWD 파일, Autodesk 뷰어 링크, BIM 360 Docs | 현장 감독자가 3D 모델을 탐색하여 공간 관계 이해 |
| 간섭 해소 승인 | 서면 메모 또는 추적기 내보내기 | 작업 시작 전 해당 구역의 모든 Critical 및 Major 간섭 해소 확인 |
| Spool 도면 | ISO 도면(등각투영), Spool 납품 일정 | 고유 Spool ID 태그로 현장에 매칭된 프리패브 어셈블리 |
| RFI 응답 | 서면 RFI 응답 (24시간 이내) | 현장 조건이 모델 또는 도면과 상이할 경우 공식 해결 |

---

## RFI 프로세스 흐름

```
현장 조건이 모델 또는 도면과 상이함
        ↓
현장 감독자가 공식 RFI 제기
(RFI 번호, 설명, 사진 할당)
        ↓
RFI가 BIM 코디네이터 / 엔지니어에게 제출됨
        ↓
BIM 코디네이터 / 엔지니어가 모델 + 현장 정보 검토
        ↓
합의된 기한 내 응답 발행 (일반적으로 24~48시간)
        ↓
모델 변경이 필요한 경우:
  → 모델 업데이트, 도면 신규 리비전으로 재발행
        ↓
RFI가 이슈 추적기에서 해결 참조와 함께 종료
        ↓
현장이 수정된 모델과 상이할 경우 편차를 As-Built 기록에 반영
```

---

## As-Built 업데이트 체크리스트

설치가 BIM 모델과 상이할 경우:
- [ ] 편차 식별 및 설명 (요소, 위치, 변경 성격)
- [ ] RFI 제기 및 공식 응답 완료
- [ ] BIM 모델러가 실제 설치 위치를 반영하여 모델 업데이트
- [ ] 업데이트된 모델 버전을 파일명에 "AS-BUILT" 포함하여 저장
- [ ] 수정된 조정 도면을 신규 리비전으로 발행
- [ ] 현장팀에 업데이트된 도면 통보 및 구 리비전 폐기
- [ ] 프로젝트 변경 등록부에 편차 기록

## Spool 도면 기본 사항

| 용어 | 의미 |
|---|---|
| Spool | 배관 또는 덕트의 사전 제작 구간 (피팅, 플랜지, 지지대 포함) — 현장 외부에서 제조 |
| ISO 도면 | 3D 경로, 모든 연결부 및 치수를 보여주는 등각투영 Spool 도면 |
| 절단 길이 | 피팅 여유를 고려한 각 직선 파이프 세그먼트의 정확한 길이 |
| Spool ID | 각 Spool에 부착된 고유 태그로 Spool 도면과 일치 — 현장에서 태그와 도면을 매칭하여 설치 |
| 단부 가공 | 각 Spool 단부의 접합 유형 (플랜지, 그루브, 맞대기 용접) |
| BIM-to-Fabrication | Spool 도면 생성 전 모델은 LOD 350 이상, 간섭 해소 완료 상태여야 함 |

## 일반적인 현장-BIM 피드백 유형

| 피드백 유형 | 일반적 원인 | BIM 대응 |
|---|---|---|
| 슬래브 편차 | 바닥 타설 레벨 불량 — 모델과 상이 | 실제 레벨 측량, 모델 레벨 조정, 조정 도면 재발행 |
| 구조물 위치 이탈 | 기둥 또는 보가 설계와 다르게 설치됨 | 구조 엔지니어에게 RFI, 승인 시 모델 업데이트, 간섭 재확인 |
| 모델에서 누락된 설비 | 설계 변경이 현장에 발행되었으나 BIM에 반영되지 않음 | 모델 업데이트 필요, 관련 조정 도면 모두 재발행 |
| 현장 여유 공간 부족 | 마감 두께로 인해 보이드가 모델보다 좁음 | 마감 적층 확인, 확정 데이터로 모델 조정, 재조정 |
| 피팅 조달 불가 | 지정된 피팅 유형 사용 불가 또는 대체 | 엔지니어 대체 승인, 대체 패밀리로 모델 업데이트 |
