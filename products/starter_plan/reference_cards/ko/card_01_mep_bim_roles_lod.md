## BIM 역할 참조표

| 역할 | 주요 책임 |
|---|---|
| BIM 매니저 | BIM 표준 수립, CDE(공통 데이터 환경) 관리, 프로젝트 전반의 모델 품질 검토 |
| BIM 코디네이터 | 간섭 감지 및 조정 수행, 이슈 트래커 관리, 조정 회의 주관 |
| MEP BIM 모델러 | HVAC(공조), 배관, 전기 등 분야별 BIM 모델 생성 및 유지관리 |
| BIM 리드 | BEP(BIM 실행계획) 관리, 납품 마일스톤 모니터링, 모델링 팀 멘토링 |
| MEP 엔지니어 | 설계 의도 및 엔지니어링 결정 제공, 조정 변경사항 승인 |
| 발주처 / 개발사 | EIR(발주처 정보 요구사항)을 통해 BIM 요구사항 설정, 납품물 및 승인 기준 정의 |

## LOD 정의

| LOD | 명칭 | 설명 | MEP 예시 |
|---|---|---|---|
| LOD 100 | 개념 | 전체 크기 및 위치 표시 — 측정 불가 | 평면도에 표시된 개략적인 덕트 구역 |
| LOD 200 | 근사 | 대략적인 형상, 크기, 형태 및 위치 | 대략적인 크기와 레벨로 표현된 덕트 경로 |
| LOD 300 | 정밀 | 정확한 형상, 확인된 치수, 연결부 | 올바른 크기, 레벨, 시스템 지정이 적용된 덕트 |
| LOD 350 | 조정 | 지지대 등 조정에 필요한 모든 요소 포함 | 행거가 포함되고 모든 분야와 간섭 검토된 덕트 |
| LOD 400 | 제작 | 제조를 위한 전체 샵드로잉 수준의 상세 | 플랜지, 지지대, 설치 공차가 포함된 덕트 섹션 |
| LOD 500 | 준공 | 현장 조건이 반영된 실측 모델 — 기록 모델 | 실제 설치된 덕트, 설계 대비 차이가 기록된 모델 |

## 주요 약어

| 약어 | 전체 명칭 | 실무 의미 |
|---|---|---|
| BEP | BIM 실행계획 (BIM Execution Plan) | 팀이 작성하는 프로젝트 문서로, 누가 어떤 모델 콘텐츠를 언제, 어떻게 납품할지 정의 |
| EIR | 발주처 정보 요구사항 (Employer's Information Requirements) | 발주처가 프로젝트의 BIM 납품물과 표준을 지정하는 문서 |
| CDE | 공통 데이터 환경 (Common Data Environment) | 모든 프로젝트 모델과 문서를 저장 및 관리하는 공유 디지털 플랫폼 |
| IFC | Industry Foundation Classes | 서로 다른 소프트웨어 플랫폼 간 BIM 데이터 공유를 위한 개방형 비독점 파일 형식 |
| NWC | Navisworks 캐시 (Navisworks Cache) | Navisworks 간섭 감지 및 검토에 사용되는 Revit 경량 모델 내보내기 파일 |
| NWD | Navisworks 문서 (Navisworks Document) | 여러 NWC 모델을 묶어 공유 및 보관용으로 패키징한 프로젝트 파일 |
| RFI | 정보 요청 (Request for Information) | 현장 조건이 모델 또는 도면과 다를 때 제기하는 공식 문의 |
